📖 Decimal Notation Overview

📜 Etymology

The word "decimal" comes from the Latin decem, meaning "ten."

The decimal number system is a positional numeral system with base 10.

📝 What is a Decimal?

The decimal number system is a positional numeral with:

  • Finitely many digits to the left of the decimal point (ones, tens, hundreds, etc.)
  • Finite or infinitely many digits to the right of the decimal point (tenths, hundredths, thousandths, etc.)

📊 Place Value Explorer

📝 Understanding Place Value

Each position in a decimal number has a specific value based on powers of 10. Moving left increases the value by 10×, moving right decreases the value by 10×.

Interactive Place Value Chart
Using Money to Represent Decimals

Money is a great model for understanding decimals!

Using Base-10 Blocks

If we use the large cube as 1 (the unit), then:

Block Represents Value
🧊 Large Cube 1 whole (unit) 1
📋 Flat One tenth 0.1
📏 Long One hundredth 0.01
🔲 Small Unit One thousandth 0.001

📚 Teaching Tip

Use multiple models to help students understand decimals:

  • Money: Dollars, dimes, pennies (limited to hundredths)
  • Base-10 Blocks: Can extend to thousandths
  • Number Line: Shows decimals as points between whole numbers

🔢 Expanded Form

📝 Writing Decimals in Expanded Form

Expanded form shows each digit multiplied by its place value. This helps students understand the value of each digit in a decimal number.

📝 Reading Decimals

📝 How to Read Decimals

To read a decimal correctly:

  • Read the whole number part
  • Say "and" for the decimal point
  • Read the decimal part as a whole number
  • Say the place value of the last digit
Examples
Decimal Read As
0.25 Twenty-five hundredths
3.7 Three and seven tenths
12.045 Twelve and forty-five thousandths
100.001 One hundred and one thousandth

⚡ Review of Exponents

📝 Exponent Rules Needed for Decimals

Understanding exponents is essential for working with place values and powers of 10.

Key Exponent Rules
Rule Formula Example
Product Rule am · an = am+n 104 · 103 = 107
Quotient Rule am / an = am-n 105 / 102 = 103
Zero Exponent a0 = 1 100 = 1
Negative Exponent a-n = 1/an 10-3 = 1/1000 = 0.001

Multiplying & Dividing by Powers of 10

Multiply by 10r

Move decimal point r places RIGHT

Example: 34.762 × 10² = 3476.2

Divide by 10r

Move decimal point r places LEFT

Example: 34.762 ÷ 10² = 0.34762

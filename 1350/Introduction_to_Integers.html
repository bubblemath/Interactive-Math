📖 What Are Integers?

Definition of Integers

ℤ = {..., -3, -2, -1, 0, 1, 2, 3, ...}

The set of integers includes all positive whole numbers, negative whole numbers, and zero.

Sets of Numbers
Set Name Symbol Examples
Natural (Counting) ℕ 1, 2, 3, 4, 5, ...
Whole Numbers W 0, 1, 2, 3, 4, 5, ...
Integers ℤ ..., -3, -2, -1, 0, 1, 2, 3, ...

🔍 Key Observations

  • Natural numbers ⊂ Whole numbers ⊂ Integers
  • Every natural number is a whole number
  • Every whole number is an integer
  • Integers extend whole numbers to include negative numbers

📐 Absolute Value

Definition

The absolute value of a number is its distance from zero on the number line.

|a| = distance from 0

Absolute value is always non-negative (≥ 0)

Absolute Value Calculator

The absolute value of

-5
| -5 | = 5

Examples

| 7 | = 7

| -7 | = 7

| 0 | = 0

🔑 Key Insight

A number and its opposite have the same absolute value because they are the same distance from zero!

| 5 | = | -5 | = 5

🔄 Opposites (Additive Inverses)

Definition

Two numbers are opposites if they are the same distance from zero but on opposite sides of the number line.

a + (-a) = 0

The sum of a number and its opposite is always zero.

Find the Opposite

The opposite of 3 is -3

3 + (-3) = 0 ✓

📝 From Your Course Materials

  • The opposite of 3 is -3
  • The opposite of -5 is 5
  • The opposite of 0 is 0 (zero is its own opposite!)

⚖️ Comparing & Ordering Integers

Key Rule

On a number line, numbers increase as you move right.

  • Any positive number is greater than any negative number
  • Any positive number is greater than zero
  • Zero is greater than any negative number
  • For negative numbers: the one closer to zero is greater

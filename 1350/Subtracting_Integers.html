📚 Understanding Integer Subtraction

🔑 The Big Idea

a − b = a + (−b)

Subtracting a number is the same as adding its opposite!

📝 From Your Course Materials

"While this is the formal definition of subtraction, we may need to find a simpler approach than to change all subtraction problems into addition."

Why "Adding the Opposite" Works

📖 Formal Definition

Definition of Subtraction of Integers

Let a, b ∈ ℤ. For any a − b, there exists a unique number N such that:

a = b + N

This means: a − b = N where b + N = a

📝 From Your Course Materials

"Since addition of integers has already been defined, subtraction is defined as the inverse of addition."

🔄 The Transformation Rule

Every subtraction can be rewritten as addition:

a − b → a + (−b)

The "Add Zero Pairs" Technique

Key Idea: If you don't have enough chips to take away, add zero pairs first!

A zero pair = + + − = 0 (doesn't change the value)

Examples from Your Materials

5 − 2 = 3

Start with 5 yellow chips, take away 2

+ + + + +

3 yellow chips remain = 3

3 − 5 = −2

Start with 3 yellow, need to take 5. Add 2 zero pairs!

+ + + + + − −

2 red chips remain = −2

−3 − (−2) = −1

Start with 3 red, take away 2 red

− − −

1 red chip remains = −1

📚 Teaching Tip

  • The key insight: adding zero pairs doesn't change the value!
  • Students often struggle with "subtracting a negative" — the chip model makes it concrete
  • Connect to the "adding the opposite" rule after students see the pattern

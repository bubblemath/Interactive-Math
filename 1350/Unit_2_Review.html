🎯 What's Covered

📘 Lesson 4: Addition & Subtraction

  • Set Model, Number Line, Measurement Model
  • Properties: Closure, Commutative, Associative, Identity
  • Basic Facts: Counting On, Doubles, Making 10
  • Algorithms: Partial Sums, Lattice, Standard
  • Subtraction: Take-Away, Missing Addend, Comparison
  • Mental Math & Other Bases

📗 Lesson 5: Multiplication

  • Repeated Addition, Array, Area Model
  • Cartesian Product Model
  • Properties: Closure, Commutative, Associative, Identity, Zero, Distributive
  • Algorithms: Partial Products, Lattice, Standard
  • Mental Math: Front-End, Compatible Numbers, Thinking Money
  • Multiplication in Other Bases

📙 Lesson 6: Division

  • Repeated Subtraction Model
  • Set (Partition) Model - Sharing Equally
  • Missing Factor Model
  • Division Algorithm: a = bq + r
  • Algorithms: Scaffold, Standard Long Division
  • Divisibility Rules (2-11)

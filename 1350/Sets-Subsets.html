📦 What is a Set?

A set is a collection of objects whose contents can clearly be defined.

The objects in the set are called elements or members.

📝 Three Ways to Describe a Set

Method Description Example
Word Description Describes the set in a written sentence Set A is the set of weekdays
Roster Method Lists elements inside curly braces { } A = {Mon, Tue, Wed, Thu, Fri}
Set Builder Notation Uses a rule to define elements A = {x | x is a weekday}

💡 Reading Set Builder: {x | x is a weekday} reads as "the set of all x such that x is a weekday"

🔣 Important Symbols

Symbol Name Meaning Example
∈ Element of Is a member of the set 3 ∈ {1, 2, 3}
∉ Not element of Is NOT a member of the set 5 ∉ {1, 2, 3}
⊆ Subset All elements are contained in another set (can be equal) {a, b} ⊆ {a, b, c}
⊂ Proper Subset All elements contained, but sets are NOT equal {a, b} ⊂ {a, b, c}
⊄ Not a subset NOT all elements are contained {a, d} ⊄ {a, b, c}
∅ or { } Empty Set A set with no elements ∅ ⊆ {a, b}
ℕ Natural Numbers The counting numbers ℕ = {1, 2, 3, 4, ...}

⚠️ Common Student Error: Element vs. Subset

Students often confuse when to use ∈ (element) vs. ⊆ (subset):

a ⊆ {a, b, c} ✗ WRONG!
a ∈ {a, b, c} ✓ CORRECT (a is an element)
{a} ∈ {a, b, c} ✗ WRONG!
{a} ⊆ {a, b, c} ✓ CORRECT ({a} is a subset)

Remember: Elements are single objects. Subsets are sets (with curly braces).

♾️ Finite vs. Infinite Sets

A finite set contains a countable number of elements.

Example: X = {x | x ∈ ℕ and x < 10} = {1, 2, 3, 4, 5, 6, 7, 8, 9} → 9 elements

An infinite set contains an unlimited/uncountable number of elements.

Example: Y = {x | x ∈ ℕ} = {1, 2, 3, 4, 5, ...} → ∞ elements

🎯 Key Insight

Ask yourself: Does the item have curly braces { } around it?

  • NO braces → It's a single object → Use ∈ (element of)
  • YES braces → It's a set → Use ⊆ (subset of)

Special case: The empty set ∅ or { } is a subset of every set!

💡 How to Determine Finite vs. Infinite

  1. Look for upper AND lower bounds: If the set has both, it's likely finite.
  2. Check for "x < n" or "x ≤ n": These create upper limits → finite
  3. Check for "x > n" or "x ≥ n" alone: No upper limit → infinite
  4. Just "x ∈ ℕ" with no bounds: All natural numbers → infinite

🔑 Sign Rules for Multiplication & Division

(+) × (+) = (+)
(+) × (−) = (−)
(−) × (+) = (−)
(−) × (−) = (+)

Same signs → Positive  |  Different signs → Negative

📋 Key Concepts

Integers (ℤ)

{..., −3, −2, −1, 0, 1, 2, 3, ...}

Opposites

a + (−a) = 0

Absolute Value

Distance from zero: |−5| = 5

Zero Pairs

(+1) + (−1) = 0

Subtraction Rule

a − b = a + (−b)

Division Definition

a ÷ b = c iff b × c = a

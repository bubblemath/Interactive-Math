📌 Remember: Whole Numbers

The set of all counting numbers and zero

📖 The Purpose of This Lesson

As future teachers, you will have the opportunity to teach many students of varying learning styles. You will need an arsenal of techniques to help students visualize how and why addition works.

Key Insight: Try to approach this section as learning the various techniques of teaching addition, not learning the skill of addition itself.

🎯 Definition of Addition

Addition combines two quantities to find a total. We can model this in multiple ways:

a + b = c

Where a and b are called addends and c is called the sum.

🔗 Set Theory Connection

For two disjoint sets A and B (sets with no elements in common):

n(A ∪ B) = n(A) + n(B)

The number of elements in the union equals the sum of elements in each set.

Why Disjoint? If sets overlap, we'd count some elements twice! This connects directly to our Set Theory work in Lessons 2 and 3.

🎨 Three Models of Addition

Each model helps different learners visualize addition:

  • Set Model: Combining groups of discrete objects
  • Number Line Model: Movement/jumps along a line
  • Measurement Model: Combining continuous lengths

🔲 Set Model: Union of Disjoint Sets

The Set Model views addition as combining two separate groups of objects. This is often the first model children encounter: "Jane has 4 blocks in one pile and 3 in another. If she combines them, how many does she have?"

🤔 How Would You Explain This?

"A student asks why we can't just dump all the blocks together without counting each group first. How would you explain the importance of knowing n(A) and n(B) separately?"

📚 Teaching Tip

Best for: Kinesthetic and visual learners who need to physically manipulate objects.

  • Use real objects (blocks, counters, toys) before pictures
  • Emphasize that sets must be disjoint (no shared objects)
  • This model connects directly to counting—students can verify by counting the union
  • Transition: Once comfortable, move to the Number Line for more abstract thinking
📏 Number Line Model

The Number Line Model shows addition as movement: start at the first addend, then "jump" forward by the second addend. This model helps students see addition as a dynamic process and prepares them for understanding negative numbers later.

4 + 3 = ?

🤔 How Would You Explain This?

"A student keeps starting their count at 0 instead of the first addend. For 4 + 3, they count '1, 2, 3, 4' then '5, 6, 7' and get 7—but they're confused about why we 'skip' to 4. How would you clarify?"

📚 Teaching Tip

Best for: Visual learners who benefit from seeing relationships between numbers spatially.

  • Use a floor number line where students can physically walk the jumps
  • Emphasize: "We START at 4, we don't count TO 4"
  • The number line prepares students for subtraction (moving left) and negative numbers
  • Connect to real-world: "If you're on step 4 and climb 3 more steps..."
📐 Measurement Model

The Measurement Model uses continuous quantities like length, weight, or volume. This helps students transition from counting discrete objects to working with measurements—a crucial skill for real-world applications.

🤔 How Would You Explain This?

"A student says 'But I can't count the ribbon like I count blocks!' How would you help them see that measuring is like counting, but with units instead of individual objects?"

📚 Teaching Tip

Best for: Students ready to transition from discrete counting to continuous measurement.

  • Use physical ribbons, string, or paper strips that students can actually combine
  • Emphasize that we're counting units (feet, inches, etc.), not individual objects
  • This model is essential for understanding fractions later (half a foot, etc.)
  • Real-world connections: cooking (cups), distance (miles), time (hours)

⚖️ Properties of Addition

These properties are the "rules" that make addition predictable and useful.

🔒 Closure Property
If a, b ∈ 𝕎 (whole numbers), then a + b ∈ 𝕎

When you add any two whole numbers, the result is always a whole number. The set is "closed" under addition—you can never "escape" the whole numbers by adding.

5 + 3 = 8 ✓ Still a whole number!

📚 Teaching Tip

Why It Matters: Closure seems obvious for addition, but it's NOT true for all operations!

  • Subtraction is NOT closed on whole numbers: 3 - 5 = -2 (not a whole number)
  • Division is NOT closed: 5 ÷ 2 = 2.5 (not a whole number)
  • This property becomes important when students learn about integers and rationals
🔄 Commutative Property
Let a, b ∈ 𝕎. Then a + b = b + a

The order of addends doesn't change the sum. "Commute" means to move—the numbers can swap places!

6 + 8 = 8 + 6 = 14

📚 Teaching Tip

Practical Application: This property makes the "Counting On" strategy work!

  • For 2 + 7, students can think 7 + 2 and count on just 2 from 7
  • Use physical objects: "Does it matter which pile we start with?"
  • Caution: Subtraction is NOT commutative (8 - 3 ≠ 3 - 8)
  • This reduces the addition facts students need to memorize by nearly half!
🔗 Associative Property
Let a, b, c ∈ 𝕎. Then (a + b) + c = a + (b + c)

When adding three or more numbers, the grouping doesn't matter. "Associate" means to group—we can regroup however we like!

( 3 + 6 ) + 9
= 9 + 9 = 18
↕️ Same result!
3 + ( 6 + 9 )
= 3 + 15 = 18

📚 Teaching Tip

Practical Application: This property enables the "Making 10" strategy!

  • For 8 + 5 + 2, students can regroup as 8 + (5 + 2) = 8 + 7, or (8 + 2) + 5 = 10 + 5
  • Combined with commutative property: (3 + 6) + 9 = (6 + 3) + 9 = 6 + (3 + 9) = 6 + 12
  • This is the foundation for mental math with multiple addends
  • Caution: Subtraction is NOT associative: (10 - 5) - 2 ≠ 10 - (5 - 2)
0️⃣ Identity Property
Let a ∈ 𝕎. Then 0 + a = a (and a + 0 = a)

Zero is the additive identity—adding zero doesn't change a number's "identity."

7 + 0 = 7 ✓ Unchanged!

📚 Teaching Tip

Why It Matters: Zero is special—it's the only number that "does nothing" when added!

  • Use the Set Model: "If you have 5 blocks and add 0 more, how many do you have?"
  • This concept prepares students for the multiplicative identity (1) later
  • Students sometimes confuse this with multiplication by zero (which DOES change the number)
  • Historical note: Zero as a number was a revolutionary mathematical concept!

Critical Concepts to Master

Key ideas that appear throughout the course

📐 Properties

Closure: Result stays in the set
Commutative: Order doesn't matter
Associative: Grouping doesn't matter
Distributive: a(b+c) = ab + ac

🔢 Key Formulas

Division Algorithm: a = bq + r
GCF × LCM = a × b
Cross-Products: a/b = c/d ⟺ ad = bc
Percent: Part = % × Whole

⚡ Common Traps

−3² = −9 but (−3)² = 9
Division by 0: UNDEFINED
Percent Addition: Only with SAME base!
GCF: MIN exponents | LCM: MAX exponents

💡 Interest Formulas

Simple: I = Prt, A = P(1 + rt)
Compound: A = P(1 + r/n)^(nt)
n values: Annual=1, Semi=2, Quarterly=4, Monthly=12

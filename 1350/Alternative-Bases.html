Understanding Number Bases

💻 Binary (Base 2)

Digits: 0, 1

The foundation of all computing! Computers use binary for arithmetic, logic operations, file compression, cryptography, digital media, and AI.

Place values: 1, 2, 4, 8, 16, 32, 64, 128...

✋ Quinary (Base 5)

Digits: 0, 1, 2, 3, 4

Based on counting with one hand! While no major culture used it extensively, it's a natural counting system.

Place values: 1, 5, 25, 125, 625, 3125...

🕐 Duodecimal (Base 12)

Digits: 0-9, T (ten), E (eleven)

More common than you think! 12 hours on a clock, 12 months, 12 in a dozen, 12 inches in a foot, 12 keys in an octave.

Place values: 1, 12, 144, 1728...

📝 Note: Some textbooks use A for 10 and B for 11 instead of T and E. Both notations are valid!

🎨 Hexadecimal (Base 16)

Digits: 0-9, A, B, C, D, E, F

Used in computing for colors (#FF5733), memory addresses, and compact binary representation.

Place values: 1, 16, 256, 4096...

🔑 The Universal Base Rule

Base n uses digits 0 through n - 1

Base 2 → digits 0-1 | Base 5 → digits 0-4 | Base 12 → digits 0-E | Base 16 → digits 0-F

💡 What is ASCII?

ASCII (American Standard Code for Information Interchange) is how computers represent text using numbers.

Each character (letter, number, symbol) has a unique number from 0-127. For example:

  • A = 65 = 01000001₂
  • a = 97 = 01100001₂
  • 0 = 48 = 00110000₂
  • Space = 32 = 00100000₂

This is how your text messages, emails, and even AI responses are stored as 1s and 0s! 🤖

🔑 Cross-Product Property

a/b = c/d   if and only if   ad = bc

A proportion is a statement that two given ratios are equal. The cross-products of a true proportion are always equal.

📝 Key Concepts

Ratio If a and b are real numbers with b ≠ 0, then the ratio of a to b is the quotient a/b
Proportion A statement that two given ratios are equal: a/b = c/d
Unit Rate A rate with a denominator of 1 (e.g., $1.25 per ticket)
Scale The ratio of length in a drawing to length of the real object
Cross-Products If a/b = c/d, then ad = bc (extremes × means)
Equivalent Ratios Ratios that represent the same relationship (2:3 = 4:6 = 6:9)

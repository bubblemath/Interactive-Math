⚠️ Critical Concepts to Master

• Percent Addition Trap: You can only add percentages with the SAME base!

• Exponent Distinction: −3² = −9 but (−3)² = 9

• Division by Zero: UNDEFINED!

• Terminating Decimals: Only when denominator = 2ᵃ × 5ᵇ

🔑 Complete Formula Reference

📘 Unit I: Sets & Numeration

n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

Number of subsets = 2ⁿ

Expanded form: dₙ×bⁿ + ... + d₀×b⁰

📙 Unit II: Operations

Division Algorithm: a = bq + r

where 0 ≤ r < b

Distributive: a(b+c) = ab + ac

n(A × B) = n(A) · n(B)

📗 Unit III: Number Theory

GCF: MIN exponents, COMMON primes

LCM: MAX exponents, ALL primes

GCF(a,b) × LCM(a,b) = a × b

a − b = a + (−b)

📕 Unit IV: Proportions

Cross-Products: a/b = c/d ⟺ ad = bc

Part = Percent × Whole

Simple: I = Prt, A = P(1+rt)

Compound: A = P(1+r/n)^(nt)

🔄 From Two Sets to Three Sets

When we add a third set C to our Venn diagram, the number of distinct regions increases from 4 to 8.

Each region represents a unique combination of membership in sets A, B, and C.

📊 The 8 Regions of a Three-Set Venn Diagram

📏 Cardinal Number Formula for Three Sets

n(A ∪ B ∪ C) = n(A) + n(B) + n(C)
− n(A ∩ B) − n(A ∩ C) − n(B ∩ C)
+ n(A ∩ B ∩ C)

Why +n(A∩B∩C)? The center region gets subtracted 3 times (once for each pairwise intersection), so we add it back once!

⚙️ Order of Operations

Just like arithmetic, set operations follow an order:

  1. Parentheses first - Evaluate expressions inside ( ) first
  2. Complement - Apply ' to individual sets
  3. Intersection and Union - Left to right

Example: (A ∪ B) ∩ C means first find A ∪ B, then intersect with C

🎯 Strategy: Work from the Center Out

When filling in a three-set Venn diagram:

  1. Start with Region V (center - all three sets)
  2. Fill in the two-set intersections (Regions II, IV, VI)
  3. Fill in the single-set regions (Regions I, III, VII)
  4. Finally, fill in Region VIII (outside all)

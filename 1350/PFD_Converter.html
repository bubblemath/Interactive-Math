📖 Understanding Percent, Fraction, Decimal

📝 From Your Course Materials

Percent: The word percent comes from the Latin phrase per centum, which means "per hundred."

Definition: n% = n/100 where n is any non-negative number. (Parts "per 100")

🔑 The Big Idea

Percent, Fraction, and Decimal are three different ways to write the same value!

75% = ¾ = 0.75

📐 Conversion Rules

Percent ↔ Decimal
Conversion Rule Example
Percent → Decimal Move decimal 2 places LEFT
(or divide by 100)		 75% → 0.75
8% → 0.08
125% → 1.25
Decimal → Percent Move decimal 2 places RIGHT
(or multiply by 100)		 0.75 → 75%
0.08 → 8%
1.25 → 125%

💡 Memory Trick

Percent to Decimal: "Percent is bigger, so make it smaller" → move LEFT

Decimal to Percent: "Decimal is smaller, so make it bigger" → move RIGHT

Fraction ↔ Percent
Conversion Rule Example
Fraction → Percent Divide numerator by denominator,
then multiply by 100		 ¾ → 3÷4 = 0.75 → 75%
⅖ → 2÷5 = 0.4 → 40%
Percent → Fraction Put over 100,
then simplify		 75% → 75/100 → ¾
40% → 40/100 → ⅖
Fraction ↔ Decimal
Conversion Rule Example
Fraction → Decimal Divide numerator by denominator ¾ → 3÷4 = 0.75
⅜ → 3÷8 = 0.375
Decimal → Fraction Use place value, then simplify 0.75 → 75/100 → ¾
0.375 → 375/1000 → ⅜

🔑 Pattern Recognition

Fifths: Multiply numerator by 20 to get percent (⅗ → 3×20 = 60%)

Fourths: Multiply numerator by 25 to get percent (¾ → 3×25 = 75%)

Eighths: Multiply numerator by 12.5 to get percent (⅜ → 3×12.5 = 37.5%)

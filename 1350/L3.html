Numeration & Number Systems

Interactive Explorations in Mathematical History

MATH 1350 - Lesson 1

📚 About This Lesson

Explore how civilizations throughout history have represented quantities and developed systems for counting. From the tally marks of prehistoric humans to the binary code powering modern computers, this lesson covers the evolution of numeration systems, place value concepts, and conversions between different bases. Each interactive applet below builds upon the previous, creating a comprehensive understanding of how numbers work.

Part 1
Numeration Systems

An exploration of how we represent quantities, focusing on the Hindu-Arabic numeration system that we use today. Learn about the importance of place value, the role of zero, and how our base-ten (decimal) system works. Includes interactive place value charts, expanded form demonstrations, and practice problems ranging from ones to billions.

Key Topics

Numerals vs Numbers Place Value Base 10 Expanded Form Powers of 10

Part 2
Historical Numbers

A journey through ancient mathematical history! Explore how different civilizations developed their own numeration systems: Tally marks (~30,000 BCE), Egyptian hieroglyphics (base 10, additive), Babylonian cuneiform (base 60, positional), Roman numerals (subtractive notation), and Mayan (modified base 20). Build numbers in each system using interactive symbol labs.

Key Topics

Tally System Egyptian Babylonian (Base 60) Roman Numerals Mayan (Base 20)
🚀 Launch Applet
💻
Part 3
Alternative Bases

A deep dive into non-decimal number systems used in computing and mathematics. Master Binary (Base 2) with interactive light switches, explore Quinary (Base 5) with hand counting, and learn Duodecimal (Base 12) using T for 10 and E for 11. Includes step-by-step converters using the expanded form and division-remainder methods, plus an ASCII mode explaining how computers represent text.

Key Topics

Binary (Base 2) Quinary (Base 5) Duodecimal (Base 12) Hexadecimal (Base 16) Base Conversions

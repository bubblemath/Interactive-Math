📚 Adding & Subtracting Fractions Overview

📝 From Your Course Materials

Your course uses concrete models to build understanding:

  • Area Model: Fraction bars and grids
  • Number Line Model: Jumps on a number line
  • Grid Model: For visualizing common denominators
Key Concepts
Situation Rule Example
Like Denominators Add/subtract numerators, keep denominator 1/8 + 3/8 = 4/8 = 1/2
Unlike Denominators Find common denominator first (use LCM) 2/3 + 1/4 = 8/12 + 3/12 = 11/12
Mixed Numbers Convert to improper OR add whole parts separately 2 1/3 + 1 1/2 = 3 5/6
Formal Definitions

Adding Fractions with Common Denominators

ab + cb = a + cb

Adding Fractions with Unlike Denominators

ab + cd = ad + bcbd

📚 Teaching Tip

The grid model is powerful because it shows WHY we need common denominators:

  • Each fraction is represented on a grid
  • The grid dimensions create the common denominator
  • Students can COUNT the shaded squares

🔢 Mixed Numbers

📝 From Your Course Materials

"A mixed number is the sum of a natural number and a fraction."

Two methods for adding mixed numbers:

  • Method 1: Convert to improper fractions, add, convert back
  • Method 2: Add whole parts and fraction parts separately

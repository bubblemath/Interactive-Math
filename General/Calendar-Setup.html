📅 Balanced Semester Calendar Generator

Create a balanced assignment calendar with automatic deadline distribution

📋 Instructions

How to Use This Generator:

  1. Set Semester Dates: Enter your semester start and end dates.
  2. Choose Deadline Days: Select which days of the week assignments are due.
  3. Schedule Exams: Add your exams with specific dates.
  4. Add Unit Assignments: Paste a vertical list of assignments for each unit.
  5. Generate Calendar: Click to auto-balance assignments across deadline slots.
  6. Adjust Dates: Use the "Edit Dates" tab to manually adjust individual deadlines.
  7. Print: Use "Print Calendar" for a single-page 8.5" x 11" printout.

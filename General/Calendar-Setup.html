Create a balanced assignment calendar with automatic deadline distribution

📋 Instructions

How to Use This Generator:

  1. Set Semester Dates: Enter your semester start and end dates.
  2. Choose Deadline Days: Select which days of the week assignments are due (e.g., Mon/Wed/Sat or Wed/Sat).
  3. Schedule Exams: Add your exams with specific dates - these will be "locked" in place.
  4. Add Unit Assignments: Paste a vertical list of assignments for each unit.
  5. Generate Calendar: The applet will automatically balance assignments across available deadline slots.
  6. Export: Copy the HTML code for your LMS or download as a file.

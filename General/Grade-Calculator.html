📋 Instructions

How to Use This Generator:

  1. Enter Course Info: Add your course name and select your LMS platform.
  2. Add Categories: Enter each grading category from your syllabus with its weight percentage.
  3. Mark the Final: Check the "Final Exam" box for the category that represents your final exam.
  4. Set Grading Scale: Adjust the minimum percentages for each letter grade.
  5. Generate Code: Click "Generate Calculator" when weights total 100%.
  6. Copy & Embed: Copy the HTML code and paste it into your LMS.

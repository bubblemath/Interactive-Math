🏛️ The Four Pillars of Reading Comprehension

📐

Main Ideas

Identify the central point or thesis of a passage. Distinguish between topic (what it's about) and main idea (what the author says about it).

🔍

Supporting Details

Find specific facts, examples, and evidence that support the main idea. Think of details as the "legs" that hold up the "table" of the main idea.

🧠

Inference

Draw logical conclusions from information that is implied but not directly stated. Use the "evidence test" to verify your inferences.

📝

Context Clues

Determine the meaning of unfamiliar words using surrounding text. Remember the IDEAS acronym: Inference, Definition, Example, Antonym, Synonym.

💡 Test-Taking Strategies

Before Reading the Passage:

  • Read the question first — Know what you're looking for before you read
  • Identify the question type — Main idea? Detail? Inference? Vocabulary?
  • Scan answer choices — Get a sense of what distinctions matter

While Reading the Passage:

  • Identify the passage type — Informational, persuasive, or narrative?
  • Find the main idea — Usually in the first or last paragraph
  • Note signal words — "However," "therefore," "for example," etc.
  • Don't get stuck on unfamiliar words — Use context clues

When Answering Questions:

  • Eliminate extremes — Words like "always," "never," "all," "none" are often wrong
  • Use the evidence test — Can you point to specific text that supports your answer?
  • Beware of "true but wrong" — An answer can be factually true but not answer the question
  • Watch for scope errors — Too broad or too narrow for the question asked

📖 Quick Reference: Signal Words

Main Idea Signals:
  • The main point is...
  • Most importantly...
  • In conclusion...
Detail Signals:
  • For example...
  • Specifically...
  • According to...
Inference Signals:
  • This suggests...
  • It can be inferred...
  • Most likely...
Context Clue Signals:
  • ...which means...
  • ...or (synonym)...
  • Unlike... (antonym)

📖 Quick Reference Vault

📐 Main Ideas

  • Main Idea = Topic + Author's Point
  • Often in first or last paragraph
  • Ask: "What is the author saying about the topic?"

🔍 Supporting Details (Table Analogy)

  • Main idea = tabletop
  • Details = legs that support it
  • Look for: facts, examples, statistics, quotes

🧠 Inference

  • Evidence Test: Can you point to supporting text?
  • Avoid extreme answers (always, never)
  • Signal words: suggests, implies, most likely

📝 Context Clues (IDEAS)

  • Inference - Use logic
  • Definition - Stated directly
  • Example - Illustrations given
  • Antonym - Contrast clues
  • Synonym - Restatement clues

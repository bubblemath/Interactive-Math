📐 Key Formulas & Rules

🪞 Reflection over x-axis

(x, y) → (x, −y)

Y-coordinate changes sign; x stays the same

🪞 Reflection over y-axis

(x, y) → (−x, y)

X-coordinate changes sign; y stays the same

🪞 Reflection over y = x

(x, y) → (y, x)

Coordinates swap positions

🔍 Dilation (center at origin)

(x, y) → (kx, ky)

k > 1: enlargement | 0 < k < 1: reduction

📐 Interior Angle (regular polygon)

(n − 2) × 180° ÷ n

n = number of sides

🧩 Tessellation Rule

Angles at vertex = 360°

Only △ (60°), □ (90°), ⬡ (120°) work alone

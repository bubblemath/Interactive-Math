📐 Key Formulas

Mean (Average)

\(\bar{x} = \frac{\sum x}{n}\)
Sum of values divided by count

Standard Deviation (Sample)

\(s = \sqrt{\frac{\sum(x-\bar{x})^2}{n-1}}\)
Measures spread around the mean

Z-Score

\(z = \frac{x - \mu}{\sigma}\)
Standard deviations from mean

IQR (Interquartile Range)

\(IQR = Q_3 - Q_1\)
Spread of middle 50%

Outlier Fences

\(Q_1 - 1.5(IQR)\) to \(Q_3 + 1.5(IQR)\)
Values outside are potential outliers

Percentile Rank

\(\frac{\text{# below } x}{n} \times 100\)
Percent of data at or below value

🔑 When to Use Each Measure of Center

📍 Mean

  • Uses ALL data values
  • Sensitive to outliers
  • Best for symmetric data
✅ Use when: Data is symmetric with no extreme outliers

📍 Median

  • Middle value when ordered
  • Resistant to outliers
  • Best for skewed data
✅ Use when: Data is skewed or has outliers (like home prices)

📍 Mode

  • Most frequent value
  • Can have multiple modes
  • Works with categorical data
✅ Use when: Data is categorical or you want "most popular"

📚 Understanding Data Types

Before choosing a graph, you must understand what type of data you have.

📋 Categorical (Qualitative) Data

  • Describes qualities or characteristics
  • Cannot be measured numerically
  • Can be sorted into groups/categories
  • Examples: colors, brands, yes/no responses

🔢 Numerical (Quantitative) Data

  • Represents measurable quantities
  • Can be counted or measured
  • Can perform arithmetic operations
  • Examples: height, age, test scores, temperature
🎓 Teaching Tip

Ask students: "Can you add these values together and get a meaningful result?" If yes, it's likely numerical. If no (like adding "red" + "blue"), it's categorical.

📊 Graph Types for Categorical Data

📊 Bar Graph

Uses rectangular bars to represent frequencies or counts of categories. Bars have equal widths and gaps between them.

Key Feature: Bars do NOT touch (gaps show categories are distinct)

🥧 Pie Chart (Circle Graph)

A circular graph divided into sectors, where each sector represents a part of the whole. Shows proportions and percentages.

Key Feature: All sectors must sum to 100% (360°)

🖼️ Pictograph

Uses pictures or symbols to represent data. Each symbol represents a specific quantity defined by a key.

Key Feature: Must include a key (e.g., 🍎 = 10 apples)

📈 Histogram (NOT for categorical!)

Similar to bar graph but for numerical data. Bars touch because data is continuous. Used for frequency distributions.

Key Difference: Histogram bars TOUCH (no gaps)

⚠️ Bar Graph vs. Histogram

This is a common point of confusion! Here's how to tell them apart:

📊 Bar Graph

  • Data Type: Categorical
  • Bars: Have GAPS between them
  • Order: Categories can be rearranged
  • X-axis: Category names
  • Example: Favorite colors, car brands

📈 Histogram

  • Data Type: Numerical (quantitative)
  • Bars: TOUCH each other (no gaps)
  • Order: Must be in numerical order
  • X-axis: Number ranges (bins)
  • Example: Test scores, ages
🎓 Memory Trick for Students

"Gaps = Groups" — Bar graphs have gaps because categories are separate groups. "Touch = Together" — Histogram bars touch because numerical data flows together continuously.

🔢 Pie Chart Math

Converting data to pie chart sectors requires calculating percentages and degrees.

📐 The Formulas

Step 1: Find the Percentage

Percentage = (Part ÷ Whole) × 100

Step 2: Find the Degrees

Degrees = (Part ÷ Whole) × 360°

📝 Example: Titanic Passengers

Total passengers: 2201

Class Count Ratio Percent Degrees
1st Class 325 325/2201 14.8% 53°
2nd Class 285 285/2201 12.9% 47°
3rd Class 706 706/2201 32.1% 115°
Crew 885 885/2201 40.2% 145°
Total 2201 — 100% 360°

📝 Quick Reference: When to Use Each Graph

📊 Use a Bar Graph When...

  • ✓ Comparing frequencies of categories
  • ✓ Data is categorical (not numerical)
  • ✓ You want to show "how many" of each
  • ✓ Order of categories doesn't matter

🥧 Use a Pie Chart When...

  • ✓ Showing parts of a whole
  • ✓ Data represents percentages
  • ✓ You have 2-6 categories (not too many!)
  • ✓ All parts sum to 100%

🖼️ Use a Pictograph When...

  • ✓ Audience is young children
  • ✓ You want visual appeal
  • ✓ Data values are multiples of a key
  • ✓ Comparing simple quantities

📈 Use a Histogram When...

  • ✓ Data is numerical (quantitative)
  • ✓ Showing frequency distributions
  • ✓ Data is grouped into ranges/bins
  • ✓ Looking for shape of distribution

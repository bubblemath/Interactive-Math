📐 Key Formulas

Multiplication Principle

\(n_1 \times n_2 \times n_3 \times \cdots\)
Use when making successive independent choices

Factorial

\(n! = n \times (n-1) \times \cdots \times 2 \times 1\)
Note: 0! = 1 by definition

Permutations (Order Matters)

\(P(n,r) = \frac{n!}{(n-r)!}\)
Arranging r items from n items

Combinations (Order Doesn't Matter)

\(C(n,r) = \frac{n!}{r!(n-r)!}\)
Selecting r items from n items

Expected Value

\(E(X) = \sum x \cdot P(x)\)
Long-run average outcome

🔑 The Key Difference

🔄 Permutations

ORDER MATTERS
  • Items selected from the same group
  • No item is used more than once
  • The order of arrangement makes a difference
🍦 Ice Cream Cone (Stacked): Chocolate-Vanilla-Strawberry is DIFFERENT from Strawberry-Vanilla-Chocolate

🎯 Combinations

ORDER DOESN'T MATTER
  • Items selected from the same group
  • No item is used more than once
  • The order of items makes NO difference
🍨 Ice Cream Bowl (Mixed): A bowl with Chocolate, Vanilla, Strawberry is THE SAME as Strawberry, Vanilla, Chocolate

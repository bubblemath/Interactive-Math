👩‍🏫 Teaching Moment

Each unit fraction equals 1 because the numerator and denominator represent the same measure. Multiplying by 1 doesn't change the value — only the units! This is why dimensional analysis works.

Perimeter Explorer

Drag vertices to reshape polygons — perimeter updates live

🖱️ How to Use

Select a shape, then drag the vertices (colored dots) to reshape it. The perimeter recalculates in real-time. For circles, use the radius slider.

👩‍🏫 Teaching Tip

Have students predict: "If I double one side of a rectangle, does the perimeter double?" This explorer lets them test conjectures instantly.

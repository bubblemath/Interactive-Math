Day 4 — Triangle Angle Relationships

  • Triangle Sum: ∠A + ∠B + ∠C = 180° — missing angle = 180° − (other two)
  • Exterior Angle: ∠ext = ∠R₁ + ∠R₂ — equals the sum of the two remote interior angles
  • Isosceles: equal sides → equal base angles. The vertex angle is not a base angle
  • Each base angle = (180° − vertex) ÷ 2
  • Equilateral: all sides equal → every angle = 60°

Day 5 — Vocabulary & Notation

  • "Can you measure it?" — a number (length / measure) uses =; a geometric object uses ≅
  • So: m∠A = 40° ✓ but m∠A ≅ 40° ✗ · AB̄ ≅ CD̄ ✓ but AB̄ = CD̄ ✗
  • Segment Addition: if B is between A and C → AB + BC = AC
  • Angle Addition: ray inside an angle → m∠ABD + m∠DBC = m∠ABC
  • Midpoint / Bisector = a command to set the two halves equal and solve
  • Angle name ∠ABC: the middle letter is the vertex

Marker I   The Watchtower · Triangle Sum
Locked

A triangular weathervane atop the watchtower has angles ∠A = 72° and ∠B = 53°. Find the third angle ∠C (in degrees, number only).

Strategy: The three interior angles of any triangle add to 180°. To find the missing one, subtract the two known angles from 180°.

Marker II   The Faded Signpost · Notation
Locked

A weather-beaten signpost lists four geometry statements. Which one uses ≅ vs. = correctly?

Strategy: Apply the "Can you measure it?" rule. A measure or length is a number → use =. A geometric object (segment, angle, triangle) → use ≅. Find the statement that pairs the right kind of thing with the right symbol.

Marker III   The Broken Roof · Exterior Angle
Locked

A collapsed roofline forms a triangle whose exterior angle is 130°. Its two remote interior angles are ∠A = (2x + 10)° and ∠B = (3x)°. Find x.

Strategy: An exterior angle equals the sum of the two remote interior angles (the ones it doesn't touch). Set that sum equal to the exterior angle and solve for x.

Marker IV   The Clocktower · Angle Addition
Locked

The clocktower's cracked face splits an angle. Ray BD is inside ∠ABC, with m∠ABD = (3x + 10)°, m∠DBC = (2x + 5)°, and m∠ABC = 75°. Find x.

Strategy: When a ray splits an angle, the two parts add to the whole: m∠ABD + m∠DBC = m∠ABC. Add the two expressions, set the total equal to the whole angle, and solve.

Marker V   The Old Chapel · Isosceles

The chapel's gable is an isosceles triangle △PQR with PQ = PR and vertex angle ∠P = 50°. Find the base angle ∠Q (number only).

Strategy: Equal sides give equal base angles — and ∠P is the vertex angle, not a base angle, so don't set it equal to ∠Q. Each base angle = (180° − vertex) ÷ 2.

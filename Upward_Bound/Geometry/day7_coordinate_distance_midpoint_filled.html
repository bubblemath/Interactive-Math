The unmasking — complete these blanks as your teacher walks through:
Pythagorean Theorem:   a² + b² = c²
In coordinates:   a = |x₂ − x₁| = |4−1| = 3   (horizontal leg)
                       b = |y₂ − y₁| = |6−2| = 4   (vertical leg)
So:   d² = (x₂ − x₁)² + (y₂ − y₁)²
Therefore:   d = √[(x₂−x₁)² + (y₂−y₁)²]

Section 3 — Distance Formula: The Unmasking
Distance Formula (= Pythagorean Theorem on a coordinate plane)
d = √[ (x₂ − x₁)² + (y₂ − y₁)² ]
(x₂ − x₁) = horizontal distance = Δx (leg a)
(y₂ − y₁) = vertical distance = Δy (leg b)
d = distance between points = hypotenuse (hypotenuse c)
📐 GeoGebra — See the Right Triangle geogebra.org/m/KUdVJFgk
Task: Drag the two points. Watch the right triangle form automatically between them.
  • What are the two legs of the triangle? The horizontal leg = |x₂−x₁|; the vertical leg = |y₂−y₁|.
  • If I move one point directly to the right, which leg changes? The horizontal leg changes; the vertical leg stays the same.
  • Set the points to create legs of 3 and 4. Predict the distance before looking. Predict: 5 (3-4-5 triple). Applet confirms 5. ✓
When legs = 3 and 4, distance = 5
📊 GeoGebra — Formula in Action geogebra.org/m/uvj3ytpu
Task: Drag the endpoints and watch the formula calculate automatically.
  • Try a pair of points — write the coordinates and the distance the applet shows: E.g. A(1,1) to B(4,5): d = √(9+16) = √25 = 5
  • Does the order of the points matter? Swap — does d change? No — (x₂−x₁)² = (x₁−x₂)² because squaring removes the sign. d stays the same.
My test: A(1,1) to B(4,5), d = 5
Example 1 — Integer Answer (guided)
Find the distance between A(1, 2) and B(4, 6).
Step 1: Label coordinates(x₁, y₁) = (1, 2)    (x₂, y₂) = (4, 6)
Step 2: Substituted = √[(4 − 1)² + (6 − 2)²]
Step 3: Compute legsd = √[3² + 4²] = √[9 + 16]
Step 4: Simplifyd = √25 = 5
This is the 3-4-5 Pythagorean triple! ✓
Example 2 — Radical Answer (try it)⚠ Watch negatives!
Find the distance between P(−2, 3) and Q(4, −1).
Step 1: Label coordinates(x₁, y₁) = (−2, 3)    (x₂, y₂) = (4, −1)
Step 2: Substituted = √[(4 − (−2))² + (−1 − 3)²]
Step 3: Compute legsd = √[6² + (−4)²] = √[36 + 16]
Step 4: Simplify radicald = √52 = √(4 × 13) = 2√13
⚠ Negative subtraction check:
4 − (−2) = 4 + 2 = 6, NOT 4 − 2 = 2

Section 4 — Midpoint Formula: Just an Average
★ Discover it before your teacher tells you:
You're at x = 2 on a number line. Your friend is at x = 8. You want to meet exactly halfway.
Where do you meet? 5    How did you find it? (2+8)÷2 = 5 (average)
Now: You're at (1, 3). Your friend is at (5, 7). Where do you meet halfway?
Midpoint x: 3    Midpoint y: 5    Meeting point: (3, 5)
Midpoint Formula (average the coordinates)
M = ( (x₁+x₂)/2 , (y₁+y₂)/2 )
Average the x-coordinates. Average the y-coordinates. Leave fractions as fractions — do not convert to decimals.
📍 GeoGebra — Midpoint Explorer geogebra.org/m/zWFNqgUe
Task: Drag the endpoints. Watch the midpoint always stay exactly centered between them.
  • As you drag an endpoint farther away, does the midpoint follow it? How far? Yes — the midpoint moves half as far as the endpoint does.
  • What is always true about the distance from each endpoint to the midpoint? Both distances are always equal — the midpoint is equidistant from each endpoint.
Key observation: The midpoint is always equidistant (equal distance) from each endpoint.
Worked examples — fill in the blanks:
Example 1: A(2,8) and B(6,4)
M = ( (2+6)/2, (8+4)/2 )
= ( 8/2, 12/2 )
= ( 4, 6 )
Example 2: P(−3,5) and Q(7,−1)
M = ( (−3+7)/2, (5+(−1))/2 )
= ( 4/2, 4/2 )
= ( 2, 2 )
Example 3: A(1,3) and B(4,8) ← fraction case
M = ( (1+4)/2, (3+8)/2 )
= ( 5/2, 11/2 ) ← leave as fraction!
= ( 5/2, 11/2 )
Section 5 — Missing Endpoint: Two Methods
Given one endpoint and the midpoint, find the other endpoint. Try both methods.
Method 1 — "Same Change" (Intuitive)
Given: A(2, 5) and midpoint M(6, 8). Find B.

From A to M: x changed by +4, y changed by +3
From M to B: same change again
B = (6 + 4, 8 + 3) = (10, 11)
Method 2 — Algebraic (Set up the average equation)
Given: A(2, 5) and midpoint M(6, 8). Find B = (x, y).

(2 + x)/2 = 6 → 2 + x = 12 → x = 10
(5 + y)/2 = 8 → 5 + y = 16 → y = 11
B = (10, 11) ✓
⚠ Common error: Students sometimes set up (M + B)/2 = A — swapping what is given and what is unknown. Always ask: "Which point is the MIDPOINT? Which is the unknown ENDPOINT?" before writing the equation.

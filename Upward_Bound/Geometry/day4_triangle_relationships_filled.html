Section 2 — 180° Discovery Lab

Your task: Draw any triangle. Tear off all three corners. Arrange them side-by-side so all the torn tips meet at one point along a line.

  • What does this prove about triangle angles? All three interior angles sum to exactly 180°.

Theorem 1 — Triangle Sum Theorem
∠A + ∠B + ∠C = 180°
The sum of the interior angles of any triangle is always 180°. Use this when you know two angles and need the third: missing angle = 180° − (sum of the other two).

Section 4 — Exterior Angle Theorem
🚪 GeoGebra — Exterior Angle Investigator geogebra.org/m/JEcprX86
Your task: Open the applet. Drag a triangle vertex to change the shape. Watch how the exterior angle and the two remote interior angles change together.
  • Is the exterior angle bigger or smaller than each individual remote interior angle? Always bigger — it equals the SUM of the two remote interior angles.
  • What operation connects the two remote interior angles to the exterior angle? Addition — exterior angle = remote angle 1 + remote angle 2.
  • Can you write a formula for this pattern before your teacher does? ∠ext = ∠A + ∠B (the two non-adjacent interior angles).
Label the diagram:
The exterior angle is ∠ACD.
The remote interior angles are ∠A and ∠B.
Theorem 2 — Exterior Angle Theorem
∠ACD = ∠A + ∠B
An exterior angle of a triangle equals the sum of the two remote interior angles (the ones it does NOT share a vertex with).

Theorem 3 — Isosceles Triangle Theorem
PQ = PR  ⟹  ∠Q = ∠R
Equal sides → Equal base angles.
Converse: Equal base angles → Equal sides.
Always identify the equal sides FIRST, then name the angles opposite them.
Special case — Equilateral Triangle:
If all 3 sides equal → all 3 angles equal.
Triangle Sum: 3 × (each angle) = 180°
Each angle = 60°
Converse — find equal sides:
In △XYZ: ∠X = 50°, ∠Y = 50°. Which sides are equal?
(Name the sides opposite the equal angles.)
Answer: YZ = XZ
Spot the Error — Isosceles Triangle
Problem: Isosceles △PQR with PQ = PR. ∠P = 50°. Find ∠Q.
Student's incorrect work:
PQ = PR → ∠P = ∠Q
∴ ∠Q = 50°
What is wrong? (circle one)
○   Used wrong theorem
○   Set vertex = base angle (not base = base)
○   Algebra error
○   Nothing is wrong
Correct solution:
PQ = PR → ∠Q = ∠R (base angles, not vertex)
∠Q = ∠R = (180° − 50°) ÷ 2 = 130° ÷ 2 = 65°
Answer: ∠Q = ∠R = 65°
∠Q = ∠R because they are the base angles.
Use Triangle Sum: (180° − vertex) ÷ 2.
The formula to find base angles of any isosceles triangle:   each base angle = (180° − vertex angle) ÷ 2

Example 1 — Triangle Sum (guided)
In △ABC: ∠A = (2x + 10)°, ∠B = (3x − 5)°, ∠C = 45°. Find x.
Step 1: IdentifyThree interior angles → Triangle Sum Theorem
Step 2: Translate(2x+10) + (3x−5) + 45 = 180
Step 3: Solve5x + 50 = 180 → 5x = 130 → x = 26
x = 26   ∠A = 62°   ∠B = 73°   ∠C = 45° ✓ (sum = 180°)
Example 2 — Isosceles (guided)
Isosceles △PQR, PQ = PR. ∠P = (4x)°. ∠Q = (3x + 18)°. Find x.
Step 1: IdentifyPQ = PR → equal sides → ∠Q = ∠R (base angles equal)
Step 2: Translate∠Q = ∠R, so also ∠Q = 3x+18. Then:
∠P + 2∠Q = 180° → 4x + 2(3x+18) = 180
Step 3: Solve4x + 6x + 36 = 180 → 10x = 144 → x = 14.4
x = 14.4   ∠P = 57.6°   ∠Q = ∠R = 61.2°
Practice Problems — Write the theorem name before each setup
P1 — Triangle Sum
In △ABC, ∠A = (5x − 3)°, ∠B = (2x + 7)°, ∠C = (3x + 8)°. Find x and all three angles.
Theorem: Triangle Sum → (5x−3)+(2x+7)+(3x+8) = 180
10x + 12 = 180 → 10x = 168 → x = 16.8
x = 16.8   ∠A = 81°   ∠B = 40.6°   ∠C = 58.4°
P2 — Exterior Angle
The exterior angle at C is 112°. Remote angles: ∠A = (3x + 4)° and ∠B = (5x)°. Find x, then find each remote angle.
Theorem: Ext. Angle → (3x+4) + 5x = 112
8x + 4 = 112 → 8x = 108 → x = 13.5
x = 13.5   ∠A = 44.5°   ∠B = 67.5°
P3 — Isosceles
Isosceles △MNP, MN = MP. ∠M = 36°. Find ∠N and ∠P.
Formula: base angle = (180° − vertex angle) ÷ 2
MN = MP → ∠N = ∠P (base angles)
∠N = ∠P = (180° − 36°) ÷ 2 = 144° ÷ 2 = 72°
∠N = 72°   ∠P = 72°
P4 — Multi-Step
In △ABC, AB = BC and the exterior angle at B is 130°. Find all three interior angles.
Hint: use Exterior Angle Theorem first, then Isosceles.
Interior ∠B = 180° − 130° = 50° (linear pair)
AB=BC → ∠A=∠C; ∠A+∠C=180°−50°=130° → ∠A=∠C=65°
∠A = 65°   ∠B = 50°   ∠C = 65°
Exit Ticket — One problem per theorem
ET1 — Triangle Sum
In △ABC, ∠A = (3x)°, ∠B = (2x + 10)°, ∠C = 50°. Find x.
3x + (2x+10) + 50 = 180
5x + 60 = 180 → 5x = 120 → x = 24
x = 24
ET2 — Exterior Angle
Exterior angle = 118°. Remote angles: (4x + 2)° and (3x)°. Find x.
(4x+2) + 3x = 118
7x + 2 = 118 → 7x = 116 → x = 116/7 ≈ 16.6
x = 116/7 ≈ 16.6
ET3 — Isosceles
Isosceles △PQR, PQ = PR, ∠P = 50°. Find ∠Q and ∠R.
∠Q = ∠R (base angles)
∠Q = ∠R = (180°−50°) ÷ 2 = 65°
∠Q = ∠R = 65°

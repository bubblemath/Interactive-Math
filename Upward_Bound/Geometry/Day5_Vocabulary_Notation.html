🌐 The Undefined Terms: Building Blocks of Geometry

Some terms in geometry are so fundamental that we can't define them using simpler terms. We call these undefined terms.

🔄 Why "Undefined"? The Circular Logic Problem

Try to define a "point" without using the word itself:

A point is a location
→
A location is a spot
→
A spot is a position
→
A position is a point!

We accept these terms as understood without formal definition.

•
Point
A location in space with no size — no length, width, or height.
Notation: Point A or just A
↔
Line
A straight path that extends infinitely in both directions.
Notation: \(\overleftrightarrow{AB}\) or line \(\ell\)
▱
Plane
A flat surface that extends infinitely in all directions.
Notation: Plane ABC or Plane P

💡 Key Insight: Three Points Define a Plane

Any three non-collinear points determine exactly one plane. Think of a tripod — three legs that don't line up will always sit flat on a surface!

📖 The Notation Decoder

Geometry has its own alphabet. Learn to read it, and diagrams become stories you can understand.

🔑 The "Can You Measure It?" Rule

This one rule solves most notation confusion:

📏

YES → It's a NUMBER

Use \(AB\) and \(=\)

🚫📏

NO → It's an OBJECT

Use \(\overline{AB}\) and \(\cong\)

📚 The Notation Family Tree

\(\overline{AB}\)
Segment
The actual piece between A and B — an object.
Use with \(\cong\) (congruent)
\(AB\)
Distance/Length
The numerical measurement — a number.
Use with \(=\) (equal)
\(AB = 5\) cm
\(\overrightarrow{AB}\)
Ray
Starts at A, passes through B, continues forever.
⚠️ Direction matters!
\(\overrightarrow{AB} \neq \overrightarrow{BA}\)
\(\overleftrightarrow{AB}\)
Line
Extends forever in BOTH directions through A and B.
\(\overleftrightarrow{AB} = \overleftrightarrow{BA}\)

⚠️ Ray Directionality Warning

Think of a flashlight: The first letter is where you HOLD it, the second letter is where it POINTS.

\(\overrightarrow{AB}\) starts at A, points toward B → Different from \(\overrightarrow{BA}\) which starts at B, points toward A!

⚖️ Congruence vs. Equality

These two symbols look similar but mean very different things!

Symbol Name Used For Example
≅ Congruent Objects (shapes, segments, angles) \(\overline{AB} \cong \overline{CD}\)
= Equal Numbers (lengths, measures) \(AB = CD\) or \(m\angle A = 45°\)

🧠 Memory Device

"Equal is for numbers on a scale. Congruent is for shapes that match — like identical twins."

❌ Common Pitfalls

❌ Wrong ✅ Correct Why
\(m\angle A \cong 40°\) \(m\angle A = 40°\) 40° is a number, not a shape
\(\overline{AB} = \overline{CD}\) \(\overline{AB} \cong \overline{CD}\) Segments are objects, not numbers
\(AB \cong CD\) \(AB = CD\) Lengths are numbers
\(\overrightarrow{AB} = \overrightarrow{BA}\) \(\overrightarrow{AB} \neq \overrightarrow{BA}\) Rays have direction!

➕ Segment Addition Postulate

If point B is between points A and C, then the whole equals the sum of its parts.

📏 Segment Addition Postulate

AB + BC = AC

The whole segment equals the sum of its parts.

🔢 Algebraic Example

Problem: Point B is between A and C. If AB = (2x + 3) and BC = (x + 5), and AC = 20, find x.

Step 1: Write the equation: AB + BC = AC

Step 2: Substitute: (2x + 3) + (x + 5) = 20

Step 3: Solve: 3x + 8 = 20 → 3x = 12 → x = 4

✓ Check: AB = 11, BC = 9, AC = 20 ✓

📐 Angle Addition Postulate

If ray BD is in the interior of ∠ABC, then the whole angle equals the sum of its parts.

📐 Angle Addition Postulate

m∠ABD + m∠DBC = m∠ABC

The whole angle equals the sum of its parts.

🔢 Algebraic Example

Problem: Ray BD is in the interior of ∠ABC. If m∠ABD = (3x + 10)° and m∠DBC = (2x + 5)°, and m∠ABC = 75°, find x.

Step 1: Write the equation: m∠ABD + m∠DBC = m∠ABC

Step 2: Substitute: (3x + 10) + (2x + 5) = 75

Step 3: Solve: 5x + 15 = 75 → 5x = 60 → x = 12

✓ Check: m∠ABD = 46°, m∠DBC = 29°, Sum = 75° ✓

✂️ Midpoint & Bisector Lab

These words are commands to write equations. When you see them, your brain should immediately think: EQUAL PARTS!

💡 The "Equal Splitter" Rule

Midpoint → Two equal segments → Set them equal!

Bisector → Two equal angles → Set them equal!

📍 Midpoint

A midpoint divides a segment into two equal parts.

Definition of Midpoint

If M is the midpoint of \(\overline{AB}\), then:

AM = MB

🔗 Interactive Exploration: Midpoint

Explore how the midpoint divides a segment into equal parts:

🔗 Open in GeoGebra (Full Screen)

🔢 Algebraic Example

Problem: M is the midpoint of \(\overline{PQ}\). If PM = (3x + 2) and MQ = (5x - 6), find x and PQ.

Step 1: Midpoint means equal parts: PM = MQ

Step 2: Set equal: 3x + 2 = 5x - 6

Step 3: Solve: 8 = 2x → x = 4

Step 4: Find PQ: PM = 14, MQ = 14, so PQ = 28

✓ Check: 3(4) + 2 = 14 and 5(4) - 6 = 14 ✓

✂️ Angle Bisector

An angle bisector divides an angle into two equal parts.

Definition of Angle Bisector

If \(\overrightarrow{BD}\) bisects \(\angle ABC\), then:

m∠ABD = m∠DBC

🔗 Interactive Exploration: Angle Bisector

See how the bisector creates two equal angles:

🔗 Open in GeoGebra (Full Screen)

🔢 Algebraic Example

Problem: \(\overrightarrow{BD}\) bisects \(\angle ABC\). If m∠ABD = (2x + 15)° and m∠DBC = (3x - 5)°, find x and m∠ABC.

Step 1: Bisector means equal parts: m∠ABD = m∠DBC

Step 2: Set equal: 2x + 15 = 3x - 5

Step 3: Solve: 20 = x → x = 20

Step 4: Find m∠ABC: Each half = 55°, so m∠ABC = 110°

✓ Check: 2(20) + 15 = 55° and 3(20) - 5 = 55° ✓

📋 Quick Reference: Statement → Reason

When You See... The Reason Is...
Two segments are equal (from a midpoint) Definition of Midpoint
Two angles are equal (from a bisector) Definition of Angle Bisector
Part + Part = Whole (segments) Segment Addition Postulate
Part + Part = Whole (angles) Angle Addition Postulate
Segments/angles have the same measure Definition of Congruent Segments/Angles

💡 Key Insight

The vocabulary you learned today IS the proof language. When you write "Definition of Midpoint" as a reason, you're saying "the two parts are equal because M is the midpoint." The definition IS the justification!

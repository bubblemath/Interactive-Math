🔄 Day 11: Transformations — The Motion Masterclass

Translation, Reflection, Rotation & Dilation

🎯 "When you walk into geometry class and the teacher says 'Today we're starting transformations,' you'll be the student who says 'Oh, I know this!' That confidence matters — it sets the tone for the whole year."

Transformations are the OPENING UNIT of most 9th-grade geometry courses!

Meeting them here first removes the "I've never seen this" panic. We'll frame transformations as "Coordinate Commands" — programming instructions for moving shapes.

  • Translation: The "Slide" — add to coordinates to move
  • Reflection: The "Flip" — mirror across a line
  • Rotation: The "Turn" — spin around a point (origin)
  • Dilation: The "Zoom" — scale up or down
  • Composition: Combining transformations — ORDER MATTERS!

Master these coordinate rules today, and you'll start geometry with confidence!

📚 The Rule Book: Coordinate Commands

Think of these rules as "programming commands" for the coordinate plane. Each transformation has a specific rule that tells you exactly where each point moves!

➡️
Translation
"The Slide"
Moves every point the same distance in the same direction
🪞
Reflection
"The Flip"
Creates a mirror image across a line
🔄
Rotation
"The Turn"

🔎

"The Zoom"

📐 Translation (The Slide)

Arrow Notation

\((x, y) \to (x + a, y + b)\)

Add \(a\) to move right (subtract to move left). Add \(b\) to move up (subtract to move down).

💡 Detective Tip

"Translation is the simplest transformation — just add or subtract from each coordinate. The shape doesn't rotate or flip, it just slides to a new location!"

🪞 Reflections (The Flip)

Reflections create mirror images. The key is knowing which coordinate changes!

Reflection Line Arrow Notation Detective Tip
x-axis \((x, y) \to (x, -y)\) The x stays; the y flips sign
y-axis \((x, y) \to (-x, y)\) The y stays; the x flips sign
y = x \((x, y) \to (y, x)\) Swap x and y — they trade places!
y = -x \((x, y) \to (-y, -x)\) Swap AND negate both!

💡 Memory Device

"For axis reflections, think about which axis is the MIRROR. If you reflect over the x-axis, the x-coordinate stays the same (it's ON the mirror line). The y-coordinate flips to the other side."

🔄 Rotations (The Turn) — Around the Origin

All rotations are counterclockwise (CCW) unless stated otherwise. We focus on rotations around the origin (0, 0).

Rotation Arrow Notation Detective Tip
90° CCW \((x, y) \to (-y, x)\) Swap, then negate the NEW first coordinate
180° \((x, y) \to (-x, -y)\) Both signs flip! (Double negation)
270° CCW (= 90° CW) \((x, y) \to (y, -x)\) Swap, then negate the NEW second coordinate

⚠️ Common Misconception

"180° rotation looks like a reflection, but it's different! Rotation preserves orientation (vertex order stays the same), while reflection reverses it."

🔎 Dilation (The Zoom)

Arrow Notation

\((x, y) \to (kx, ky)\)

Multiply both coordinates by the scale factor \(k\)

Scale Factor (k) Effect Example
\(k > 1\) Enlargement \(k = 2\): (3, 4) → (6, 8)
\(0 < k < 1\) Reduction \(k = 0.5\): (6, 8) → (3, 4)
\(k = 1\) No change (3, 4) → (3, 4)

💡 Key Insight

"Dilation is the only transformation that changes SIZE. The others (translation, reflection, rotation) are RIGID MOTIONS — they preserve size and shape!"

🔍 Dilation Station

Dilation is the "zoom" transformation — it changes SIZE while keeping the same SHAPE. This connects directly to Day 10's similarity concepts!

The Dilation Rule

\((x, y) \to (kx, ky)\)

Where \(k\) is the scale factor (centered at the origin)

📊 Scale Factor Effects

🔎
Enlargement
\(|k| > 1\)
Shape gets BIGGER
Example: k = 2 doubles all distances
🔍
Reduction
\(0 < |k| < 1\)
Shape gets SMALLER
Example: k = ½ halves all distances
📐
Congruent
\(k = 1\)
No size change
Same as the original!

📝 Worked Examples

Example 1: Enlargement (k = 3)

Dilate point A(2, -4) by scale factor k = 3 centered at the origin.

A(2, -4) → A'(3·2, 3·(-4)) = A'(6, -12)

Example 2: Reduction (k = ½)

Dilate point B(8, -6) by scale factor k = ½ centered at the origin.

B(8, -6) → B'(½·8, ½·(-6)) = B'(4, -3)

Example 3: Finding Scale Factor

Point C(3, 5) is dilated to C'(9, 15). What is the scale factor?

k = 9/3 = 3 (or k = 15/5 = 3) ✓

💡 Connection to Day 10

Dilation creates similar figures! The scale factor k in dilation is the same as the scale factor we used in Day 10's similarity problems. Dilated figures have:

  • Congruent angles (same shape)
  • Proportional sides (scaled by k)

🔗 Composition Station: Order Matters!

A composition is when you apply two or more transformations in sequence. The key insight: ORDER MATTERS!

⚠️ Critical Concept

"Composition is like following a recipe — the order of steps matters! 'Bake then frost' gives you a different result than 'frost then bake.'"

📝 Worked Example: Step-by-Step Composition

Apply to point (2, 3): First reflect over the x-axis, then translate by (x+1, y+2)

1
Start with the original point
(2, 3)
2
Apply Transformation 1: Reflect over x-axis → (x, y) → (x, -y)
(2, 3) → (2, -3)
3
Apply Transformation 2: Translate (x+1, y+2)
(2, -3) → (3, -1)

Final Answer:

(2, 3) → (3, -1)

⚠️ Proof That Order Matters

Let's apply the SAME two transformations to point (2, 1) but in DIFFERENT orders:

Sequence A: Translate THEN Reflect
Start: (2, 1)
Translate (x+3, y): (2, 1) → (5, 1)
Reflect over y-axis: (5, 1) → (-5, 1)
Final: (-5, 1)
Sequence B: Reflect THEN Translate
Start: (2, 1)
Reflect over y-axis: (2, 1) → (-2, 1)
Translate (x+3, y): (-2, 1) → (1, 1)
Final: (1, 1)

💡 Different Orders = Different Results!

(-5, 1) ≠ (1, 1) — Same transformations, different order, different answer!

📝 More Composition Examples

Example: Rotation then Dilation

Apply to (4, 0): Rotate 90° CCW, then dilate by k = 2

Step 1: (4, 0) → (-0, 4) = (0, 4) [90° CCW]
Step 2: (0, 4) → (0, 8) [Dilation k=2]

Final: (4, 0) → (0, 8)

Example: Double Reflection

Apply to (3, -2): Reflect over x-axis, then reflect over y-axis

Step 1: (3, -2) → (3, 2) [x-axis reflection]
Step 2: (3, 2) → (-3, 2) [y-axis reflection]

Final: (3, -2) → (-3, 2)

💡 Notice: This is the same as a 180° rotation!

Section 2 — Slope Discovery: Rise Over Run
★ Physical Intuition First — Think before the formula:
Imagine walking right along a hill. For every 5 feet forward, the hill rises 3 feet.
How steep is the hill? Write it as a ratio: 3 ÷ 5 = 3/5
Now: the hill drops 3 feet for every 5 feet forward. What changes? The sign becomes negative (−3/5)
Derive the formula — fill in the blanks:
Rise = vertical change = y₂ − y₁ (which two coordinates?)
Run = horizontal change = x₂ − x₁ (which two coordinates?)
Slope = rise ÷ run = y₂ − y₁ / x₂ − x₁
Slope Formula
m = (y₂−y₁)/(x₂−x₁)
= rise / run
= vertical change / horizontal change
Always reduce the answer!
Connection to Day 7:
Distance formula: square the differences, add, √
Slope formula: divide the differences
Same legs — different operation!
📈 GeoGebra — Rise Over Run Explorer geogebra.org/m/qnTcazst
Task: Drag the two points. Watch the slope triangle form. Observe how the rise and run change independently.
  • If you move one point straight up (only y changes), which part changes? The numerator (rise) changes; the denominator (run) stays the same.
  • If you move one point directly right (only x changes), which part changes? The denominator (run) changes; the numerator (rise) stays the same.
  • Can you make a slope of exactly 2? What coordinates work? E.g. (0,0) and (1,2): rise=2, run=1 → m=2/1=2. ✓
  • Can you create slope −1/3? What must rise and run be? Rise = −1, run = 3 (or rise = 1, run = −3). E.g. (0,3) and (3,2).

Example (guided): A(2, 3) and B(5, 9)

⚠ Zero vs. Undefined: Zero slope = HORIZONTAL line (y = k). Undefined slope = VERTICAL line (x = k). A flat floor has slope zero. A vertical wall has undefined slope — you can't walk on it!

Section 3 — Parallel Lines: Same Slope, Same Direction
🛤️ GeoGebra — Parallel Lines Explorer geogebra.org/m/RZFajaVH
Task: Open the applet and drag one of the parallel lines. Watch both slope values change together. Observe the relationship.
  • What do you notice about the two slope values when the lines are parallel? They are always equal — the two slope values are identical.
  • If you change slope so it's NO longer parallel, what happens? The slopes become unequal, and the lines will eventually intersect.
  • Complete this sentence: "Parallel lines always have _________ slopes." EQUAL / IDENTICAL
  • Why does different slope mean the lines eventually cross? Different slopes mean the lines are going in different directions — they must eventually meet at a point.
∥ Parallel Lines Rule
m₁ = m₂
Same slope = same direction = lines never meet.
Same slope, different y-intercept.
Day 3 connection: equal slopes ↔ equal corresponding angles
Parallel lines examples — fill in the missing slope:

Section 4 — Perpendicular Lines: The Negative Reciprocal

⊥ Perpendicular Lines Rule
m₁ × m₂ = −1
Flip the fraction AND change the sign.
This is called the negative reciprocal.
Quick check: if product = −1, they're ⊥ ✓

Section 5 — Equation Builder: Point-Slope Form
Point-Slope Form — "Plug and Play"
y − y₁ = m(x − x₁)
Use when: you have a point (x₁, y₁) and a slope m.
Plug in — then distribute to get y = mx + b form.
Where does this formula come from?
Start with slope formula: m = (y − y₁)/(x − x₁)
Multiply both sides by (x − x₁):
m(x − x₁) = y − y₁ → this IS point-slope!
It's just the slope formula rearranged — no new formula to memorize.
★ Always write Step 1 first:
Step 1: Identify the slope
Basic problem: slope given → m = ___
Parallel problem: same slope → m = (copy from given line)
Perpendicular: flip & negate → m = negative reciprocal
⚠ Do NOT skip Step 1. Wrong slope = wrong equation.

Step 1: Slopem = 3 (given directly)
Step 2: Plug iny − 4 = 3(x − 1)
Step 3: Distributey − 4 = 3x − 3
Step 4: Slope-Int. formy = 3x + 1
Example 2 — Fraction slope (try it)m = −2/3, point (6, −1)

Step 3: Distributey + 1 = −2/3x + 4

⚠ y − (−1) = y + 1, NOT y − 1

Step 2: Plug iny − 1 = 2(x − 3)

P1 — Slope + Equation
Find slope of A(−2, 5) to B(4, −1). Then write the equation of the line through A with that slope.
m = (−1−5)/(4−(−2)) = −6/6 = −1
y−5 = −1(x−(−2)) → y−5 = −x−2 → y = −x+3

P2 — Parallel or Perpendicular?
Line 1 through (0,0) and (4,6). Line 2 through (1,5) and (5,8). Find both slopes and state the relationship.
m₁ = 6/4 = 3/2   m₂ = (8−5)/(5−1) = 3/4
3/2 ≠ 3/4, product = 3/2 × 3/4 = 9/8 ≠ −1
m₁ = 3/2   m₂ = 3/4   Relationship: Neither ∥ nor ⊥

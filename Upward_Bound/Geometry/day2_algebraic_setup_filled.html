Segment Addition Postulate
AB + BC = AC
If B is between A and C, the whole = sum of its parts.
Use when: a point lies between two other points on a segment.
Section 2 (continued) — Segment Addition: Algebraic Example
Guided — Case File: Segment Addition
B is between A and C. AB = (2x + 1), BC = (3x), AC = 21. Find x.
Step 1: IDENTIFYB is between A and C → Segment Addition Postulate
Step 2: TRANSLATEAB + BC = AC → (2x + 1) + (3x) = 21
Step 3: SOLVE5x + 1 = 21  →  5x = 20  →  x = 4
Check: AB = 9, BC = 12, AC = 21 ✓
Try it — Your TurnUse the 3 steps
M is between P and Q. PM = (x + 5), MQ = (2x − 1), PQ = 25. Find x, PM, and MQ.
Step 1: IDENTIFYM is between P and Q → Segment Addition Postulate
Step 2: TRANSLATEPM + MQ = PQ → (x + 5) + (2x − 1) = 25
Step 3: SOLVE3x + 4 = 25 → 3x = 21 → x = 7
x = 7   PM = 12   MQ = 13   Check: 12+13=25 ✓
Key Insight: The Segment Addition Postulate is just common sense — the whole equals the sum of its parts. When you see algebraic expressions for segment lengths, use this postulate to set up the equation.

Angle Relationship Decoder
Five angle relationships — identify the type, then write the equation automatically
Name: Answer Key
📐 GeoGebra — Explore Angle Relationships (before reading the cards below) geogebra.org/m/a2kf7ypd
Task: Open the applet and drag the angles. Observe how each pair behaves.
  • When two angles are complementary, what does their sum always equal? Their sum always equals 90°. You can see this because complementary angles together form a right angle.
  • What is different about supplementary angles versus a linear pair? Both add to 180°, but a linear pair must be adjacent (sharing a side, on a straight line). Supplementary angles don't need to be next to each other — they just need to sum to 180°.
  • When two lines cross, vertical angles are formed. What do you predict is true about them? Vertical angles are always equal (congruent) — they are opposite each other and have the same measure.
  • Can adjacent angles ever be equal to each other? Yes — if the shared ray bisects the larger angle, the two adjacent angles are equal. For example, if a ray bisects a 90° angle, the two adjacent angles are each 45°.
The Five Angle Relationships — completed equation setup column
Complementary
Two angles whose sum is 90°. Often found inside a right angle.
∠1 + ∠2 = 90°
Supplementary
Two angles whose sum is 180°. Do not need to be adjacent.
∠1 + ∠2 = 180°
Linear Pair
Adjacent angles on a straight line. Always supplementary — sum = 180°.
∠1 + ∠2 = 180°
Vertical Angles
Formed by two intersecting lines. Opposite (vertical) angles are always congruent (equal).
∠1 = ∠1   →   set the two angle expressions equal
Adjacent Angles
Share a common vertex B and a common side BD. No overlap.
No automatic sum — additional info must be given

The single most important rule: Identify the relationship first — then the equation writes itself. Opposite sides → set equal. Complementary → sum to 90. Supplementary or linear pair → sum to 180.

The 3-Step Detective Method
1
IDENTIFY
What geometric relationship do you see?
Complementary? Supplementary? Vertical? Linear pair? Segment addition?
2
TRANSLATE
Write the equation.
Equal → set expressions equal. Sum rule → add to 90 or 180. Segment → add parts to get whole.
3
SOLVE & VERIFY
Solve for x. Check your answer.
Substitute x back in. Does the relationship hold?
Complementary / Supplementary
(∠1) + (∠2) = 90 or 180
Vertical Angles
(expr 1) = (expr 2)
Segment Addition
AB + BC = AC
Example 1 — Complementary (guided)
Two angles are complementary. ∠1 = (3x − 5)°, ∠2 = (x + 15)°. Find x.
Step 1: IDENTIFYComplementary → sum = 90°
Step 2: TRANSLATE(3x − 5) + (x + 15) = 90
Step 3: SOLVE4x + 10 = 90  →  4x = 80  →  x = 20
Check: ∠1 = 55°, ∠2 = 35°, sum = 90° ✓
Example 2 — Supplementary (try it)fill in the blanks
Two angles are supplementary. ∠1 = (2x + 40)°, ∠2 = (3x + 10)°. Find x.
Step 1: IDENTIFYSupplementary → sum = 180°
Step 2: TRANSLATE(2x + 40) + (3x + 10) = 180
Step 3: SOLVE5x + 50 = 180  →  5x = 130  →  x = 26
x = 26   ∠1 = 92°   ∠2 = 88°   Check: sum = 180° ✓
Spot the Error Challenge
Problem: Two angles form a linear pair. One is (2x + 40)° and the other is (3x + 10)°. Find x.
Student's incorrect work:
(2x + 40) + (3x + 10) = 90
5x + 50 = 90
x = 8
What is wrong? (circled answer)
○   Should multiply instead of add
○   Linear pairs add to 180°, not 90°
○   Should set the angles equal
○   Nothing is wrong
Correct solution:
(2x+40)+(3x+10)=180 → 5x+50=180 → x=26
Answer: x = 26
Always name the pair type BEFORE writing the equation. The type tells you which equation to write.
Practice Problems
P1 — Vertical Angles
Vertical angles: (3x + 15)° and (5x − 25)°. Find x and both angle measures.
3x + 15 = 5x − 25 → −2x = −40 → x = 20
x = 20   each angle = 75°
P2 — Complementary
Complementary: (4x)° and (x + 15)°. Find x and both angle measures.
4x + (x+15) = 90 → 5x + 15 = 90 → x = 15
x = 15   ∠1 = 60°   ∠2 = 30°
P3 — Linear Pair
Linear pair: (4x − 10)° and (x + 40)°. Find x and both angles.
(4x−10)+(x+40)=180 → 5x+30=180 → x=30
x = 30   ∠1 = 110°   ∠2 = 70°
P4 — Segment Addition (Midpoint)
M is the midpoint of PQ. PM = (3x + 2), MQ = (5x − 6). Find x and PQ.
Midpoint → PM = MQ (set equal)
3x+2 = 5x−6 → −2x=−8 → x=4
x = 4   PM = MQ = 14   PQ = 28

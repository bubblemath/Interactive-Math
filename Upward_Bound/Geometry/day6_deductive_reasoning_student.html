StatementReason

Key Insight: Proofs are not new math — you already know how to solve equations. A two-column proof is just your solution with every step labeled. The structure is: Statement (what you did) | Reason (why you were allowed to do it).

Section 4 — Logic Lab: IF-THEN Reasoning
The Logic Machine — every theorem is IF-THEN:
IF M is the midpoint of AB    THEN AM = MB
IF angles are vertical            THEN they are ≅
IF angles are supplementary    THEN sum = 180°
Write the IF-THEN form for the Exterior Angle Theorem:
IF THEN
The Converse — flip IF and THEN. Is it still true?
Original (IF → THEN)True?Converse (THEN → IF)True?
IF square THEN rectangle T IF rectangle THEN square
IF vertical angles THEN congruent T IF congruent THEN vertical
IF 2 sides equal THEN base angles = T IF base angles = THEN 2 sides equal
IF parallel lines THEN corresponding ∠s = T IF corresponding ∠s = THEN parallel
Converse Rule: Just because a statement is true does NOT mean its converse is automatically true. You must prove each separately! Example: IF rectangle THEN square is false — a rectangle can be any proportions, but a square must have equal sides.

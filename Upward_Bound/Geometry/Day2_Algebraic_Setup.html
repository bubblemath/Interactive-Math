📏 Segment Addition Postulate

If point B is between points A and C, then: AB + BC = AC

Drag point B to see how the segments change:

A 0
B 8
C 20
AB = 8 + BC = 12 = AC = 20

🔍 Case File: Segment Addition

Point B is between A and C.

AB = 2x + 1    BC = 3x    AC = 21

Find the value of x.

Step 1: IDENTIFY B is between A and C → Use Segment Addition Postulate
Step 2: TRANSLATE AB + BC = AC → (2x + 1) + 3x = 21
Step 3: SOLVE 5x + 1 = 21 → 5x = 20 → x = 4

🎯 Key Insight

The Segment Addition Postulate is just common sense written as math: the whole equals the sum of its parts. When you see algebraic expressions for segment lengths, set up the equation and solve!

🎯 The Pattern

Step 1: Identify the relationship (complementary, supplementary, linear pair, vertical, or adjacent)

Step 2: Write the equation based on that relationship

Step 3: Solve for the variable

🔴 Tampered Evidence: Spot the Error!

Problem: Two angles form a linear pair. One is (2x + 40)° and the other is (3x + 10)°.

Student's work: (2x + 40) + (3x + 10) = 90

What's wrong with this setup?

A) Should multiply instead of add
B) Linear pairs add to 180°, not 90°
C) The angles should be set equal to each other
D) Nothing is wrong

⏱️ Why Time Yourself?

Shape Detectives, timing yourself serves two important purposes:

  1. Fluency Building: In geometry, you'll encounter these setups dozens of times per unit. Timing helps you move from "thinking through each step" to "automatic recognition" — the same way you don't have to think about spelling common words.
  2. Exam Readiness: Standardized tests and classroom exams have time limits. Practicing under mild time pressure now builds confidence for when it counts.
💡 Tip: Start with the timer OFF while learning. Turn it ON once you feel comfortable to build speed. The timer counts UP (not down) to reduce anxiety.

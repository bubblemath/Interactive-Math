📍 Case 1: Coordinate Evidence

Three corners of the monument's base have been located at A(2, 1), B(8, 1), and C(8, 5). Your mission: Find the fourth corner and verify the shape.

Task 1: Find the Fourth Vertex

If ABCD is a rectangle, what are the coordinates of vertex D?

Hint: In a rectangle, opposite sides are parallel and equal in length.

Task 2: Calculate the Perimeter

Use the Distance Formula to find the perimeter of rectangle ABCD.

Distance Formula: \(d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}\)

Task 3: Find the Center Point

Find the center of the monument using the Midpoint Formula on diagonal AC.

Midpoint Formula: \(M = \left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)\)

Task 4: Verify the Rectangle

Prove ABCD is a rectangle by showing that adjacent sides are perpendicular. What is the slope of AB? What is the slope of BC?

Perpendicular lines have slopes that are negative reciprocals (or one is 0 and one is undefined).

📐 Case 2: The Angle Investigation

A security laser grid protects the monument. The lasers form parallel lines cut by a transversal. Decode the angles to disable the grid!

Task 1: Corresponding Angles

Two parallel lines are cut by a transversal. ∠1 = (5x + 10)° and ∠5 = (3x + 40)° are corresponding angles.

Find the value of x.

Hint: Corresponding angles are EQUAL when lines are parallel.

Task 2: Find the Angle Measure

Using your value of x from Task 1, find the measure of ∠1.

°

Task 3: Co-Interior Angles

∠3 = (2y + 20)° and ∠5 = (4y - 10)° are co-interior (same-side interior) angles.

Find the value of y.

Hint: Co-interior angles are SUPPLEMENTARY (add to 180°).

Task 4: Exterior Angle Theorem

A triangular warning sign has an exterior angle of 140°. One remote interior angle is 65°.

Find the other remote interior angle.

Hint: Exterior angle = sum of remote interior angles.

🗺️ Case 3: The Scale Model

The detectives find an ancient map with a scale of 1 unit : 15 meters. Use proportional reasoning to find the actual dimensions!

Task 1: Calculate Actual Perimeter

From Case 1, the monument's perimeter on the map is 20 units.

Using the scale 1 unit : 15 meters, find the actual perimeter in meters.

meters

Task 2: Find the Scale Factor

A similar triangular chamber is shown on the map with sides 2, 3, and 4 units.

The actual base (corresponding to the 3-unit side) is 45 meters.

What is the scale factor from map to actual?

Task 3: Find Missing Sides

Using the scale factor from Task 2, find the actual lengths of the other two sides of the triangular chamber.

meters meters

Task 4: Proportion Problem

A doorway on the map is \(\frac{x}{12}\) units wide. The actual doorway is 10 meters wide.

If the scale is 1 unit : 15 meters, set up and solve a proportion to find x.

Hint: \(\frac{x/12}{1} = \frac{10}{15}\)

🔄 Case 4: The Final Shift

The treasure chamber has shifted over centuries due to geological activity. Track the transformations to find its current location!

Task 1: Translation

The original entrance was at point E(3, 2). It has been translated by the rule (x - 5, y + 3).

Find the new location of the entrance.

Task 2: Rotation

A key artifact at F(4, -1) was rotated 90° counterclockwise around the origin.

Where is it now?

Rule: (x, y) → (-y, x)

Task 3: Reflection

The final treasure location is found by reflecting point G(6, 2) over the line y = x.

What are the coordinates of the treasure?

Rule: (x, y) → (y, x)

Task 4: Composition

A secret marker at H(2, -4) underwent two transformations:

1. First, reflected over the x-axis

2. Then, translated by (x + 3, y - 1)

Find the final location.

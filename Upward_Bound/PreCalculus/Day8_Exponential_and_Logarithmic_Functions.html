📖 Log & Exp Rules

🔄 Form Conversion

Exponential ↔ Log
\(b^x = y \iff \log_b(y) = x\)

📐 Log Properties

Product Rule
\(\log_b(xy) = \log_b(x) + \log_b(y)\)
Quotient Rule
\(\log_b(\frac{x}{y}) = \log_b(x) - \log_b(y)\)
Power Rule
\(\log_b(x^n) = n \cdot \log_b(x)\)
Change of Base
\(\log_b(x) = \frac{\ln(x)}{\ln(b)}\)

⚡ Special Values

Log of 1
\(\log_b(1) = 0\) (any base)
Log of Base
\(\log_b(b) = 1\)
Inverse Property
\(b^{\log_b(x)} = x\) and \(\log_b(b^x) = x\)

🎮 Warm-Up: Growth or Decay?

Identify whether each function represents exponential growth or decay.

Function 1: \(f(x) = 3^x\)

✓ Growth! Base 3 > 1, so the function increases as x increases.

Function 2: \(f(x) = (0.5)^x\)

✓ Decay! Base 0.5 < 1, so the function decreases as x increases.

Function 3: \(f(x) = e^{-x}\)

✓ Decay! The negative exponent means \(e^{-x} = (1/e)^x\), and 1/e < 1.

Function 4: \(f(x) = 2 \cdot 1.05^x\)

✓ Growth! Base 1.05 > 1 (5% growth rate). The coefficient 2 is just the initial value.

📚 Key Concepts

🔄 The Base Switcher

A logarithm IS an exponent! Watch the numbers rearrange between forms.

Exponential Form:

2 3 = 8
⇅

Logarithmic Form:

log 2 ( 8 ) = 3

Key Insight: "log base b of y equals x" means "b to the x power equals y"

\(\log_b(y) = x \iff b^x = y\)

📊 Common Logarithm

\(\log(x) = \log_{10}(x)\)

Base 10 (calculator's LOG button)

Example: \(\log(100) = 2\) because \(10^2 = 100\)

🌿 Natural Logarithm

\(\ln(x) = \log_e(x)\)

Base e ≈ 2.71828 (calculator's LN button)

Example: \(\ln(e) = 1\) because \(e^1 = e\)

🪞 Inverse Relationship

Exponential and logarithmic functions are inverses—they "undo" each other!

Exponential → Log

\(\log_b(b^x) = x\)

Log "undoes" the exponential

Log → Exponential

\(b^{\log_b(x)} = x\)

Exponential "undoes" the log

🧮 Log Property Lab

Toggle between expanded and condensed forms to see how log properties work.

✖️ Product Rule

Logs turn multiplication into addition!

Condensed

\(\log_b(xy)\)
⟷

Expanded

\(\log_b(x) + \log_b(y)\)
Example: \(\log_2(8 \cdot 4) = \log_2(8) + \log_2(4) = 3 + 2 = 5\)

➗ Quotient Rule

Logs turn division into subtraction!

Condensed

\(\log_b\left(\frac{x}{y}\right)\)
⟷

Expanded

\(\log_b(x) - \log_b(y)\)
Example: \(\log_3(27/9) = \log_3(27) - \log_3(9) = 3 - 2 = 1\)

⬆️ Power Rule

Exponents become coefficients!

Condensed

\(\log_b(x^n)\)
⟷

Expanded

\(n \cdot \log_b(x)\)
Example: \(\log_2(8^4) = 4 \cdot \log_2(8) = 4 \cdot 3 = 12\)

🔀 Change of Base Formula

Convert any log to natural log or common log for your calculator!

\(\log_b(x) = \frac{\ln(x)}{\ln(b)} = \frac{\log(x)}{\log(b)}\)
Example: \(\log_5(20) = \frac{\ln(20)}{\ln(5)} \approx \frac{2.996}{1.609} \approx 1.861\)

🚀 Calculus Preview: Why "e" is Special

The function \(f(x) = e^x\) has a remarkable property:

\(\frac{d}{dx}[e^x] = e^x\)

It's the only function that equals its own derivative! This makes it fundamental to calculus, differential equations, and modeling continuous growth.

You'll explore this deeply in Week 3 when we study derivatives.

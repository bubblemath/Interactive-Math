📜 The Curator's Manual

Study these transformation techniques to restore the gallery.

🎨 The 6 Parent Functions

\(f(x) = x^2\) — Quadratic (U-shape)

\(f(x) = |x|\) — Absolute Value (V-shape)

\(f(x) = \sqrt{x}\) — Square Root (half-curve)

\(f(x) = x^3\) — Cubic (S-shape)

\(f(x) = \frac{1}{x}\) — Reciprocal (hyperbola)

\(f(x) = e^x\) — Exponential (growth curve)

📐 The a, b, h, k Framework

\(g(x) = a \cdot f(b(x - h)) + k\)

a = Vertical stretch/compress & reflect over x-axis

b = Horizontal stretch/compress & reflect over y-axis

h = Horizontal shift (opposite sign!)

k = Vertical shift

⚠️ The "Opposite" Trap:

\(f(x - 3)\) shifts RIGHT 3

\(f(x + 2)\) shifts LEFT 2

The sign is OPPOSITE what you see!

🔄 The H-S-R-V Order

Apply transformations in this order:

  1. Horizontal shift (h)
  2. Stretch/Compress (a, b)
  3. Reflect (negative a or b)
  4. Vertical shift (k)

Example: \(-2(x-3)^2 + 4\)
H: right 3, S: stretch ×2, R: flip, V: up 4

🪞 Inverse Functions

The Switcher Method:

  1. Replace \(f(x)\) with \(y\)
  2. Swap \(x\) and \(y\)
  3. Solve for \(y\)
  4. Replace \(y\) with \(f^{-1}(x)\)

Key Property:

\(f\) and \(f^{-1}\) are reflections over \(y = x\)

If \(f(a) = b\), then \(f^{-1}(b) = a\)

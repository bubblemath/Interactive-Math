🚫 Warm-Up: Spot the Illegal Cancel

Each problem shows a "simplification." Click the button that identifies whether it's LEGAL or ILLEGAL!

Problem 1

\(\frac{x + 5}{x} = 5\)
🚫 ILLEGAL! You cannot cancel terms that are being ADDED. The \(x\) in the numerator is part of a sum \((x + 5)\), not a factor. You can only cancel common FACTORS, not terms!

Problem 2

\(\frac{3x}{6x^2} = \frac{1}{2x}\)
✓ LEGAL! Both numerator and denominator share the factor \(3x\). Canceling gives \(\frac{1}{2x}\). This is valid because we're canceling a common FACTOR.

Problem 3

\(\frac{x^2 - 9}{x - 3} = x - 3\)
🚫 ILLEGAL! The numerator factors to \((x+3)(x-3)\), so after canceling \((x-3)\), the answer should be \(x + 3\), NOT \(x - 3\). Always factor first!

Problem 4

\(\frac{(x+2)(x-1)}{(x+2)(x+5)} = \frac{x-1}{x+5}\)
✓ LEGAL! The factor \((x+2)\) appears in both numerator and denominator, so it can be canceled. Just remember: \(x \neq -2\) is still a restriction!

📚 Key Concepts & Formulas

🔤 Essential Vocabulary

Term Definition
Rational Expression A fraction where the numerator and/or denominator contains a polynomial: \(\frac{P(x)}{Q(x)}\)
Domain Restriction Values of \(x\) that make the denominator equal to zero (these are EXCLUDED)
LCD Least Common Denominator - the smallest expression that all denominators divide into evenly
Extraneous Solution A "solution" that makes the original denominator zero - must be rejected!

🎯 Finding Domain Restrictions

Set each denominator factor equal to zero and solve:

\(\frac{x+1}{(x-2)(x+3)}\)

Restrictions: \(x \neq 2\) and \(x \neq -3\)

💡 Even after canceling, restrictions from the ORIGINAL expression still apply!

✂️ Simplifying Rational Expressions

\(\frac{x^2 - 4}{x^2 + 4x + 4} = \frac{(x-2)(x+2)}{(x+2)(x+2)} = \frac{x-2}{x+2}\)

Steps:

  1. Factor numerator completely
  2. Factor denominator completely
  3. Cancel common FACTORS only
  4. State restrictions

⚠️ The #1 Mistake: Illegal Canceling

❌ WRONG:

\(\frac{x + 3}{x} \neq 3\)

Can't cancel TERMS across addition!

✓ RIGHT:

\(\frac{x \cdot 3}{x} = 3\)

CAN cancel FACTORS (multiplication)!

🧪 The LCD Recipe

1 Factor all denominators completely
2 List all unique factors from all denominators
3 Use the highest power of each factor that appears
4 Multiply all factors together = LCD!

Example: Find LCD of \(\frac{1}{x^2-9}\) and \(\frac{1}{x^2+5x+6}\)

\(x^2-9 = (x+3)(x-3)\)

\(x^2+5x+6 = (x+2)(x+3)\)

LCD = \((x+3)(x-3)(x+2)\)

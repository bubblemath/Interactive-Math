📚 KEY DEFINITIONS: The Unit Circle
\(x = \cos\theta\)
\(y = \sin\theta\)
\(r = 1\) (always!)

Every point on the unit circle is \((\cos\theta, \sin\theta)\)

📐 The Six Trig Functions:

Sine
\(\sin\theta = \frac{\text{opp}}{\text{hyp}}\)
= y-coordinate
Cosine
\(\cos\theta = \frac{\text{adj}}{\text{hyp}}\)
= x-coordinate
Tangent
\(\tan\theta = \frac{\text{opp}}{\text{adj}}\)
= y/x = sin/cos
csc θ ↔ sin θ
\(\csc\theta = \frac{1}{\sin\theta}\)
sec θ ↔ cos θ
\(\sec\theta = \frac{1}{\cos\theta}\)
cot θ ↔ tan θ
\(\cot\theta = \frac{1}{\tan\theta}\)
🧭 QUADRANT SIGNS & CONVERSIONS

ASTC: "All Students Take Calculus"

S Sin+ only
A All positive
T Tan+ only
C Cos+ only

🔄 Degree ↔ Radian Conversion

Degrees
⇄
Radians
Key: \(180° = \pi\) radians
Deg → Rad: multiply by \(\frac{\pi}{180}\)
Rad → Deg: multiply by \(\frac{180}{\pi}\)
⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"Reciprocal Confusion"
\(\sec\theta = \frac{1}{\sin\theta}\) 		Secant is the reciprocal of COSINE, not sine. The "c" in sec goes with the "c" in cos! \(\sec\theta = \frac{1}{\cos\theta}\)
\(\csc\theta = \frac{1}{\sin\theta}\)
"Calculator Mode"
Getting 0.0175 for \(\sin(1)\) 		Calculator was in degree mode when you meant radians! \(\sin(1°) \approx 0.0175\) but \(\sin(1 \text{ rad}) \approx 0.841\) Always check DEG/RAD mode before calculating!
"Quadrant Blindness"
\(\sin(210°) = \frac{1}{2}\) 		210° is in Quadrant III where sine is NEGATIVE. The reference angle gives the value, but ASTC gives the sign. \(\sin(210°) = -\frac{1}{2}\)
(Q3: only tan is positive)

🔧 PROCEDURE: Finding Reference Angles

The reference angle \(\theta'\) is the acute angle between the terminal side and the x-axis.

Q1 (0° - 90°)
\(\theta' = \theta\)
Q2 (90° - 180°)
\(\theta' = 180° - \theta\)
Q3 (180° - 270°)
\(\theta' = \theta - 180°\)
Q4 (270° - 360°)
\(\theta' = 360° - \theta\)
🔧 PROCEDURE: Evaluating Trig Functions at Any Angle
  1. Identify the Quadrant — Where does the terminal side land?
  2. Find the Reference Angle — Use the formulas above to find \(\theta'\)
  3. Determine the Sign — Use ASTC to decide if the answer is + or −
  4. Evaluate — Find the trig value of the reference angle, then apply the sign
✏️ WORKED EXAMPLES

Example 1: Find \(\sin(210°)\)

Step 1: Identify Quadrant
210° is between 180° and 270° → Quadrant III
Step 2: Find Reference Angle
\(\theta' = 210° - 180° = 30°\)
Step 3: Determine Sign
Q3: Only Tan is positive → sin is NEGATIVE
Step 4: Evaluate
\(\sin(30°) = \frac{1}{2}\), so \(\sin(210°) = -\frac{1}{2}\)
\(\sin(210°) = -\frac{1}{2}\)

Example 2: Find \(\cos\left(\frac{3\pi}{4}\right)\)

Step 1: Identify Quadrant
\(\frac{3\pi}{4} = 135°\) → between 90° and 180° → Quadrant II
Step 2: Find Reference Angle
\(\theta' = \pi - \frac{3\pi}{4} = \frac{\pi}{4}\) (or 45°)
Step 3: Determine Sign
Q2: Only Sin is positive → cos is NEGATIVE
Step 4: Evaluate
\(\cos\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}\), so \(\cos\left(\frac{3\pi}{4}\right) = -\frac{\sqrt{2}}{2}\)
\(\cos\left(\frac{3\pi}{4}\right) = -\frac{\sqrt{2}}{2}\)

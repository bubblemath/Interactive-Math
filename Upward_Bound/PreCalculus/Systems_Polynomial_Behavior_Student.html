Day 9: Multiplicity - Cross vs. Bounce

Section 3 - Multiplicity: Cross vs. Bounce

The multiplicity of a root tells us HOW the graph behaves at that x-intercept:

✖️ CROSS (Odd Multiplicity)

Multiplicity 1, 3, 5, ...

Graph passes the x-axis

"Screen door—walk right through"

🔄 BOUNCE (Even Multiplicity)

Multiplicity 2, 4, 6, ...

Graph and bounces back

"Trampoline—bounce back"

⚠️ Multiplicity 3+ Creates "Flattening"

Higher odd multiplicities (3, 5, ...) still cross, but the graph "flattens out" at the root—like pushing through a heavy door.

Section 4 - The 5-Step Polynomial Sketching Procedure
📋 THE 5-STEP RECIPE
  1. Find the zeros: Set \(f(x) = 0\) and solve (factor if needed)
  2. Determine multiplicity: Look at the exponent of each factor
  3. Check end behavior: Use the Leading Coefficient Test
  4. Plot the y-intercept: Evaluate \(f(0)\)
  5. Connect the dots: Use Cross/Bounce behavior at each root

Section 5 - What is a System of Equations?
📖 Definition

A system of equations is a set of two or more equations with the same variables. The solution is the point(s) where ALL equations are true simultaneously.

Section 6 - Solution Types

A system of two linear equations can have three types of solutions:

⚠️ How to Recognize Special Cases

No Solution: You get a FALSE statement like \(0 = 5\) (variables disappear, statement is false)

Infinite Solutions: You get a TRUE statement like \(0 = 0\) (variables disappear, statement is true)

Day 9: Solving Systems of Equations

Section 7 - Three Methods Comparison
Method Best When... Key Idea
Substitution One variable is already or easy to isolate Solve for one variable, plug into the other equation
Elimination Coefficients are or easy to make opposites Add/subtract equations to eliminate a variable
Graphing You need to the solution or estimate Graph both lines, find intersection point
Section 8 - The 7-Step Elimination Recipe
📋 THE ELIMINATION PROCEDURE
  1. Align: Write both equations in standard form (\(Ax + By = C\))
  2. Target: Choose which variable to eliminate
  3. Multiply: Multiply one or both equations so coefficients are opposites
  4. Add: Add the equations to eliminate the target variable
  5. Solve: Solve for the remaining variable
  6. Back-substitute: Plug back into either original equation
  7. CHECK: Verify your solution in BOTH original equations!
Worked Example: Solve the System

\(\begin{cases} 2x + 3y = 12 \\ 4x - 3y = 6 \end{cases}\)

Step 1-2: Already aligned. Target: \(y\) (coefficients are already opposites!)

Step 3: No multiplication needed.

Step 4: Add equations: \((2x + 3y) + (4x - 3y) = 12 + 6\) → \(6x = 18\)

Step 5: Solve: \(x = 3\)

Step 6: Back-substitute into Equation 1: \(2(3) + 3y = 12\) → \(6 + 3y = 12\) → \(y = 2\)

Solution: \((3, 2)\)

✓ CHECK in BOTH equations:

Eq 1: \(2(3) + 3(2) = 6 + 6 = 12\) ✓    Eq 2: \(4(3) - 3(2) = 12 - 6 = 6\) ✓

Day 9: Substitution Method & Common Pitfalls

Section 9 - The Substitution Method
Worked Example: Solve Using Substitution

\(\begin{cases} y = 2x - 1 \\ 3x + y = 9 \end{cases}\)

Step 1: \(y\) is already isolated in Equation 1.

Step 2: Substitute \(y = 2x - 1\) into Equation 2:

\(3x + (2x - 1) = 9\)

Step 3: Solve: \(5x - 1 = 9\) → \(5x = 10\) → \(x = 2\)

Step 4: Back-substitute: \(y = 2(2) - 1 = 3\)

Solution: \((2, 3)\)

Section 10 - Common Pitfalls
❌ The Trap ✅ The Fix
Forgetting to Distribute
Only multiplying the first term		 "Distributive Justice" — EVERY term gets the multiplier!
\(2(3x + y = 5)\) becomes \(6x + 2y = 10\), NOT \(6x + y = 5\)
Sign Errors in Elimination
Subtracting incorrectly		 When subtracting, change ALL signs in the second equation first, then add.
Or: multiply by \(-1\) to make addition easier.
Thinking \(0 = 5\) means \(x = 0\) \(0 = 5\) is a FALSE statement → No Solution (parallel lines)
Not Checking Both Equations A solution must work in BOTH equations. Always verify!

🚀 Calculus Connection: Where This Leads

End Behavior → Limits at Infinity

In Calculus, you'll write:

\(\lim_{x \to \infty} f(x) = \infty\)

This formalizes "what happens at the ends."

Multiplicity → Local Extrema

Even multiplicity roots often indicate local minima or maxima. The First Derivative Test will help you find these precisely!

Systems → Area Between Curves: Finding intersection points is essential for \(\int_a^b [f(x) - g(x)] dx\).

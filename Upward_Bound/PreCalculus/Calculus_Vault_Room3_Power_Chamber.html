📜 The Power Manual

Study these ancient power formulas to survive the chamber.

⚡ Exponent Rules

Product: \(x^a \cdot x^b = x^{a+b}\)

Quotient: \(\frac{x^a}{x^b} = x^{a-b}\)

Power: \((x^a)^b = x^{ab}\)

Zero: \(x^0 = 1\) (if \(x \neq 0\))

Negative: \(x^{-n} = \frac{1}{x^n}\)

Power of Product: \((xy)^n = x^n y^n\)

📊 Logarithm Properties

Product: \(\log(ab) = \log(a) + \log(b)\)

Quotient: \(\log\left(\frac{a}{b}\right) = \log(a) - \log(b)\)

Power: \(\log(a^n) = n\log(a)\)

Identity: \(\log_b(b) = 1\)

Zero: \(\log_b(1) = 0\)

Change of Base:
\(\log_b(x) = \frac{\ln(x)}{\ln(b)}\)

🔄 Exp ↔ Log Conversion

\(b^y = x \iff \log_b(x) = y\)

Example:

\(2^5 = 32 \iff \log_2(32) = 5\)

Natural Log:

\(\ln(x) = \log_e(x)\)

\(e \approx 2.71828...\)

🚀 Calculus Preview:

\(\frac{d}{dx}[e^x] = e^x\)

The only function equal to its own derivative!

🪤 Power Traps

Negative base + even root = NOT REAL

\((-4)^{1/2} = \sqrt{-4}\) → ❌ Not real!

But odd roots of negatives ARE real:

\((-8)^{1/3} = \sqrt[3]{-8} = -2\) → ✓ Real!

⚠️ Common Errors:

\(\log(a+b) \neq \log(a) + \log(b)\)

\((\log(a))^n \neq n\log(a)\)

\(\log(0)\) is UNDEFINED

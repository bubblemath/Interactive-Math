Day 11: The 5-Point Graphing Method

Section 5 - The 5-Point Anchor Method
📋 HOW TO GRAPH ANY SINUSOID
  1. Find the Midline: Draw the horizontal line \(y = D\).
  2. Calculate the Period: Period = \(\frac{2\pi}{|B|}\), then find "Jump Size" = Period ÷ 4.
  3. Determine the Starting Point: The phase shift \(C\) tells you where to begin.
  4. Plot the 5 Key Points: Start → Quarter → Middle → Three-Quarter → End.
  5. Connect with a Smooth Curve: No sharp corners or "V" shapes!
🎯 The 5 Anchor Points Pattern

For SINE (starting at midline, going UP):

Midline → Maximum → Midline → Minimum → Midline

For COSINE (starting at maximum):

Maximum → Midline → Minimum → Midline → Maximum

Worked Example: Graph \(y = 2\sin(x) + 1\)

Step 1: Midline: \(y = 1\)

Step 2: Period = \(\frac{2\pi}{1} = 2\pi\), Jump Size = \(\frac{2\pi}{4} = \frac{\pi}{2}\)

Step 3: Starting point: \(x = 0\) (no phase shift)

Step 4: The 5 x-values: \(0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi\)

Step 5: The 5 y-values (midline ± amplitude): \(1, 3, 1, -1, 1\)

Section 7 - Common Graphing Pitfalls
❌ The Trap ✅ The Fix
Period = B Period = \(\frac{2\pi}{|B|}\). If B = 2, the period is \(\pi\), not 2!
Wrong Phase Direction Always factor to standard form first. \((2x - \pi) = 2(x - \frac{\pi}{2})\)
Forgetting Reflection Negative A flips the wave. Sine starts DOWN, cosine starts at MIN.
Sharp Corners Trig waves are SMOOTH curves. No "V" shapes at max/min!

Day 11: Solving Trig Equations

Section 8 - The 5-Step Solving Strategy
📋 HOW TO SOLVE TRIG EQUATIONS
  1. Isolate the Trig Function: Get sin(x), cos(x), or tan(x) alone on one side.
  2. Find the Reference Angle: Use inverse trig or special angles.
  3. Identify the Quadrants: Use ASTC to find where the function has that sign.
  4. Apply the "Two-Stop" Rule: Find ALL solutions in \([0, 2\pi)\).
  5. Write the General Solution: Add \(+ 2\pi n\) (or \(+ \pi n\) for tangent).
Section 9 - The "Two-Stop" Rule
🚏 Most Trig Values Have TWO Solutions Per Period!

The wave goes up AND down, so it usually hits the same height TWICE in one lap around the circle.

Function Positive Value (+) Negative Value (−)
sin(x) = k Q I and Q II Q III and Q IV
cos(x) = k Q I and Q IV Q II and Q III
tan(x) = k Q I and Q III Q II and Q IV
⚠️ Exception: When k = 0, ±1

When the value is 0, 1, or -1, there's only ONE solution per period (on an axis).

Worked Example: Solve \(\sin(x) = \frac{1}{2}\) for \(x \in [0, 2\pi)\)

Step 1: Already isolated.

Step 2: Reference angle: \(\sin^{-1}(\frac{1}{2}) = \frac{\pi}{6}\)

Step 3: Sine is positive in Q I and Q II.

Step 4: Q I: \(x = \frac{\pi}{6}\)    Q II: \(x = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\)

Solutions: \(x = \frac{\pi}{6}, \frac{5\pi}{6}\)

Day 11: Complex Equations

Section 10 - Equations with Bx Inside
Worked Example: Solve \(2\cos(2x) - \sqrt{3} = 0\) for \(x \in [0, 2\pi)\)

Step 1: Isolate. \(\cos(2x) = \frac{\sqrt{3}}{2}\)

Step 2: Reference angle. \(\cos^{-1}(\frac{\sqrt{3}}{2}) = \frac{\pi}{6}\)

Step 3: Quadrants. Cosine is positive in Q I and Q IV.

Step 4: Solve for 2x first!

Q I: \(2x = \frac{\pi}{6}\)    Q IV: \(2x = 2\pi - \frac{\pi}{6} = \frac{11\pi}{6}\)

Step 5: But wait! The period of cos(2x) is π, so we need MORE solutions!

Add \(2\pi\) to each: \(2x = \frac{\pi}{6} + 2\pi = \frac{13\pi}{6}\) and \(2x = \frac{11\pi}{6} + 2\pi = \frac{23\pi}{6}\)

Step 6: Divide by 2 to get x:

\(x = \frac{\pi}{12}, \frac{11\pi}{12}, \frac{13\pi}{12}, \frac{23\pi}{12}\)

✓ Check: All four solutions are in \([0, 2\pi)\) ✓

⚠️ The "B Multiplier" Rule

If the equation has \(\sin(Bx)\) or \(\cos(Bx)\), expect B times as many solutions in \([0, 2\pi)\)!

B = 2 → Usually 4 solutions  |  B = 3 → Usually 6 solutions

Section 11 - General Solutions
🔄 Writing the General Solution

To find ALL solutions (not just in one interval), add the period:

For sine and cosine: \(x = \text{solution} + 2\pi n\), where \(n\) is any integer

For tangent: \(x = \text{solution} + \pi n\), where \(n\) is any integer

Day 11: Summary & Exit Ticket

🚀 Calculus Connection: Simple Harmonic Motion

Why Trig Graphs Matter

These waves model EVERYTHING that oscillates:

• Springs and pendulums

• Sound waves and music

• Tides and seasons

• AC electrical current

The Calculus Preview

\(\frac{d}{dx}[\sin x] = \cos x\)

\(\frac{d}{dx}[\cos x] = -\sin x\)

The derivative of a wave is another wave! This is why trig functions are so important in physics.

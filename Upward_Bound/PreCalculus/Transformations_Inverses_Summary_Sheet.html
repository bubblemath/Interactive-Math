📚 KEY DEFINITIONS
Inverse Function \(f^{-1}(x)\) A function that "undoes" what \(f(x)\) does. If \(f(a) = b\), then \(f^{-1}(b) = a\).
If \(f(x) = 2x + 3\), then \(f^{-1}(x) = \frac{x-3}{2}\)
\(f\) adds 3 then doubles → \(f^{-1}\) halves then subtracts 3
One-to-One Function A function where each output comes from exactly ONE input. Only one-to-one functions have inverses!
❌ \(f(x) = x^2\) is NOT one-to-one
Both \(f(2) = 4\) and \(f(-2) = 4\)

THE INVERSE RELATIONSHIP

\(f(x)\)
Forward
⇄
\(f^{-1}(x)\)
Reverse

Domain of \(f\) = Range of \(f^{-1}\)  |  Range of \(f\) = Domain of \(f^{-1}\)

📐 GRAPHICAL RELATIONSHIP: Reflection over \(y = x\)

Point on \(f(x)\)
\((a, b)\)
← reflects over →
\(y = x\)
Point on \(f^{-1}(x)\)
\((b, a)\)

Example: If \((2, 5)\) is on \(f\), then \((5, 2)\) is on \(f^{-1}\)

🎯 THE HORIZONTAL LINE TEST (HLT)

If ANY horizontal line crosses the graph more than once, the function is NOT one-to-one and has NO inverse (unless we restrict the domain).

📈

\(f(x) = 2x + 1\) (Linear)

Every horizontal line hits once

✓ ONE-TO-ONE — Has Inverse
⋃

\(f(x) = x^2\) (Parabola)

Horizontal lines hit twice

✗ NOT One-to-One
💡 Domain Restriction Fix: For \(f(x) = x^2\), restrict to \(x \geq 0\) → Now it passes HLT!
Inverse: \(f^{-1}(x) = \sqrt{x}\) with domain \(x \geq 0\)
⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"Inverse = Reciprocal"
\(f^{-1}(x) = \frac{1}{f(x)}\) 		The \(-1\) is NOT an exponent! \(f^{-1}\) means "inverse function," not "one over." \(f^{-1}(x)\) undoes \(f(x)\)
\(\frac{1}{f(x)}\) is the reciprocal
"Swap & Stop"
Forgetting to solve for \(y\) 		After swapping \(x\) and \(y\), you MUST isolate the new \(y\) to get the inverse formula. Complete ALL steps: swap, then solve for \(y\), then rename as \(f^{-1}(x)\)
"Domain Blindness"
Ignoring domain restrictions 		The inverse's domain equals the original's range. Restrictions carry over! Always state domain of \(f^{-1}\) based on range of \(f\)

The Algebraic "Switcher" Method

🔧 PROCEDURE: Finding an Inverse Algebraically

THE "SWITCHER" METHOD

1
\(f(x) = ...\)
Start with function
2
\(y = ...\)
Replace \(f(x)\) with \(y\)
3
\(x \leftrightarrow y\)
SWAP \(x\) and \(y\)
4
Solve for \(y\)
Isolate the new \(y\)
5
\(f^{-1}(x) = ...\)
Rename as inverse
  1. Check HLT first! Verify the function is one-to-one (or note domain restrictions needed).
  2. Replace \(f(x)\) with \(y\) to make the algebra easier.
  3. Swap every \(x\) with \(y\) and every \(y\) with \(x\).
  4. Solve for the new \(y\) using inverse operations.
  5. Rename \(y\) as \(f^{-1}(x)\) and state the domain.
  6. Verify by checking that \(f(f^{-1}(x)) = x\) and \(f^{-1}(f(x)) = x\).
✏️ WORKED EXAMPLE: Find the inverse of \(f(x) = \sqrt{x - 2} + 1\) for \(x \geq 2\)
Step 1: Check HLT
Square root function (with restricted domain) passes HLT ✓
Domain: \(x \geq 2\) Range: \(y \geq 1\)
Step 2: Replace \(f(x)\) with \(y\)
\(y = \sqrt{x - 2} + 1\)
Step 3: Swap \(x\) and \(y\)
\(x = \sqrt{y - 2} + 1\)
Step 4: Solve for \(y\)
\(x - 1 = \sqrt{y - 2}\)   (subtract 1)
\((x - 1)^2 = y - 2\)   (square both sides)
\(y = (x - 1)^2 + 2\)   (add 2)
Step 5: Rename & State Domain
\(f^{-1}(x) = (x - 1)^2 + 2\)
Domain: \(x \geq 1\) (from original range)

✓ VERIFICATION: Composition Check

\(f(f^{-1}(x)) = f((x-1)^2 + 2) = \sqrt{((x-1)^2 + 2) - 2} + 1 = \sqrt{(x-1)^2} + 1 = (x-1) + 1 = x\) ✓
\(f^{-1}(f(x)) = f^{-1}(\sqrt{x-2} + 1) = ((\sqrt{x-2}+1) - 1)^2 + 2 = (\sqrt{x-2})^2 + 2 = x\) ✓
✓ VERIFIED — Both compositions equal \(x\)
✅ QUICK REFERENCE CHECKLIST
  • Did I verify the function passes HLT (or note restrictions)?
  • Did I swap EVERY \(x\) and \(y\)?
  • Did I completely solve for the new \(y\)?
  • Did I rename \(y\) as \(f^{-1}(x)\)?
  • Did I state the domain of \(f^{-1}\) (= range of \(f\))?
  • Did I verify with \(f(f^{-1}(x)) = x\)?
  • For graphs: Are \(f\) and \(f^{-1}\) symmetric over \(y = x\)?
  • Did I avoid confusing \(f^{-1}\) with \(\frac{1}{f}\)?
🚀 CALCULUS PREVIEW: Derivatives of Inverse Functions
📐

Inverse Function Theorem

In Calculus, if \(f\) and \(f^{-1}\) are both differentiable:

\(\frac{d}{dx}[f^{-1}(x)] = \frac{1}{f'(f^{-1}(x))}\)
🔗

Why This Matters

This theorem lets you find derivatives of inverse trig functions like \(\sin^{-1}(x)\) and \(\ln(x)\) (the inverse of \(e^x\))!

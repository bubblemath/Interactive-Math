📖 Lesson Overview - Read Before Teaching

Day 5 marks the transition from algebraic manipulation (Week 1) to functional thinking (Week 2). The central conceptual shift is moving students away from seeing \(f(x)\) as "f times x" and toward understanding it as a process—an input-output machine that transforms values according to a rule.

The lesson's signature feature is the "Function Machine" metaphor, which becomes essential for understanding the Chain Rule in Calculus. When students see \(f(g(x))\), they need to visualize the input going through the "inner machine" \(g\) first, then the output feeding into the "outer machine" \(f\). This "inside-out" thinking is the conceptual foundation for differentiation of composite functions.

The interactive lesson at bubblemath.github.io provides the Vertical Line Test visualizer, the Function Machine simulator, domain/range explorers, and the Composition Lab where students can see the step-by-step flow of values through nested functions.

⚙️ THE FUNCTION MACHINE ⚙️
Input \(x\) → \(g(x)\) → \(f(\text{result})\) → Output \(f(g(x))\)

The inner function acts FIRST, then the outer function acts on that result.

🎯 Learning Objectives
  • Distinguish between relations and functions using the Vertical Line Test
  • Correctly interpret function notation: \(f(x)\) means "f of x," not "f times x"
  • Evaluate functions at numerical values, expressions, and other functions
  • Identify domain and range from graphs and algebraic expressions
  • Compose two functions and understand that \(f(g(x)) \neq g(f(x))\) in general
  • Connect composition to the "inside-out" structure needed for the Chain Rule

Section 1 - Warm-Up: Vertical Line Test Gallery 10 min

Purpose:

This visual warm-up establishes the fundamental definition of a function: each input has exactly ONE output. Students apply the Vertical Line Test to 6 graphs and classify each as "Function" or "Not a Function."

✅ Function (Passes VLT)

Any vertical line crosses the graph at most ONCE.

Examples: \(y = x^2\), \(y = \sin(x)\), \(y = 2x + 1\)

❌ Not a Function (Fails VLT)

At least one vertical line crosses the graph MORE than once.

Examples: \(x = y^2\), circles, sideways parabolas

📢 Script - Say This
"Before we dive into function notation, let's make sure we know what a function IS. Open Tab 1 on the interactive hub. You'll see 6 graphs. For each one, imagine drawing vertical lines across it. If ANY vertical line hits the graph more than once, it's NOT a function. You have 5 minutes to classify all 6."
Section 2 - Function Notation: The "Empty Box" Method 15 min

The Big Misconception:

Students see \(f(x)\) and think "f times x." This leads to errors like \(f(x+h) = fx + fh\). We need to rewire their thinking.

The "Empty Box" Method

Write the function with a box instead of \(x\):

\(f(\boxed{\phantom{xx}}) = 3\boxed{\phantom{xx}}^2 - 2\boxed{\phantom{xx}} + 1\)

Whatever goes IN the parentheses goes IN every box!

Example: Find \(f(x+h)\)

\(f(x+h) = 3(x+h)^2 - 2(x+h) + 1\)

The entire expression \((x+h)\) replaces every \(x\)!

📢 Script - The Key Insight
"Here's the secret: \(f(x)\) doesn't mean 'f times x.' It means 'f OF x'—what happens to x when it goes through the function machine. Think of \(f\) as a VERB, not a number. When I write \(f(2)\), I'm asking: 'What does the function DO to 2?' When I write \(f(x+h)\), I'm asking: 'What does the function DO to the expression \(x+h\)?'"
💡 Teaching Tip: Have students physically write the "empty box" version of every function before evaluating. This kinesthetic step prevents the "distribute f" error.
Section 3 - Domain & Range: Forbidden Inputs 15 min

The Three "Forbidden Input" Scenarios:

Function Type Forbidden Input Example
Rational (fractions) Denominator = 0 \(f(x) = \frac{1}{x-3}\) → \(x \neq 3\)
Even Root (square root, etc.) Radicand < 0 \(g(x) = \sqrt{x+2}\) → \(x \geq -2\)
Logarithmic Argument ≤ 0 \(h(x) = \ln(x)\) → \(x > 0\)
📢 Script - Domain as "Allowed Inputs"
"Think of domain as the 'guest list' for your function party. Some inputs are NOT allowed in. For fractions, we can't divide by zero—so any x that makes the denominator zero is banned. For square roots, we can't take the square root of a negative—so any x that makes the inside negative is banned. Always ask: 'What would break this function?'"
Section 4 - Composition: The "Inside-Out" Method 20 min

The Core Concept:

\(f(g(x))\) means: "First apply \(g\) to \(x\), then apply \(f\) to the result."

The inner function acts FIRST. The outer function acts SECOND.

⚙️ COMPOSITION FLOW ⚙️
\(x = 3\) → \(g(x) = x + 1\)
\(g(3) = 4\) → \(f(x) = x^2\)
\(f(4) = 16\) → \(f(g(3)) = 16\)
Finding \(f(g(x))\) Algebraically

Given: \(f(x) = x^2\), \(g(x) = x + 1\)

Step 1: Write \(f(\boxed{\phantom{xx}}) = \boxed{\phantom{xx}}^2\)

Step 2: Put \(g(x)\) in the box:

\(f(g(x)) = (x+1)^2 = x^2 + 2x + 1\)

Finding \(g(f(x))\) Algebraically

Given: \(f(x) = x^2\), \(g(x) = x + 1\)

Step 1: Write \(g(\boxed{\phantom{xx}}) = \boxed{\phantom{xx}} + 1\)

Step 2: Put \(f(x)\) in the box:

\(g(f(x)) = x^2 + 1\)

🚨 Critical Point: \(f(g(x)) \neq g(f(x))\) in general! In the example above, \(x^2 + 2x + 1 \neq x^2 + 1\). Order matters!
📢 Script - Calculus Preview
"In Calculus, you'll learn the Chain Rule for derivatives. It says: to differentiate a composite function, you differentiate the outer function, then multiply by the derivative of the inner function. But you can't do that if you don't know which function is 'outer' and which is 'inner.' That's why we're drilling this today—you need to SEE the layers."
⚠️ Common Pitfalls and Teaching Strategies
1. The Multiplication Trap

Students see \(f(x+h)\) and write \(fx + fh\), treating \(f\) as a variable to distribute.

Strategy: The "Empty Box" method. Write \(f(\boxed{\phantom{xx}})\) and have students physically replace the box with the entire input expression. "Whatever is in the parentheses goes in EVERY box."
2. Composition Order Confusion

Students think \(f(g(x))\) means "do \(f\) first, then \(g\)."

Strategy: Use the "Nested Boxes" visual. Draw \(g\) as a small box inside a larger \(f\) box. "The input goes to the innermost box first, then works its way out."
3. Assuming \(f(g(x)) = g(f(x))\)

Students assume composition is commutative like multiplication.

Strategy: Immediate counterexample with numbers. "If \(f(x) = x^2\) and \(g(x) = x+1\), find \(f(g(2))\) and \(g(f(2))\). Are they equal?" (No: 9 vs. 5)
4. Domain of Composition Oversight

Students forget that the input must survive the inner function before reaching the outer function.

Strategy: "Two checkpoints" rule. For \(f(g(x))\): (1) Can \(x\) enter \(g\)? (2) Can \(g(x)\) enter \(f\)?
5. Negative Input Errors

When evaluating \(f(-2)\) for \(f(x) = x^2 - 3x\), students write \(-2^2 - 3(-2) = -4 + 6 = 2\) instead of \((-2)^2 - 3(-2) = 4 + 6 = 10\).

Strategy: Require parentheses around ALL substituted values. "Parentheses are your seatbelt—always wear them when substituting!"
👥 Partner and Small Group Activities
1. Human Function Machine Groups of 3

Student A is the "Input," Student B is "Function \(g\)," and Student C is "Function \(f\)." Student A writes a number on a card and hands it to B. B performs their operation (e.g., "add 2") and hands the result to C. C performs their operation (e.g., "square it") and announces the final output. This physically demonstrates the sequence of composition.

Materials: Index cards, markers
Reinforces: The "inside-out" order of composition
2. "What's My Rule?" Pairs

One student secretly chooses a linear function (e.g., \(f(x) = 2x + 1\)). The partner provides inputs, and the first student provides outputs. The partner tries to guess the function rule in as few guesses as possible. Switch roles after each round.

Materials: Whiteboards or paper
Reinforces: Understanding functions as input-output relationships
3. Domain "Forbidden Zone" Sort Pairs

Provide cards with various functions (radicals, rationals, polynomials, logarithms). Partners must sort them into categories based on their "Forbidden Inputs": "No restrictions," "Cannot be zero," "Cannot be negative," "Cannot be ≤ 0," etc.

Materials: Function cards (20-30), category labels
Reinforces: Quick domain identification by function type
4. Composition Relay Teams of 4

Each team is given \(f(x)\) and \(g(x)\). Student 1 finds \(g(2)\), Student 2 takes that answer and finds \(f(\text{Ans})\), Student 3 finds \(f(g(x))\) algebraically, and Student 4 verifies by substituting \(x = 2\) into the composition. First team with a verified match wins!

Materials: Function cards, team answer sheets, timer
Reinforces: Both numerical and algebraic composition
5. "Order Matters" Showdown Pairs

Give pairs two functions \(f\) and \(g\). Each partner independently computes one composition: Partner A finds \(f(g(x))\), Partner B finds \(g(f(x))\). They compare results. If the results are different (they usually are!), they must explain WHY order matters for these specific functions.

Materials: Function pair cards, comparison worksheet
Reinforces: Non-commutativity of composition
6. Vertical Line Test Gallery Walk Small Groups

Post 8-10 graphs around the room. Groups rotate with sticky notes, marking each graph as "Function" or "Not a Function" with a brief justification. After the walk, discuss any disagreements as a class.

Materials: Posted graph sheets, sticky notes, timer
Reinforces: Visual function identification
7. "Build the Composition" Challenge Groups of 3 - Advanced

Give groups a target composite function (e.g., \(h(x) = \sqrt{x^2 + 1}\)). They must decompose it into \(f(x)\) and \(g(x)\) such that \(h(x) = f(g(x))\). Multiple correct answers are possible! Groups present their decompositions and discuss which is most "natural."

Materials: Target function cards, presentation space
Reinforces: Recognizing composition structure (Chain Rule prep!)
🔗 Calculus Connections
Pre-Calc Skill Calculus Application
Function Composition The Chain Rule: \(\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)\). Students must identify "outer" and "inner" functions.
Evaluating \(f(x+h)\) Difference Quotient: \(\frac{f(x+h) - f(x)}{h}\) is the foundation of the derivative definition.
Domain Restrictions Continuity & Differentiability: Functions can only be differentiated where they're continuous.
Function Notation Leibniz vs. Lagrange Notation: Understanding \(f'(x)\) vs. \(\frac{dy}{dx}\) requires solid notation fluency.
Decomposing Compositions u-Substitution: Integration technique that reverses the Chain Rule by identifying the "inner function."
📢 Calculus Preview - Say This
"Here's a sneak peek at why this matters. In Calculus, you'll learn the Chain Rule: to find the derivative of \(f(g(x))\), you take the derivative of the outer function, leave the inner function alone, then multiply by the derivative of the inner function. But you can't do ANY of that if you can't identify which function is outer and which is inner. Today's composition work is literally training your brain to see those layers."
✅ Formative Assessment Checkpoints
1 After Warm-Up: Can students correctly apply the Vertical Line Test to all 6 graphs?
2 During Notation Section: Given \(f(x) = x^2\), can students correctly find \(f(x+h)\)? (Answer: \(x^2 + 2xh + h^2\), NOT \(x^2 + h\))
3 During Domain Section: Can students identify the domain of \(g(x) = \frac{1}{\sqrt{x-4}}\)? (Answer: \(x > 4\))
4 During Composition Lab: Given \(f(x) = 2x\) and \(g(x) = x + 5\), can students find both \(f(g(3))\) and \(g(f(3))\) and recognize they're different? (Answers: 16 and 11)
📝 Exit Ticket Answers (Tab 6)

ET1: Does the graph pass the Vertical Line Test? [Graph of \(x = y^2\)]

Answer: No. A vertical line at \(x = 4\) crosses at \(y = 2\) and \(y = -2\).

ET2: If \(f(x) = 3x^2 - 1\), find \(f(-2)\).

Answer: \(f(-2) = 3(-2)^2 - 1 = 3(4) - 1 = 12 - 1 = 11\)

ET3: Find the domain of \(h(x) = \sqrt{5 - x}\).

Answer: \(5 - x \geq 0\) → \(x \leq 5\) → Domain: \((-\infty, 5]\)

ET4: Given \(f(x) = x + 1\) and \(g(x) = x^2\), find \(f(g(x))\).

Answer: \(f(g(x)) = f(x^2) = x^2 + 1\)

ET5: Using the same functions, find \(g(f(x))\) and explain why it's different from \(f(g(x))\).

Answer: \(g(f(x)) = g(x+1) = (x+1)^2 = x^2 + 2x + 1\). It's different because composition is not commutative—the order in which you apply the functions matters.

🎚️ Differentiation Strategies

For Struggling Students:

  • Provide a "Function Machine" graphic organizer
  • Start with numerical composition before algebraic
  • Use color-coding: inner function in blue, outer in red
  • Allow calculator use for arithmetic during composition

For Advanced Students:

  • "Build the Composition" decomposition challenges
  • Domain of composite functions (two-checkpoint analysis)
  • Preview: "What would \(\frac{d}{dx}[f(g(x))]\) look like?"
  • Explore when \(f(g(x)) = g(f(x))\) (commuting functions)

📚 KEY DEFINITIONS: Anatomy of a Sinusoid

🎯 THE MASTER FORMULA

\(y = {\color{red}A} \sin\left({\color{blue}B}(x - {\color{teal}C})\right) + {\color{violet}D}\)

Same structure for cosine: \(y = A\cos(B(x - C)) + D\)

Amplitude
\(|A|\)
Height from midline
B
Period Factor
Period = \(\frac{2\pi}{|B|}\)
Horizontal stretch/compress
C
Phase Shift
Shift = \(C\)
Horizontal shift (right if +)
D
Midline
\(y = D\)
Vertical shift

📉 The Parent Wave Gallery:

Sine: \(y = \sin(x)\)

Starts at origin (0, 0)
Period: \(2\pi\)

Cosine: \(y = \cos(x)\)

Starts at max (0, 1)
Period: \(2\pi\)

Tangent: \(y = \tan(x)\)

Asymptotes at \(\pm\frac{\pi}{2}\)
Period: \(\pi\)
🔧 PROCEDURE: The 5-Point Graphing Method
  1. Find the Period: Calculate \(\text{Period} = \frac{2\pi}{|B|}\)
  2. Find the Phase Shift: The graph starts at \(x = C\) (not at 0)
  3. Mark 5 Key Points: Divide one period into 4 equal parts (start, ¼, ½, ¾, end)
  4. Apply Amplitude & Midline: Max = \(D + |A|\), Min = \(D - |A|\)
  5. Connect with Smooth Curve: Follow the parent shape (sine starts mid, cosine starts max)

⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"The Period Trap"
Period = B 		\(B\) is the frequency factor, not the period! The period is inversely related to \(B\). Period = \(\frac{2\pi}{|B|}\)
(For tangent: \(\frac{\pi}{|B|}\))
"Phase Shift Sign"
\(\sin(x + 2)\) shifts right 		The formula is \((x - C)\), so \((x + 2) = (x - (-2))\) means \(C = -2\), which shifts LEFT. \((x - C)\): positive \(C\) → right
\((x + C)\): shifts LEFT by \(C\)
"Tangent Blindness"
Treating tan like sin/cos 		Tangent has period \(\pi\) (not \(2\pi\)), vertical asymptotes, and no amplitude! Tan: Period = \(\frac{\pi}{|B|}\)
Asymptotes where \(\cos = 0\)

🔧 PROCEDURE: Solving Trig Equations
  1. Isolate the Trig Function: Get \(\sin(\text{stuff})\), \(\cos(\text{stuff})\), or \(\tan(\text{stuff})\) alone on one side
  2. Find the Reference Angle: Use inverse trig to find the acute angle where the function equals that value
  3. Identify All Quadrants: Use ASTC to determine which quadrants give the correct sign
  4. Write All Solutions: Find angles in each valid quadrant within the given interval
  5. Apply General Solution (if needed): Add \(+ 2\pi n\) for sin/cos or \(+ \pi n\) for tan

🔄 THE GENERAL SOLUTION

\(\theta = \theta_{\text{ref}} + 2\pi n\)   or   \(\theta = (\pi - \theta_{\text{ref}}) + 2\pi n\)
where \(n\) is any integer (for all solutions) | For tangent: use \(+ \pi n\)
✏️ WORKED EXAMPLE: Solve \(2\cos(2x) - \sqrt{3} = 0\) on \([0, 2\pi)\)
Step 1: Isolate the Trig Function
\(2\cos(2x) - \sqrt{3} = 0\)
\(2\cos(2x) = \sqrt{3}\)
\(\cos(2x) = \frac{\sqrt{3}}{2}\)
Step 2: Find Reference Angle
\(\cos^{-1}\left(\frac{\sqrt{3}}{2}\right) = \frac{\pi}{6}\)
Reference angle: \(\theta' = \frac{\pi}{6}\)
Step 3: Identify Quadrants
Cosine is positive in Q1 and Q4
So \(2x = \frac{\pi}{6}\) or \(2x = 2\pi - \frac{\pi}{6} = \frac{11\pi}{6}\)
Step 4: Solve for x (divide by 2)
From \(2x = \frac{\pi}{6}\): \(x = \frac{\pi}{12}\)
From \(2x = \frac{11\pi}{6}\): \(x = \frac{11\pi}{12}\)
Step 5: Find ALL Solutions in \([0, 2\pi)\)
Since the period of \(\cos(2x)\) is \(\pi\), add \(\pi\) to each:
\(x = \frac{\pi}{12} + \pi = \frac{13\pi}{12}\)
\(x = \frac{11\pi}{12} + \pi = \frac{23\pi}{12}\)
Solutions:
\(x = \frac{\pi}{12}, \frac{11\pi}{12}, \frac{13\pi}{12}, \frac{23\pi}{12}\)

Position, Velocity, Acceleration

If position is \(s(t) = A\sin(\omega t)\), then:

Velocity: \(v(t) = A\omega\cos(\omega t)\)
Acceleration: \(a(t) = -A\omega^2\sin(\omega t)\)

The Derivative Cycle

Trig derivatives cycle through each other:

\(\sin \xrightarrow{d/dx} \cos \xrightarrow{d/dx} -\sin \xrightarrow{d/dx} -\cos \xrightarrow{d/dx} \sin\)

🔧 PROCEDURE: Building a Model from Scratch
  1. Read & Identify: What quantity do you want to optimize or express? This is your dependent variable.
  2. Define Variables: Assign letters to all unknown quantities. Write "Let \(x\) = ..." statements.
  3. Find Relationships: Use geometry, rates, or given constraints to connect variables.
  4. Write the Function: Express the dependent variable in terms of ONE independent variable.
  5. State the Domain: Determine realistic restrictions based on the context.
⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"Unit Mismatch"
Mixing feet and inches 		Adding 5 ft + 3 in directly gives nonsense. Units must match before combining. Convert first: 5 ft + 0.25 ft = 5.25 ft
or 60 in + 3 in = 63 in
"Domain Neglect"
Allowing \(x = -3\) for length 		Physical quantities like length, time, and population can't be negative! State domain: \(x > 0\) for lengths
\(t \geq 0\) for time
"Variable Confusion"
Using \(x\) for two things 		If \(x\) means both "width" and "number of items," your model is ambiguous. Use distinct variables:
\(w\) = width, \(n\) = number of items

📚 KEY DEFINITIONS: What is a Limit?

🎯 THE LIMIT CONCEPT

\(\lim_{x \to c} f(x) = L\)

"As \(x\) approaches \(c\), \(f(x)\) approaches \(L\)"

← ← ←
From the LEFT
\(x \to c^-\)

From the RIGHT

⚠️ The limit exists ONLY if both sides agree!

Key Insight The limit describes what \(f(x)\) approaches, not necessarily what \(f(c)\) equals. The function doesn't even need to be defined at \(x = c\) for the limit to exist!
🔧 PROCEDURE: Evaluating Limits Numerically
  1. Create a Table: Choose values approaching \(c\) from both sides (e.g., 1.9, 1.99, 1.999 and 2.1, 2.01, 2.001)
  2. Calculate \(f(x)\): Evaluate the function at each value in your table
  3. Observe the Pattern: Do the outputs approach the same number from both sides?
  4. State the Limit: If both sides agree, that's your limit. If not, the limit DNE (Does Not Exist)

✏️ WORKED EXAMPLE: Modeling + Limit Analysis

Problem: A farmer has 100 ft of fencing to enclose a rectangular pen against a barn (3 sides). Find the area function and analyze what happens as width approaches 0.

Step 1: Define Variables
Let \(w\) = width (perpendicular to barn)
Let \(\ell\) = length (parallel to barn)
Step 2: Find Constraint
Fencing: \(2w + \ell = 100\)
So: \(\ell = 100 - 2w\)
Step 3: Write Area Function
\(A(w) = w \cdot \ell = w(100 - 2w)\)
\(A(w) = 100w - 2w^2\)
Step 4: State Domain
Width must be positive: \(w > 0\)
Length must be positive: \(100 - 2w > 0 \Rightarrow w < 50\)
Domain: \(0 < w < 50\)
Step 5: Limit Analysis
\(\lim_{w \to 0^+} A(w) = \lim_{w \to 0^+} (100w - 2w^2) = 0\)
As width → 0, area → 0 ✓
Model: \(A(w) = 100w - 2w^2\)
Domain: \((0, 50)\)
Limit: \(\lim_{w \to 0^+} A(w) = 0\)
✅ QUICK REFERENCE CHECKLIST

📐 Modeling:

  • Did I clearly define all variables?
  • Did I use the constraint to eliminate a variable?
  • Is my function in terms of ONE variable?
  • Did I state the domain with context?
  • Do my units make sense?

🎯 Limits:

  • Did I check from BOTH sides?
  • Do left and right limits agree?
  • Did I distinguish limit from function value?
  • Did I check for asymptotes or holes?
  • Does my answer make physical sense?
🚀 CALCULUS PREVIEW: The Difference Quotient

📈 THE DERIVATIVE IS A LIMIT!

\(f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)
This limit gives the instantaneous rate of change — the slope of the tangent line!

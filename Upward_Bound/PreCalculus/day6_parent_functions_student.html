★ The Anchor Point Principle
Every parent function has 3–4 anchor points — the simplest coordinates that define its shape. Once you know these, you can transform any function just by moving the anchor points.

The four transformations never change the basic shape — they only move, flip, and resize it.

Inside-Outside Framework  ·  a, b, h, k Parameters  ·  Transformation Rules
Name:                         
The General Transformation Form
y = a · f( b(x − h) ) + k
a, k → OUTSIDE changes (affect y-values)
Apply directly to each output — multiply then add.
b, h → INSIDE changes (affect x-values)
⚠ Counter-intuitive — do the OPPOSITE of what you expect!
Section 2 — The Inside-Outside Framework
📤 OUTSIDE  — affects y (intuitive)
📥 INSIDE  — affects x (counter-intuitive)
a — Vertical
|a| > 1 → (stretch)
0 < |a| < 1 → (compress)
a < 0 → reflect over
b — Horizontal
|b| > 1 → (compress — OPPOSITE!)
0 < |b| < 1 → (stretch — OPPOSITE!)
b < 0 → reflect over
k — Vertical Shift
+k → shift
−k → shift
h — Horizontal Shift
(x − h) → shift   ← counter-intuitive!
(x + h) → shift   ← counter-intuitive!
⚠ The Inside Rule: "Inside does the opposite!"   (x − 3) shifts right 3, not left.  (x + 2) shifts left 2, not right. This trips up every student — write a reminder here:

📚 KEY DEFINITIONS
Exponential Function \(f(x) = b^x\) where \(b > 0\) and \(b \neq 1\). The variable is in the exponent.
\(f(x) = 2^x\), \(g(x) = e^x\), \(h(x) = \left(\frac{1}{3}\right)^x\)
Logarithmic Function \(f(x) = \log_b(x)\) — the inverse of \(b^x\). Answers: "What power of \(b\) gives \(x\)?"
\(\log_2(8) = 3\) because \(2^3 = 8\)

📋 Essential Log Properties:

Product Rule
\(\log_b(xy) = \log_b(x) + \log_b(y)\)
Quotient Rule
\(\log_b\left(\frac{x}{y}\right) = \log_b(x) - \log_b(y)\)
Power Rule
\(\log_b(x^n) = n \cdot \log_b(x)\)
Change of Base
\(\log_b(x) = \frac{\ln(x)}{\ln(b)}\)
⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"Log Addition Trap"
\(\log(A+B) = \log A + \log B\) 		The Product Rule applies to \(\log(A \cdot B)\), NOT \(\log(A + B)\). There's no rule for log of a sum! \(\log(A \cdot B) = \log A + \log B\)
\(\log(A + B) \neq\) anything simpler
"Negative Argument"
\(\log(-5) = ?\) 		You cannot take the log of a negative number or zero. The domain of \(\log_b(x)\) is \(x > 0\). Domain: \(x > 0\) only
Check for extraneous solutions!
"Power Misplacement"
\(\log(x^2) = (\log x)^2\) 		The Power Rule brings the exponent DOWN as a coefficient, not outside the log. \(\log(x^2) = 2\log(x)\)
\((\log x)^2\) is something different!

🔧 PROCEDURE: Solving Exponential Equations

🟠 Method 1: Same Base

Use when both sides CAN be written with the same base
  1. Rewrite both sides with the same base
  2. Set the exponents equal
  3. Solve the resulting equation
If \(b^m = b^n\), then \(m = n\)

🔵 Method 2: Take the Log

Use when bases CANNOT be made the same
  1. Isolate the exponential expression
  2. Take \(\ln\) (or \(\log\)) of both sides
  3. Use Power Rule to bring down exponent
  4. Solve for the variable
🔧 PROCEDURE: Solving Logarithmic Equations
  1. Isolate the logarithm (or combine logs using properties).
  2. Convert to exponential form using the Heart Method: \(\log_b(x) = y \Rightarrow b^y = x\)
  3. Solve the resulting equation for the variable.
  4. CHECK for extraneous solutions — plug back into original to verify argument is positive!
🔥
EXTRANEOUS SOLUTION ALERT! Any solution that makes the argument of a log ≤ 0 must be REJECTED. Always check your answers!
✏️ WORKED EXAMPLE: Solve \(5 + 2e^{3x} = 11\)
Step 1: Isolate the Exponential
\(5 + 2e^{3x} = 11\)
\(2e^{3x} = 6\)   (subtract 5)
\(e^{3x} = 3\)   (divide by 2)
Step 2: Take Natural Log of Both Sides
\(\ln(e^{3x}) = \ln(3)\)
Step 3: Apply Power Rule
\(3x \cdot \ln(e) = \ln(3)\)
\(3x \cdot 1 = \ln(3)\)   (since \(\ln(e) = 1\))
\(3x = \ln(3)\)
Step 4: Solve for x
\(x = \frac{\ln(3)}{3}\)
Exact Answer:
\(x = \frac{\ln(3)}{3}\)
Approximate Answer:
\(x \approx \frac{1.0986}{3} \approx 0.366\)
Step 5: Verify (Optional)
\(5 + 2e^{3(0.366)} = 5 + 2e^{1.098}\)
\(\approx 5 + 2(3) = 11\) ✓
✅ QUICK REFERENCE CHECKLIST
  • Is the base positive and not equal to 1?
  • Did I isolate the exponential/log before converting?
  • Did I apply log properties correctly (Product, Quotient, Power)?
  • Did I use the correct method (Same Base vs. Take Log)?
  • Did I check for EXTRANEOUS solutions (log domain)?
  • Is my answer in exact form when required?
  • Did I distinguish between \(\log\) (base 10) and \(\ln\) (base \(e\))?
  • Did I verify my answer by substitution?
🚀 CALCULUS PREVIEW: The Magic of \(e\)
✨

The Only Function = Its Own Derivative

\(\frac{d}{dx}[e^x] = e^x\)

This is WHY \(e\) is the "natural" base — it makes calculus beautiful!

🔗

The Natural Log Derivative

\(\frac{d}{dx}[\ln(x)] = \frac{1}{x}\)

This connects logarithms to the integral \(\int \frac{1}{x}\,dx = \ln|x| + C\)

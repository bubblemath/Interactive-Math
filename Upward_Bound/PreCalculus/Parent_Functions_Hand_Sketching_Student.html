Section 2 - The Master Transformation Formula
🔧 THE TRANSFORMATION BLUEPRINT
\(g(x) = a \cdot f(x - h) + k\)
\(h\) = Horizontal \(a\) = Stretch/Reflect \(k\) = Vertical
Section 3 - The Inside-Outside Rule

🔷 INSIDE (affects \(x\))

Changes to \(x\) work OPPOSITE to what you expect!

  • \(f(x - 3)\) → moves 3
  • \(f(x + 2)\) → moves 2

🔶 OUTSIDE (affects \(y\))

Changes to \(y\) work EXACTLY as expected!

  • \(f(x) + 4\) → moves 4
  • \(f(x) - 5\) → moves 5
⚠️ The "Opposite Land" Rule

Everything INSIDE the function lives in "Opposite Land." Ask yourself: "What value of \(x\) makes the inside equal zero?" That's where the new "home base" is!

Day 6: The H-S-R-V Method

Section 4 - The H-S-R-V Sketching Order

To avoid "Order Chaos," ALWAYS apply transformations in this specific order:

H
Horizontal Shift

Left/Right

S
Stretch/Shrink

Multiply \(y\)

R
Reflection

Flip over axis

V
Vertical Shift

Up/Down

🎯 H-S-R-V PROCEDURE
  1. H - Horizontal Shift: Move the "Ghost Parent" left or right based on \(h\). Remember: opposite of the sign!
  2. S - Stretch/Shrink: Multiply ALL \(y\)-values by \(|a|\). If \(|a| > 1\), taller. If \(|a| < 1\), shorter.
  3. R - Reflection: If \(a < 0\), flip ALL points over the \(x\)-axis (negate \(y\)-values).
  4. V - Vertical Shift: Add \(k\) to ALL \(y\)-values. Positive = up, Negative = down.
Section 5 - The "Ghost Parent" Technique
👻 THE GHOST PARENT METHOD

Before drawing your final graph, LIGHTLY sketch the parent function in pencil. This "ghost" helps you:

  • Remember the basic shape
  • Track your anchor points through each transformation
  • Catch mistakes before they compound
Parameter What It Does Effect on Points Example
\(h\) in \((x - h)\) Horizontal shift Add \(h\) to all \(x\)-values \((x-3)\) → Right 3
\(a\) (when \(|a| > 1\)) Vertical stretch Multiply \(y\) by \(a\) \(2f(x)\) → twice as tall
\(a\) (when \(0 < |a| < 1\)) Vertical shrink Multiply \(y\) by \(a\) \(\frac{1}{2}f(x)\) → half as tall
\(a\) (when \(a < 0\)) Reflection over \(x\)-axis Negate all \(y\)-values \(-f(x)\) → flip upside down
\(k\) Vertical shift Add \(k\) to all \(y\)-values \(f(x)+4\) → Up 4

Section 6 - Common Pitfalls
❌ The Mistake Why It Happens ✅ The Correction
The Sign Trap Seeing \((x + 3)\) and moving RIGHT instead of left. Inside moves are ! \((x+3)\) means LEFT 3.
Order Chaos Doing vertical shift BEFORE stretch, getting wrong final position. ALWAYS follow H-S-R-V order!
Reflection Blur Forgetting to flip points BEFORE shifting up/down. Reflect over -axis BEFORE vertical shift.
Anchor Drift Losing track of key points during transformations. Use a transformation TABLE to track each point!

Day 6: Summary & Exit Ticket

🚀 Calculus Connection: Curve Sketching

In Calculus, you'll use derivatives to find exactly where graphs turn (maximums/minimums) and where they change concavity (inflection points). Understanding parent functions NOW allows you to:

  • Predict the general SHAPE before doing any calculus
  • Know how many turning points to expect (quadratic = 1, cubic = 2)
  • Recognize transformations in derivative rules (Chain Rule!)

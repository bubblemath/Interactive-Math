Section 1 - Key Definitions
Function

A rule that assigns each exactly output.

Domain

The set of all (allowed inputs).

Range

The set of all (possible outputs).

Section 2 - The Function Machine
⚙️ FUNCTION = INPUT-OUTPUT MACHINE ⚙️
Input \(x\) → Process: \(f\) → Output \(f(x)\)

The function \(f\) is a VERB—it DOES something to the input!

🔑 THE BIG IDEA

\(f(x)\) means " of \(x\)" — NOT "\(f\) \(x\)"!

\(f\) is the function NAME, \(x\) is the INPUT, \(f(x)\) is the OUTPUT VALUE.

Section 3 - Common Mistakes
❌ Mistake ✅ Correction
Multiplication Trap: \(f(x+h) = fx + fh\) \(f(x+h)\) means substitute \((x+h)\) for EVERY \(x\) in the function!
Negative Input Error: \(f(-2) = -2^2 = -4\) Use parentheses! \(f(-2) = (-2)^2 = 4\)
Composition Order: \(f(g(x))\) means "do \(f\) first" The INNER function \(g\) acts FIRST, then \(f\) acts on the result!
Section 4 - The Vertical Line Test (VLT)

Rule: If ANY vertical line crosses the graph more than once, it is NOT a function.

✅ FUNCTION (Passes VLT)

Every vertical line crosses at most ONCE.

Examples: \(y = x^2\), \(y = 2x + 1\), \(y = |x|\)

❌ NOT A FUNCTION (Fails VLT)

At least one vertical line crosses MORE than once.

Examples: \(x = y^2\), circles, sideways parabolas

Day 5: Evaluating Functions & Domain

Section 5 - Evaluating Functions: The "Empty Box" Method
🎯 3-STEP PROCEDURE
  1. Write the function with a box: \(f(\boxed{\phantom{xx}}) = 3\boxed{\phantom{xx}}^2 - 2\boxed{\phantom{xx}} + 1\)
  2. Put the INPUT in every box (use parentheses!)
  3. Simplify the result
Example: If \(f(x) = x^2 - 3x\), find \(f(x + h)\)

Step 1: \(f(\boxed{\phantom{xx}}) = \boxed{\phantom{xx}}^2 - 3\boxed{\phantom{xx}}\)

Step 2: \(f(x+h) = (x+h)^2 - 3(x+h)\)

Step 3: \(= x^2 + 2xh + h^2 - 3x - 3h\)

Day 5: Function Composition

Section 7 - Function Composition: The "Inside-Out" Method
🔗 COMPOSITION NOTATION

\(f(g(x))\) or \((f \circ g)(x)\) means: "First apply , then apply to the result."

⚙️ COMPOSITION FLOW ⚙️
Input \(x\) → Inner: \(g(x)\) → Outer: \(f(\text{result})\) → Output \(f(g(x))\)

The INNER function acts FIRST!

Section 8 - Composition Procedure & Worked Example
🎯 COMPOSITION PROCEDURE
  1. Identify the INNER function (the one closest to \(x\))
  2. Write the OUTER function with an empty box: \(f(\boxed{\phantom{xx}})\)
  3. Put the ENTIRE inner function in the box
  4. Simplify

Given: \(f(x) = x^2\) and \(g(x) = x + 1\)

Find \(f(g(x))\)

Outer: \(f\), Inner: \(g\)

\(f(\boxed{\phantom{xx}}) = \boxed{\phantom{xx}}^2\)

Put \(g(x) = x + 1\) in the box:

\(f(g(x)) = (x+1)^2\)

\(=\)

Find \(g(f(x))\)

Outer: \(g\), Inner: \(f\)

\(g(\boxed{\phantom{xx}}) = \boxed{\phantom{xx}} + 1\)

Put \(f(x) = x^2\) in the box:

\(g(f(x)) = x^2 + 1\)

\(=\)

⚠️ ORDER MATTERS!

\(f(g(x)) = x^2 + 2x + 1\) but \(g(f(x)) = x^2 + 1\)

These are NOT equal! Composition is NOT commutative.

Day 5: Numerical Composition Practice

Section 9 - Numerical Composition

Given: \(f(x) = x^2 - 1\) and \(g(x) = 3x + 2\)

Example: Find \(f(g(2))\)

Step 1: Find \(g(2)\) first (inner function):

\(g(2) = 3(2) + 2 = 8\)

Step 2: Now find \(f(8)\) (outer function):

\(f(8) = (8)^2 - 1 = 64 - 1 = 63\)

Answer: \(f(g(2)) = 63\)

🔗 Calculus Connection: The Chain Rule

In Calculus, you'll learn the Chain Rule for derivatives:

\(\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)\)

"Derivative of outer (leave inner alone) × derivative of inner"

You MUST be able to identify the outer and inner functions to use this rule. That's exactly what you practiced above!

Day 5: Summary & Key Formulas

Section 11 - Summary Checklist
  • I can apply the Vertical Line Test to determine if a graph is a function.
  • I understand that \(f(x)\) means "f OF x," not "f TIMES x."
  • I can evaluate functions using the "Empty Box" method with parentheses.
  • I can find the domain by identifying "forbidden inputs."
  • I can compose functions using the "Inside-Out" method.
  • I understand that \(f(g(x)) \neq g(f(x))\) in general—order matters!
  • I can decompose a composite function into its outer and inner parts.
📝 KEY FORMULAS TO REMEMBER

Function Notation: \(f(x)\) = "f of x"

Composition: \(f(g(x))\) = outer(inner(x))

Circle Notation: \((f \circ g)(x) = f(g(x))\)

Domain Restrictions:

• Fractions: denominator ≠ 0

• Even roots: radicand ≥ 0

• Logarithms: argument > 0

⚠️ COMMON MISTAKES TO AVOID
  • \(f(x)\) does NOT mean \(f \times x\)
  • \(f(g(x)) \neq g(f(x))\) — order matters!
  • Always use parentheses when substituting negative numbers
  • Check BOTH restrictions when you have a fraction with a radical

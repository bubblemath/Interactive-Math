📖 The Astronomer's Almanac

Study these celestial charts to navigate the sanctum.

🌙 Degree ↔ Radian

\(180° = \pi \text{ radians}\)

Deg → Rad: multiply by \(\frac{\pi}{180}\)

Rad → Deg: multiply by \(\frac{180}{\pi}\)

Key Angles:

\(30° = \frac{\pi}{6}\), \(45° = \frac{\pi}{4}\), \(60° = \frac{\pi}{3}\), \(90° = \frac{\pi}{2}\)

⭐ The Six Trig Functions

\(\sin\theta = y\), \(\cos\theta = x\), \(\tan\theta = \frac{y}{x}\)

Reciprocals:

\(\csc\theta = \frac{1}{\sin\theta}\)

\(\sec\theta = \frac{1}{\cos\theta}\)

\(\cot\theta = \frac{1}{\tan\theta} = \frac{\cos\theta}{\sin\theta}\)

🧭 ASTC — "All Students Take Calculus"

QII: S (sin+) | QI: A (all+)
QIII: T (tan+) | QIV: C (cos+)

💡 Reciprocal Signs:

csc has same sign as sin

sec has same sign as cos

cot has same sign as tan

📐 Special Angle Values

30° (π/6): \(\left(\frac{\sqrt{3}}{2}, \frac{1}{2}\right)\)

45° (π/4): \(\left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}\right)\)

60° (π/3): \(\left(\frac{1}{2}, \frac{\sqrt{3}}{2}\right)\)

Remember: (cos, sin) = (x, y)

🔢 Exponential Equations

Common Base:

If \(b^m = b^n\), then \(m = n\)

Uncommon Base:

Take ln of both sides:

\(a^x = c\) → \(x = \frac{\ln(c)}{\ln(a)}\)

Leave answers in exact log form!

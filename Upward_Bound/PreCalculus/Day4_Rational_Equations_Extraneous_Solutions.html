🔎 Warm-Up: Spot the Missing Step

Each problem shows a student's work. Find the ERROR!

Problem 1: Partial Multiplication

Solve: \(\frac{2}{x} + 3 = \frac{5}{x}\)

Student's work: Multiply by \(x\):

\(2 + 3 = 5\) ← "So it's always true!"

✓ Correct: B! The student forgot to multiply the middle term (3) by x.

Correct work: \(2 + 3x = 5\) → \(3x = 3\) → \(x = 1\)

Problem 2: Missing Check

Solve: \(\frac{x}{x-3} = \frac{3}{x-3} + 2\)

Student's work:

\(x = 3 + 2(x-3)\) → \(x = 3 + 2x - 6\) → \(-x = -3\) → \(x = 3\) ✓

✓ Correct: C! When \(x = 3\), the denominator \((x-3) = 0\)!

This is an EXTRANEOUS solution. The equation has NO SOLUTION.

Problem 3: LCD Error

Solve: \(\frac{1}{x+2} + \frac{1}{x-2} = \frac{4}{x^2-4}\)

Student's work: "LCD is \((x+2)(x-2)(x^2-4)\)"

✓ Correct: A! The student didn't recognize that \(x^2-4 = (x+2)(x-2)\).

Correct LCD: \((x+2)(x-2)\) — which IS \(x^2-4\)!

📚 Key Concepts

🔧 The 4-Step Strategy for Rational Equations

  1. FACTOR all denominators completely (Day 1 skills!)
  2. FIND the LCD using all unique factors (Day 2 skills!)
  3. MULTIPLY every term by the LCD — denominators vanish!
  4. SOLVE the resulting polynomial equation
⚠️ STEP 5: ALWAYS CHECK YOUR SOLUTIONS!

🔥 What is an Extraneous Solution?

A solution that emerges from the algebra but makes an original denominator equal to zero.

Division by zero is UNDEFINED

Action: These solutions must be REJECTED — they are not valid!

❓ Why Do Extraneous Solutions Occur?

When we multiply both sides by the LCD, we're assuming the LCD ≠ 0.

If our solution makes the LCD = 0, we violated that assumption!

Original: \(\frac{x}{x-2} = \frac{2}{x-2}\)

If \(x = 2\): Both sides are \(\frac{2}{0}\) = UNDEFINED!

📝 Complete Example

Solve: \(\frac{2}{x-3} + \frac{1}{x} = \frac{5}{x(x-3)}\)

Step 1: Factor denominators

Already factored: \((x-3)\), \(x\), \(x(x-3)\)

Step 2: Find LCD

LCD = \(x(x-3)\)

Step 3: Multiply every term by LCD

\(x(x-3) \cdot \frac{2}{x-3} + x(x-3) \cdot \frac{1}{x} = x(x-3) \cdot \frac{5}{x(x-3)}\)
\(2x + (x-3) = 5\)

Step 4: Solve

\(2x + x - 3 = 5\) → \(3x = 8\) → \(x = \frac{8}{3}\)

Step 5: CHECK!

Does \(x = \frac{8}{3}\) make any denominator = 0?

\(x - 3 = \frac{8}{3} - 3 = -\frac{1}{3} \neq 0\) ✓

\(x = \frac{8}{3} \neq 0\) ✓

Solution: \(x = \frac{8}{3}\) is VALID!

⚠️ Before You Start Solving...

Identify the domain restrictions FIRST!

Set each denominator = 0 and solve. These values are automatically EXCLUDED.

For \(\frac{2}{x-3} + \frac{1}{x} = \frac{5}{x(x-3)}\): Restrictions are \(x \neq 0\) and \(x \neq 3\)

📖 Trig Graph Reference

🌊 General Form

Sinusoidal Function
\(y = A\sin(B(x - h)) + k\)
\(y = A\cos(B(x - h)) + k\)

📊 Parameters

A = Amplitude
|A| = vertical stretch
A < 0 = reflection over x-axis
B = Frequency
Period = 2π/|B|
B > 1 = horizontal compression
h = Phase Shift
h > 0 = shift right
h < 0 = shift left
k = Midline
Vertical shift up/down

🎯 Key Points (Sine)

Start: (0, 0) → midline
Max: (π/2, 1) → top
Middle: (π, 0) → midline
Min: (3π/2, -1) → bottom
End: (2π, 0) → midline

⚖️ Solving Equations

Substitution Method
1. Let u = (inside)
2. Solve for trig(u)
3. Find all u values
4. Substitute back
General Solutions
Add + 2πn for all solutions

📚 Key Concepts

🌊 The Sinusoidal Toolkit

\(y = A\sin(B(x - h)) + k\)   or   \(y = A\cos(B(x - h)) + k\)

|A| = Amplitude

Distance from midline to max/min

Amplitude = \(\frac{\text{max} - \text{min}}{2}\)

If A < 0: reflection over x-axis

Period = 2π/|B|

Length of one complete cycle

B = \(\frac{2\pi}{\text{Period}}\)

B > 1: compressed, B < 1: stretched

h = Phase Shift

Horizontal translation

h > 0: shift RIGHT
h < 0: shift LEFT

⚠️ Watch the sign: (x - h)

k = Midline

Vertical translation (center line)

Midline: y = k

Max = k + |A|, Min = k - |A|

🔄 Sine vs Cosine: Starting Points

y = sin(x)

Starts at midline

Going UP first

Key point: (0, 0)

y = cos(x)

Starts at maximum

Going DOWN first

Key point: (0, 1)

📊 Reading Parameters from a Graph

  1. Find the midline: y = (max + min) / 2 → this is k
  2. Find the amplitude: |A| = (max - min) / 2
  3. Find the period: Distance for one complete cycle → B = 2π/period
  4. Find the phase shift: Where does the "standard" pattern start? → this is h
  5. Determine sine or cosine: Does it start at midline (sine) or max/min (cosine)?

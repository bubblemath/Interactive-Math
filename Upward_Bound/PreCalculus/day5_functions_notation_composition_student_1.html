VLT in words: A graph represents a function if and only if every x-value maps to exactly one y-value. The moment any vertical line hits two points, two y-values share the same x-input — that breaks the definition of a function.

Section 2 — Domain Restrictions: What "Breaks" the Machine?
The domain is the set of all valid inputs. Three expression types restrict the domain — fill in the condition for each.
√ Square Root
√(expression)
Condition: expression 0
Why? You can't take √ of a negative in the real numbers.
Example: f(x) = √(x − 3)
Set x − 3 0
Domain: x
÷ Fraction
1/(expression)
Condition: expression 0
Why? Division by zero is undefined.
Example: f(x) = 1/(x − 1)
Set x − 1 0
Domain: x
log Logarithm
log(expression)
Condition: expression 0
Why? log(0) and log(negative) are undefined.
Example: f(x) = log(x + 5)
Set x + 5 0
Domain: x
Memory device: √ needs ≥ 0 (can include zero — zero under a root is fine). Fraction needs ≠ 0 (exclude zero only). Log needs > 0 (must be strictly positive). All three say "don't let the expression do the forbidden thing."

🔥 Warm-Up: Spot the Mistake

Each problem below contains a common error. Can you find it?

Problem 1: Factor \(x^2 + 5x + 6\)

Student work: \(x^2 + 5x + 6 = (x + 2)(x + 3)\) ✓

✓ This one is actually correct! The factors of 6 that add to 5 are 2 and 3.

Problem 2: Factor \(x^2 - 5x + 6\)

Student work: \(x^2 - 5x + 6 = (x - 2)(x + 3)\) ✗
The Error: Sign mistake! When the middle term is negative and the last term is positive, BOTH factors must be negative.

Correct Answer: \((x - 2)(x - 3)\)
Check: \((-2) \times (-3) = +6\) ✓ and \((-2) + (-3) = -5\) ✓

Problem 3: Solve \(x^2 + 3x = 10\)

Student work: \(x(x + 3) = 10\), so \(x = 10\) or \(x + 3 = 10\) ✗
The Error: Must set equal to ZERO first before factoring!

Correct Process:
\(x^2 + 3x - 10 = 0\)
\((x + 5)(x - 2) = 0\)
\(x = -5\) or \(x = 2\)

📚 Key Concepts & Formulas

🔤 Essential Vocabulary

These four terms all describe the SAME thing:

Term Definition
Roots The x-values where \(f(x) = 0\)
Zeros Same as roots - where the function equals zero
X-intercepts Where the graph crosses the x-axis
Solutions Values that make the equation true

📐 Standard Form

\(ax^2 + bx + c = 0\)

Where \(a \neq 0\)

  • \(a\) = leading coefficient
  • \(b\) = linear coefficient
  • \(c\) = constant term

🎯 The Discriminant

\(D = b^2 - 4ac\)
  • D > 0 (perfect square) → Factorable!
  • D > 0 (not perfect square) → Use quadratic formula
  • D = 0 → One repeated root
  • D < 0 → No real roots

📊 Quadratic Formula

\(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

Works for ANY quadratic equation - your backup plan when factoring fails!

💎 AC Method (Diamond)

Find factors of \(a \cdot c\) that add to \(b\)

Steps:

  1. Multiply \(a \times c\)
  2. Find factor pair that adds to \(b\)
  3. Rewrite middle term
  4. Factor by grouping

🧭 Method Selection Guide: Which Method Should I Use?

GCF First Always check for a Greatest Common Factor before anything else!
Difference of Squares \(a^2 - b^2 = (a+b)(a-b)\) — No middle term, both terms are perfect squares
Square Root Method When the variable is already isolated: \(x^2 = k\) → \(x = \pm\sqrt{k}\)
AC/Diamond Method When discriminant is a perfect square — the equation IS factorable
Quadratic Formula When discriminant is NOT a perfect square — always works!

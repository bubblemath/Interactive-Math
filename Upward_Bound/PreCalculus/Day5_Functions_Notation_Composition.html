🎮 Warm-Up: Function or Relation?

Use the Vertical Line Test! If any vertical line crosses the graph more than once, it's NOT a function.

Graph 1: Parabola opening upward

✓ FUNCTION! Every vertical line crosses the parabola at most once.

Graph 2: Circle

✗ NOT A FUNCTION! A vertical line through the center crosses the circle TWICE.

Graph 3: Straight line with negative slope

✓ FUNCTION! Every vertical line crosses a non-vertical line exactly once.

Graph 4: Sideways parabola (horizontal)

✗ NOT A FUNCTION! A horizontal parabola fails the vertical line test — most vertical lines cross it twice.

📚 Key Concepts

📝 Function Notation Decoder

Notation Meaning Example
\(f(x)\) "f of x" — the output when input is x If \(f(x) = 2x + 1\), then \(f(3) = 7\)
\(f(a)\) Replace every x with a \(f(a) = 2a + 1\)
\(f(x+h)\) Replace every x with (x+h) \(f(x+h) = 2(x+h) + 1\)
\(f(g(x))\) Composition: plug g(x) into f Evaluate g first, then f

🚫 Domain Restrictions

The domain is all valid inputs. Watch for these restrictions:

Square Root: \(\sqrt{expression}\)

expression ≥ 0

Fraction: \(\frac{1}{expression}\)

expression ≠ 0

Logarithm: \(\log(expression)\)

expression > 0

🔗 Composition Basics

\((f \circ g)(x) = f(g(x))\)

"f composed with g" or "f of g of x"

The Process:

  1. Evaluate the INNER function first: \(g(x)\)
  2. Take that result and plug it into the OUTER function: \(f(\text{result})\)
⚠️ Order Matters!

\(f(g(x)) \neq g(f(x))\) in most cases!

📝 Complete Example

Given \(f(x) = x^2 + 1\) and \(g(x) = 2x - 3\), find \(f(g(2))\).

Step 1: Evaluate the INNER function first

\(g(2) = 2(2) - 3 = 4 - 3 = 1\)

Step 2: Plug that result into the OUTER function

\(f(g(2)) = f(1) = (1)^2 + 1 = 1 + 1 = 2\)

Answer: \(f(g(2)) = 2\)

🚫 Domain Explorer

Try inputs that might "break" the machine!

Try: f(x) = √(x - 2)

What happens when x = 1?

The machine will "jam" because √(1-2) = √(-1) is undefined in real numbers!

Domain: x ≥ 2

Try: f(x) = 1/(x - 1)

What happens when x = 1?

The machine will "jam" because 1/(1-1) = 1/0 is undefined!

Domain: x ≠ 1

📈 Teacher Notes: Day 8

Exponential & Logarithmic Functions | Pre-Calculus Bridge Course | Week 2, Day 4
📖 Lesson Overview

The Big Picture: This lesson bridges algebraic manipulation with transcendental functions. The conceptual anchor is the inverse relationship between exponentials and logarithms—they "undo" each other just like squaring and square roots.

The "Heart Method": A visual mnemonic for converting between log and exponential forms. Students trace a heart shape from base → exponent → result to remember that \(\log_b(x) = y\) means \(b^y = x\).

Key Insight: Logarithms are just exponents in disguise. When students internalize this, log properties become intuitive extensions of exponent rules they already know.

Calculus Connection: The natural base \(e\) is the ONLY function that is its own derivative: \(\frac{d}{dx}[e^x] = e^x\). This makes \(e\) the "star" of calculus!

🎯 Learning Objectives
  • Convert between exponential and logarithmic forms using the Heart Method.
  • Identify and graph exponential growth vs. decay based on the base \(b\).
  • Apply the Product, Quotient, and Power properties to expand and condense log expressions.
  • Use the Change of Base formula to evaluate non-standard logarithms.
  • Solve exponential equations using the "Same Base" method or by taking the log of both sides.
  • Solve logarithmic equations and check for extraneous solutions (domain restrictions).

Day 8: The Heart Method & Properties

💕 THE HEART METHOD

\(\log_b(x) = y \quad \Longleftrightarrow \quad b^y = x\)

Trace a heart: Start at the base \(b\) → loop to the answer \(y\) → end at the argument \(x\)

"The base raised to the answer equals the argument"

🎤 Teacher Script: The Heart Method
"A logarithm is just an exponent in disguise. When you see \(\log_b(x) = y\), think of it as asking: 'What power do I raise \(b\) to in order to get \(x\)?' The answer is \(y\). The 'Heart' helps you remember: start at the base, loop to the answer, and land on the argument. It draws a heart that says \(b^y = x\)."
🔢 Growth vs. Decay
📈 EXPONENTIAL GROWTH

\(b > 1\)

Examples: \(2^x\), \(e^x\), \(10^x\)

Graph rises from left to right

📉 EXPONENTIAL DECAY

\(0 < b < 1\)

Examples: \((0.5)^x\), \((1/3)^x\)

Graph falls from left to right

Common Confusion: Students think a negative exponent means decay. Remind them: \(2^{-x} = (1/2)^x\). It's the BASE that determines growth/decay, not the sign of the exponent!

📋 The "Big Three" Log Properties
Product Rule
\(\log_b(MN) = \log_b M + \log_b N\)

Multiply inside → Add outside

Quotient Rule
\(\log_b\left(\frac{M}{N}\right) = \log_b M - \log_b N\)

Divide inside → Subtract outside

Power Rule
\(\log_b(M^p) = p \cdot \log_b M\)

Exponent inside → Multiplier outside

Analogy: These mirror exponent rules! Multiplying bases = adding exponents, so multiplying inside a log = adding two logs.

🔄 CHANGE OF BASE FORMULA

\(\log_b(x) = \frac{\log(x)}{\log(b)} = \frac{\ln(x)}{\ln(b)}\)

Use this to evaluate any log on a calculator (which only has \(\log\) and \(\ln\) buttons)

🔑 Special Logarithms

Common Log

\(\log(x) = \log_{10}(x)\)

Base 10 (calculator default)

Natural Log

\(\ln(x) = \log_e(x)\)

Base \(e \approx 2.718\) (calculus star!)

Day 8: Solving Equations & Pitfalls

🧮 Solving Exponential Equations

Method 1: Same Base

If you can rewrite both sides with the same base, set exponents equal.

Example: \(8^x = 32\)

\((2^3)^x = 2^5\) → \(3x = 5\) → \(x = \frac{5}{3}\)

Method 2: Take the Log

When same base isn't possible, take \(\log\) or \(\ln\) of both sides.

Example: \(5^x = 12\)

\(\ln(5^x) = \ln(12)\) → \(x \ln 5 = \ln 12\) → \(x = \frac{\ln 12}{\ln 5}\)

🔓 Solving Logarithmic Equations

The 4-Step Procedure:

  1. Isolate the log expression (or combine logs using properties)
  2. Convert to exponential form using the Heart Method
  3. Solve the resulting equation
  4. CHECK for extraneous solutions (arguments must be positive!)

⚠️ DOMAIN ALARM

You CANNOT take the log of zero or a negative number! Always check that your solution makes ALL original log arguments positive.

⚠️ Common Pitfalls & Teaching Strategies
1. The Log Addition Trap

Students write \(\log(A + B) = \log A + \log B\). This is FALSE!

Strategy: Emphasize: "Logs turn MULTIPLICATION into addition, not addition into addition." Use a numerical counterexample: \(\log(2 + 8) = \log(10) = 1\), but \(\log 2 + \log 8 = \log 16 \neq 1\).
2. Negative Argument Error

Students plug in solutions without checking if the log argument is positive.

Strategy: Use the "Domain Alarm" visual. Before solving, have students identify the "forbidden zone" where arguments would be ≤ 0. Circle these restrictions!
3. Power Rule Misplacement

Students write \(\log(x + 3)^2 = 2\log(x+3)\) when the exponent only applies to part of the argument.

Strategy: The power rule ONLY works when the ENTIRE argument is raised to the power. Use parentheses carefully: \(\log(x^2 + 3) \neq 2\log(x+\sqrt{3})\).
4. Confusing \(\ln\) and \(\log\)

Students mix up natural log (base \(e\)) and common log (base 10).

Strategy: Create a "Log ID Card" visual. \(\ln\) = "natural" = base \(e\). \(\log\) (no subscript) = base 10. When in doubt, write the base explicitly!

Day 8: Calculus Connections & Assessment

🔗 Calculus Connections
Pre-Calculus Skill Calculus Application
Natural Log (\(\ln\)) Essential for integration: \(\int \frac{1}{x} dx = \ln|x| + C\). The antiderivative of \(1/x\) is the natural log!
Log Properties Logarithmic Differentiation: Simplify complex products/quotients before differentiating by taking \(\ln\) of both sides.
Base \(e\) The ONLY function that is its own derivative: \(\frac{d}{dx}[e^x] = e^x\). This makes \(e\) the "natural" choice for growth models.
Exponential Equations Solving differential equations like \(\frac{dy}{dx} = ky\) leads to \(y = Ce^{kx}\) (exponential growth/decay).
Change of Base Deriving \(\frac{d}{dx}[a^x] = a^x \ln(a)\) requires rewriting \(a^x = e^{x \ln a}\).
✅ Formative Assessment Checkpoints
  • Fist to Five: "How confident are you using the Heart Method to rewrite \(\log_3(81) = 4\) as an exponential?"
  • Whiteboard Flash: "Expand \(\log\left(\frac{x^2}{y}\right)\)." Look for: \(2\log x - \log y\).
  • Error Analysis: Write \(\log(x) + \log(y) = \log(x+y)\) on the board. Ask students to "arrest the illegal math."
  • Quick Poll: "Is \(\log_2(-8)\) defined?" (No! Negative argument.)
📝 Exit Ticket & Answers
Exit Ticket Problems
  1. Convert to exponential form: \(\log_2(32) = 5\)
    Answer: \(2^5 = 32\)
  2. Solve: \(3^{x-1} = 27\)
    Answer: \(3^{x-1} = 3^3\) → \(x - 1 = 3\) → \(\boxed{x = 4}\)
  3. Expand: \(\ln(ab^3)\)
    Answer: \(\ln a + \ln(b^3) = \boxed{\ln a + 3\ln b}\)
  4. Solve: \(\log_4(x) + \log_4(x-3) = 1\)
    Answer: \(\log_4[x(x-3)] = 1\) → \(x^2 - 3x = 4\) → \(x^2 - 3x - 4 = 0\) → \((x-4)(x+1) = 0\)
    Solutions: \(x = 4\) or \(x = -1\). Check: \(x = -1\) makes \(\log_4(-1)\) undefined. \(\boxed{x = 4}\) only.
🎚️ Differentiation Strategies
🔴 For Struggling Students
  • Focus on the Heart Method with simple bases (2, 3, 10)
  • Provide a "Powers Reference Sheet" (powers of 2, 3, 5, 10)
  • Use color-coding: base in blue, exponent in red, result in green
  • Start with evaluation before properties (\(\log_2 8 = ?\))
  • Allow calculator for Change of Base problems
🟢 For Advanced Students
  • Introduce nested logs: \(\log_2(\log_3 81)\)
  • Explore why \(e\) is "natural" (compound interest limit)
  • Challenge: Prove \(\log_a b \cdot \log_b a = 1\)
  • Preview logarithmic differentiation
  • Real-world: pH scale, Richter scale, decibels

Day 8: Partner & Small Group Activities

👥 Partner & Small Group Activities
1. Heart Method Speed Dating 💕
Rotating Pairs
Half the class gets cards with LOG form (e.g., \(\log_3(27) = 3\)), the other half gets EXPONENTIAL form (e.g., \(3^3 = 27\)). Students rotate to find their "match" by converting their form. Once matched, they verify together and get a new card.
📦 Materials: Matching card sets (20+ pairs), timer
✨ Reinforces: Heart Method conversions, mental math fluency
2. Log Property Scavenger Hunt 🔍
Groups of 3
Hide "condensed" log expressions around the room in envelopes. Groups must find them, EXPAND them fully using all applicable properties, and the expanded form reveals a code/clue for the next station. First team to complete all 5 stations wins!
📦 Materials: Envelopes with expressions, answer verification sheets, small prizes
✨ Reinforces: Product, Quotient, Power rules; teamwork under pressure
3. "Same Base" Relay Race 🏁
Teams of 4
A multi-step exponential equation is posted. Each team member handles ONE step:
• Student 1: Identify the common base
• Student 2: Rewrite both sides with that base
• Student 3: Set exponents equal and solve for \(x\)
• Student 4: Verify by substitution
Paper passes down the line. First team with correct, verified answer wins the round!
📦 Materials: Relay problem cards, team answer sheets, timer
✨ Reinforces: Same Base method, step-by-step accountability
4. The "Forbidden Zone" Audit 🚫
Pairs
Provide 6 "solved" logarithmic equations—some correct, some with extraneous solutions that weren't caught. Partners must "audit" each solution by checking if the answers fall into the "Forbidden Zone" (making log arguments ≤ 0). They write an "Audit Report" for each problem.
📦 Materials: Pre-solved problem sheets, "Audit Report" templates
✨ Reinforces: Domain restrictions, checking for extraneous solutions
5. Base \(e\) vs. Common Log Sort 📊
Groups of 3
Card sort activity: Students categorize real-world scenarios by whether they typically use \(\log_{10}\) (common log) or \(\ln\) (natural log). Scenarios include: compound interest, pH scale, population growth, sound decibels, radioactive decay, Richter scale, bacterial growth.
📦 Materials: Scenario cards, sorting mat with two columns
✨ Reinforces: Real-world applications, distinguishing log types
6. Create the "Monster" Log 👹
Pairs (Advanced)
Challenge pairs to create the most complex SINGLE log expression they can, using all three properties (product, quotient, power). They then swap with another pair who must CONDENSE it back to a single log. Award points for complexity AND correct condensing.
📦 Materials: Whiteboards, markers, complexity rubric
✨ Reinforces: All log properties, working forwards AND backwards
7. Powers of 2 "War" ⚔️
Pairs
A card game where each card shows a power of 2 in different forms: \(2^5\), \(\log_2(32)\), \(32\), etc. Players flip simultaneously; the higher value wins both cards. Ties are resolved by a "war" (flip 3 more). Builds instant recognition of equivalent forms.
📦 Materials: Custom card deck (powers of 2, 3, and 5 in various forms)
✨ Reinforces: Mental math, equivalent form recognition, fluency

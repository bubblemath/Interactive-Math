📚 KEY DEFINITIONS
Function A rule that assigns exactly one output to each input. Every \(x\) gets only one \(y\).
✅ \(f(x) = x^2\)
❌ \(x = y^2\) (one \(x\) → two \(y\)'s)
Domain All possible input values (\(x\)-values) that make the function work.
\(f(x) = \sqrt{x}\)
Domain: \(x \geq 0\)
Range All possible output values (\(y\)-values) the function can produce.
\(f(x) = x^2\)
Range: \(y \geq 0\)

THE FUNCTION MACHINE

INPUT
\(x = 3\)
→
⚙️ \(f(x) = x^2 + 1\)
→
OUTPUT
\(f(3) = 10\)

One input in → One output out. That's what makes it a function!

⚠️ CRITICAL: \(f(x)\) does NOT mean "\(f\) times \(x\)"! The parentheses indicate function input, not multiplication.

🎯 THE VERTICAL LINE TEST

If ANY vertical line crosses the graph more than once, it's NOT a function.

📈

Parabola \(y = x^2\)

Every vertical line hits once

✓ IS A FUNCTION
⭕

✗ NOT A FUNCTION
⚠️ COMMON MISTAKES & CORRECTIONS
❌ The Mistake Why It's Wrong ✅ The Correction
"Multiplication Trap"
\(f(x) = f \cdot x\) 		The parentheses in function notation indicate INPUT, not multiplication! \(f(x)\) means "the output of function \(f\) when \(x\) is the input"
"Negative Input Error"
\(f(-2) = -f(2)\) 		You must substitute \(-2\) for EVERY \(x\) in the formula, including inside exponents If \(f(x) = x^2\), then \(f(-2) = (-2)^2 = 4\), not \(-4\)
"Composition Order"
\(f(g(x)) = g(f(x))\) 		Composition is NOT commutative! The order matters. Always work "inside-out": evaluate the inner function first

🔧 PROCEDURE: Evaluating Functions

To find \(f(a)\) where \(a\) is any value or expression:

  1. Identify the input value (what's inside the parentheses).
  2. Substitute that value for EVERY \(x\) in the function formula. Use parentheses!
  3. Simplify using order of operations (PEMDAS).
Example: If \(f(x) = 2x^2 - 3x + 1\), find \(f(-2)\):
\(f(-2) = 2(-2)^2 - 3(-2) + 1 = 2(4) + 6 + 1 = 8 + 6 + 1 = 15\)
🔗 PROCEDURE: Function Composition (The "Inside-Out" Method)

\((f \circ g)(x) = f(g(x))\) — "f of g of x"

\(x\)
→
\(g(x)\)
→
\(f(\text{result})\)
→
Final Answer

Start with the INNER function, then feed its output into the OUTER function

  1. Identify the inner function (the one closest to \(x\)) and the outer function.
  2. Evaluate the inner function first (or write its expression).
  3. Substitute the inner function's result into the outer function (replace every \(x\)).
  4. Simplify the final expression completely.
✏️ WORKED EXAMPLE: Given \(f(x) = x^2 + 3\) and \(g(x) = 2x - 1\)

Find \(f(g(x))\)

Step 1: Identify
Inner: \(g(x) = 2x - 1\)
Outer: \(f(x) = x^2 + 3\)
Step 2: Substitute \(g(x)\) into \(f\)
\(f(g(x)) = f(2x-1)\)
\(= (2x-1)^2 + 3\)
Step 3: Expand & Simplify
\(= 4x^2 - 4x + 1 + 3\)
\(f(g(x)) = 4x^2 - 4x + 4\)

Find \(g(f(x))\)

Step 1: Identify
Inner: \(f(x) = x^2 + 3\)
Outer: \(g(x) = 2x - 1\)
Step 2: Substitute \(f(x)\) into \(g\)
\(g(f(x)) = g(x^2+3)\)
\(= 2(x^2+3) - 1\)
Step 3: Distribute & Simplify
\(= 2x^2 + 6 - 1\)
\(g(f(x)) = 2x^2 + 5\)

⚠️ Notice: \(f(g(x)) \neq g(f(x))\) — ORDER MATTERS!

🚀 CALCULUS PREVIEW: The Chain Rule
🔗

Composition → Chain Rule

In Calculus, when you take the derivative of a composite function like \(f(g(x))\), you use the Chain Rule:

\(\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)\)
🎯

Why This Matters

The Chain Rule is one of the most-used derivative rules. Mastering composition NOW means you'll breeze through it later!

Example: To differentiate \((3x+1)^5\), recognize it as \(f(g(x))\) where \(f(u) = u^5\) and \(g(x) = 3x+1\).

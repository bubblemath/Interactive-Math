Section 1 - What is an Inverse Function?

An inverse function "undoes" what the original function does. If \(f\) takes you from \(a\) to \(b\), then \(f^{-1}\) takes you from \(b\) back to \(a\).

📖 Key Definition: Inverse Function

If \(f(a) = b\), then \(f^{-1}(b) = \)

The Domain/Range Swap:

  • The Domain of \(f\) becomes the of \(f^{-1}\)
  • The Range of \(f\) becomes the of \(f^{-1}\)
Section 2 - One-to-One Functions & The Horizontal Line Test

Not every function has an inverse! A function must be one-to-one (each output comes from exactly ONE input) to have an inverse.

🔍 The Horizontal Line Test (HLT)

A function has an inverse if and only if no horizontal line intersects its graph more than time(s).

✅ PASSES HLT

Has an inverse function

\(f(x) = 2x + 3\)
\(f(x) = \sqrt{x}\)
\(f(x) = x^3\)
❌ FAILS HLT

NO inverse function

\(f(x) = x^2\)
\(f(x) = |x|\)
\(f(x) = \sin(x)\)

Day 7: The "Switcher" Method

Section 4 - The "Switcher" Method (Algebraic Procedure)

To find the inverse of a function algebraically, follow these 5 steps:

🔄 THE "SWITCHER" METHOD
  1. Replace \(f(x)\) with \(y\)
  2. Swap every \(x\) and \(y\) in the equation
  3. Solve the new equation for \(y\) (isolate it!)
  4. Rename the result as \(f^{-1}(x)\)
  5. State any domain restrictions (from the original range)
Section 5 - Worked Example #1: Linear Function
Example: Find the inverse of \(f(x) = \frac{2x - 3}{5}\)

Step 1: Replace \(f(x)\) with \(y\)

\(y = \frac{2x - 3}{5}\)

Step 2: Swap \(x\) and \(y\)

\(x = \frac{2y - 3}{5}\)

Step 3: Solve for \(y\)

\(5x = 2y - 3\) → \(5x + 3 = 2y\) → \(y = \frac{5x + 3}{2}\)

Step 4: Rename as \(f^{-1}(x)\)

\(\boxed{f^{-1}(x) = \frac{5x + 3}{2}}\)

Step 5: Domain

No restrictions (linear function). Domain: \((-\infty, \infty)\)

Day 7: Radical & Rational Inverses

Section 6 - Worked Example #2: Radical Function
Example: Find the inverse of \(f(x) = \sqrt{x - 2} + 1\)

Step 1: Replace \(f(x)\) with \(y\)

\(y = \sqrt{x - 2} + 1\)

Step 2: Swap \(x\) and \(y\)

\(x = \sqrt{y - 2} + 1\)

Step 3: Solve for \(y\)

\(x - 1 = \sqrt{y - 2}\) (subtract 1)

\((x - 1)^2 = y - 2\) (square both sides)

\(y = (x - 1)^2 + 2\) (add 2)

Step 4: Rename as \(f^{-1}(x)\)

\(f^{-1}(x) = (x - 1)^2 + 2\)

Step 5: Domain Restriction ⚠️

Original \(f(x)\) has range \([1, \infty)\) (starts at 1, goes up)

So \(f^{-1}(x)\) has domain: \(\boxed{x \geq 1}\)

Final Answer: \(\boxed{f^{-1}(x) = (x-1)^2 + 2, \quad x \geq 1}\)

Section 7 - Verification by Composition

To PROVE two functions are inverses, show that composing them (in BOTH directions) gives \(x\):

🔗 THE VERIFICATION TEST

\(f(f^{-1}(x)) = x\) AND \(f^{-1}(f(x)) = x\)

If BOTH compositions equal \(x\), they are true inverses!

Day 7: Verification & Graphical Symmetry

Section 8 - Verification Example
Example: Verify that \(f(x) = 2x + 3\) and \(g(x) = \frac{x-3}{2}\) are inverses

Check #1: \(f(g(x))\)

\(f\left(\frac{x-3}{2}\right)\)

\(= 2 \cdot \frac{x-3}{2} + 3\)

\(= (x - 3) + 3\)

\(= x \quad \checkmark\)

Check #2: \(g(f(x))\)

\(g(2x + 3)\)

\(= \frac{(2x + 3) - 3}{2}\)

\(= \frac{2x}{2}\)

\(= x \quad \checkmark\)

Both equal \(x\), so they ARE inverses! ✅

Section 9 - Graphical Symmetry

The graph of \(f^{-1}(x)\) is a reflection of \(f(x)\) across the line \(y = x\).

🪞 The Reflection Rule

If the point \((a, b)\) is on the graph of \(f\), then the point \((\)\(,\)\()\) is on the graph of \(f^{-1}\).

Why? Because \(f(a) = b\) means \(f^{-1}(b) = a\). The coordinates SWAP!

Day 7: Summary & Exit Ticket

🚀 Calculus Connection: The Inverse Function Theorem

In Calculus, you'll learn that the derivative (slope) of an inverse function is related to the derivative of the original function:

\((f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))}\)

This means: If \(f\) is growing quickly, then \(f^{-1}\) is growing . The slopes are reciprocals at corresponding points!

📖 Reference Sheet

Quadratic Formula

\(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

Discriminant: \(b^2 - 4ac\)

Exponent Rules

\(x^m \cdot x^n = x^{m+n}\)
\(\frac{x^m}{x^n} = x^{m-n}\)
\((x^m)^n = x^{mn}\)

Radical ↔ Exponent

\(\sqrt[n]{x^m} = x^{m/n}\)
\(x^{-n} = \frac{1}{x^n}\)

Logarithm Rules

\(\log_b(xy) = \log_b x + \log_b y\)
\(\log_b(x/y) = \log_b x - \log_b y\)
\(\log_b(x^n) = n \log_b x\)

Transformation Form

\(y = a \cdot f(b(x - h)) + k\)

a = vertical stretch, h = horizontal shift, k = vertical shift

Unit Circle Values

\(\sin(30°) = \frac{1}{2}, \cos(30°) = \frac{\sqrt{3}}{2}\)
\(\sin(45°) = \cos(45°) = \frac{\sqrt{2}}{2}\)
\(\sin(60°) = \frac{\sqrt{3}}{2}, \cos(60°) = \frac{1}{2}\)

Sinusoidal Functions

\(y = A\sin(Bx) + k\)

Amplitude = |A|, Period = \(\frac{2\pi}{|B|}\)

Degree ↔ Radian

\(\text{Radians} = \text{Degrees} \times \frac{\pi}{180}\)
\(180° = \pi \text{ rad}\)

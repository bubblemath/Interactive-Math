📊 Parent Functions & Transformations

Page 1: The Parent Function Gallery — 15 Functions in 5 Families
📈 Family 1: Polynomials

Linear

f(x) = x
D: (−∞, ∞) R: (−∞, ∞) Pts: (−1,−1), (0,0), (1,1)

Quadratic

f(x) = x²
D: (−∞, ∞) R: [0, ∞) Pts: (−1,1), (0,0), (1,1)

Cubic

f(x) = x³
D: (−∞, ∞) R: (−∞, ∞) Pts: (−1,−1), (0,0), (1,1)

Absolute Value

f(x) = |x|
D: (−∞, ∞) R: [0, ∞) Pts: (−1,1), (0,0), (1,1)
💡 Memory: Linear = diagonal line | Quadratic = U-shape | Cubic = S-shape | Abs Value = V-shape
√ Family 2: Radicals

Square Root

f(x) = √x
D: [0, ∞) — Can't take √ of negative! R: [0, ∞) Pts: (0,0), (1,1), (4,2), (9,3)

Cube Root

f(x) = ³√x
D: (−∞, ∞) — CAN take ³√ of negative! R: (−∞, ∞) Pts: (−8,−2), (−1,−1), (0,0), (1,1), (8,2)
💡 Memory: Square root starts at origin, goes right | Cube root passes through origin like a "lazy S"
÷ Family 3: Rationals

Reciprocal

f(x) = 1/x
D: (−∞, 0) ∪ (0, ∞) — x ≠ 0 R: (−∞, 0) ∪ (0, ∞) Asymptotes: VA: x = 0, HA: y = 0 Pts: (−1,−1), (1,1)

Reciprocal Squared

f(x) = 1/x²
D: (−∞, 0) ∪ (0, ∞) — x ≠ 0 R: (0, ∞) — Always positive! Asymptotes: VA: x = 0, HA: y = 0 Pts: (−1,1), (1,1)
💡 Memory: 1/x = hyperbola in Q1 & Q3 | 1/x² = both branches above x-axis (always positive)
eˣ Family 4: Exponential & Logarithmic

Exponential (b > 1)

f(x) = bˣ (e.g., 2ˣ)
D: (−∞, ∞) R: (0, ∞) HA: y = 0 Pts: (0,1), (1,b)

Natural Exp

f(x) = eˣ
D: (−∞, ∞) R: (0, ∞) HA: y = 0 Pts: (0,1), (1,e≈2.72)

Logarithmic

f(x) = log_b(x)
D: (0, ∞) R: (−∞, ∞) VA: x = 0 Pts: (1,0), (b,1)

Natural Log

f(x) = ln(x)
D: (0, ∞) R: (−∞, ∞) VA: x = 0 Pts: (1,0), (e,1)
💡 Memory: Exp & Log are INVERSES — reflect over y = x! Exp has HA at y=0; Log has VA at x=0
∿ Family 5: Trigonometric

Sine

f(x) = sin(x)
D: (−∞, ∞) R: [−1, 1] Period: 2π Starts: at origin (0,0)

Cosine

f(x) = cos(x)
D: (−∞, ∞) R: [−1, 1] Period: 2π Starts: at max (0,1)

Tangent

f(x) = tan(x)
D: x ≠ π/2 + nπ R: (−∞, ∞) Period: π VA: at odd π/2
💡 Memory: Sine starts at 0 | Cosine starts at 1 | Tangent has vertical asymptotes every π

Page 2: The Transformation Toolkit — g(x) = a · f(b(x − h)) + k

🎯 The Inside-Outside Rule — Master This First!

INSIDE the function: f(bx − h)
Affects the X-axis (horizontal)

⚠️ OPPOSITE of what you expect!

h = 3 → moves RIGHT | b = 2 → compresses

OUTSIDE the function: a·f(x) + k
Affects the Y-axis (vertical)

✓ Works as expected!

k = 3 → moves UP | a = 2 → stretches

🔧 The Four Transformation Types

↔️ Shifts (Translations)

f(x) + k Shift UP k units
f(x) − k Shift DOWN k units
f(x − h) Shift RIGHT h units ⚠️
f(x + h) Shift LEFT h units ⚠️

🪞 Reflections (Flips)

−f(x) Reflect over X-AXIS
(negative OUTSIDE)
f(−x) Reflect over Y-AXIS
(negative INSIDE)

↕️ Vertical Stretch/Compress (a)

a·f(x), |a| > 1 Vertical STRETCH
(taller/skinnier)
a·f(x), 0 < |a| < 1 Vertical COMPRESS
(shorter/fatter)

↔️ Horizontal Stretch/Compress (b)

f(bx), |b| > 1 Horizontal COMPRESS ⚠️
(pushed toward y-axis)
f(bx), 0 < |b| < 1 Horizontal STRETCH ⚠️
(pulled from y-axis)

📋 Order of Transformations (H-S-R-V Method)

1 Horizontal Shift
2 Stretch/Compress
3 Reflection
4 Vertical Shift
💡 Mnemonic: "High School Really Valuable" or "Horizontal Stuff, Reflection, Vertical"
Transformation Notation Effect on Points (x, y) Example
Shift Up k f(x) + k (x, y) → (x, y + k) x² + 3: vertex (0,0) → (0,3)
Shift Right h f(x − h) (x, y) → (x + h, y) (x−2)²: vertex (0,0) → (2,0)
Reflect x-axis −f(x) (x, y) → (x, −y) −x²: opens down instead of up
Vertical Stretch a·f(x), a > 1 (x, y) → (x, ay) 2x²: (1,1) → (1,2)
Horiz. Compress f(bx), b > 1 (x, y) → (x/b, y) (2x)²: (2,4) → (1,4)

🚀 Calculus Preview — Why This Matters!

Recognizing Parent Functions
→ Quickly identify derivative/integral patterns
Horizontal Shifts
→ u-substitution in integration
Vertical Stretch (a)
→ Constant multiple rule: d/dx[af(x)] = a·f'(x)
Chain Rule Setup
→ f(g(x)) requires recognizing "inside" function

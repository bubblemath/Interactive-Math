Section 2 — Product Rule
✖ Product Rule "First d-Second  plus  Second d-First"
\(\dfrac{d}{dx}[f(x)\cdot g(x)] = f'(x)\cdot g(x) + f(x)\cdot g'(x)\)
Geometric proof (Growing Rectangle)
Rectangle sides \(u\) and \(v\) both grow.
New area = top strip + right strip + corner.
Corner (\(du\cdot dv\)) → _______ as \(h\to0\).
Result: \(u\,dv + v\,du\) → \(f'g + fg'\) ✓
Algebraic proof — key trick
Add & subtract \(f(x+h)g(x)\) in the numerator. This creates two recognisable difference quotients — one for \(g'\) and one for \(f'\).
As \(h\to0\): \(f(x+h)\to f(x)\).
Always label \(f\), \(g\), \(f'\), \(g'\) before applying the rule:
Guided: \(\frac{d}{dx}[x^2\sin x]\)
\(f = x^2\)
\(g = \sin x\)
\(f' = 2x\)
\(g' = \cos x\)
\(= f'g + fg'\)
\(= 2x\sin x + x^2\cos x\) ✓

Section 3 — Quotient Rule
➗ Quotient Rule "Low d-High  minus  High d-Low  over  Low-Low"
\(\dfrac{d}{dx}\!\left[\dfrac{f(x)}{g(x)}\right] = \dfrac{f'(x)\cdot g(x) - f(x)\cdot g'(x)}{[g(x)]^2}\)
The mnemonic — say it aloud before writing:
"Low \(d\)·High  minus  High \(d\)·Low, all over Low²"
High = numerator \(f\)  ·  Low = denominator \(g\)
Proof: Quotient Rule from Product Rule
Rewrite: \(\frac{f}{g} = f\cdot g^{-1}\). Apply Product Rule. For \(\frac{d}{dx}[g^{-1}]\), apply Chain Rule: \(-g^{-2}\cdot g'\). Combine over \(g^2\).
🚨 The Sign Error Trap — Gets Half the Class on Every Exam
✗ Wrong order:  \(\dfrac{fg' - f'g}{g^2}\)   (subtraction backwards)
✓ Correct order:  \(\dfrac{f'g - fg'}{g^2}\)   "Low d-High" comes first
Always label High, Low, d(High), d(Low) before applying:
Guided: \(\frac{d}{dx}\!\left[\frac{x^2}{\sin x}\right]\)
High \(f = x^2\)
Low \(g = \sin x\)
\(f' = 2x\)
\(g' = \cos x\)
\(= \dfrac{2x\sin x - x^2\cos x}{\sin^2 x}\) ✓

Classic: Prove \(\frac{d}{dx}[\tan x] = \sec^2 x\) using the Quotient Rule
Write \(\tan x = \frac{\sin x}{\cos x}\). Apply Quotient Rule. Use \(\sin^2 x + \cos^2 x = 1\) at the end.
Result: \(\frac{\cos^2 x + \sin^2 x}{\cos^2 x} = \frac{1}{\cos^2 x} = \sec^2 x\) ✓ (verify this path)
Section 4 — Chain Rule
🔗 Chain Rule "Outer prime (keep inner)  ×  inner prime"
\(\dfrac{d}{dx}[f(g(x))] = f'(g(x))\cdot g'(x)\)
Leibniz form:  \(\dfrac{dy}{dx} = \dfrac{dy}{du}\cdot\dfrac{du}{dx}\)
where \(y=f(u)\) and \(u=g(x)\)

In plain English:
Differentiate the outer — keep the inner untouched. Multiply by the derivative of the inner.
Proof — the \(\frac{\Delta u}{\Delta u}\) trick:
\(\dfrac{dy}{dx} = \lim_{\Delta x\to0}\dfrac{\Delta y}{\Delta x}\)
\(= \lim_{\Delta x\to0}\dfrac{\Delta y}{\Delta u}\cdot\dfrac{\Delta u}{\Delta x}\)
\(\to\dfrac{dy}{du}\cdot\dfrac{du}{dx}\) ✓

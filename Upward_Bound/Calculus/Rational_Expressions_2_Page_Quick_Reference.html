🚫 Domain Restrictions Procedure
1

Factor all denominators completely

2

Set each factor equal to zero

3

Solve for x in each equation

4

Write: x ≠ [forbidden values]

⚠️ Do this BEFORE canceling any factors! Restrictions persist even after simplifying!

📌 Example: Find restrictions for (x + 2)/(x² − 9)

Factor: x² − 9 = (x − 3)(x + 3)
Set = 0: x − 3 = 0, x + 3 = 0
Solve: x = 3, x = −3
Answer: x ≠ 3, x ≠ −3
🕳️ vs 📊 Holes vs. Vertical Asymptotes

🕳️ HOLE (Removable Discontinuity)

Cause: Factor cancels completely from num & denom
Graph: Open circle — a single missing point
Behavior: Function approaches a finite value
🔮 Calculus: The LIMIT EXISTS! You can evaluate it by canceling and substituting.

📊 VERTICAL ASYMPTOTE

Cause: Factor remains in denominator after simplifying
Graph: Vertical line — graph goes to ±∞
Behavior: Function increases/decreases without bound
🔮 Calculus: The limit is ±∞ (or DNE). One-sided limits may differ!

📌 Example: f(x) = [(x − 2)(x + 3)] / [(x − 2)(x − 1)]

Hole at x = 2 — the (x − 2) factor cancels
Asymptote at x = 1 — the (x − 1) factor remains in denominator
❌ The "Illegal Cancel" — DON'T DO THIS!

❌ WRONG — Canceling TERMS

(x + 3)/(x + 5) = 3/5
(x² + 4)/x = x + 4

Terms are connected by + or −

✓ CORRECT — Canceling FACTORS

(x + 3)(x − 2)/(x + 3) = (x − 2)
x(x + 4)/x = (x + 4)

Factors are connected by × (multiplication)

💡 Memory Trick: "Factors cancel, terms don't!" — You can only cancel things being MULTIPLIED!

📐 Finding the LCD (Least Common Denominator)

① Factor ALL denominators completely
② List each unique factor
③ Take each to its HIGHEST power
④ Multiply together = LCD
⚡ Solving Rational Equations — "Clear the Denominators"
1

Find the LCD of ALL denominators

2

Multiply EVERY term by the LCD

3

Simplify — denominators will cancel!

4

Solve the resulting equation

5

CHECK for extraneous solutions!

🔥 Extraneous Solutions — THE TRAP!

What is an Extraneous Solution?

A solution that appears algebraically correct but makes the ORIGINAL equation undefined.

Why does this happen? When we multiply by the LCD, we assume LCD ≠ 0. If our answer makes LCD = 0, we've violated our own assumption!

✅ Verification Checklist

  • Found domain restrictions FIRST?
  • Multiplied EVERY term by LCD?
  • Solved the equation correctly?
  • Checked EACH solution in ORIGINAL?
  • Rejected solutions that make denom = 0?

📌 Classic Trap Example: Solve x/(x − 2) = 2/(x − 2) + 1

Algebra: Multiply by (x − 2):
x = 2 + (x − 2) → x = x
Always true... so x = 2?
CHECK: If x = 2, then
denominator = 2 − 2 = 0
→ UNDEFINED! ❌
x = 2 is EXTRANEOUS! → Answer: NO SOLUTION

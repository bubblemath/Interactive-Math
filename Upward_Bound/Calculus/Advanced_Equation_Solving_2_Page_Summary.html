✨ Advanced Equation Solving

Radicals · Absolute Value · Exponentials · Logarithms

🎯 The Universal Strategy: Undo with Inverses

\(\text{Operation} \xrightarrow{\text{inverse}} \text{Undone}\)

Every equation type has an inverse operation that "unlocks" the variable!

📚 Key Definitions

🔄 Inverse Operations

Operations that "undo" each other. Squaring ↔ Square Root, Exponential ↔ Logarithm. The key to isolating variables!

⚠️ Extraneous Solution

A "fake" answer that emerges algebraically but does NOT satisfy the original equation. Always check your work!

📐 Principal Root

The non-negative root. \(\sqrt{25} = 5\), not \(\pm 5\). This is why \(\sqrt{x} = -3\) has no solution!

🔀 Absolute Value

Distance from zero: \(|x| = x\) if \(x \geq 0\), and \(|x| = -x\) if \(x < 0\). Creates TWO cases to solve!

📐 Essential Formulas & Inverse Pairs

√ Radical Equations

\(\sqrt[n]{x} = a \implies x = a^n\)

Raise both sides to the index power to eliminate the radical.

| | Absolute Value
\(|u| = a \implies u = a \text{ OR } u = -a\)

Split into two cases (only when \(a \geq 0\)).

📈 Exponential Equations

\(b^x = a \implies x = \log_b(a)\)

Take the logarithm of both sides to bring down the exponent.

📉 Logarithmic Equations

\(\log_b(x) = a \implies x = b^a\)

Rewrite in exponential form to solve for x.

💡 Memory Trick: "Logs and Exponents are INVERSES" — just like addition/subtraction or multiplication/division!
🔧 Step-by-Step Procedures
√ Radical Equations
  1. Isolate the radical on one side
  2. Raise both sides to the index power
  3. Solve the resulting equation
  4. Check ALL solutions in original!
| | Absolute Value
  1. Isolate the absolute value
  2. Check if right side is negative (no solution!)
  3. Split into positive and negative cases
  4. Solve both equations
📈 Exponential Equations
  1. Isolate the exponential term
  2. Take log of both sides (any base)
  3. Apply power rule: \(\log(b^x) = x\log(b)\)
  4. Solve for the variable
📉 Logarithmic Equations
  1. Isolate the logarithm (or combine logs)
  2. Convert to exponential form
  3. Solve the resulting equation
  4. Check domain: argument must be > 0!

Common Mistakes, Worked Examples & Calculus Connections
⚠️ Common Mistakes & Corrections
The Mistake Why It's Wrong The Correction
\((x+3)^2 = x^2 + 9\) Forgot the middle term when squaring a binomial (FOIL!) \((x+3)^2 = x^2 + 6x + 9\)
\(\sqrt{x} = -5 \implies x = 25\) Principal square root is NEVER negative No Solution (∅)
\(|x-2| = 5 \implies x = 7\) only Forgot the second case (negative) \(x = 7\) OR \(x = -3\)
\(\log(x) + \log(3) = \log(x+3)\) Log of a sum ≠ sum of logs \(\log(x) + \log(3) = \log(3x)\)
Not checking for extraneous solutions Squaring can introduce fake answers ALWAYS plug back into original equation!
\(\log(-4)\) is acceptable Logarithm argument must be positive Reject any solution where argument ≤ 0
✏️ Worked Examples
Example 1: Radical Equation — \(\sqrt{2x+7} - x = 2\)
Isolate: \(\sqrt{2x+7} = x + 2\)
Square: \(2x + 7 = (x+2)^2 = x^2 + 4x + 4\)
Rearrange: \(0 = x^2 + 2x - 3\)
Factor: \(0 = (x+3)(x-1)\)
Solutions: \(x = -3\) or \(x = 1\)
✓ Check \(x=1\): \(\sqrt{9} - 1 = 2\) ✓
✗ Check \(x=-3\): \(\sqrt{1} + 3 = 4 \neq 2\)
Answer: \(x = 1\)
Example 2: Exponential Equation — \(3^{2x-1} = 15\)
Take ln: \(\ln(3^{2x-1}) = \ln(15)\)
Power Rule: \((2x-1)\ln(3) = \ln(15)\)
Solve: \(2x - 1 = \frac{\ln(15)}{\ln(3)}\)
Calculate: \(2x - 1 \approx 2.465\)
Answer: \(x \approx 1.732\)
Example 3: Logarithmic Equation — \(\log_2(x) + \log_2(x-2) = 3\)
Combine: \(\log_2[x(x-2)] = 3\)
Exponential form: \(x(x-2) = 2^3 = 8\)
Expand: \(x^2 - 2x - 8 = 0\)
Factor: \((x-4)(x+2) = 0\)
Solutions: \(x = 4\) or \(x = -2\)
✗ \(x = -2\): \(\log_2(-2)\) undefined!
Answer: \(x = 4\)
🚀 Calculus Preview — Why This Matters
📐 Finding Critical Points
\(f'(x) = 0\)

Setting derivatives equal to zero often produces radical, exponential, or logarithmic equations. These skills find maximums and minimums!

📈 Exponential Growth/Decay
\(A = Pe^{rt}\)

Solving for time (t) requires logarithms. Population growth, radioactive decay, compound interest — all use these techniques!

🔄 Implicit Differentiation
\(\frac{d}{dx}[\ln y] = \frac{y'}{y}\)

Logarithmic differentiation simplifies complex products. Your log skills will be essential!

∫ Integration Techniques
\(\int \frac{1}{x}dx = \ln|x| + C\)

The natural log appears constantly in integration. Understanding log properties makes integration much easier!

💡 The #1 Rule: Always check your answers in the ORIGINAL equation! Squaring and logarithms can create extraneous solutions.

Reference Angle Strategy — Step by Step
Guided: Solve \(\sin\theta = \tfrac{1}{2}\) on \([0,2\pi)\)
Step 1:Reference angle: \(\sin^{-1}\!\left(\tfrac{1}{2}\right) = \frac{\pi}{6}\)
Step 2:Sign: \(\frac{1}{2}>0\) → sine positive → Q I & Q II (S in ASTC)
Step 3:Q I: \(\theta = \frac{\pi}{6}\)
Step 4:Q II: \(\theta = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\)
Answer:\(\theta = \frac{\pi}{6}\) and \(\theta = \frac{5\pi}{6}\)

★ The Three Common Traps — These Lose Points on Every Exam
🚨 Trap 1 — The Division Trap   Never divide both sides by a trig function
✗ Wrong \(\sin x\cos x = \sin x\)
Divide by \(\sin x\):
\(\cos x = 1 \Rightarrow x = 0\) only
✓ Correct — Move & Factor \(\sin x\cos x - \sin x = 0\)
\(\sin x(\cos x - 1) = 0\)
\(\sin x = 0 \Rightarrow x = 0, \pi\)
\(\cos x = 1 \Rightarrow x = 0\)
Solutions: \(x = 0\) and \(x = \pi\)
Rule: never divide by a trig function — always move to one side and factor. Dividing by \(\sin x\) silently assumes \(\sin x \neq 0\), throwing away every solution where \(\sin x = 0\).
🔢 Trap 2 — Forgetting Solutions   Most equations have TWO solutions in \([0,2\pi)\)
✗ Incomplete \(\cos x = \frac{1}{2}\)
Reference angle \(\frac{\pi}{3}\), cosine positive in Q I:
\(x = \frac{\pi}{3}\) — missing \(\frac{5\pi}{3}\)
✓ Both Solutions Cosine positive in Q I and Q IV
Q I: \(x = \frac{\pi}{3}\)
Q IV: \(x = 2\pi - \frac{\pi}{3} = \frac{5\pi}{3}\)
Both answers required.
Habit check: a single solution to a trig equation should feel suspicious. Ask: "Are there other quadrants with the right sign?"
🔄 Trap 3 — The Multiple Angle Trap   Expand the interval when the argument is \(nx\)
✗ Wrong Interval Solve \(\sin(2x) = \frac{\sqrt{3}}{2}\) on \([0,2\pi)\)
Treat as \(\sin(\theta) = \frac{\sqrt{3}}{2}\):
\(2x = \frac{\pi}{3}\) and \(2x = \frac{2\pi}{3}\)
\(x = \frac{\pi}{6}\) and \(x = \frac{\pi}{3}\) — only 2 of 4
✓ Substitute & Expand Let \(u = 2x\). Since \(x \in [0,2\pi)\),  \(u \in [0,4\pi)\)
Solve \(\sin u = \frac{\sqrt{3}}{2}\) on \([0,4\pi)\):
\(u = \frac{\pi}{3},\frac{2\pi}{3},\frac{\pi}{3}+2\pi,\frac{2\pi}{3}+2\pi\)
Divide by 2: \(x = \frac{\pi}{6}, \frac{\pi}{3}, \frac{7\pi}{6}, \frac{4\pi}{3}\)

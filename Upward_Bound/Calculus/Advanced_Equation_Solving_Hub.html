📚 Prerequisite Bridge: The "Inverse" Foundation

The Golden Rule of Equation Solving

To solve an equation, undo operations using their inverses!

Every equation type has a specific "undoing" strategy.

🔄 Inverse Operation Pairs

Square ↔ Square Root

\(x^2\)
⟷
\(\sqrt{x}\)

Cube ↔ Cube Root

\(x^3\)
⟷
\(\sqrt[3]{x}\)

Exponential ↔ Logarithm

\(e^x\)
⟷
\(\ln(x)\)

Base-10 Exp ↔ Common Log

\(10^x\)
⟷
\(\log(x)\)

Absolute Value → Two Cases

\(|x| = c\)
⟷
\(x = c\) or \(x = -c\)

🚨 Domain Restrictions: The "Danger Zones"

Before solving, know what values are IMPOSSIBLE!

√ Even Roots

\(\sqrt{x}\) requires \(x \geq 0\)

Output is always ≥ 0!

log Logarithms

\(\ln(x)\) requires \(x > 0\)

Can't take log of 0 or negative!

eˣ Exponentials

\(e^x > 0\) always

Output is NEVER zero or negative!

🔍 Radical & Absolute Value Discovery Lab

📐 Absolute Value: The "Two-Path" Strategy

Absolute value equations split into TWO cases!

\(|ax + b| = c\) means the expression is \(c\) units from zero

This happens in TWO places on the number line!

Example: Solve \(|2x - 3| = 7\)

Path 1: Positive Case

\(2x - 3 = 7\)

\(2x = 10\)

\(x = 5\)

OR
Path 2: Negative Case

\(2x - 3 = -7\)

\(2x = -4\)

\(x = -2\)

Solutions: \(x = 5\) and \(x = -2\)

⚠️ Special Cases

\(|x| = 0\) → One solution: \(x = 0\)

\(|x| = -5\) → NO solution! (absolute value can't be negative)

√ Radical Equations: Even vs. Odd Index

Even Index (√, ⁴√, etc.)

Output is ALWAYS ≥ 0

\(\sqrt{x} = 4\) → \(x = 16\) ✓

\(\sqrt{x} = -4\) → NO SOLUTION!

Even roots can't equal negatives!

Odd Index (³√, ⁵√, etc.)

Output can be ANY real number

\(\sqrt[3]{x} = 2\) → \(x = 8\) ✓

\(\sqrt[3]{x} = -2\) → \(x = -8\) ✓

Odd roots CAN be negative!

🎬 The "Square Both Sides" Animation

Squaring can introduce "ghost" solutions — always CHECK!

Solve:

\(\sqrt{x+3} = x - 3\)

1
Isolate the radical

Radical is already isolated: \(\sqrt{x+3} = x - 3\)

2
Square both sides

\(x + 3 = (x-3)^2 = x^2 - 6x + 9\)

3
Solve the resulting equation

\(x^2 - 7x + 6 = 0\) → \((x-1)(x-6) = 0\) → \(x = 1\) or \(x = 6\)

4
CHECK each solution!

\(x=1\): \(\sqrt{4} = -2\)? NO! 🔥 | \(x=6\): \(\sqrt{9} = 3\)? YES! ✓

5
Final Answer

\(x = 6\) only (x = 1 is extraneous)

🧮 Exponential & Logarithmic Workshop

The Fundamental Relationship

\(b^x = y \Longleftrightarrow \log_b(y) = x\)

Exponentials and logarithms are INVERSES of each other!

🔄 The "Base Switcher" — Converting Forms

Exponential → Log

\(2^5 = 32\)

becomes

\(\log_2(32) = 5\)

Log → Exponential

\(\log_3(81) = 4\)

becomes

\(3^4 = 81\)

📋 Log Properties Toolkit

Use these to simplify before solving!

Product Rule

\(\log(xy) = \log(x) + \log(y)\)

Example: \(\log(2) + \log(5) = \log(10) = 1\)

Quotient Rule

\(\log\left(\frac{x}{y}\right) = \log(x) - \log(y)\)

Example: \(\log(100) - \log(10) = \log(10) = 1\)

Power Rule

\(\log(x^n) = n \cdot \log(x)\)

Example: \(\log(10^3) = 3\log(10) = 3\)

Change of Base

\(\log_b(x) = \frac{\ln(x)}{\ln(b)}\)

Useful for calculator evaluation!

🚫 Common Log Mistakes

❌ WRONG

\(\log(a + b) \neq \log(a) + \log(b)\)

No "sum rule" for logs!

✅ RIGHT

\(\log(a \cdot b) = \log(a) + \log(b)\)

Product rule only!

🎬 Solving Exponential Equations

Solve:

\(e^{2x} = 15\)

1
Take ln of both sides

\(\ln(e^{2x}) = \ln(15)\)

2
Simplify using \(\ln(e^a) = a\)

\(2x = \ln(15)\)

3
Solve for x

\(x = \frac{\ln(15)}{2} \approx 1.354\)

🎬 Solving Logarithmic Equations

Solve:

\(\log(x) + \log(x-3) = 1\)

1
Combine logs using Product Rule

\(\log(x(x-3)) = 1\)

2
Convert to exponential form

\(x(x-3) = 10^1 = 10\)

3
Solve the quadratic

\(x^2 - 3x - 10 = 0\) → \((x-5)(x+2) = 0\) → \(x = 5\) or \(x = -2\)

4
CHECK domain restrictions!

\(x = -2\): \(\log(-2)\) is undefined! 🔥 | \(x = 5\): Both logs defined ✓

5
Final Answer

\(x = 5\) only

🔗 Calculus Connection

📈 Derivatives

The derivative of \(e^x\) is itself! And \(\frac{d}{dx}[\ln(x)] = \frac{1}{x}\)

\(\frac{d}{dx}[e^x] = e^x\)

∫ Integration

Exponential and log functions are essential for integration!

\(\int \frac{1}{x}dx = \ln|x| + C\)

📊 Growth/Decay

Differential equations like \(\frac{dy}{dt} = ky\) have exponential solutions!

🔥 The Extraneous Solution Detective

The Most Critical Skill!

Some algebraic operations can introduce "ghost" solutions that don't actually work!

You MUST check every solution in the ORIGINAL equation!

👻 Why Do "Ghost" Solutions Appear?

Squaring Both Sides

Squaring can make a negative equal a positive

\((-3)^2 = 9 = 3^2\)

Log Domain Violations

Solutions may make log arguments ≤ 0

\(\log(-2)\) is undefined!

Absolute Value Cases

One "path" may produce invalid solutions

Check both cases!

🔥 The "Solution Burner" — Interactive Checker

Original Equation:

\(\sqrt{x+2} = x\)

Candidate solutions: \(x = 2\) and \(x = -1\)

x = 2

Check: \(\sqrt{2+2} = \sqrt{4} = 2\)

Does \(2 = 2\)? YES!

⏳
x = -1

Check: \(\sqrt{-1+2} = \sqrt{1} = 1\)

Does \(1 = -1\)? NO!

⏳

✅ Final Answer

\(x = 2\) only

✅ Verification Checklist

  • ☐ Did you isolate the radical/log/exponential first?
  • ☐ Did you apply the inverse operation correctly?
  • ☐ Did you solve the resulting equation?
  • ☐ Did you check EVERY solution in the ORIGINAL equation?
  • ☐ Did you verify domain restrictions (no negative under even roots, no non-positive in logs)?
  • ☐ Did you reject all extraneous solutions?

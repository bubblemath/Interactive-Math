📚 Prerequisite Bridge: Understanding Limits

The Big Idea

A limit describes what value a function approaches — not necessarily what it equals!

\(\lim_{x \to a} f(x) = L\)

"As x gets closer and closer to a, f(x) gets closer and closer to L"

🔤 Notation Guide

Click each part to understand the limit notation:

\(\lim\)
"The limit" — we're finding what value is being approached
\(x \to a\)
"As x approaches a" — x gets infinitely close to a
\(f(x)\)
"The function output" — what we're tracking
\(= L\)
"Equals L" — the value being approached

⚠️ Critical Insight

The limit can exist even if the function is undefined at that point!

Example: \(f(x) = \frac{x^2 - 4}{x - 2}\)

\(f(2)\) = UNDEFINED (division by zero!)

But \(\lim_{x \to 2} f(x) = 4\) (the limit exists!)

🎯 Key Concept: Approaching vs. Arriving

Function Value: \(f(a)\)

The actual output AT the point

May be undefined, or different from the limit!

Limit: \(\lim_{x \to a} f(x)\)

What value we're APPROACHING

Depends on behavior NEAR the point, not AT it!

🔗 Calculus Connection

📈 Derivatives

The derivative is defined as a limit! It measures instantaneous rate of change.

\(f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)

∫ Integrals

The definite integral is also a limit — of Riemann sums as rectangles get infinitely thin.

\(\int_a^b f(x)dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i)\Delta x\)

🎯 Why Limits Matter

Limits are the foundation of ALL of calculus. Every major concept builds on this idea!

Three Ways to Find Limits

Explore limits using tables, graphs, and algebraic manipulation!

📊 Method 1: Tables (Numerical Approach)

Watch values converge as x approaches the target!

Function:

\(f(x) = \frac{x^2 - 4}{x - 2}\)

Find: \(\lim_{x \to 2} f(x)\)

From the LEFT (x → 2⁻) From the RIGHT (x → 2⁺)
x f(x) x f(x)
1.9 3.9 2.1 4.1
1.99 3.99 2.01 4.01
1.999 3.999 2.001 4.001
1.9999 3.9999 2.0001 4.0001

✅ Conclusion

Both sides approach 4, so \(\lim_{x \to 2} f(x) = 4\)

Note: \(f(2)\) is undefined, but the limit still exists!

📈 Method 2: Graphs (Visual Approach)

Use the graph to visualize limits. Try graphing: \(y = \frac{x^2-4}{x-2}\)

Try these functions:

  • \(y = \frac{x^2-4}{x-2}\) — Hole at x = 2
  • \(y = \frac{1}{x-2}\) — Vertical asymptote at x = 2
  • \(y = \frac{|x|}{x}\) — Jump discontinuity at x = 0

🧮 Method 3: Algebra (Analytical Approach)

Factor and simplify to evaluate limits!

Find:

\(\lim_{x \to 2} \frac{x^2 - 4}{x - 2}\)

1
Direct substitution gives \(\frac{0}{0}\)

This is an indeterminate form — we need to simplify!

2
Factor the numerator

\(\frac{x^2 - 4}{x - 2} = \frac{(x-2)(x+2)}{x-2}\)

3
Cancel common factors

\(\frac{(x-2)(x+2)}{x-2} = x + 2\) (when \(x \neq 2\))

4
Now substitute!

\(\lim_{x \to 2} (x + 2) = 2 + 2 = 4\)

🔗 Calculus Connection

📈 The \(\frac{0}{0}\) Form

When you get \(\frac{0}{0}\), the limit might still exist! This is called an "indeterminate form."

Factor, simplify, then substitute!

🕳️ Holes = Removable

When a factor cancels, you get a "hole" — the limit exists even though the function is undefined.

📊 L'Hôpital's Rule

In calculus, you'll learn a powerful shortcut for \(\frac{0}{0}\) forms using derivatives!

↔️ Sided Limits & Continuity

↔️ One-Sided Limits

Sometimes we need to approach from just ONE direction!

Left-Hand Limit

\(\lim_{x \to a^-} f(x)\)

Approaching from values less than a

x = 1.9, 1.99, 1.999...

⟷

Right-Hand Limit

\(\lim_{x \to a^+} f(x)\)

Approaching from values greater than a

x = 2.1, 2.01, 2.001...

The Limit Exists Rule

\(\lim_{x \to a} f(x) = L\) exists if and only if:

\(\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L\)

Both sides must approach the SAME value!

🚶 The "Traveler" Visualizer

Watch a point approach from each side!

← From Left Target: x = a From Right →

✅ Continuity: The 3-Condition Checklist

A function is continuous at x = a if ALL THREE conditions are met:

Condition 1: \(f(a)\) exists

Is the function defined at this point?

Condition 2: \(\lim_{x \to a} f(x)\) exists

Do both sides approach the same value?

Condition 3: \(\lim_{x \to a} f(x) = f(a)\)

Does the limit equal the function value?

🕳️ vs 📊 Hole vs. Asymptote Visualizer

🕳️ Hole (Removable)

Factor cancels completely

\(f(x) = \frac{x^2-4}{x-2}\)

Limit EXISTS at x = 2

Condition 1 or 3 fails

📊 Vertical Asymptote

Factor remains in denominator

\(f(x) = \frac{1}{x-2}\)

Limit is ±∞ at x = 2

Condition 2 fails (limit DNE)

Try graphing both functions to see the difference!

🔗 Calculus Connection

📈 Differentiability

A function must be continuous to be differentiable! Discontinuities mean no derivative exists there.

∫ Integrability

Continuous functions on closed intervals are always integrable — a key theorem in calculus!

🎯 IVT & MVT

The Intermediate Value Theorem and Mean Value Theorem both require continuity!

📈 Derivative Discovery: The Limit Definition

The Derivative IS a Limit!

The derivative measures instantaneous rate of change using limits.

\(f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)

📐 Secant → Tangent Animation

Watch the secant line become the tangent line as h → 0!

Function: \(f(x) = x^2\) at \(x = 1\)

h = 1.00
Secant Line Slope

\(\frac{f(x+h) - f(x)}{h}\)

3.00
Tangent Line Slope (Derivative)

\(f'(1) = 2(1) = 2\)

2.00

As h gets smaller, the secant slope approaches the tangent slope!

🧮 Finding a Derivative Using the Limit Definition

Find \(f'(x)\) for:

\(f(x) = x^2\)

1
Write the difference quotient

\(f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}\)

2
Expand the numerator

\(= \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}\)

3
Simplify

\(= \lim_{h \to 0} \frac{2xh + h^2}{h} = \lim_{h \to 0} \frac{h(2x + h)}{h}\)

4
Cancel h and evaluate the limit

\(= \lim_{h \to 0} (2x + h) = 2x\)

5
Final Answer

\(f'(x) = 2x\)

🔗 Calculus Connection

📈 Shortcut Rules

In calculus, you'll learn shortcuts (power rule, product rule, etc.) so you don't always need the limit definition!

Power Rule: \(\frac{d}{dx}[x^n] = nx^{n-1}\)

🎯 Applications

Derivatives give us velocity, acceleration, optimization, and so much more!

∫ Fundamental Theorem

The Fundamental Theorem of Calculus connects derivatives and integrals — both defined using limits!

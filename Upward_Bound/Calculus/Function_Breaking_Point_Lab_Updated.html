The Three "Breaking Points" of Functions

Every function has its limits — literally! Before you can master calculus, you need to understand where and why functions break down. Today, you become a Function Detective.

🕳️

The Black Hole

The Crime: Division by Zero

\( f(x) = \frac{1}{x - 3} \)

When \( x = 3 \), the denominator becomes zero, creating a vertical asymptote — a "black hole" the function can never cross.

Calculus Connection: Infinite limits, vertical asymptotes

🚧

The Imaginary Wall

The Crime: Square Root of Negatives

\( g(x) = \sqrt{x + 4} \)

When \( x < -4 \), we'd need the square root of a negative — impossible in the real numbers!

Calculus Connection: Domain restrictions on derivatives

⚠️

The Logarithmic Void

The Crime: Log of Zero or Negatives

\( h(x) = \ln(x) \)

Logarithms only accept positive inputs. At \( x = 0 \) and below, the function ceases to exist.

Calculus Connection: Limits approaching boundaries

📝 Important: Interval Notation vs. Inequality Notation

You'll need to express domains in both formats. Here's a quick reference:

Description Inequality Notation Interval Notation
All \( x \) greater than 2 \( x > 2 \) \( (2, \infty) \)
All \( x \) less than or equal to 5 \( x \leq 5 \) \( (-\infty, 5] \)
All \( x \) between -3 and 3 (inclusive) \( -3 \leq x \leq 3 \) \( [-3, 3] \)
All \( x \) except \( x = 2 \) \( x \neq 2 \) \( (-\infty, 2) \cup (2, \infty) \)
All \( x \) between 1 and 4, excluding both \( 1 < x < 4 \) \( (1, 4) \)
Key Symbols:
• Parentheses \( ( \) or \( ) \) = endpoint NOT included (open)
• Brackets \( [ \) or \( ] \) = endpoint IS included (closed)
• \( \cup \) = "union" (combines separate intervals)
• \( \infty \) always uses parentheses (you can't "reach" infinity)

🎯 Today's Learning Goals

  • Identify all three types of "breaking points" in any function
  • Test values in a calculator to discover domain restrictions BEFORE graphing
  • Distinguish between holes (removable) and asymptotes (non-removable)
  • Write domains in both interval AND inequality notation
  • Connect these concepts to limits in calculus

Rational Ruptures: Holes vs. Asymptotes

Focus: What's the difference between a hole and an asymptote?

\( g(x) = \frac{(x-2)(x+3)}{(x-2)(x-4)} \)
📝 Instructions:
  1. Type the function: ((x-2)(x+3))/((x-2)(x-4))
  2. Zoom in near \( x = 2 \) — What do you see? (Hint: look for a "hollow dot")
  3. Zoom in near \( x = 4 \) — What do you see? (Hint: the graph goes to infinity)
  4. The factor \( (x-2) \) cancels — this creates a HOLE
  5. The factor \( (x-4) \) doesn't cancel — this creates an ASYMPTOTE

Record on your handout: Location of hole: _____ | Location of asymptote: _____ | Simplified function: _____

B

Radical Restrictions: Even vs. Odd Roots

Focus: Why do some roots "break" and others don't?

\( \text{Compare: } \sqrt{x} \text{ vs. } \sqrt[3]{x} \)
📝 Instructions:
  1. Type y = sqrt(x) (square root — even root)
  2. Type y = x^(1/3) (cube root — odd root)
  3. Compare: Which one exists for negative x-values?
  4. Try y = x^(1/4) (4th root) and y = x^(1/5) (5th root)

Record on your handout: Even roots require x ≥ _____ | Odd roots work for _____ x-values

C

Logarithmic Limits: The Unreachable Horizon

Focus: What happens as x approaches 0?

\( y = \ln(x) \text{ as } x \to 0^+ \)

📖 How to Create a Slider in Desmos

  1. Type a = 1 in a new line — Desmos will automatically create a slider!
  2. Type y = ln(a) to see the output value
  3. Click on the slider and change the range: set min to 0.001 and max to 10
  4. Slowly drag the slider toward 0 and watch what happens to y
  5. Also type (a, ln(a)) to plot the point on the graph of y = ln(x)
📝 Record these values as you slide:
  • When \( a = 1 \): \( \ln(a) = \) _____
  • When \( a = 0.1 \): \( \ln(a) = \) _____
  • When \( a = 0.01 \): \( \ln(a) = \) _____
  • When \( a = 0.001 \): \( \ln(a) = \) _____

This IS a limit! Write it: \( \displaystyle\lim_{x \to 0^+} \ln(x) = \) _____

D

Piecewise Surgery: Healing the Break

Focus: How do we make a piecewise function continuous?

\( f(x) = \begin{cases} x + k & \text{if } x < 2 \\ x^2 & \text{if } x \geq 2 \end{cases} \)

📖 How to Create a Piecewise Function in Desmos

  1. First, create a slider: Type k = 0
  2. For the first piece, type: y = x + k {x < 2}
    (The curly braces create a domain restriction!)
  3. For the second piece, type: y = x^2 {x >= 2}
  4. Now adjust the slider k until the two pieces connect at \( x = 2 \)
  5. Hint: What is \( f(2) \) using the second piece? The first piece must equal this!
📝 Solve algebraically:
  • Using the second piece: \( f(2) = 2^2 = \) _____
  • For continuity, the first piece at \( x = 2 \): \( 2 + k = \) _____
  • Solving for k: \( k = \) _____

Calculus Connection: This is exactly what limits formalize! We need \( \displaystyle\lim_{x \to 2^-} f(x) = \lim_{x \to 2^+} f(x) = f(2) \)

💼 When Functions Break in Real Life

Understanding function behavior isn't just math class — it's critical in many careers!

🏗️ Structural Engineer

The Break: Resonance frequency (division by zero in vibration models)

Real Consequence: The Tacoma Narrows Bridge collapsed in 1940 when wind matched its resonance frequency!

\( A(f) = \frac{1}{f^2 - f_0^2} \)

💻 Software Developer

The Break: Integer overflow, division by zero errors

Real Consequence: The Y2K bug, Ariane 5 rocket explosion, countless system crashes

💊 Pharmacologist

The Break: Drug concentration approaching zero

Real Consequence: Dosing calculations depend on exponential decay models with logarithmic analysis

\( C(t) = C_0 \cdot e^{-kt} \)

📈 Economist

The Break: Supply/demand curves with asymptotic behavior

Real Consequence: Market predictions, price elasticity, understanding economic limits

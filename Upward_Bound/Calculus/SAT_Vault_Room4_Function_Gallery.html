📋 Quick Review — Functions & Circles

Study these concepts before exploring the exhibits.

📝 Function Notation

\(f(x)\) means "the output when the input is \(x\)"
  • \(f(3)\) → substitute 3 for every \(x\)
  • \(f(a+1)\) → substitute \((a+1)\) for every \(x\)
⚠️ SAT Trap

\(f(x) = x^2 + 1\). Find \(f(x+2)\).
WRONG: \(x^2 + 1 + 2 = x^2 + 3\)
RIGHT: \((x+2)^2 + 1 = x^2 + 4x + 5\)

🔗 Function Composition

\(f(g(x))\) = "plug \(g(x)\) into \(f\)"
Work inside-out
  1. Evaluate the inner function first: find \(g(x)\)
  2. Use that result as the input for \(f\)
💡 Order Matters!

\(f(g(x)) \neq g(f(x))\) in general. Always check which function is "inside."

🚫 Domain & Range

Domain: All valid inputs (x-values)
Range: All possible outputs (y-values)

Common restrictions:

  • Fractions: denominator ≠ 0
  • Square roots: radicand ≥ 0
  • Logarithms: argument > 0

🔄 Transformations

\(y = a \cdot f(b(x - h)) + k\)
  • \(h\) → horizontal shift (right if +, left if −)
  • \(k\) → vertical shift (up if +, down if −)
  • \(a\) → vertical stretch/compress; negative = reflect over x-axis
  • \(b\) → horizontal stretch/compress; negative = reflect over y-axis
⚠️ The Horizontal Trap

Horizontal changes are OPPOSITE what you'd expect!
\(f(x-3)\) shifts RIGHT 3 (not left).
\(f(2x)\) compresses by ½ (not stretches by 2).

⭕ Circle Equations

\((x-h)^2 + (y-k)^2 = r^2\)
Center: \((h, k)\), Radius: \(r\)

Completing the Square:

  1. Group x-terms and y-terms
  2. Take half of the linear coefficient, square it
  3. Add to both sides
  4. Factor each perfect square trinomial
💡 SAT Shortcut

If the equation is \(x^2 + y^2 + Dx + Ey + F = 0\):
Center = \((-D/2, -E/2)\), Radius = \(\sqrt{(D/2)^2 + (E/2)^2 - F}\)

📐 Arc Length & Sector Area

Arc Length: \(s = r\theta\) (θ in radians)
Sector Area: \(A = \frac{1}{2}r^2\theta\) (θ in radians)
Arc Length: \(s = \frac{\theta}{360} \cdot 2\pi r\) (θ in degrees)
Sector Area: \(A = \frac{\theta}{360} \cdot \pi r^2\) (θ in degrees)
⚠️ Radian vs Degree

Check which formula matches your angle unit! The SAT may give angles in either form.

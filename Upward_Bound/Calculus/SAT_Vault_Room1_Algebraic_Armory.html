📋 Quick Review — Heart of Algebra

Study these key concepts before entering the forges.

📈 Slope-Intercept Form

\(y = mx + b\)
  • m = slope = rate of change = rise/run
  • b = y-intercept = starting value (when x = 0)
💡 SAT Tip

In word problems, slope = "per unit" rate. Y-intercept = initial/starting amount.

📐 Finding Slope

\(m = \frac{y_2 - y_1}{x_2 - x_1}\)
  • Positive slope → line goes up ↗
  • Negative slope → line goes down ↘
  • Zero slope → horizontal line —
  • Undefined slope → vertical line |

🔗 Parallel & Perpendicular

  • Parallel lines: same slope (\(m_1 = m_2\))
  • Perpendicular lines: negative reciprocal slopes (\(m_1 \cdot m_2 = -1\))
⚠️ Common SAT Trap

Perpendicular ≠ just "opposite." It's the negative reciprocal. If m = 2, the perpendicular slope is −1/2, NOT −2.

⚙️ Systems of Equations

Substitution: Solve one equation for a variable, plug into the other.

Elimination: Add/subtract equations to cancel a variable.

Number of Solutions:
• Different slopes → 1 solution (lines cross)
• Same slope, different intercept → 0 solutions (parallel)
• Same slope, same intercept → ∞ solutions (same line)

🎯 SAT Strategy: Word Problems

  1. Identify the variables (what do x and y represent?)
  2. Find the rate of change → that's your slope
  3. Find the starting value → that's your y-intercept
  4. Write the equation: \(y = mx + b\)
💡 Key Phrases

"per hour" → slope | "initial fee" → y-intercept | "total cost" → y

🧮 SAT Reference Formulas

Standard Form: \(Ax + By = C\)
Point-Slope: \(y - y_1 = m(x - x_1)\)
Slope: \(m = \frac{y_2 - y_1}{x_2 - x_1}\)
💡 Converting Forms

SAT often gives standard form. Solve for y to get slope-intercept!

Page 1: Key Definitions & The Shrinking Interval Method

🤔 The Big Question That Invented Calculus

"How do you measure speed at a single instant if speed requires distance AND time?"

📖 Key Definitions — Average vs. Instantaneous

Average Rate of Change (AROC)

AROC = (d₂ − d₁) / (t₂ − t₁)
Speed over an interval of time
= Slope of the SECANT line

Instantaneous Rate of Change (IROC)

IROC = lim(Δt→0) [Δd / Δt]
Speed at a single moment
= Slope of the TANGENT line
📐 Slope Foundation:
Slope = Rise / Run = Δy / Δx = (y₂ − y₁) / (x₂ − x₁)
🔢 The Shrinking Interval Method — Step by Step
  1. Identify the Target: Choose the specific time (t) where you want the instantaneous speed.
  2. Calculate AROC for Large Interval: Find average speed from your target to a point far away (e.g., Δt = 1 sec).
  3. Shrink the Interval: Repeat with progressively smaller intervals (Δt = 0.5, 0.1, 0.01, 0.001...).
  4. Organize in a Table: Record each Δt and its corresponding average velocity.
  5. Observe the Pattern: Watch the values converge toward a specific number.
  6. State the Limit: The number the averages approach IS the instantaneous rate!
💡 Key Insight: We never actually let Δt = 0 (that's undefined!). We observe what happens as Δt approaches zero.
⚠️ Common Mistakes & Corrections
❌ Common Error ✅ Correction
Using only ONE point to calculate rate of change. AROC requires TWO points. Slope = (y₂−y₁)/(x₂−x₁). One point gives position, not rate!
Confusing "Position" with "Speed". Position = where the marble IS. Speed = how fast position is CHANGING. They're related but different!
Plugging in Δt = 0 and getting "undefined". We find the LIMIT as Δt → 0, not the value AT zero. Look at the pattern of values approaching!
Stopping at one or two intervals. Use at least 4-5 shrinking intervals to clearly see the convergence pattern.

📝 Worked Example: The Rolling Marble
Scenario: A marble rolls down a ramp following the position function d(t) = 2t² (distance in cm, time in seconds). Find the instantaneous velocity at t = 3 seconds.

Step 1: Note that d(3) = 2(3)² = 18 cm. Now calculate AROC for shrinking intervals starting at t = 3:

Interval [3, t₂] t₂ d(t₂) = 2t₂² Δt Δd = d(t₂) − 18 AROC = Δd/Δt
[3, 4] 4 32 1.0 14 14.0
[3, 3.5] 3.5 24.5 0.5 6.5 13.0
[3, 3.1] 3.1 19.22 0.1 1.22 12.2
[3, 3.01] 3.01 18.1202 0.01 0.1202 12.02
[3, 3.001] 3.001 18.012002 0.001 0.012002 12.002

Step 2: Observe the pattern: 14 → 13 → 12.2 → 12.02 → 12.002 → ... The values are converging toward 12!

Result: The instantaneous velocity at t = 3 seconds is 12 cm/sec
This is the slope of the tangent line to d(t) = 2t² at the point (3, 18).
✅ Quick Reference Checklist
📹 Data Collection (Physical Lab)
  • Camera set to high FPS (240 fps ideal)
  • Ramp angle is fixed and measured
  • Scale/ruler visible in frame
  • Start time (t = 0) clearly defined
  • Multiple trials for accuracy
🧮 Calculation Verification
  • Units are consistent (cm/s or m/s)
  • At least 4-5 shrinking intervals used
  • Intervals shrink by factor of 10
  • Pattern clearly shows convergence
  • Final answer stated as a limit
📊 Graph Analysis
  • Position vs. time curve plotted
  • Secant lines drawn for each interval
  • Secant lines approach tangent line
  • Tangent line touches at ONE point
🎯 Conceptual Understanding
  • Can explain AROC vs. IROC difference
  • Can explain why Δt ≠ 0 but → 0
  • Can identify secant vs. tangent lines
  • Can connect to "derivative" concept
🚀 Calculus Preview — The Derivative Shortcut
The Formal Definition
f'(x) = lim(h→0) [f(x+h) − f(x)] / h

This is exactly what we did with the shrinking interval method — but written in formal notation!

The Power Rule Shortcut
If f(x) = axⁿ, then f'(x) = n·axⁿ⁻¹

In calculus, you'll learn rules that skip the table entirely!

🔗 Connecting Our Example to the Power Rule:

Our function: d(t) = 2t²
Using Power Rule: d'(t) = 2 · 2t¹ = 4t
At t = 3: d'(3) = 4(3) = 12 cm/sec ✓
The limit we found by shrinking intervals matches the derivative!

💡 The Big Takeaway: The derivative IS the instantaneous rate of change. The "shrinking interval" method you learned today is exactly how calculus defines the derivative — you just discovered it experimentally!

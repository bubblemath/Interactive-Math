Page 1: The Core Rules — Power, Constant, Sum & Difference

🎯 The Big Idea: Shortcuts from the Limit Definition

f'(x) = limh→0 [f(x+h) − f(x)] / h

The derivative rules are SHORTCUTS — they give the same answer as the limit definition, but much faster!

📖 The Four Core Derivative Rules

⚡ Power Rule

d/dx [xn] = n · xn−1

Example: d/dx [x⁵] = 5x⁴

"Bring down the exponent, reduce by 1"

🔒 Constant Rule

d/dx [c] = 0

Example: d/dx [7] = 0

"Constants don't change — derivative is zero"

➕ Sum Rule

d/dx [f + g] = f' + g'

Example: d/dx [x² + x³] = 2x + 3x²

"Derivative of a sum = sum of derivatives"

➖ Difference Rule

d/dx [f − g] = f' − g'

Example: d/dx [x⁴ − x²] = 4x³ − 2x

"Derivative of a difference = difference of derivatives"

🎢
Visual Mnemonic: "Slide Down the Exponent"

Picture the exponent sliding down to become the coefficient, then the exponent shrinks by 1. For x⁵: the 5 slides down → 5x, then the exponent becomes 4 → 5x⁴

🔢 The Constant Multiple Rule
d/dx [c · f(x)] = c · f'(x)

"Constants can be pulled out — they're just along for the ride!"

Example:

d/dx [3x⁴] = 3 · d/dx [x⁴] = 3 · 4x³ = 12x³

🔢 Differentiating Polynomials — Step by Step
  1. Rewrite if needed: Convert roots to fractional exponents, move terms from denominator using negative exponents. Example: √x = x1/2, 1/x² = x−2
  2. Apply Sum/Difference Rule: Differentiate each term separately.
  3. Apply Constant Multiple Rule: Pull out any coefficients.
  4. Apply Power Rule: For each xn term, bring down n and reduce exponent by 1.
  5. Simplify: Combine like terms, convert back to radical form if required.
💡 Pro Tip: Always rewrite BEFORE differentiating! It's much easier to apply the Power Rule to x−2 than to 1/x².
⚠️ Common Mistakes & Corrections
🚨 Forgetting to Reduce the Exponent
❌ d/dx [x⁵] = 5x⁵
✓ d/dx [x⁵] = 5x⁴ (exponent goes from 5 to 4)
🚨 Derivative of x is NOT 0
❌ d/dx [x] = 0
✓ d/dx [x] = d/dx [x¹] = 1·x⁰ = 1
🚨 Not Rewriting Before Differentiating
❌ d/dx [1/x³] = "stuck"
✓ Rewrite as x−3, then: −3x−4 = −3/x⁴
🚨 Confusing Constant vs. Variable
❌ d/dx [π²] = 2π (treating π as variable)
✓ d/dx [π²] = 0 (π² is a constant!)

📝 Worked Example: Full Process
Problem: Find the derivative of f(x) = 4x³ − 2√x + 5/x² − 7
Step 1 — Rewrite: f(x) = 4x³ − 2x1/2 + 5x−2 − 7
Step 2 — Apply rules term by term:
d/dx [4x³] = 4 · 3x² = 12x²
d/dx [−2x1/2] = −2 · (1/2)x−1/2 = −x−1/2
d/dx [5x−2] = 5 · (−2)x−3 = −10x−3
d/dx [−7] = 0
Step 3 — Combine: f'(x) = 12x² − x−1/2 − 10x−3
✓ Final Answer: f'(x) = 12x² − 1/√x − 10/x³

✅ Quick Reference Checklist
📝 Before Differentiating
  • Did I rewrite roots as fractional exponents?
  • Did I rewrite fractions using negative exponents?
  • Did I expand any products if needed?
  • Is each term in the form cxn?
⚡ During Differentiation
  • Did I bring down the exponent as coefficient?
  • Did I reduce the exponent by 1?
  • Did I keep constants outside?
  • Did I get 0 for all constant terms?
✓ After Differentiating
  • Did I simplify coefficients?
  • Did I convert back to radical form if needed?
  • Did I convert negative exponents to fractions?
  • Does my answer make sense dimensionally?
📐 Trig Verification
  • Did I use the correct trig derivative?
  • Did I include the negative for co-functions?
  • Did I remember sec² and csc² (not sec and csc)?
  • Did I check for Chain Rule needs?

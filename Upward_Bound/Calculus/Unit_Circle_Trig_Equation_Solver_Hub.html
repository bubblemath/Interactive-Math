\(\sin(\theta) = y\)

\(\cos(\theta) = x\)

🎯 Reference Angle Strategy

To find ALL solutions in \([0, 2\pi)\):

  1. Find the reference angle (the acute angle to the x-axis)
  2. Determine which quadrants give the correct sign
  3. Use symmetry to find all solutions

Example: Solve \(\sin(\theta) = \frac{1}{2}\)

Reference angle: \(\frac{\pi}{6}\) (where sin = ½)

Sine is positive in Q I and Q II

Solutions: \(\theta = \frac{\pi}{6}\) and \(\theta = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\)

🧠 Key Observations

  • Sine starts at 0, reaches maximum at \(\frac{\pi}{2}\), returns to 0 at \(\pi\)
  • Cosine starts at 1, reaches 0 at \(\frac{\pi}{2}\), minimum at \(\pi\)
  • Cosine is just sine shifted left by \(\frac{\pi}{2}\): \(\cos(\theta) = \sin(\theta + \frac{\pi}{2})\)
  • Both waves have period \(2\pi\) — one complete trip around the circle

🔧 Identity Toolkit

Choose the right identity for more complex equations:

Pythagorean Identities
\(\sin^2(x) + \cos^2(x) = 1\)

Use when you see sin² or cos²

Double Angle
\(\sin(2x) = 2\sin(x)\cos(x)\)

Use when you see sin(2x) or 2sin·cos

Quotient Identity
\(\tan(x) = \frac{\sin(x)}{\cos(x)}\)

Use to convert between functions

Cofunction
\(\sin(x) = \cos(\frac{\pi}{2} - x)\)

Use to relate sin and cos

⚠️ Common Traps & Mistakes

🚨 The Division Trap

Never divide both sides by a trig function — you'll lose solutions!

❌ WRONG

\(\sin(x)\cos(x) = \sin(x)\)

Divide by sin(x):

\(\cos(x) = 1\)

Missing solutions where sin(x) = 0!

✅ RIGHT

\(\sin(x)\cos(x) = \sin(x)\)

\(\sin(x)\cos(x) - \sin(x) = 0\)

\(\sin(x)(\cos(x) - 1) = 0\)

sin(x) = 0 OR cos(x) = 1

🔢 The "Forgetting Solutions" Trap

Remember: most trig equations have multiple solutions in [0, 2π)!

Example: \(\cos(x) = \frac{1}{2}\)

Reference angle: \(\frac{\pi}{3}\)

Cosine is positive in Q I and Q IV

Don't forget: \(x = \frac{\pi}{3}\) AND \(x = 2\pi - \frac{\pi}{3} = \frac{5\pi}{3}\)

🔄 The "Multiple Angle" Trap

For equations like sin(2x) = value, you need to expand your interval!

Example: Solve \(\sin(2x) = \frac{\sqrt{3}}{2}\) on [0, 2π)

Let u = 2x. If x ∈ [0, 2π), then u ∈ [0, 4π)

Solve sin(u) = √3/2 on [0, 4π):

u = π/3, 2π/3, π/3 + 2π, 2π/3 + 2π

x = π/6, π/3, π/6 + π, π/3 + π = π/6, π/3, 7π/6, 4π/3

🎯 Spot the Error Challenge

Which solution method is CORRECT for: \(2\sin^2(x) - \sin(x) = 0\)?

Method A

\(2\sin^2(x) = \sin(x)\)

Divide by sin(x):

\(2\sin(x) = 1\)

\(\sin(x) = \frac{1}{2}\)

Method B

\(2\sin^2(x) - \sin(x) = 0\)

Factor: \(\sin(x)(2\sin(x) - 1) = 0\)

\(\sin(x) = 0\) or \(\sin(x) = \frac{1}{2}\)

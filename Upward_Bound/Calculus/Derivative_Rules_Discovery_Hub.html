🌉 Prerequisite Bridge

The Foundation: Limit Definition of the Derivative

Every derivative rule we'll discover comes from this single definition:

\(f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)

The rules are shortcuts — but understanding WHY they work makes you a stronger mathematician!

🔤 Derivative Notation Guide

You'll see derivatives written in several equivalent ways:

\(f'(x)\)
"f prime of x" — Lagrange notation
\(\frac{dy}{dx}\)
"dy dx" — Leibniz notation
\(\frac{d}{dx}[f(x)]\)
"derivative of f(x)" — operator notation
\(y'\)
"y prime" — shorthand

🔑 Special Limits (Needed for Trig Proofs)

These limits are proven using the Squeeze Theorem:

Special Limit #1
\(\lim_{h \to 0} \frac{\sin h}{h} = 1\)

This is the foundation for all trig derivatives!

Special Limit #2
\(\lim_{h \to 0} \frac{\cos h - 1}{h} = 0\)

This limit equals zero because cosine approaches 1 as h approaches 0.

📐 Algebraic Proofs: The "Why" Behind the Rules

Proof Strategy

Every proof follows the same pattern: Start with the limit definition, substitute, simplify, evaluate!

📏 Proof: The Constant Rule

Why is the derivative of a constant always zero?

The Constant Rule

\(\frac{d}{dx}[c] = 0\)

The derivative of any constant is zero.

1
Start with the limit definition
\(f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)
2
Substitute \(f(x) = c\)
\(= \lim_{h \to 0} \frac{c - c}{h}\)

Since f(x) = c for all x, f(x+h) = c too!

3
Simplify the numerator
\(= \lim_{h \to 0} \frac{0}{h} = \lim_{h \to 0} 0\)
4
Evaluate the limit
\(= 0\) ✓

⚡ Proof: The Power Rule

Discover the pattern through the limit definition!

The Power Rule

\(\frac{d}{dx}[x^n] = nx^{n-1}\)

Bring down the exponent, reduce the power by 1.

1
Start with the limit definition
\(\frac{d}{dx}[x^2] = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}\)
2
Expand the numerator
\(= \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}\)
3
Simplify (cancel \(x^2\) terms)
\(= \lim_{h \to 0} \frac{2xh + h^2}{h} = \lim_{h \to 0} \frac{h(2x + h)}{h}\)
4
Cancel h and evaluate
\(= \lim_{h \to 0} (2x + h) = 2x\)
5
Verify the pattern!
\(\frac{d}{dx}[x^2] = 2x^1 = 2x\) ✓ (Power Rule works!)

🎯 Pattern Discovery Table

Function Derivative Pattern
\(x^2\) \(2x\) 2 comes down, power → 1
\(x^3\) \(3x^2\) 3 comes down, power → 2
\(x^4\) \(4x^3\) 4 comes down, power → 3
\(x^n\) \(nx^{n-1}\) n comes down, power → n-1

➕ Proof: Sum & Difference Rules

The derivative of a sum is the sum of the derivatives!

Sum & Difference Rules

\(\frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)\)

Differentiate term by term!

1
Start with the limit definition
\(\frac{d}{dx}[f(x) + g(x)] = \lim_{h \to 0} \frac{[f(x+h) + g(x+h)] - [f(x) + g(x)]}{h}\)
2
Regroup the terms
\(= \lim_{h \to 0} \frac{[f(x+h) - f(x)] + [g(x+h) - g(x)]}{h}\)
3
Split into two limits
\(= \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} + \lim_{h \to 0} \frac{g(x+h) - g(x)}{h}\)
4
Recognize the derivatives!
\(= f'(x) + g'(x)\) ✓

📊 Geometric Proofs: See WHY the Rules Work

Visual Understanding

The derivative is the SLOPE of the tangent line. Let's see what that means for each rule!

📏 Geometric Proof: Constant Rule

Why is the derivative of a constant zero? Look at the graph!

🔍 Visual Insight

A constant function is a horizontal line. What's the slope of a horizontal line?

The Graph
Horizontal line at y = c
The Tangent
Also horizontal everywhere!
The Slope
Always 0
Therefore
\(\frac{d}{dx}[c] = 0\)

Try graphing \(y = 5\) — notice the tangent line is always horizontal (slope = 0)!

⚡ Geometric Proof: Power Rule

Watch how the slope changes as you move along \(y = x^2\)!

🔍 Visual Insight: \(f(x) = x^2\) and \(f'(x) = 2x\)

At x = -2
Slope = 2(-2) = -4
At x = 0
Slope = 2(0) = 0 (vertex!)
At x = 1
Slope = 2(1) = 2
At x = 3
Slope = 2(3) = 6

The slope increases linearly — that's why the derivative is \(2x\)!

Graph \(y = x^2\) and \(y = 2x\) together. Notice: when \(x^2\) has a minimum, \(2x = 0\)!

➕ Geometric Proof: Sum Rule

The slope of a sum is the sum of the slopes!

🔍 Visual Insight: Stacking Slopes

If \(s(x) = f(x) + g(x)\), then at any point:

  • The height of s(x) = height of f(x) + height of g(x)
  • The slope of s(x) = slope of f(x) + slope of g(x)
  • Therefore: \(s'(x) = f'(x) + g'(x)\)

Example: \(s(x) = x^2 + x\)

Function Derivative
\(f(x) = x^2\) \(f'(x) = 2x\)
\(g(x) = x\) \(g'(x) = 1\)
\(s(x) = x^2 + x\) \(s'(x) = 2x + 1\)

Graph \(y = x^2\), \(y = x\), and \(y = x^2 + x\). See how the slopes "stack"!

🎯 Key Geometric Relationships

Function ↔ Derivative Connections

f(x) has a MAX or MIN
f'(x) = 0 (horizontal tangent)
f(x) is INCREASING
f'(x) > 0 (positive slope)
f(x) is DECREASING
f'(x) < 0 (negative slope)
f(x) is CONSTANT
f'(x) = 0 everywhere

🌊 Trig Derivatives: The Beautiful Connection

The Trig Derivative Family

Sine and cosine are derivatives of each other — with a twist!

Sine → Cosine

\(\frac{d}{dx}[\sin x] = \cos x\)

💡 Sine's slope is cosine!

Cosine → Negative Sine

\(\frac{d}{dx}[\cos x] = -\sin x\)

💡 Cosine's slope is NEGATIVE sine!

Tangent

\(\frac{d}{dx}[\tan x] = \sec^2 x\)

💡 Derived using quotient rule

Cotangent

\(\frac{d}{dx}[\cot x] = -\csc^2 x\)

💡 Note the negative!

Secant

\(\frac{d}{dx}[\sec x] = \sec x \tan x\)

💡 Secant stays, tangent joins

Cosecant

\(\frac{d}{dx}[\csc x] = -\csc x \cot x\)

💡 Negative, csc stays, cot joins

🌊 Proof: Derivative of Sine

Using the limit definition and sum identity!

1
Start with the limit definition
\(\frac{d}{dx}[\sin x] = \lim_{h \to 0} \frac{\sin(x+h) - \sin x}{h}\)
2
Apply the sine sum identity
\(= \lim_{h \to 0} \frac{\sin x \cos h + \cos x \sin h - \sin x}{h}\)

Using: \(\sin(x+h) = \sin x \cos h + \cos x \sin h\)

3
Regroup terms
\(= \lim_{h \to 0} \frac{\sin x(\cos h - 1) + \cos x \sin h}{h}\)
4
Split into two limits
\(= \sin x \cdot \lim_{h \to 0} \frac{\cos h - 1}{h} + \cos x \cdot \lim_{h \to 0} \frac{\sin h}{h}\)
5
Apply the special limits!
\(= \sin x \cdot (0) + \cos x \cdot (1)\)

Using: \(\lim_{h \to 0} \frac{\cos h - 1}{h} = 0\) and \(\lim_{h \to 0} \frac{\sin h}{h} = 1\)

6
Simplify
\(= \cos x\) ✓

📊 Geometric Visualization: Why sin' = cos

🔍 Visual Insight

sin(x) at maximum (x = π/2)
cos(π/2) = 0 (horizontal tangent!)
sin(x) increasing fastest (x = 0)
cos(0) = 1 (steepest positive slope!)
sin(x) at minimum (x = 3π/2)
cos(3π/2) = 0 (horizontal tangent!)
sin(x) decreasing fastest (x = π)
cos(π) = -1 (steepest negative slope!)

Graph \(y = \sin(x)\) and \(y = \cos(x)\) together. Notice: when sin has a peak, cos = 0!

🎯 Memory Pattern: The Trig Derivative Cycle

\(\sin x \xrightarrow{d/dx} \cos x \xrightarrow{d/dx} -\sin x \xrightarrow{d/dx} -\cos x \xrightarrow{d/dx} \sin x\)

The derivatives cycle through with alternating signs!

📋 Derivative Rules Cheat Sheet (Save for Fall!)

📏 Constant Rule

\(\frac{d}{dx}[c] = 0\) Derivative of any constant is 0
\(\frac{d}{dx}[cf(x)] = c \cdot f'(x)\) Constants factor out

⚡ Power Rule

\(\frac{d}{dx}[x^n] = nx^{n-1}\) Works for ALL real n
\(\frac{d}{dx}[x] = 1\) Special case: n = 1

➕ Sum & Difference Rules

\(\frac{d}{dx}[f + g] = f' + g'\) Differentiate term by term
\(\frac{d}{dx}[f - g] = f' - g'\) Same for subtraction

🌊 Trig Derivatives

\(\frac{d}{dx}[\sin x] = \cos x\) \(\frac{d}{dx}[\cos x] = -\sin x\)
\(\frac{d}{dx}[\tan x] = \sec^2 x\) \(\frac{d}{dx}[\cot x] = -\csc^2 x\)
\(\frac{d}{dx}[\sec x] = \sec x \tan x\) \(\frac{d}{dx}[\csc x] = -\csc x \cot x\)

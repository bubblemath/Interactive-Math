💡 Memory Trick: Wall = can't enter | Hole = can jump over | Asymptote = can never reach
✅ The 3-Part Continuity Test — Is f(x) continuous at x = c?
1
Is f(c) DEFINED?

Can you plug in c and get a real number?

f(c) = some number ✓
2
Does the LIMIT exist?

Do left and right limits match?

lim(x→c⁻) = lim(x→c⁺)
3
Do they EQUAL?

Does the limit equal the function value?

lim(x→c) f(x) = f(c)
💡 All THREE must be YES for continuity! If ANY fails, you have a breaking point (discontinuity).

Page 2: The Breaking Point Solver's Toolkit — Detection & Analysis
🛠️ The 4-Step Algebraic Detection Strategy
1
FACTOR

Factor numerator AND denominator completely

2
IDENTIFY HOLES

Set CANCELED factors = 0 → These are holes

3
IDENTIFY VA's

Set REMAINING denominator factors = 0 → These are asymptotes

4
CHECK RADICALS

Set radicand ≥ 0 for even roots → Domain walls

📌 Example: Find all breaking points of f(x) = (x² − 9) / (x² − x − 6)

Factor: f(x) = (x+3)(x−3) / (x−3)(x+2)
Hole: (x−3) cancels → x = 3 is a HOLE
VA: (x+2) remains → x = −2 is a VERTICAL ASYMPTOTE
Simplified: f(x) = (x+3)/(x+2), x ≠ 3
Domain: (−∞, −2) ∪ (−2, 3) ∪ (3, ∞)
⚠️ Common Pitfalls & Traps — Don't Fall In!
🚨 The "Hidden Hole" Trap

Forgetting that a canceled factor still creates a break!

❌ f(x) = (x−2)/(x−2) = 1 for all x
✓ f(x) = 1 for x ≠ 2 (hole at x = 2!)
🚨 The "Zero vs. Undefined" Trap

Confusing when numerator = 0 vs. denominator = 0

❌ f(x) = (x−3)/(x−5): "breaks" at x = 3
✓ x = 3 gives f(3) = 0 (just a zero!)
x = 5 is the actual break (VA)
🚨 The "Piecewise Boundary" Trap

Not checking if left and right limits match at boundaries

❌ Assuming piecewise is continuous
✓ Always check: lim(x→c⁻) = lim(x→c⁺)?
🚨 The "Even Root" Trap

Forgetting domain restrictions on square roots

❌ f(x) = √(x−4) has domain (−∞, ∞)
✓ Domain: x − 4 ≥ 0 → x ≥ 4 → [4, ∞)
🎯 Finding a Hole's Location — The y-coordinate!

📌 Example: Find the exact location of the hole in f(x) = (x² − 4)/(x − 2)

Factor: f(x) = (x+2)(x−2)/(x−2)
Cancel: f(x) = x + 2, x ≠ 2
x-coordinate: x = 2 (from canceled factor)
y-coordinate: Plug x = 2 into SIMPLIFIED: y = 2 + 2 = 4
Hole Location: (2, 4)
💡 Key Insight: The y-coordinate of a hole = the LIMIT as x approaches that point. Use the simplified function!
🚀 Calculus Preview — Why Breaking Points Matter!
📐 Continuity → Differentiability

If a function is NOT continuous at x = c, it is automatically NOT differentiable there.

Discontinuous ⟹ Not Differentiable
📈 Limits at Breaking Points

Evaluating limits at holes uses the simplified function. Limits at VA's require one-sided analysis.

lim(x→c) f(x) = L (if hole exists)
⚡ Sharp Turns (Cusps)

Even continuous functions can be non-differentiable at "sharp corners" like |x| at x = 0.

Continuous ⇏ Differentiable
∫ Integration Boundaries

Improper integrals occur when integrating across vertical asymptotes — requires special techniques!

∫₀² 1/(x−1) dx = improper!
💡 The Big Picture: Differentiable ⊂ Continuous ⊂ Has Limits Defined — Each level is more restrictive!
✅ Function Detective's Quick Checklist
□ Check for RADICALS: Set radicand ≥ 0 (even roots)
□ Check for FRACTIONS: Set denominator ≠ 0
□ Check for LOGARITHMS: Set argument > 0
□ Check for PIECEWISE: Test boundary points
□ FACTOR completely: Identify holes vs. asymptotes
□ WRITE domain: Both interval AND inequality notation

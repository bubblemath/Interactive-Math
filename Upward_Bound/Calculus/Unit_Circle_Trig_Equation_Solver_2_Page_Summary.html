🔺 The "Big Three" Identity Families
Pythagorean
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ
Quotient
tan θ = sin θ / cos θ
cot θ = cos θ / sin θ
Reciprocal
csc θ = 1 / sin θ
sec θ = 1 / cos θ
cot θ = 1 / tan θ
📐 Reference Angle Guide — Finding θ'
Q I (0° – 90°)
θ' = θ
Q II (90° – 180°)
θ' = π − θ
Q III (180° – 270°)
θ' = θ − π
Q IV (270° – 360°)
θ' = 2π − θ
💡 Reference Angle: Always the acute angle to the x-axis. Same trig values, just adjust the sign!

🎯 The 4-Step Solving Strategy
1
ISOLATE

Get the trig function alone on one side

2
IDENTIFY

Find the reference angle using the unit circle

3
DETERMINE

Which quadrants? (Use ASTC for sign)

4
STATE

Write all solutions (+ 2nπ for general)

📌 Example: Solve cos(x) = 1/2 on [0, 2π)

Already isolated ✓
Reference angle: π/3 (from unit circle)
Cosine positive in Q I and Q IV
Q I: x = π/3
Q IV: x = 2π − π/3 = 5π/3
Answer: x = π/3, 5π/3
⚠️ Common Traps & Mistakes — Avoid These!
🚨 The Division Trap

Never divide by a trig function!

sin(x)cos(x) = sin(x)
÷ sin(x): cos(x) = 1
✓ Factor: sin(x)(cos(x) − 1) = 0
sin(x) = 0 OR cos(x) = 1
🔢 Forgetting Solutions

Most equations have 2+ solutions!

cos(x) = 1/2 on [0, 2π)

x = π/3 only
✓ x = π/3 AND x = 5π/3
(Q I and Q IV)
🔄 Multiple Angle Trap

Expand the interval first!

sin(2x) = √3/2 on [0, 2π)

✓ Let u = 2x
If x ∈ [0, 2π), then u ∈ [0, 4π)
Find ALL u values, then divide by 2
🔙 Inverse Trig Restrictions — Critical for Principal Values!
Function Domain (Input) Range (Output) Quadrants
arcsin(x) or sin⁻¹(x) [−1, 1] [−π/2, π/2] Q I and Q IV
arccos(x) or cos⁻¹(x) [−1, 1] [0, π] Q I and Q II
arctan(x) or tan⁻¹(x) (−∞, ∞) (−π/2, π/2) Q I and Q IV
💡 Why Restrictions? Inverse functions must pass the horizontal line test. We restrict the range to get ONE output per input!
🔄 Solving Multiple Angle Equations — Don't Divide the Angle Yet!

📌 Example: Solve sin(2x) = √3/2 on [0, 2π)

Step 1: Let u = 2x
Step 2: If x ∈ [0, 2π), then u ∈ [0, 4π)
Step 3: Solve sin(u) = √3/2 on [0, 4π)
u = π/3, 2π/3 (first period)
u = π/3 + 2π, 2π/3 + 2π (second period)
u = π/3, 2π/3, 7π/3, 8π/3
Step 4: Divide by 2: x = u/2
x = π/6, π/3, 7π/6, 4π/3
💡 Key Rule: For sin(nx) or cos(nx), multiply the interval by n to find ALL solutions, then divide back!
🚀 Calculus Preview — Why Unit Circle Fluency Matters!
Evaluating Limits
→ lim(x→0) sin(x)/x = 1 requires understanding sin behavior near 0
Trig Derivatives
→ d/dx[sin(x)] = cos(x), d/dx[cos(x)] = −sin(x)
Critical Points
→ Setting f'(x) = 0 often yields trig equations to solve
Trig Substitution
→ Integration technique using x = sin(θ), x = tan(θ), etc.
💡 Calculus Insight: The unit circle values you memorize now become the "answers" you'll need instantly when solving calculus problems. Fluency here = speed later!

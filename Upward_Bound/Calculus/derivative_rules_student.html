Section 4 (Part 1) — Trig Derivatives: The Derivative Cycle & Sine Proof
The Trig Derivative Cycle — Memorise This
\(\sin x \;\xrightarrow{d/dx}\; \cos x \;\xrightarrow{d/dx}\; -\sin x \;\xrightarrow{d/dx}\; -\cos x \;\xrightarrow{d/dx}\; \sin x\)
Every 4 derivatives you return to the start  ·  The co-functions (cos, cot, csc) always pick up a _______ sign
Sine proof outline — fill in the steps (using the special limits from page 1):
1 \(\frac{d}{dx}[\sin x] = \lim_{h\to0}\frac{\sin(x+h)-\sin x}{h}\)
2 Apply \(\sin(x+h) = \sin x\cos h + \cos x\sin h\):  \(= \lim_{h\to0}\frac{\sin x\cos h + \cos x\sin h - \sin x}{h}\)
3 Regroup: \(= \lim_{h\to0}\dfrac{\sin x(\cos h - 1) + \cos x\sin h}{h}\)
4 Split: \(= \sin x\cdot\lim_{h\to0}\dfrac{\cos h-1}{h} + \cos x\cdot\lim_{h\to0}\dfrac{\sin h}{h}\)
5 Apply special limits: \(= \sin x\cdot(\underline{\phantom{x}}) + \cos x\cdot(\underline{\phantom{x}}) = \) _________ ✓
Geometric check:  At \(x=\frac{\pi}{2}\): sine is at its ________ (slope = 0), so \(\cos\!\left(\frac{\pi}{2}\right) = \) _____ ✓  ·  At \(x=0\): sine is climbing steeply (slope = 1), so \(\cos(0) = \) _____ ✓

📊 Lesson 4: Measures of Center

Mode, Mean, Median, Quartiles, Box Plots & Ogives

📚 Key Terms & Definitions

Measures of Central Tendency

Mode

The data value that occurs most frequently. If no value repeats, there is no mode.

Mode = Most frequent value

Mean (Average)

The sum of all values divided by the number of values. The "balance point" of the data.

x̄ = Σxn (sample)    μ = ΣxN (population)

Median

The middle value when data is ordered. Measures the "center" of the data.

Middle value of sorted data

📝 Important Symbols

Σ (Sigma) = "Sum of" - add up all values

x̄ (x-bar) = Sample mean

μ (mu) = Population mean

n = Sample size    N = Population size

⚠️ Finding the Median

Odd number of values: The median is the single middle number.

Even number of values: The median is the average of the two middle numbers.

Mode Classifications

Unimodal

Data has one mode - a single value with the highest frequency.

Bimodal

Data has two modes - two values tied for highest frequency.

Multimodal

Data has three or more modes - multiple values tied for highest frequency.

No Mode

No value repeats - every value appears exactly once.

Grouped Data Formulas

Midpoint (m)

The center value of a class interval.

m = Lower + Upper2

Grouped Mean

Weighted average using midpoints and frequencies.

x̄ = Σ(m · f)Σf

Grouped Mode

The midpoint of the class with the highest frequency.

Grouped Median

The midpoint of the class containing the 50th percentile.

Quartiles & Five-Number Summary

Quartiles

Values that divide sorted data into four equal parts.

Q1 = 25th percentile (median of lower half)

Q2 = 50th percentile (median)

Q3 = 75th percentile (median of upper half)

Five-Number Summary

The five key values that describe a data set:

Min, Q1, Median, Q3, Max

IQR (Interquartile Range)

The spread of the middle 50% of data.

IQR = Q3 - Q1

Outliers & Fences

Outliers

Data values that are unusually far from the rest of the data. Values beyond the fences.

Lower Fence

Values below this are potential outliers.

Lower Fence = Q1 - 1.5 × IQR

Upper Fence

Values above this are potential outliers.

Upper Fence = Q3 + 1.5 × IQR

Box Plot (Box and Whisker)

📦 Box Plot Components

• Box: Spans from Q1 to Q3 (contains middle 50% of data)

• Line in box: The median (Q2)

• Whiskers: Extend to min and max (or to fences if outliers exist)

• Dots/Stars: Represent outliers beyond the fences

Ogive (Cumulative Frequency Graph)

Ogive

A line graph showing cumulative relative frequency on the y-axis and data values on the x-axis. Used to find percentiles and determine what proportion of data falls at or below a certain value.

Reading an Ogive: Find a value on the x-axis, go up to the curve, then read across to the y-axis to find the cumulative relative frequency (percentile).

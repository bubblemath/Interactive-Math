📊 Conditional Probability

P(E|F) = P(E ∩ F) / P(F)

The probability of event E given that event F has occurred. The vertical bar "|" means "given."

Example: Roll a die. Find P(3 | less than 5).
P(3 ∩ less than 5) = 1/6, P(less than 5) = 4/6
P(3 | less than 5) = (1/6) / (4/6) = 1/4

🔗 Independent Events

P(A ∩ B) = P(A) · P(B)

Events are independent if the occurrence of one does not affect the probability of the other.

Test: Events are independent if P(B|A) = P(B)

Example: Drawing with replacement.
P(Red₁ ∩ Red₂) = P(Red) · P(Red) = 3/10 · 3/10 = 9/100

⛓️ Dependent Events

P(A ∩ B) = P(A) · P(B|A)

Events are dependent if the occurrence of one affects the probability of the other.

Example: Drawing without replacement from 3 red, 7 blue.
P(Red₁ ∩ Red₂) = P(Red₁) · P(Red₂|Red₁)
= 3/10 · 2/9 = 6/90 = 1/15

🌳 Tree Diagram Method

Multiply along branches, Add across paths

Tree diagrams visualize sequential events. Marginal probabilities are on first branches, conditional on second branches.

Joint Probability: Multiply probabilities along a complete path from start to end.
Total Probability: Add joint probabilities of all paths leading to the same outcome.

🔄 With vs. Without Replacement

With: Independent | Without: Dependent

With replacement: Item returned before next draw → probabilities stay the same (independent).

Without replacement: Item not returned → probabilities change (dependent).

Jar: 4 white, 3 red, 3 blue (10 total)
With: P(R₂|R₁) = 3/10
Without: P(R₂|R₁) = 2/9

✅ Testing for Independence

Independent if P(B|A) = P(B)

To test if events A and B are independent, check if knowing A occurred changes the probability of B.

Example: If P(Female) = 0.52 and P(Female|Jogger) = 0.40, then Female and Jogger are dependent because 0.40 ≠ 0.52.

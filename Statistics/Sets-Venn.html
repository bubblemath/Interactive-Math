📚 Key Terms & Concepts

Set

A collection of distinct objects called elements or members.

Example: A = {1, 2, 3, 4, 5}

Element / Member

An object that belongs to a set. We use ∈ to show membership.

3 ∈ A means "3 is an element of A"

Empty Set

A set with no elements. Also called the null set.

∅ or { }

Universal Set (U)

The set containing ALL possible elements in a given context.

U = {all students in the class}

Subset (⊆)

Set A is a subset of B if every element of A is also in B.

A ⊆ B

Proper Subset (⊂)

A is a proper subset of B if A ⊆ B and A ≠ B.

A ⊂ B

Set Operations

Union (A ∪ B) - "A OR B"

All elements that are in A OR B OR both. The word "or" indicates union.

A ∪ B = {x : x ∈ A or x ∈ B}

Both circles are shaded for Union (OR)

Intersection (A ∩ B) - "A AND B"

All elements that are in BOTH A AND B. The word "and" indicates intersection.

A ∩ B = {x : x ∈ A and x ∈ B}

Only the overlap is shaded for Intersection (AND)

Complement (A' or Aᶜ)

All elements in U that are NOT in A.

A' = {x : x ∈ U and x ∉ A}

Everything outside A is shaded for Complement

Disjoint Sets

Two sets with NO elements in common.

A ∩ B = ∅

Circles don't overlap for Disjoint Sets

Cardinality

n(A) - Number of Elements

The cardinality of a set is the count of elements in that set.

If A = {2, 4, 6, 8}, then n(A) = 4

⚠️ Important: "AND" vs "OR" in Set Theory

"AND" = Intersection (∩) - Elements must be in BOTH sets

"OR" = Union (∪) - Elements can be in EITHER set (or both)

Example: "Students in Math AND Science" = Math ∩ Science

Example: "Students in Math OR Science" = Math ∪ Science

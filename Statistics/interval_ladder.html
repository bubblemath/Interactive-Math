Confidence intervals · one skeleton, three types

Every interval is estimate ± margin. The margin is where it breaks.

A confidence interval is never a guess about a formula — it is always the same shape: a point estimate with a band around it. What changes from a Z to a T to a proportion is only which critical value you pull and how you build the standard error. Get those two right and the interval assembles itself.

How it works

Climb the assembly, one piece at a time

Eleven rungs run from "which interval is this?" up to building a complete interval end-to-end. Nothing is locked — start anywhere, or jump to the piece you keep dropping.

01

Isolate one piece

Each rung drills a single step: the critical value, the standard error, the margin, the bounds. When the whole interval feels like too much, this is where you find the leak.

02

Earn each badge

Get four in a row on a rung to earn its badge. Pick answers with the mouse or press 1–4; type numbers and hit Enter. Miss one and the streak resets.

03

Report to 4 places

Round the standard error, margin, and bounds to 4 decimal places. Use critical values to 3 places (1.960, 2.576, and the t-table). Round the same way the ladder does and the numbers line up.

The three standard errors
Z-interval · mean
SE = σ / √n
Population σ is known. Critical value is z*.
T-interval · mean
SE = s / √n
Population σ is unknown; use sample s. Critical value is t* at df = n − 1.
Proportion
SE = √( p̂(1−p̂) / n )
Estimating a proportion. Critical value is z*.

The point estimate is the exact midpoint — always. Once you have an interval, you get two free checks: the estimate sits dead center (estimate = (L+U)/2) and the margin is exactly half the width (ME = (U−L)/2). For a proportion, the bounds also cannot leave [0, 1]. If any of those fail, it's arithmetic — not statistics — and you caught it before it was graded.

Why it matters

An honest answer has a width

A point lies by omission

"The mean is 70.2 inches" pretends to a precision the data never had. Thirty-six students cannot pin a population to a single number. The interval is the honest version: here is the estimate, and here is how far off it can reasonably be.

Confidence is about the method

"95% confident" is a statement about the procedure, not this one interval. Build intervals this way over and over and about 95% of them will catch the true value. This particular one either did or didn't — you just don't get to know which.

Width is a design choice

Want a narrower interval? Collect more data — the standard error shrinks with √n. Want more confidence? The interval widens. The sample-size rungs are exactly this trade, run backwards from the precision you need.

Every interval on this ladder is the same three moves: pick the critical value · build the standard error · assemble estimate ± critical × SE. The Z, the T, and the proportion differ only in the middle move. Once the skeleton is automatic, a confidence interval stops being a formula to memorize and becomes what it actually is — a measured statement about how much you don't know.

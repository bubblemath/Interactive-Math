Key Terms & Formulas

Experiment Event Sample Space Complement Union Mutually Exclusive Odds

🎯 Basic Probability

P(E) = favorable / total

Sample Space (S): The set of all possible outcomes.

Event (E): A specific outcome or set of outcomes.

🔄 Complement Rule

P(E') = 1 − P(E)

The probability that event E does not occur.

Tip: Use for "at least" problems!

⚡ Union Rule (General)

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

For any two events, add probabilities and subtract overlap.

🚫 Mutually Exclusive

P(A ∪ B) = P(A) + P(B)

When events cannot occur simultaneously.

📊 Probability to Odds

Odds = P(E) : P(E')

Convert probability to odds in favor.

🔢 Odds to Probability

P(E) = m / (m + n)

If odds are m to n, use this formula.

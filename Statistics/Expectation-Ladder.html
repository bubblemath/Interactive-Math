The Expectation Ladder

Lesson 12 · Discrete Distributions & Expected Value
The signature first move

Prove the table is legal. Then build a column and add it.

Before you compute a single thing, check the distribution: every P(x) is between 0 and 1, and ΣP(x) = 1. A table that fails this test cannot be fixed by arithmetic downstream.

Once it's legal, everything you'll ever be asked for is one column, multiplied down and added up:

E(X) = Σ[ x · P(x) ]  — multiply, then add
σ² = Σ[ (x − μ)² · P(x) ]  — same move, new column
net gain: bake the cost into the x's, then E(X) again
fair game ⟺ E(X) = 0

📊 Lesson 23: Two-Sample Inference with Means

Independent Samples & Matched Pairs Practice

🎯 Two-Sample Inference Overview

The goal is to compare responses to two treatments or compare characteristics of two populations.

Feature Independent Samples (2-Sample) Matched Pairs (Dependent)
Data Structure Two separate, unrelated groups Same subjects measured twice OR naturally paired subjects
Random Variable Difference in sample means: \(\bar{x}_1 - \bar{x}_2\) Mean of the differences: \(\bar{d}\)
Examples Boys vs. girls, Brand A vs. Brand B (different people) Before/after treatment, Left hand vs. right hand
Key Phrase "Two independent groups" "Same subjects" or "paired/matched"

📝 Hypothesis Statements

For Independent Samples:

  • \(H_0\): \(\mu_1 = \mu_2\)  or  \(\mu_1 - \mu_2 = 0\) (no difference)
  • \(H_a\): \(\mu_1 < \mu_2\) (left-tailed), \(\mu_1 > \mu_2\) (right-tailed), or \(\mu_1 \neq \mu_2\) (two-tailed)

For Matched Pairs:

  • \(H_0\): \(\mu_d = 0\) (mean difference is zero)
  • \(H_a\): \(\mu_d < 0\), \(\mu_d > 0\), or \(\mu_d \neq 0\)

✅ Assumptions to Check

  • ✓ Have two simple random samples (SRS) from populations OR two randomly assigned treatment groups
  • ✓ Samples are independent (for 2-sample) OR properly paired (for matched pairs)
  • ✓ Both distributions are approximately normal
    • Large sample sizes (n ≥ 30 for each group), OR
    • Graph BOTH sets of data to check for normality

🔢 Test Statistic Formula

Independent Samples:

\[t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}\]
Matched Pairs:

\[t = \frac{\bar{d}}{s_d / \sqrt{n}}\]
where \(\bar{d}\) = mean of differences, \(s_d\) = standard deviation of differences

📊 Decision Rule

  • If p-value < α: Reject \(H_0\) → Evidence supports \(H_a\)
  • If p-value ≥ α: Fail to reject \(H_0\) → Not enough evidence for \(H_a\)

⚠️ Type I and Type II Errors

Error Type What Happens When It Occurs Probability
Type I Error
(False Positive)		 Reject \(H_0\) when \(H_0\) is actually TRUE We conclude there IS a difference when there really ISN'T α (significance level)
Type II Error
(False Negative)		 Fail to reject \(H_0\) when \(H_0\) is actually FALSE We conclude there is NO difference when there really IS one β

Memory Tip: Type I = "I made a mistake by rejecting the truth" | Type II = "II missed the truth by not rejecting"

💡 How to Identify Paired vs. Independent

  • Paired: Same individuals measured twice, before/after studies, twins, left/right comparisons
  • Independent: Two completely separate groups with no natural pairing

Ask yourself: "Can I match each data point in group 1 with a specific data point in group 2?"

Lesson 22 Summary: Hypothesis Tests with Proportions

1-Sample Proportion Test

  • Assumptions: SRS, normal approximation (np₀ ≥ 10, n(1-p₀) ≥ 10)
  • Hypotheses: H₀: p = p₀, Hₐ: p <, >, or ≠ p₀
  • Test Statistic: z = (p̂ - p₀) / sqrt(p₀(1-p₀)/n)
  • Use the 1-sample applet for calculations.

2-Sample Proportion Test

  • Assumptions: Two independent SRS, normal approximation (at least 5 successes and failures in each group)
  • Hypotheses: H₀: p₁ = p₂ (or p₁ - p₂ = 0), Hₐ: p₁ <, >, or ≠ p₂
  • Test Statistic: z = (p̂₁ - p̂₂) / sqrt(p_c(1-p_c)(1/n₁ + 1/n₂)), where p_c is the pooled proportion
  • Use the 2-sample applet for calculations.

Type I and II Errors

  • Type I: Rejecting H₀ when it is true
  • Type II: Not rejecting H₀ when it is false

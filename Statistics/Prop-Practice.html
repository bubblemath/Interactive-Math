Hypothesis Tests with Proportions Practice
Reminders: Fractions, Decimals, Percents
  • Fraction → Decimal: Divide numerator by denominator. (e.g., 3/4 = 0.75)
  • Decimal → Percent: Multiply by 100 and add %. (e.g., 0.75 × 100 = 75%)
  • Percent → Decimal: Divide by 100. (e.g., 75% ÷ 100 = 0.75)
Hypothesis Test Steps & p-value
  1. Identify if it’s a 1-sample or 2-sample proportion test.
  2. State the null (H₀) and alternative (Hₐ) hypotheses.
  3. Check assumptions (random sample, normal approximation, independence).
  4. Calculate test statistic (z) and p-value (use the applet):
    1-sample applet | 2-sample applet
  5. Compare p-value to significance level (α):
    • If p-value < α: Reject H₀ (evidence for Hₐ)
    • If p-value ≥ α: Fail to reject H₀ (not enough evidence for Hₐ)
  6. State conclusion in context.
Type I and Type II Errors
  • Type I Error (False Positive): Rejecting H₀ when it is actually true.
  • Type II Error (False Negative): Failing to reject H₀ when it is actually false.
  • Mnemonic: Type I = "I made a mistake by rejecting the truth"; Type II = "I missed the truth by not rejecting"

📋 The Hypothesis Testing Framework (Unit IV)

  1. State hypotheses: \(H_0\) (null) and \(H_a\) (alternative)
  2. Choose \(\alpha\): Significance level (usually 0.05)
  3. Check conditions and calculate test statistic
  4. Find p-value
  5. Conclude in context: If \(p \leq \alpha\), reject \(H_0\)

📌 Final Exam Quick Tips

  • Know your notation: \(\mu\) vs. \(\bar{x}\), \(\sigma\) vs. \(s\), \(p\) vs. \(\hat{p}\)
  • Skewed data? Use median (resistant to outliers)
  • Standard Error: \(\frac{\sigma}{\sqrt{n}}\) for means, \(\sqrt{\frac{pq}{n}}\) for proportions
  • \(p\text{-value} \leq \alpha\)? Reject \(H_0\) and conclude in context
  • Correlation ≠ Causation: \(r\) measures association, not cause
  • Chi-Square conditions: All expected counts \(\geq 5\)

📋 Quick Reference Formulas

Unit I: Descriptive Statistics

Sample Mean:
\(\bar{x} = \frac{\sum x}{n}\)
Sample Standard Deviation:
\(s = \sqrt{\frac{\sum(x - \bar{x})^2}{n-1}}\)
IQR:
\(IQR = Q_3 - Q_1\)

Unit II: Probability

Complement:
\(P(A') = 1 - P(A)\)
Addition Rule:
\(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
Combination:
\(_nC_r = \frac{n!}{r!(n-r)!}\)
Binomial:
\(P(X=k) = {_nC_k} \cdot p^k \cdot q^{n-k}\)
Expected Value:
\(E(X) = \sum [x \cdot P(x)]\)

Unit III: Inference

Z-score:
\(z = \frac{x - \mu}{\sigma}\)
Standard Error (means):
\(SE_{\bar{x}} = \frac{\sigma}{\sqrt{n}}\)
Standard Error (proportions):
\(SE_{\hat{p}} = \sqrt{\frac{pq}{n}}\)
Confidence Interval:
\(\text{estimate} \pm z^* \times SE\)
LSRL:
\(\hat{y} = a + bx\)

Unit IV: Hypothesis Testing

Z-test (σ known):
\(z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}\)
T-test (σ unknown):
\(t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}\)
Z-test for proportion:
\(z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0 q_0}{n}}}\)
Chi-Square:
\(\chi^2 = \sum \frac{(O - E)^2}{E}\)

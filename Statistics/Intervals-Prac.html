🔄 Tier 4: Work Backwards

Find the point estimate and margin of error:

Hint: p̂ = (Lower + Upper) / 2. ME = (Upper - Lower) / 2.
📊 Tier 5: Determine Sample Size

What sample size is necessary to be within ±0.04 of the true proportion with 95% confidence?

For proportion: n = (z*)² × p̂(1-p̂) / ME²
Worst case (no prior estimate): use p̂ = 0.5
Hint: Use worst case p̂ = 0.5. For 95% CI, z* = 1.96. n = (1.96)² × 0.5 × 0.5 / (0.04)². Round UP.

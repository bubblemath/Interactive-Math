Understanding Hypothesis Testing Errors

🔴 Type I Error (\(\alpha\))

Definition: Rejecting the null hypothesis when it is actually true.

Symbol: \(\alpha\) (alpha) — also called the level of significance

Memory Aid: "False Positive" — You conclude there IS an effect when there ISN'T one.

Example: A jury convicts an innocent person. (H₀: defendant is innocent)
\(\alpha = P(\text{Type I error}) = P(\text{Reject } H_0 \mid H_0 \text{ is true})\)

🟡 Type II Error (\(\beta\))

Definition: Failing to reject the null hypothesis when it is actually false.

Symbol: \(\beta\) (beta)

Memory Aid: "False Negative" — You conclude there is NO effect when there IS one.

Example: A jury acquits a guilty person. (H₀: defendant is innocent)
\(\beta = P(\text{Type II error}) = P(\text{Fail to reject } H_0 \mid H_0 \text{ is false})\)

💪 Power of the Test

\(\text{Power} = 1 - \beta\)

The power is the probability of correctly rejecting a false null hypothesis. We want power to be high (close to 1).

How to increase power: Increase the sample size (n)

Decision Rule: p-value vs. α

  • If p-value < α, reject H₀ (statistically significant)
  • If p-value ≥ α, do not reject H₀ (not statistically significant)
Memory Aid: "If the p is low, the null must go. If the p is high, the null must fly."

Test Statistic Formulas

z = \frac{\bar{x} - \mu_0}{\sigma/\sqrt{n}}     t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}

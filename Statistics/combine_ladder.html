Combining random variables · sums, differences, transformations

Means add. Variances add too — even when you subtract.

Put two independent random variables together and the means behave exactly as you'd guess: E(X + Y) = E(X) + E(Y), and for a difference they subtract. Variability doesn't play along. Combining two uncertain things always makes the result more variable, never less — so the variances add either way, and the standard deviation comes from the square root at the end. Standard deviations themselves never add.

The rules
Sums and differences (independent)
E(X ± Y) = E(X) ± E(Y)
Var(X ± Y) = Var(X) + Var(Y)
Then SD = √Var.
Linear transformations
E(aX + b) = a·E(X) + b
SD(aX + b) = |a|·SD(X)
Var(aX + b) = a²·Var(X)
The mistake to avoid
SD(X + Y) ≠ SD(X) + SD(Y). Adding standard deviations always overstates the spread. Square first.
n independent copies
Total of n items: Var = n·Var(X), so SD = √n·SD(X). But SD(nX) = n·SD(X) — not the same thing.
➕

Why does variance still add for a difference? Because variance measures how much a value bounces around, and subtracting doesn't calm anything down — both variables are still wobbling, and the gap between them wobbles from both sources. Uncertainty accumulates no matter which operation you use: combining never reduces variability.

Why it matters

This is where every standard error comes from

It explains the formulas

Why does the standard error of a mean divide by √n? Because a total of n independent copies has variance n·Var(X), and dividing by n scales the SD by 1/n. The formula isn't arbitrary — it's this rule.

Differences are everywhere

Profit is revenue minus cost; a gap is one time minus another; a treatment effect is one group minus another. Every one of those needs variances added, not subtracted.

Independence is the price

These rules for variance hold because the variables don't influence each other. When they do, the arithmetic changes — which is exactly why you check independence before adding.

The whole topic reduces to a habit: handle the means directly, and handle the spread through variance. Add or subtract expected values as the situation asks, add the variances whichever operation you used, square-root at the end, and never add standard deviations. Once that's automatic, two-sample standard errors and the Central Limit Theorem stop looking like new formulas and start looking like this rule applied again.

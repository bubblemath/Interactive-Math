Key Terms & Formulas

Random Variable Probability Distribution Expected Value Histogram

🎲 Random Variable

X = numerical outcome

A random variable is a function that assigns a numerical value to each outcome of an experiment.

Example: Toss 2 coins
X = number of heads
Possible values: X = 0, 1, or 2

📋 Probability Distribution

\[ \sum P(x) = 1 \]

A table listing all possible values of a random variable with their corresponding probabilities.

Key Property: All probabilities must sum to exactly 1.

Requirements:
• Each P(x) must be between 0 and 1
• Sum of all P(x) = 1

📊 Expected Value Formula

\[ E(X) = \sum [x \cdot P(x)] \]

The expected value is the long-run average outcome if the experiment is repeated many times.

Calculation:
E(X) = x₁·P(x₁) + x₂·P(x₂) + ... + xₙ·P(xₙ)
Multiply each outcome by its probability, then sum.

🎯 Interpreting E(X)

E(X) = "average" outcome

Expected value represents what you would expect on average over many repetitions.

Example: If E(X) = 1.5 for coin tosses
"On average, you expect 1.5 heads per 2 tosses"
(Note: E(X) doesn't have to be a possible outcome!)

📈 Histogram

Visual representation of distribution

A histogram displays the probability distribution graphically with bars.

• X-axis: possible values of X
• Y-axis: probabilities P(x)
• Bar heights = probabilities

💡 Decision Making with E(X)

Choose option with better E(X)

When comparing options, calculate E(X) for each and choose the one with the higher expected value (for gains) or lower expected value (for costs).

Example: Life insurance options
Option 1: E(X) = $17,600
Option 2: E(X) = $19,200
Choose Option 2 (higher expected payout)

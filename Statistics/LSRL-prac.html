3
Residual Plots - Assess Fit Quality
📊 Instructions: Click to add points and see both the scatterplot with LSRL and the residual plot. A random scatter in the residual plot indicates a linear relationship. A pattern suggests a non-linear relationship!

Scatterplot with LSRL

Residual Plot

✓ Random Pattern (Linear)

Points scattered randomly around y = 0 with no clear pattern indicates the linear model is appropriate.

✗ Curved Pattern (Non-Linear)

A U-shape or curve in the residual plot suggests the relationship is not linear and LSRL may not be appropriate.

📊 Single Sample Hypothesis Tests

🔬

Hypothesis Tests: Z-Test, T-Test, Z-Prop

Learn the foundational framework for testing claims about a single population mean or proportion.

  • Z-Test (σ known): z = (x̄ − μ₀) / (σ/√n)
  • T-Test (σ unknown): t = (x̄ − μ₀) / (s/√n)
  • Z-Prop: z = (p̂ − p₀) / √(p₀q₀/n)
  • Setting up H₀ and Hₐ correctly
  • One-tailed vs. Two-tailed tests

Two Sample Tests: Means & Proportions

Compare two populations using independent means, dependent (paired) means, or proportions.

  • Independent Means: t = (x̄₁ − x̄₂) / SE
  • Paired/Dependent: t = d̄ / (s_d/√n)
  • Two Proportions: z = (p̂₁ − p̂₂) / SE_pooled
  • Choosing between independent vs. paired designs
  • Pooled vs. unpooled standard errors

📋 Chi-Square Tests for Categorical Data

📋

Chi-Square Tests

Analyze categorical data using Chi-Square tests for goodness of fit, independence, and homogeneity.

  • Goodness of Fit: Does data match expected distribution?
  • Independence: Are two categorical variables related?
  • Homogeneity: Same distribution across populations?
  • Formula: χ² = Σ(O − E)² / E
  • Expected counts: E = (row total × col total) / n

📌 Hypothesis Testing Quick Reference

The 5-Step Process

  1. State H₀ and Hₐ
  2. Choose significance level (α)
  3. Check conditions & calculate test statistic
  4. Find p-value
  5. State conclusion in context

Decision Rule

  • If p-value ≤ α: Reject H₀
  • If p-value > α: Fail to reject H₀
  • Common α values: 0.01, 0.05, 0.10
  • Always state conclusion in context!

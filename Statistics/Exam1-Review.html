📋 Key Concepts Summary

🎯 Data Storytelling Framework

  • Population: The entire group we want information about
  • Sample: A part of the population we actually examine
  • Parameter: A numerical measurement describing a population (μ, σ)
  • Statistic: A numerical measurement describing a sample (x̄, s)
  • Census: A complete count of the entire population

📊 Variable Types

  • Qualitative (Categorical): Identifies characteristics - names, labels, categories
  • Quantitative (Numerical): Measurements that make sense to average
    • Discrete: Countable, listable values (usually counts)
    • Continuous: Any value in a range (usually measurements)

📏 Levels of Measurement

  • Nominal: Categories with no order (colors, names, labels)
  • Ordinal: Categories with meaningful order (rankings, ratings)
  • Interval: Ordered numerical data, no true zero (temperature in °F/°C)
  • Ratio: Ordered numerical data with true zero (height, weight, age, money)

🎲 Sampling Methods

  • Simple Random Sample (SRS): Every individual has equal chance of selection
  • Stratified: Divide into homogeneous groups (strata), then SRS from each
  • Systematic: Select every kth individual after random start
  • Cluster: Randomly select locations, sample everyone there
  • Convenience: Sample whoever is easy to reach (introduces bias)

📈 Measures of Center

Mean: x̄ = Σx / n (sample)    μ = Σx / N (population)
Median: Middle value when data is ordered
Mode: Most frequently occurring value
Grouped Mean: Σ(f·m) / Σf where m = midpoint, f = frequency

📉 Measures of Variation

Range: Maximum - Minimum
IQR: Q3 - Q1
Population Variance: σ² = Σ(x - μ)² / N
Sample Variance: s² = Σ(x - x̄)² / (n-1)
Standard Deviation: σ = √σ² or s = √s²

📊 Five-Number Summary & Boxplots

  • Minimum: Smallest value
  • Q1 (Lower Quartile): 25th percentile
  • Median (Q2): 50th percentile
  • Q3 (Upper Quartile): 75th percentile
  • Maximum: Largest value

🔔 Distribution Shapes

  • Symmetric: Both sides mirror each other; Mean ≈ Median
  • Right-Skewed (Positive): Tail extends right; Mean > Median
  • Left-Skewed (Negative): Tail extends left; Mean < Median
  • Uniform: All values have approximately equal frequency
  • Bimodal: Two distinct peaks

Rule: Use median for skewed data; use mean for symmetric data

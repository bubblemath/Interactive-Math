📊 Bar Graph

When to Use a Bar Graph

Bar graphs are ideal for comparing categorical data - data that falls into distinct groups or categories. Each bar represents a category, and the height shows the frequency or value.

📊 Summary Statistics

🥧 Pie Chart

When to Use a Pie Chart

Pie charts show parts of a whole. Each slice represents a category's proportion of the total. Best used when you have 2-7 categories and want to emphasize relative sizes.

📊 Summary Statistics

🌿 Stem and Leaf Plot

How to Create a Stem and Leaf Plot

  1. Determine the min and max of the data set
  2. Decide the stem (10's digit vs 100's)
  3. Stem and leaf plots use 2 digits (1 for stem, 1 for leaf)
  4. Identify the key for the table
  5. Draw a vertical line with stem on left, leaves on right in order

Enter numerical values only. Works best with 2-3 digit numbers.

Enter data and click "Create" to generate your stem and leaf plot.

⚫ Dot Plot

How to Create a Dot Plot

  1. Organize your data
  2. Collect frequency data for repeated observations
  3. Draw an X and Y axis
  4. Label the x-axis based on data, y-axis based on highest frequency
  5. Place a dot over each unique data value

Enter numerical values. Repeated values will stack as dots.

Enter data and click "Create" to generate your dot plot.

📈 Histogram

How to Create a Histogram

  1. Organize your data, note the min and max
  2. Decide on the number of bins (classes)
  3. Calculate bin width: (max - min) / number of bins
  4. Count frequencies for each bin
  5. Draw bars with no gaps between them

Enter numerical values. The histogram will automatically create bins.

📊 Summary Statistics

📦 Box Plot (Box and Whisker)

Understanding Box Plots

A box plot displays the five-number summary: minimum, Q1 (25th percentile), median (Q2), Q3 (75th percentile), and maximum. The "box" shows the middle 50% of data (IQR), and "whiskers" extend to the min and max.

Enter numerical values to see the distribution.

📊 Hypothesis Testing Unit Review

Master the concepts of statistical inference and hypothesis testing

🎯 Hypothesis Testing Fundamentals

  • Null Hypothesis (H₀): The starting assumption that specifies a population parameter value; we assume it's true until evidence suggests otherwise. Always contains an equality (=, ≤, or ≥)
  • Alternative Hypothesis (Hₐ): The claim we're testing for; contains the values we consider plausible if we reject H₀. Never contains an equality
  • p-value: The probability of obtaining results as extreme as observed, assuming H₀ is true
  • Significance Level (α): The threshold for rejecting H₀; commonly 0.05 or 0.01
Decision Rule: If p-value < α → Reject H₀ | If p-value ≥ α → Fail to reject H₀

⚠️ Type I and Type II Errors

  • Type I Error (α): Rejecting H₀ when it is actually TRUE (False Positive)
  • Type II Error (β): Failing to reject H₀ when it is actually FALSE (False Negative)
  • Power of the Test: 1 - β; probability of correctly rejecting a false H₀
Memory Aid: "If the p is low, the null must go. If the p is high, the null will fly."

📐 Test Statistics & Formulas

One-Sample Tests

z-test (σ known)
\[ z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} \]
t-test (σ unknown)
\[ t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}} \]
df = n - 1
One-Sample Proportion
\[ z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}} \]

Two-Sample Tests

Two-Sample t-test (Independent)
\[ t = \frac{(\bar{x}_1 - \bar{x}_2) - 0}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}} \]
Two-Sample Proportion
\[ z = \frac{(\hat{p}_1 - \hat{p}_2) - 0}{\sqrt{\hat{p}(1-\hat{p})\left(\frac{1}{n_1} + \frac{1}{n_2}\right)}} \]
where \(\hat{p} = \frac{x_1 + x_2}{n_1 + n_2}\)
Matched Pairs t-test
\[ t = \frac{\bar{d} - 0}{s_d / \sqrt{n}} \]
df = n - 1 (pairs)

Chi-Square Tests

Chi-Square Test Statistic
\[ \chi^2 = \sum \frac{(O - E)^2}{E} \]
O = Observed, E = Expected
Expected Count (Independence)
\[ E = \frac{(\text{row total})(\text{column total})}{\text{grand total}} \]
Degrees of Freedom for Chi-Square
Goodness of Fit: df = k - 1 (k = categories)
Independence/Homogeneity: df = (r - 1)(c - 1)

📊 Types of Hypothesis Tests

  • One-Sample z-test: Compare sample mean to population mean when σ is known
  • One-Sample t-test: Compare sample mean to population mean when σ is unknown
  • One-Sample Proportion: Test claims about population proportions
  • Two-Sample t-test: Compare means of two independent populations
  • Two-Sample Proportion: Compare proportions from two populations
  • Matched Pairs: Compare measurements from the same subjects (before/after)
  • Chi-Square Goodness of Fit: Test if observed frequencies match expected
  • Chi-Square Test of Independence: Test if two categorical variables are related
  • Chi-Square Test of Homogeneity: Compare distributions across populations

✅ Assumptions to Check

For Means (z and t tests)
  • Random sample (SRS)
  • Independence: population ≥ 10n
  • Normality: n ≥ 30 OR population normal
For Proportions
  • Random sample (SRS)
  • Independence: population ≥ 10n
  • np ≥ 10 AND n(1-p) ≥ 10
For Chi-Square
  • Random sample (SRS)
  • Independence
  • All expected counts ≥ 5

📝 Writing Conclusions

If you REJECT H₀: "Since the p-value (____) < α (____), we reject H₀. There is sufficient evidence to support the claim that [alternative hypothesis in context]."
If you FAIL TO REJECT H₀: "Since the p-value (____) > α (____), we fail to reject H₀. There is not sufficient evidence to support the claim that [alternative hypothesis in context]."

Remember: Never say "accept H₀" - we only "fail to reject" it!

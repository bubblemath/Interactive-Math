Union Rule, Complement & Odds

Master the fundamental rules for combining and manipulating probabilities.

  • Complement Rule: P(A') = 1 − P(A)
  • Addition Rule: P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
  • Mutually Exclusive: P(A ∩ B) = 0
  • Odds in favor: P(A) : P(A')
  • Converting between odds and probability

Conditional & Independence

Understand how the occurrence of one event affects the probability of another.

  • Conditional Probability: P(A|B) = P(A ∩ B) / P(B)
  • Independent Events: P(A|B) = P(A)
  • Dependent Events: P(A|B) ≠ P(A)
  • Multiplication Rule: P(A ∩ B) = P(A) · P(B|A)
  • Tree diagrams for sequential events

Counting: Permutations & Combinations

Learn systematic methods for counting possible arrangements and selections.

  • Fundamental Counting Principle
  • Factorial: n! = n × (n−1) × ... × 1
  • Permutation: Order matters → nPr = n!/(n−r)!
  • Combination: Order doesn't matter → nCr = n!/[r!(n−r)!]
  • Distinguishing "arrange" vs. "choose"

Binomial Distribution

Model experiments with exactly two outcomes repeated a fixed number of times.

  • Criteria: Fixed n, two outcomes, constant p, independent trials
  • Formula: P(X=k) = nCk · p^k · (1−p)^(n−k)
  • Mean: μ = n · p
  • Standard Deviation: σ = √(n · p · q)

Discrete Distributions

Build probability distribution tables and calculate expected values for discrete random variables.

  • Random Variable (X) – Numerical outcome
  • Probability Distribution – All X values with P(X)
  • Valid Distribution: All P(X) ≥ 0 and ΣP(X) = 1
  • Expected Value: E(X) = Σ[X · P(X)]

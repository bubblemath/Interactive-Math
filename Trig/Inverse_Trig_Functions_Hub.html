🌟 Why Does This Matter?

Inverse trig functions answer the question: "What angle gives this ratio?"

  • 🧭 Navigation: Finding angles from coordinate ratios (GPS, surveying)
  • 🚀 Physics: Calculating launch angles from projectile data
  • 🏗️ Engineering: Determining angles in structural analysis
  • 🎮 Computer Graphics: Rotation angles from transformation matrices
  • 📡 Signal Processing: Phase angle calculations in electronics

❓ "Why Restrict?" Visualizer

If sin(30°) = 0.5 and sin(150°) = 0.5, which angle should sin⁻¹(0.5) return?

The Problem: Many angles have the same sine value!

The Solution: We choose the principal value from a restricted domain.

For arcsin: We use [−π/2, π/2], so sin⁻¹(0.5) = π/6 (30°)

🪞 The Reflection Discovery Lab

🌟 Creating Inverse Functions

To create an inverse function, we:

  1. Restrict the domain to make the function one-to-one
  2. Reflect across y = x to swap inputs and outputs

The result is a valid function where x and y roles are reversed!

🪞 "Mirror on y = x" Animation

Watch the restricted sine curve reflect to create arcsin:

Original (Restricted)

y = sin(x), x ∈ [−π/2, π/2]

Domain: [−π/2, π/2]

Range: [−1, 1]

Inverse

y = arcsin(x)

Domain: [−1, 1]

Range: [−π/2, π/2]

🔗 Composition Workshop

⚠️ The Cancellation Trap

Students often assume inverse functions always "cancel"—but there's a catch!

  • ✓ Always true: sin(arcsin(x)) = x (if x ∈ [−1, 1])
  • ⚠️ Sometimes false: arcsin(sin(x)) = x (only if x ∈ [−π/2, π/2]!)

📐 Composition Unwrapper

Step-by-step breakdown of expressions like cos(arcsin(3/5)):

1 Let θ = arcsin(3/5)

This means sin(θ) = 3/5 and θ is in [−π/2, π/2]

2 Draw a right triangle

Opposite = 3, Hypotenuse = 5

3 Find the adjacent side

Adjacent = √(5² − 3²) = √16 = 4

4 Calculate cos(θ)

cos(θ) = Adjacent/Hypotenuse = 4/5

cos(arcsin(3/5)) = 4/5

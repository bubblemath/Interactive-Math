Inverse trig functions "reverse" the process—instead of finding a ratio from an angle, we find an angle from a ratio. This requires understanding domain restrictions.

🌟 Why Does This Matter?

Inverse trig functions are essential for:

  • 🧭 Navigation: Finding angles from coordinate ratios (GPS, surveying)
  • 🚀 Physics: Calculating launch angles from projectile data
  • 🏗️ Engineering: Determining angles in structural analysis
  • 🎮 Computer Graphics: Rotation angles from transformation matrices
  • 📊 Calculus: Derivatives and integrals involving inverse trig

⚠️ "Why Restrict?" Preview

Here's the problem: multiple angles can have the same trig value!

Which angles have sin(θ) = 1/2?

30° (π/6) and 150° (5π/6) both have sin = 1/2

Plus infinitely more: 390°, 510°, −210°, −330°, ...

❓ The Big Question

If arcsin(1/2) could equal 30° OR 150° OR infinitely many others...

Which one should it return?

We need to RESTRICT the domain to get a unique answer!

❓ "Why This Angle?" Explainer

Why does arcsin(1/2) = π/6 and NOT 5π/6?

1 Both sin(π/6) = 1/2 and sin(5π/6) = 1/2 are true
2 But arcsin is restricted to output only angles in [−π/2, π/2]
3 π/6 ≈ 0.52 is in [−π/2, π/2] ✓
4 5π/6 ≈ 2.62 is NOT in [−π/2, π/2] ✗

Therefore: arcsin(1/2) = π/6

🔗 Composition Workshop (The "Cancellation Trap")

⚠️ THE CANCELLATION TRAP

sin(arcsin(x)) = x ✓ (always, if x ∈ [−1, 1])

arcsin(sin(x)) = x ⚠️ (ONLY if x ∈ [−π/2, π/2]!)

📐 Right Triangle Builder

Evaluate compositions like cos(arcsin(3/5)) using a right triangle:

Example: Find cos(arcsin(3/5))

1 Let θ = arcsin(3/5), so sin θ = 3/5
2 Draw a right triangle with opposite = 3, hypotenuse = 5
3 Find adjacent using Pythagorean theorem: √(5² − 3²) = √16 = 4
4 cos θ = adjacent/hypotenuse = 4/5

cos(arcsin(3/5)) = 4/5

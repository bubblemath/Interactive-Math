🔬 Sine & Tangent Discovery Lab

📊 The Complete Identity Set

Sine Identities:

sin(A + B) = sin A cos B + cos A sin B

sin(A − B) = sin A cos B − cos A sin B

Tangent Identities:

tan(A + B) = (tan A + tan B) / (1 − tan A tan B)

tan(A − B) = (tan A − tan B) / (1 + tan A tan B)

🎯 Exact Value Workshop

🌟 Finding Exact Values

Use sum/difference identities to find exact values for non-standard angles by decomposing them into standard angles (30°, 45°, 60°).

🧮 Step-by-Step: sin(75°)

1 Decompose the angle:

75° = 45° + 30°

2 Apply the sum identity:

sin(45° + 30°) = sin 45° cos 30° + cos 45° sin 30°

3 Substitute exact values:

= (√2/2)(√3/2) + (√2/2)(1/2)

4 Simplify:

= √6/4 + √2/4 = (√6 + √2)/4

sin(75°) = (√6 + √2)/4 ≈ 0.966

🧮 Step-by-Step: tan(15°)

1 Decompose the angle:

15° = 45° − 30°

2 Apply the difference identity:

tan(45° − 30°) = (tan 45° − tan 30°) / (1 + tan 45° tan 30°)

3 Substitute exact values:

= (1 − √3/3) / (1 + 1·√3/3) = (1 − √3/3) / (1 + √3/3)

4 Simplify (multiply by 3/3):

= (3 − √3) / (3 + √3) = 2 − √3

tan(15°) = 2 − √3 ≈ 0.268

⚠️ Tangent "Undefined" Explorer

The tangent identity has a restriction—when is it undefined?

⚠️ Division by Zero Alert!

tan(A + B) = (tan A + tan B) / (1 − tan A tan B)

Undefined when: 1 − tan A tan B = 0

This happens when: tan A tan B = 1

Example: tan(45°) · tan(45°) = 1 · 1 = 1

So tan(45° + 45°) = tan(90°) is undefined ✓

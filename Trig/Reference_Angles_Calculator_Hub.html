🔑 The Big Idea

A reference angle is the acute angle formed between the terminal side and the x-axis. Since all reference angles are acute (0° to 90°), we can use our special angle values from Lesson 5!

The only new skill: determine the correct sign (+/−) based on the quadrant.

📐 Reference Angle Formulas

Quadrant θ Range (Degrees) θ Range (Radians) Reference Angle θʹ
I 0° < θ < 90° 0 < θ < π/2 θʹ = θ
II 90° < θ < 180° π/2 < θ < π θʹ = 180° − θ (or π − θ)
III 180° < θ < 270° π < θ < 3π/2 θʹ = θ − 180° (or θ − π)
IV 270° < θ < 360° 3π/2 < θ < 2π θʹ = 360° − θ (or 2π − θ)

Before every calculation, verify your calculator is in DEGREE mode (not RADIAN). Look for "DEG" on your display. Wrong mode = wrong answer!

🔄 Reciprocal Functions

Calculators don't have sec, csc, or cot buttons. Use reciprocal identities:

sec θ

Enter: 1 ÷ cos(θ)

Example: sec 60° → 1 ÷ cos(60) = 2

csc θ

Enter: 1 ÷ sin(θ)

Example: csc 30° → 1 ÷ sin(30) = 2

cot θ

Enter: 1 ÷ tan(θ)

Example: cot 45° → 1 ÷ tan(45) = 1

📐 DMS (Degrees-Minutes-Seconds)

Converting DMS to Decimal Degrees

38°42' means 38 degrees and 42 minutes.

Formula: Decimal = Degrees + Minutes/60 + Seconds/3600

Example: 38°42' = 38 + 42/60 = 38 + 0.7 = 38.7°

🔍 Inverse Functions (Finding Angles)

Using sin⁻¹, cos⁻¹, tan⁻¹

To find an angle when you know the trig value, use inverse functions:

  • If sin θ = 0.5, then θ = sin⁻¹(0.5) = 30°
  • If cos θ = 0.866, then θ = cos⁻¹(0.866) ≈ 30°
  • For sec/csc/cot: first find the reciprocal, then use inverse

Note: Inverse functions return angles in [0°, 90°] for our purposes.

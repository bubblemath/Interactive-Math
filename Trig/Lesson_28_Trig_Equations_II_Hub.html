📝 Substitution Step-by-Step Builder

Solve sin(2x) = √3/2 in [0, 2π) step by step:

1 Substitute: Let u = 2x

Equation becomes: sin(u) = √3/2

2 ⚠️ ADJUST THE INTERVAL!

If x ∈ [0, 2π), then u = 2x ∈ [0, 4π)

This is the critical step!

3 Solve sin(u) = √3/2 in [0, 4π)

In [0, 2π): u = π/3, 2π/3

In [2π, 4π): u = π/3 + 2π, 2π/3 + 2π = 7π/3, 8π/3

All u values: π/3, 2π/3, 7π/3, 8π/3

4 Solve for x: divide by 2

x = π/6, π/3, 7π/6, 4π/3

Solutions: x = π/6, π/3, 7π/6, 4π/3

🚨 Don't Forget to Divide!

After finding all u values, you must divide by the coefficient to get x!

If u = 2x, then x = u/2

➗ Half-Angle Equation Workshop

📏 Interval Halver Tool

For equations with x/2, the interval gets smaller:

If x ∈ [0, 2π), then x/2 ∈ [0, π)

💡 Key Insight: When solving sin(x/2) = k, you find solutions for x/2 in [0, π), then multiply by 2 to get x values in [0, 2π).

📝 Half-Angle Equation Example

Solve 2sin²(x/2) − sin(x/2) = 0 in [0, 2π):

1 Factor: sin(x/2)[2sin(x/2) − 1] = 0
2 Set each factor = 0:

sin(x/2) = 0 OR sin(x/2) = 1/2

3 Solve in [0, π): (since x/2 ∈ [0, π) when x ∈ [0, 2π))

sin(x/2) = 0 → x/2 = 0 → x = 0

sin(x/2) = 1/2 → x/2 = π/6 → x = π/3

Solutions: x = 0, π/3

🔄 Using Half-Angle Identities

Sometimes you need to convert using half-angle identities:

sin²(x/2) = 1/4

↓ Use: sin²(x/2) = (1 − cos x)/2

(1 − cos x)/2 = 1/4

↓ Solve for cos x

cos x = 1/2

Quick Check: Which identity would you use to solve cos²(x/2) = 3/4?

🔄 Before/After Transformer

See how identity substitution transforms equations:

cos(2x) = sin x

↓ Use: cos(2x) = 1 − 2sin²x

1 − 2sin²x = sin x

↓ Rearrange

2sin²x + sin x − 1 = 0

Now it's a quadratic in sin x!

📝 Complete Solution: cos(2x) = sin x

1 Substitute identity: 1 − 2sin²x = sin x
2 Rearrange: 2sin²x + sin x − 1 = 0
3 Factor: (2sin x − 1)(sin x + 1) = 0
4 Solve:

sin x = 1/2 → x = π/6, 5π/6

sin x = −1 → x = 3π/2

Solutions in [0, 2π): x = π/6, 5π/6, 3π/2

🔬 Law of Cosines Discovery Lab

🎯 The Big Question

The Pythagorean Theorem tells us that for a right triangle: c² = a² + b²

But what happens when angle C is NOT 90°? How do we "correct" the formula?

📐 Interactive Triangle Explorer

Adjust the angle C and watch how the Law of Cosines adapts:

c² = a² + b² − 2ab·cos(C)

Pythagorean (if C = 90°): c² = 64 + 36 = 100 → c = 10

Law of Cosines: c² = 64 + 36 − 96·cos(90°) = 100 − 0 = 100 → c = 10

Correction term: −2ab·cos(C) = 0

🔍 What to Notice:
  • When C = 90°, cos(90°) = 0, so the correction term disappears!
  • When C < 90° (acute), cos(C) > 0, so we subtract → c gets smaller
  • When C > 90° (obtuse), cos(C) < 0, so we add → c gets larger

📜 Derivation Walkthrough

Click through to see how the Law of Cosines is derived:

Step 1: Place triangle ABC with C at the origin and B on the positive x-axis.
Step 2: Coordinates: C = (0, 0), B = (a, 0), A = (b·cos C, b·sin C)
Step 3: Use the distance formula for side c (from A to B):
c² = (b·cos C − a)² + (b·sin C)²
Step 4: Expand:
c² = b²cos²C − 2ab·cos C + a² + b²sin²C
Step 5: Use the Pythagorean identity sin²C + cos²C = 1:
c² = a² + b²(cos²C + sin²C) − 2ab·cos C
c² = a² + b² − 2ab·cos C

All Three Forms of the Law of Cosines

a² = b² + c² − 2bc·cos A

b² = a² + c² − 2ac·cos B

c² = a² + b² − 2ab·cos C

Use when given: SAS (two sides and included angle) or SSS (all three sides)

🌍 Real-World Application Simulator

🏙️ City Triangle Calculator

Calculate the area of the triangle formed by three cities using Heron's Formula!

Heron's Formula

Semiperimeter: s = (a + b + c) / 2

Area = √[s(s−a)(s−b)(s−c)]

Use when all three sides are known (SSS) and no angle is given.

📍 Example: LA – New York – Montreal

The distances between these cities form a triangle:

  • LA to New York: 2,451 miles
  • New York to Montreal: 331 miles
  • Montreal to LA: 2,427 miles
Step 1: Identify the sides
a = 2,451 mi, b = 331 mi, c = 2,427 mi
Step 2: Calculate semiperimeter
s = (2,451 + 331 + 2,427) / 2 = 5,209 / 2 = 2,604.5
Step 3: Calculate (s − a), (s − b), (s − c)
s − a = 2,604.5 − 2,451 = 153.5
s − b = 2,604.5 − 331 = 2,273.5
s − c = 2,604.5 − 2,427 = 177.5
Step 4: Apply Heron's Formula
Area = √[2,604.5 × 153.5 × 2,273.5 × 177.5]
Area = √[1.612 × 10¹¹] ≈ 401,500 mi²

🏔️ Surveying Scenario: SAS Problem

Scenario: A surveyor needs to find the distance across a lake. She measures:

  • Distance from point A to point B: 150 meters
  • Distance from point A to point C: 200 meters
  • Angle at A (between AB and AC): 65°

Find the distance BC across the lake.

Given: b = 150 m, c = 200 m, A = 65° (SAS case)
Apply Law of Cosines:
a² = b² + c² − 2bc·cos A
a² = 150² + 200² − 2(150)(200)·cos(65°)
a² = 22,500 + 40,000 − 60,000 × 0.4226
a² = 62,500 − 25,356 = 37,144
Solution:
a = √37,144 ≈ 192.7 meters

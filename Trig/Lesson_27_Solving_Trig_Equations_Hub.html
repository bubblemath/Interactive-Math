🌟 Why Does This Matter?

Solving trig equations is essential for:

  • 🌊 Physics: Finding times when oscillating systems reach specific positions
  • 🏗️ Engineering: Determining angles for structural loads and forces
  • 🧭 Navigation: Calculating bearings, headings, and positions
  • 🎵 Music/Sound: Finding frequencies that produce specific harmonics
  • 📊 Calculus: Solving optimization problems involving trig functions

⚠️ Key Distinction: Identity vs. Equation

Identity

True for ALL values in the domain

sin²x + cos²x = 1

We VERIFY identities

Equation

True for SOME values only

sin x = 1/2

We SOLVE equations

🔄 Equation Transformation Workshop

📐 The "Substitution" Simulator

For equations like sin(2θ) = 1/2, use substitution:

Solve: sin(2x) = 1/2 in [0, 2π)

1 Let u = 2x

The equation becomes: sin(u) = 1/2

2 Adjust the interval!

If x ∈ [0, 2π), then u = 2x ∈ [0, 4π)

⚠️ This is the key step many students miss!

3 Solve sin(u) = 1/2 in [0, 4π)

u = π/6, 5π/6, π/6 + 2π, 5π/6 + 2π

u = π/6, 5π/6, 13π/6, 17π/6

4 Solve for x: divide by 2

x = π/12, 5π/12, 13π/12, 17π/12

Solutions: x = π/12, 5π/12, 13π/12, 17π/12

⚠️ "The Division Trap" Detector

See why dividing by a trig function loses solutions:

Solve: sin x cos x = cos x in [0, 2π)

❌ WRONG: Divide by cos x

sin x cos x = cos x

sin x = 1 (divided both sides by cos x)

x = π/2

Only 1 solution found!

Lost solutions where cos x = 0

✓ CORRECT: Factor

sin x cos x − cos x = 0

cos x (sin x − 1) = 0

cos x = 0 OR sin x = 1

All 3 solutions found!

x = π/2, 3π/2 (from cos x = 0)

x = π/2 (from sin x = 1)

🚨 THE DIVISION TRAP

When you divide by a trig expression, you lose all solutions where that expression equals zero!

ALWAYS factor instead of dividing!

🧩 Factoring Quadratic Trig Equations

Treat trig functions like variables when factoring:

Solve: 2cos²x + cos x − 1 = 0

1 Recognize the quadratic form

Let u = cos x: 2u² + u − 1 = 0

2 Factor

(2u − 1)(u + 1) = 0

3 Solve each factor

2u − 1 = 0 → u = 1/2 → cos x = 1/2 → x = π/3, 5π/3

u + 1 = 0 → u = −1 → cos x = −1 → x = π

Solutions in [0, 2π): x = π/3, π, 5π/3

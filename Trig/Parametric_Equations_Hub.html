🌟 Why Does This Matter?

Parametric equations describe motion through space over time—they tell us not just where something is, but when it gets there:

  • 🎬 Computer Animation: Every animated character, from Pixar films to video games, moves along parametric paths. Animators define x(t) and y(t) to control motion.
  • 🚀 Physics & Projectile Motion: When you throw a ball, its position at time t is naturally parametric: x(t) = v₀t·cos(θ), y(t) = v₀t·sin(θ) − ½gt²
  • 🤖 Robotics: Robot arms move through space using parametric equations to ensure smooth, precise movements.
  • 📍 GPS Navigation: Your location over time is a parametric curve—GPS tracks x(t), y(t) as you travel.
  • 🎢 Roller Coaster Design: Engineers use parametric equations to design smooth, thrilling tracks with controlled g-forces.
  • 🎵 Music & Sound: Lissajous figures (parametric curves) visualize the relationship between musical frequencies.

Parametric Equations: The Big Idea

Instead of y = f(x), we have:

x = f(t) and y = g(t)

where t is the parameter (often representing time)

🌟 Why Does This Matter?

Parametric equations capture motion—they tell us where an object is at each moment in time. This is fundamentally different from y = f(x), which only describes a static curve.

  • Direction matters: The same curve can be traced in different directions
  • Speed matters: The same path can be traveled at different speeds
  • Time matters: We know exactly when the object reaches each point

🔄 Same Curve, Different Parameterizations

These three parameterizations all trace the same parabola y = x², but with different behaviors:

Parameterization Direction Speed
x = t, y = t² Left to right → Constant in x
x = −t, y = t² Right to left ← Constant in x
x = t³, y = t⁶ Left to right → Slower near origin

🔄 Eliminating the Parameter

🌟 Why Does This Matter?

Sometimes we need to convert parametric equations back to rectangular form (y as a function of x) to:

  • Recognize the type of curve (line, parabola, circle, etc.)
  • Find intersections with other curves
  • Apply calculus techniques that require y = f(x)

Two Main Techniques

1. Substitution Method:

Solve one equation for t, substitute into the other

2. Pythagorean Identity:

Use cos²t + sin²t = 1 for trig parametrics

📐 Step-by-Step Converter

Select a parametric equation to see the elimination process:

Eliminate t from: x = 2t + 1, y = 3t − 2

Step 1: Solve the x-equation for t

x = 2t + 1 → 2t = x − 1 → t = (x − 1)/2

Step 2: Substitute into the y-equation

y = 3t − 2 = 3·(x − 1)/2 − 2

Step 3: Simplify

y = (3x − 3)/2 − 2 = (3x − 3 − 4)/2 = (3x − 7)/2

Rectangular form: y = (3x − 7)/2

This is a line with slope 3/2

🎨 Parametric Curve Gallery

🌟 Why Does This Matter?

Parametric equations can describe curves that are impossible to write as y = f(x). This gallery showcases the beautiful variety of parametric curves used in engineering, physics, and art.

📏 Line

x = 1 + 2t, y = 2 + t

Direction determined by coefficients of t

⭕ Circle

x = r·cos(t), y = r·sin(t)

Traced counterclockwise as t increases

🥚 Ellipse

x = a·cos(t), y = b·sin(t)

a = horizontal radius, b = vertical radius

🎢 Cycloid

x = t − sin(t), y = 1 − cos(t)

Path of a point on a rolling wheel!

🎵 Lissajous Figure

x = A·sin(at + δ), y = B·sin(bt)

Created by combining two perpendicular oscillations—used in oscilloscopes and music visualization!

🎵 Lissajous Figures in the Real World

  • Oscilloscopes: Display Lissajous patterns to compare signal frequencies
  • Music: Visualize the relationship between musical intervals (octave = 2:1, fifth = 3:2)
  • Laser shows: Create patterns by oscillating mirrors at different frequencies
  • Physics: Model coupled harmonic oscillators and pendulums

💡 Why Vectors? Until now, we've worked with scalars—quantities with only magnitude (like temperature or distance). Vectors have both magnitude AND direction, making them essential for describing forces, velocities, and displacements in physics and engineering.

🔬 Vector Fundamentals Discovery Lab

🎯 What is a Vector?

A vector is a quantity with both magnitude (size) and direction.

  • Scalar examples: 50 mph, 98.6°F, $25
  • Vector examples: 50 mph north, 10 N force at 30°

See how multiplying a vector by a scalar changes its length (and possibly direction):

Original: v = ⟨3, 2⟩, |v| = 3.61

Scaled: kv = ⟨3, 2⟩, |kv| = 3.61

🔍 Key Observations:
  • When k > 1: Vector stretches (same direction)
  • When 0 < k < 1: Vector shrinks (same direction)
  • When k < 0: Vector reverses direction!
  • When k = 0: Result is the zero vector ⟨0, 0⟩

Vector Notation

Component Form: v = ⟨a, b⟩

Magnitude: |v| = √(a² + b²)

Direction Angle: θ = tan⁻¹(b/a) (adjust for quadrant!)

⚠️ Quadrant Adjustment for Direction Angles:
Quadrant Signs (a, b) Adjustment
I (+, +) θ = tan⁻¹(b/a)
II (−, +) θ = 180° + tan⁻¹(b/a)
III (−, −) θ = 180° + tan⁻¹(b/a)
IV (+, −) θ = 360° + tan⁻¹(b/a)

Conversion Formulas

Components from Magnitude/Direction:

a = |v| · cos(θ)     b = |v| · sin(θ)

Magnitude/Direction from Components:

|v| = √(a² + b²)     θ = tan⁻¹(b/a) (+ quadrant adjustment)

📐 Angle Between Vectors (Dot Product)

Find the angle between two vectors using the dot product:

Vector u

Vector v

Dot Product: u · v = (3)(2) + (4)(1) = 10

Magnitudes: |u| = 5, |v| = √5 ≈ 2.24

Formula: cos θ = (u · v) / (|u| |v|) = 10 / (5 × 2.24) ≈ 0.894

θ = cos⁻¹(0.894) ≈ 26.6°

Vector Operation Formulas

Addition: ⟨a₁, b₁⟩ + ⟨a₂, b₂⟩ = ⟨a₁ + a₂, b₁ + b₂⟩

Subtraction: ⟨a₁, b₁⟩ − ⟨a₂, b₂⟩ = ⟨a₁ − a₂, b₁ − b₂⟩

Scalar Multiplication: k⟨a, b⟩ = ⟨ka, kb⟩

Unit Vector: û = v / |v|

Dot Product: u · v = a₁a₂ + b₁b₂ = |u||v| cos θ

🧭 Navigation Problem

Scenario: A plane flies at 400 mph on a heading of 60° (measured from east). A wind blows at 50 mph from the west (heading 0°). Find the plane's actual ground speed and direction.

Plane velocity: ⟨400·cos(60°), 400·sin(60°)⟩ = ⟨200, 346.4⟩

Wind velocity: ⟨50·cos(0°), 50·sin(0°)⟩ = ⟨50, 0⟩

Resultant: ⟨200 + 50, 346.4 + 0⟩ = ⟨250, 346.4⟩

Ground speed: √(250² + 346.4²) = √(62500 + 119993) ≈ 427.4 mph

Actual direction: tan⁻¹(346.4/250) ≈ 54.2°

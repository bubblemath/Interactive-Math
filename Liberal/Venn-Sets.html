🌐 Universal Set

The Universal Set, denoted as U, consists of all elements considered in the discussion or problem.

Example: If we survey a class about pets, U = all students in the class.

🔣 Set Operations

Operation Symbol Meaning
Union A ∪ B All elements in A or B (or both) - "OR"
Intersection A ∩ B Elements in both A and B - "AND"
Complement A' Elements not in A
Difference A − B Elements in A but not in B

📏 Cardinal Number Formula

n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

Why subtract? Elements in the intersection get counted twice (once in A, once in B), so we subtract them once to avoid double-counting.

📊 Venn Diagram Regions

A two-set Venn diagram has 4 distinct regions:

  • Region I: A ∩ B' - elements in A but not B
  • Region II: A ∩ B - elements in both A and B
  • Region III: A' ∩ B - elements in B but not A
  • Region IV: A' ∩ B' - elements in neither (outside both)

⚖️ De Morgan's Laws

These laws show how complement interacts with union and intersection:

(A ∪ B)' = A' ∩ B'
(A ∩ B)' = A' ∪ B'

📊 Statistics Explorer: Central Tendency & Dispersion 📈

📖 Statistics Concepts Overview

This applet covers Measures of Central Tendency (mean, median, mode, midrange) and Measures of Dispersion (standard deviation). These help us understand the "center" and "spread" of data!

📊 Mean (Average)

Add all data values and divide by the number of values.

Mean = Σx / n

Sensitive to extreme values (outliers)

📍 Median (Middle)

The middle value when data is arranged in order.

Middle position = (n+1)/2

If n is even, average the two middle values

🎯 Mode (Most Frequent)

The value that occurs most often in the data set.

Highest frequency wins!

A data set can have no mode, one mode, or multiple modes

↔️ Midrange

The average of the highest and lowest values.

Midrange = (max + min) / 2

Quick estimate of center, but sensitive to outliers

📏 Standard Deviation

Measures how spread out the data is from the mean.

s = √[Σ(x-x̄)² / (n-1)]

Low s = data close to mean; High s = data spread out

🔢 Standard Deviation Steps

1. Find the mean (x̄)

2. Find each deviation: x - x̄

3. Square each deviation

4. Sum the squared deviations

5. Divide by n-1

6. Take the square root

📏 Standard Deviation

The standard deviation measures how spread out data is from the mean.

s = √[Σ(x - x̄)² / (n - 1)]

Steps to Calculate:

1. Find the mean (x̄)

2. Find each deviation: (data item - mean)

3. Square each deviation: (data item - mean)²

4. Sum all squared deviations: Σ(x - x̄)²

5. Divide by (n - 1)

6. Take the square root

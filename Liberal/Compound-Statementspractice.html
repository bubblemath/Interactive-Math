📚 Key Concepts

🔗 What are Compound Statements?

Statements formed by combining two or more simple statements are called compound statements. To connect these statements, we use connectives such as and, or, if…then, and if and only if.

📋 The Five Connectives

Type Symbol Read As Example
Negation ~ not p ~p: "It is not 6am"
Conjunction ∧ p and q p ∧ q: "It is 6am and the kids are awake"
Disjunction ∨ p or q p ∨ q: "It is 6am or the kids are awake"
Conditional → if p, then q p → q: "If it is 6am, then the kids are awake"
Biconditional ↔ p if and only if q p ↔ q: "It is 6am if and only if the kids are awake"

➡️ Conditional Statement Forms

The conditional p → q can be expressed in many ways:

FormExample (p: I study, q: I pass)
If p, then qIf I study, then I pass.
q if pI pass if I study.
p only if qI study only if I pass.
Only if q, pOnly if I pass, I study.
p is sufficient for qStudying is sufficient for passing.
q is necessary for pPassing is necessary for studying.

Terminology: In p → q, p is the hypothesis and q is the conclusion.

⚠️ Common Mistake: "p only if q"

"p only if q" means p → q, NOT q → p!

Think of it as: "The only way p can happen is if q happens" = "If p, then q"

Example: "I will go only if you go" means "If I go, then you go" (p → q)

⚠️ Common Mistake: Inclusive "Or"

In logic, "or" is inclusive — it means one or the other or BOTH can be true.

p ∨ q is TRUE when p is true, q is true, or both are true.

⚠️ Key Truth Table Facts

  • p ∧ q is TRUE only when BOTH p and q are TRUE
  • p ∨ q is FALSE only when BOTH p and q are FALSE
  • p → q is FALSE only when p is TRUE and q is FALSE
  • p ↔ q is TRUE only when p and q have the SAME truth value

See how parentheses change the meaning of statements!

Why Parentheses Matter

Parentheses are symbolic notations that mean the statements are grouped together. Negation only applies to the symbol immediately after the negation unless parentheses group more symbols.

~p ∧ q

~p ∧ q

English: "Not p, and q"

Only p is negated

Example: "She is not happy and she is wealthy"
~(p ∧ q)

~(p ∧ q)

English: "It is not true that p and q"

The entire conjunction is negated

Example: "It is not true that she is happy and wealthy"

❌ These are NOT the same!

~p ∧ q ≠ ~(p ∧ q)

When p is FALSE and q is TRUE:

  • ~p ∧ q = TRUE ∧ TRUE = TRUE
  • ~(p ∧ q) = ~(FALSE ∧ TRUE) = ~FALSE = TRUE

But when p is TRUE and q is TRUE:

  • ~p ∧ q = FALSE ∧ TRUE = FALSE
  • ~(p ∧ q) = ~(TRUE ∧ TRUE) = ~TRUE = FALSE

They happen to match here, but try p=TRUE, q=FALSE!

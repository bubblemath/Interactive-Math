📖 Statistics Concepts Overview

This interactive applet covers key concepts in organizing and displaying data. Learn how to create frequency distributions, grouped frequency tables, histograms, and stem-and-leaf plots!

📋 Frequency Distribution

A table that shows each data value and how many times it occurs (its frequency).

Example: Counting letter grades in a class

📊 Grouped Frequency Distribution

Data organized into classes (intervals) when there are many different values.

Class Width = difference between consecutive lower class limits

📈 Histogram

A bar graph where bars touch, showing frequency of data in each class interval.

Height of bar = frequency of that class

🌿 Stem-and-Leaf Plot

Displays data by splitting each value into a stem (tens digit) and leaf (ones digit).

Shows shape of distribution while keeping actual values

📉 Frequency Polygon

A line graph connecting the midpoints of histogram bars.

Shows the shape of the distribution

⚠️ Misleading Graphs

Watch for: missing titles, manipulated scales, 3D effects that distort data, and truncated axes.

📊 Grouped Frequency Distributions

When data has many different values, we group them into classes (intervals).

Class Width = Upper Limit - Lower Limit + 1

Lower class limit: smallest value in a class | Upper class limit: largest value in a class

🌿 Stem-and-Leaf Plots

A stem-and-leaf plot organizes data by splitting each value into:

• Stem: The tens digit (or higher place values)

• Leaf: The ones digit

Example: 47 → Stem: 4, Leaf: 7

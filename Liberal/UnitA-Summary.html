📊 Quick Reference Cheat Sheet

💡 Study Tip: Use this consolidated reference while practicing. Memorizing these rules will dramatically speed up your problem-solving!

🔢 A1: Signed Number Rules

Multiplication/Division:
Same signs → Positive
Different signs → Negative
Examples:
\((-3) \times (-4) = +12\)
\((-3) \times (+4) = -12\)
\((-12) \div (-3) = +4\)

🍕 A2: Fraction Rules

Multiply: \(\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}\)

Divide: Keep-Change-Flip
\(\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}\)
Add/Subtract: Need common denominators!
\(\frac{1}{3} + \frac{1}{4} = \frac{4}{12} + \frac{3}{12} = \frac{7}{12}\)

⚡ A3: Exponent Rules

\(a^m \cdot a^n = a^{m+n}\)

PEMDAS Order:
Parentheses → Exponents → Multiply/Divide → Add/Subtract
⚠️ Critical Distinction:
\((-2)^4 = 16\) (negative inside)
\(-2^4 = -16\) (negative outside)

📐 A4: Polynomial Rules

Adding Multiplying
Add coefficients
Keep exponents		 Multiply coefficients
Add exponents
\(6x + 7x = 13x\) \(6x \cdot 7x = 42x^2\)
Distributive: \(a(b + c) = ab + ac\)

√ A5: Radical Rules

Product Rule:
\(\sqrt{ab} = \sqrt{a} \cdot \sqrt{b}\)

Even vs. Odd Index:
Even root of negative = No real solution
Odd root of negative = Negative result
\(\sqrt{-16}\) = No real solution
\(\sqrt[3]{-8} = -2\)

⚠️ Common Pitfalls to Avoid

🎯 Learn from Mistakes: These are the most frequent errors students make. Study each one carefully to avoid them on your work!

❌ A3: Negative Base vs. Negative Outside

WRONG: Thinking \(-3^2 = 9\)
CORRECT: \(-3^2 = -(3^2) = -9\)
But \((-3)^2 = (-3)(-3) = 9\)
💡 Why? Without parentheses, the negative sign is NOT part of the base. The exponent only applies to the 3, then the negative is applied after. Parentheses make the negative part of the base.

❌ A4: Confusing Addition with Multiplication

WRONG: \(6x + 7x = 42x^2\)
CORRECT: \(6x + 7x = 13x\)
(When ADDING, add coefficients and keep exponents the same)
💡 Why? This error confuses the addition rule with the multiplication rule. When multiplying: \(6x \cdot 7x = 42x^2\) (multiply coefficients, ADD exponents). When adding: just add the coefficients!

❌ A5: Adding Under the Radical

WRONG: \(\sqrt{9} + \sqrt{16} = \sqrt{25} = 5\)
CORRECT: \(\sqrt{9} + \sqrt{16} = 3 + 4 = 7\)
💡 Why? You cannot add numbers under separate radicals! Evaluate each radical first, then add the results. The product rule \(\sqrt{ab} = \sqrt{a} \cdot \sqrt{b}\) works for multiplication, NOT addition.

❌ A5: Even Root of a Negative

WRONG: \(\sqrt{-16} = -4\)
CORRECT: \(\sqrt{-16}\) has NO real solution
But \(\sqrt[3]{-8} = -2\) (odd roots of negatives ARE real)
💡 Why? No real number times itself (an even number of times) gives a negative result. However, odd roots of negatives DO have real solutions because a negative times itself an odd number of times stays negative.

❌ A5: Not Fully Simplifying Radicals

WRONG: \(\sqrt{48} = 4\sqrt{4}\)
CORRECT: \(\sqrt{48} = \sqrt{16 \cdot 3} = 4\sqrt{3}\)
💡 Why? The radicand must not contain any perfect square factors. \(\sqrt{4} = 2\), so \(4\sqrt{4}\) is not fully simplified. Always find the LARGEST perfect square factor.

❓ Frequently Asked Questions

How do I use the interactive applets?

Click the "Launch Applet" button for any lesson to open the interactive practice tool. Each applet includes reference materials, practice problems, and instant feedback. Work through problems in sets of 5, and use the timer to build speed and confidence.

What order should I study the lessons?

We recommend following the sequence A1 → A2 → A3 → A4 → A5. Each lesson builds on concepts from previous lessons. For example, exponent rules (A3) are essential for understanding polynomial multiplication (A4) and radical simplification (A5).

How do I enter special symbols like exponents and radicals?

Use the Symbol Template provided in the Practice tab. Click inside the answer box, then click the symbol buttons to insert exponents (², ³, ⁴) or radicals (√, ∛, ∜). You can also type "sqrt" for square root or use the ^ symbol for exponents.

The math formulas aren't displaying correctly. What should I do?

The applets use MathJax to render mathematical notation. If formulas appear as code, try refreshing the page or using a different browser (Chrome, Firefox, or Edge work best). Make sure JavaScript is enabled in your browser settings.

Can I use these applets on my phone or tablet?

Yes! All applets are mobile-responsive and work on smartphones and tablets. For the best experience on smaller screens, use landscape orientation when working on practice problems.

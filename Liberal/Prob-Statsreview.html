📖 Unit Overview

This unit covers Counting Methods, Probability, and Statistics. Review the key concepts below!

🔢 Permutations

Ordered arrangements where order MATTERS.

ₙPᵣ = n! / (n-r)!

🎯 Combinations

Selections where order does NOT matter.

ₙCᵣ = n! / [r!(n-r)!]

🚫 Complement ("Not")

Everything NOT in event E.

P(not E) = 1 - P(E)

⊕ Mutually Exclusive

Events that CANNOT happen together.

P(A or B) = P(A) + P(B)

∩ Not Mutually Exclusive

Events that CAN happen together.

P(A or B) = P(A) + P(B) - P(A∩B)

🔗 Independent Events

One event doesn't affect the other.

P(A and B) = P(A) × P(B)

⛓️ Dependent Events

One event affects the other.

P(A and B) = P(A) × P(B|A)

❓ Conditional Probability

Probability of B given A occurred.

P(B|A) = P(A and B) / P(A)

📊 Mean

Sum divided by count.

Mean = Σx / n

📍 Median

Middle value when ordered.

Position = (n+1)/2

🎯 Mode & Midrange

Mode: Most frequent value

Midrange = (max+min)/2

📏 Standard Deviation

Measures spread from mean.

s = √[Σ(x-x̄)²/(n-1)]

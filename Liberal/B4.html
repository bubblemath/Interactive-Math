📚 Quick Reference Guide

🎯 Lesson Objectives: Master the two forms of linear equations, find intercepts, calculate slope using the detailed formula, and graph lines using slope-intercept form.

📝 Two Forms of Linear Equations

Standard Form
\(Ax + By = C\)
Slope-Intercept Form
\(y = mx + b\)
Examples:
Standard: \(2x + 3y = 6\)
Slope-Intercept: \(y = \frac{2}{3}x - 4\)

🎯 Finding Intercepts

X-Intercept: Set \(y = 0\), solve for \(x\)
Form: \((x, 0)\)
Y-Intercept: Set \(x = 0\), solve for \(y\)
Form: \((0, y)\)

⚠️ Common Error: Don't confuse which variable to set to zero! X-intercept → set Y to zero.

📐 Detailed Slope Formula

\(m = \frac{y_2 - y_1}{x_2 - x_1}\)
  • \((x_1, y_1)\) = first point
  • \((x_2, y_2)\) = second point
  • Numerator = rise (vertical change)
  • Denominator = run (horizontal change)
Example: Points \((2, 3)\) and \((5, 9)\)
\(m = \frac{9-3}{5-2} = \frac{6}{3} = 2\)

📊 Graphing from Slope-Intercept

Step 1: Identify \(m\) (slope) and \(b\) (y-intercept)
Step 2: Plot the y-intercept \((0, b)\)
Step 3: Use slope \(\frac{rise}{run}\) to find more points
Step 4: Draw the line through the points
Example: \(y = \frac{3}{4}x - 2\)
• \(m = \frac{3}{4}\), \(b = -2\)
• Plot \((0, -2)\), then rise 3, run 4

✍️ Writing Equations

Given slope \(m\) and y-intercept \(b\):
Write \(y = mx + b\)
Given slope \(m\) and a point \((x_1, y_1)\):
1. Use \(y = mx + b\)
2. Substitute \(m\), \(x_1\), \(y_1\)
3. Solve for \(b\)
4. Write final equation
Example: \(m = 2\), point \((3, 5)\)
\(5 = 2(3) + b\) → \(b = -1\)
Equation: \(y = 2x - 1\)

🔄 Converting Between Forms

Standard → Slope-Intercept:
Solve for \(y\)
Example: \(2x + 3y = 6\)
\(3y = -2x + 6\)
\(y = -\frac{2}{3}x + 2\)

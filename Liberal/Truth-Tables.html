📚 Key Concepts

📊 What is a Truth Table?

A truth table is an organized table for systematically evaluating all possible truth value combinations of a compound statement. It shows every possible scenario and the resulting truth value.

🔢 Number of Rows Formula

The number of rows in a truth table depends on the number of simple statements:

StatementsFormulaRows
2 (p, q)2² = 4 rows
3 (p, q, r)2³ = 8 rows
4 (p, q, r, s)2⁴ = 16 rows
n statements2ⁿ = 2ⁿ rows

🔗 The Five Connectives - Truth Rules

Connective Symbol TRUE When... FALSE When...
Negation ~p p is FALSE p is TRUE
Conjunction p ∧ q BOTH are TRUE At least one is FALSE
Disjunction p ∨ q At least one is TRUE BOTH are FALSE
Conditional p → q p is FALSE, OR q is TRUE ONLY when p=T and q=F
Biconditional p ↔ q p and q have SAME value p and q have DIFFERENT values

⚠️ Common Mistake: The Conditional

p → q is FALSE only when p is TRUE and q is FALSE!

Many students think "If p then q" should be false when p is false. But in logic, a conditional with a false hypothesis is always TRUE (vacuously true).

💡 Quick Reference

  • ∧ (AND): TRUE only when BOTH are TRUE
  • ∨ (OR): FALSE only when BOTH are FALSE
  • → (IF-THEN): FALSE only when TRUE → FALSE
  • ↔ (IFF): TRUE only when SAME values
  • ~ (NOT): Flips the truth value

🔍 The Pattern Rule:
  • First column (p): Half TRUE, then half FALSE
  • Second column (q): Alternating pairs (TT, FF, TT, FF...)
  • Third column (r): Alternating singles (T, F, T, F...)
  • Each column alternates twice as fast as the previous!

📝 Example: "I eat green eggs" (p) and "I eat ham" (q)

pqMeaning
TTI eat green eggs AND I eat ham
TFI eat green eggs but NOT ham
FTI don't eat green eggs but I eat ham
FFI don't eat green eggs AND I don't eat ham

📐 Column Order Guide

Learn which columns to complete first!

🎯 The Golden Rule: Work from the INSIDE OUT

Always complete columns in this order:

1
Simple Statements — Fill in p, q, r columns with the standard T/F pattern
2
Negations of Simple Statements — Calculate ~p, ~q, ~r
~p just flips all values in the p column
3
Innermost Parentheses — Work on expressions inside parentheses first
In [(p ∨ q) ∧ ~p] → q, do (p ∨ q) first
4
Outer Operations — Combine results moving outward
Then do (p ∨ q) ∧ ~p
5
Main Connective — The final column uses the main connective
Finally, apply → q to get the result

⚠️ Common Mistake: Wrong Column Order

Students often try to fill in the final column first. This leads to errors!

Always work from simple statements → negations → inner parentheses → outer operations → final result

❌ Common Mistake: Skipping Steps

Don't try to calculate the final answer directly! Always:

  1. Start with the given values of p and q
  2. Calculate negations (~p, ~q)
  3. Calculate inner operations (p ∧ q)
  4. Combine for the final result

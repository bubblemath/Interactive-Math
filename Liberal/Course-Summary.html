📋 Course Overview

Welcome to Liberal Arts Mathematics! This course develops essential mathematical reasoning skills through four interconnected units. You'll explore the foundations of set theory and logical reasoning, master practical financial mathematics for real-world decision making, and learn to analyze data using probability and statistics. Each unit builds upon the previous, creating a comprehensive toolkit for critical thinking and quantitative literacy.

Use this portal to navigate through all course materials, interactive applets, and practice resources. Each unit summary page contains detailed lessons with hands-on activities designed to reinforce your understanding.

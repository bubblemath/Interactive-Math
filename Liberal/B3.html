Cartesian Coordinates, Intercepts, and Slope

📚 Quick Reference Guide

🎯 Lesson Objectives: Understand the Cartesian coordinate system, plot points, find intercepts, and calculate slope.

📍 Cartesian Coordinate System

  • x-axis: Horizontal axis
  • y-axis: Vertical axis
  • Origin: Point (0, 0) where axes intersect
  • Ordered Pair: (x, y) - horizontal first, vertical second

⚠️ Order matters! (3, 5) and (5, 3) are different points!

🗺️ The Four Quadrants

  • Quadrant I (+, +)
  • Quadrant II (−, +)
  • Quadrant III (−, −)
  • Quadrant IV (+, −)
Examples:
(3, 4) → Quadrant I
(-2, 5) → Quadrant II
(-1, -3) → Quadrant III
(4, -2) → Quadrant IV

🎯 X-Intercept & Y-Intercept

X-intercept: Where line crosses x-axis
y = 0, point is (x, 0)

Y-intercept: Where line crosses y-axis
x = 0, point is (0, y)
To find intercepts:
• X-int: Set y = 0, solve for x
• Y-int: Set x = 0, solve for y

📐 Slope Formula

\(m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{rise}}{\text{run}}\)
Example: Points (1, 2) and (4, 8)
\(m = \frac{8-2}{4-1} = \frac{6}{3} = 2\)

📊 Types of Slope

↗ Positive: rises left to right
↘ Negative: falls left to right
→ Zero: horizontal line
↑ Undefined: vertical line

📝 Slope-Intercept Form

\(y = mx + b\)

m = slope
b = y-intercept
Example: \(y = 2x + 3\)
Slope = 2, Y-intercept = (0, 3)

📊 Rules & Math Facts Reference

💡 Key Insight: The most common error is confusing addition rules with multiplication rules. Study the comparison table below carefully!

⚠️ CRITICAL: Adding vs. Multiplying Polynomials

Operation Rule Example
ADDING Add coefficients
Keep exponents SAME		 6x + 7x = 13x
MULTIPLYING Multiply coefficients
ADD exponents		 6x · 7x = 42x²
❌ Common Error: Writing 6x + 7x = 42x² (WRONG!)
This confuses addition with multiplication. When ADDING, you only add the coefficients!

🏷️ Algebraic Properties Quick Reference

Commutative Property

a + b = b + a
a · b = b · a
3 + 5 = 5 + 3 = 8

Associative Property

(a + b) + c = a + (b + c)
(a · b) · c = a · (b · c)
(2 + 3) + 4 = 2 + (3 + 4)

Distributive Property

a(b + c) = ab + ac
3(x + 4) = 3x + 12

Identity Properties

a + 0 = a
a · 1 = a
7 + 0 = 7, 7 · 1 = 7

🔹 Like Terms

Terms are like terms if they have the same variable raised to the same exponent.

3x² and 9x² ✓ Like
3x² and 9x³ ✗ Not Like

🔹 Exponent Rule for Multiplication

When multiplying same bases, add the exponents.

xm · xn = xm+n
x³ · x⁴ = x3+4 = x⁷

📦 Lesson 2: Sets & Subsets

SymbolNameMeaning
∈Element ofa ∈ A means "a is in set A"
⊆SubsetA ⊆ B means "all of A is in B"
⊂Proper SubsetA ⊂ B means "A ⊆ B but A ≠ B"
∅Empty SetThe set with no elements
n(A)CardinalityNumber of elements in A

Key Formula: Counting Subsets

If n(A) = k, then A has 2k subsets
Example: {a, b, c} has 2³ = 8 subsets

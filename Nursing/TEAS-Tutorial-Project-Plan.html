04

Phased Rollout

suggested build sequence
PHASE 1
Science core

Body Systems hub + Cardiovascular, Respiratory, Nervous, Eukaryotic Cell, Macromolecules, Punnett Square, Periodic Table, Bonding, Balancing Equations, pH & Solutions.

PHASE 2
Reading high-yield

Main Idea, Inferences, Context Clues, Reading Charts & Graphics, Argument & Evidence.

PHASE 3
English high-yield

Punctuation, Subject–Verb Agreement, Fragments & Run-ons, Pronoun Agreement, Vocabulary in Context, Word Parts.

PHASE 4
Fill out coverage

Remaining body systems plus the medium- and lower-yield topics across all three sections.

PHASE 5
Polish & unify

Section landing pages, a unified TEAS home page linking all four sections, and timed full-section practice modes.

05

Design Notes

carried over from the Math resource
  • Keep the existing card model: icon + title + short description + skill tags + Launch button.
  • Maintain the three-part applet structure — Key Concepts → Worked Examples → Practice with instant feedback.
  • Mirror the Math page layout on each section landing page for a consistent look across the suite.
  • Science benefits most from visual interactivity (clickable diagrams, drag-to-label) — this is where the format earns its keep.
  • Add a top-level TEAS Prep home page linking Math, Reading, Science, and English to tie the suite together.

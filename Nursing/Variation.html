Understanding Variability, Variance, Standard Deviation & Choosing the Right Measure

📚 Measures of Variability - Key Concepts

🤔 Why Study Variability?

Houston Airport Example: In the 1990s, passengers at George Bush Intercontinental Airport complained about waiting ~7 minutes for bags. The solution? They moved the arrival gates farther from baggage claim, so passengers walked longer but waited less. Same total time, less perceived waiting!

Variability helps us:

  • Distinguish between usual and unusual values
  • Understand the spread or consistency of data
  • Make better decisions based on data patterns

Measures of Variability

Range

The simplest measure of spread - the difference between the highest and lowest values.

Range = Max − Min

Interquartile Range (IQR)

The spread of the middle 50% of data. Less affected by outliers than range.

IQR = Q3 − Q1

Deviation

How far each data value is from the mean. Can be positive or negative.

Deviation = x − x̄

Variance & Standard Deviation

Population Variance (σ²)

The average of squared deviations from the mean for an entire population.

σ² = Σ(x − μ)²N

Sample Variance (s²)

Estimate of population variance using sample data. Divide by (n-1) for unbiased estimate.

s² = Σ(x − x̄)²n − 1

Population Standard Deviation (σ)

Square root of population variance. In the same units as the original data.

σ = √σ²

Sample Standard Deviation (s)

Square root of sample variance. Most commonly used measure of spread.

s = √s²

⚠️ Key Difference: Population vs Sample

Population: Divide by N (total population size)

Sample: Divide by n − 1 (sample size minus 1) — this corrects for bias!

Steps to Calculate Variance & Standard Deviation

📝 Step-by-Step Process

  1. Calculate the mean (x̄ or μ)
  2. Find each deviation (x − mean) for every data value
  3. Square each deviation (x − mean)²
  4. Sum the squared deviations Σ(x − mean)²
  5. Divide by N (population) or n−1 (sample) to get variance
  6. Take the square root to get standard deviation

💡 Why Square the Deviations?

If we just added up the deviations, they would sum to zero (positives cancel negatives). Squaring makes all values positive and emphasizes larger deviations.

📈 Describing Data: Choosing the Right Measure

📺 Video Reference

This content is based on concepts from: Mean, Median, Mode - When to Use Each

Measures of Center Review

Mean (Average)

Add all values and divide by the count. Affected by every value in the data set.

x̄ = Σx ÷ n

Median (Middle)

The middle value when data is ordered. Not affected by extreme values.

Middle value of sorted data

Mode (Most Frequent)

The value that occurs most often. Can be used with categorical data.

Most frequent value

When to Use Each Measure

✅ Use the MEAN when:

  • Data is spread out evenly (symmetric distribution)
  • There are no extreme values (outliers)
  • Data values are close together
  • Example: Test scores of 78, 82, 85, 79, 81

📊 Use the MEDIAN when:

  • Data contains extreme values (outliers)
  • Data is skewed (not symmetric)
  • Values are far apart
  • Example: Salaries of $35K, $42K, $38K, $45K, $950K (CEO)

🔢 Use the MODE when:

  • Data is categorical (not numerical)
  • About half or more of the data points are the same value
  • You want to know the most popular or most common response
  • Example: Favorite colors, weather types, shoe sizes

The Effect of Outliers

Example: How Outliers Affect the Mean

Data Set A: 3, 5, 1

Mean = (3 + 5 + 1) ÷ 3 = 3

Median = 3

✓ Mean and median are the same - good representation!

Data Set B: 3, 5, 19

Mean = (3 + 5 + 19) ÷ 3 = 9

Median = 5

⚠️ The outlier (19) pulls the mean up - median is better!

Quick Decision Guide

🎯 Ask Yourself These Questions:

  1. Is the data categorical? → Use Mode
  2. Are there extreme values or outliers? → Use Median
  3. Is the data symmetric with no outliers? → Use Mean
  4. Does one value repeat much more than others? → Consider Mode

Step-by-step: three classics

1. Affect vs. Effect

Affect is almost always a verb — an action. Effect is almost always a noun — a result.
Read the sentence: "The new medication will ______ her blood pressure."
Ask: is this an action (something the med does)? Yes — so you need the verb.
Memory trick: Affect = Action. Both start with A.

✓ "The new medication will affect her blood pressure." (Its effect — the result — is lower numbers.)

2. Their / There / They're

Their = belonging to them (it has the word "heir" in it — an heir owns things).
There = a place (it contains "here" — both are places).
They're = "they are." The apostrophe replaces the missing a. If "they are" fits, use they're.
Test: "______ charting their vitals over ______." Try "they are" in the first blank — it works. Second blank is a place.

✓ "They're charting their vitals over there."

3. The spelling trap: "separate"

Most people write "seperate." The middle vowel is actually an a.
Memory trick: there's a rat in sepa rate.
Picture separating things — and spot the rat hiding in the middle.

✓ Correct spelling: separate (sep-A-rate).

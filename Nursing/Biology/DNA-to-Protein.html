🧬 TEAS Biology: DNA to Protein

How the instructions in your genes become working proteins

🧬 The Central Dogma

Your DNA is an instruction manual for building proteins, and proteins do most of the work in your body. Getting from genes to proteins takes two steps: transcription (DNA is copied into mRNA) and translation (the mRNA is read to build a protein). Biologists sum it up as:

DNA→RNA→Protein

🔗 Base-Pairing Rules

  • In DNA: A pairs with T, and G pairs with C.
  • Transcription (DNA → mRNA): the DNA template is copied into RNA, but U replaces T. So DNA A → mRNA U, DNA T → mRNA A, DNA G → mRNA C, DNA C → mRNA G.

🔤 Codons

  • The ribosome reads mRNA in groups of three bases called codons.
  • Each codon codes for one amino acid. tRNA brings the matching amino acid.
  • AUG is the start codon (it also codes for methionine); special stop codons end the protein.

🌟 High-Yield TEAS Facts

  • Flow: DNA → (transcription) → mRNA → (translation) → protein.
  • Transcription happens in the nucleus; translation happens at the ribosome.
  • RNA uses U (uracil) instead of T (thymine).
  • DNA pairs: A–T, G–C. A codon = 3 bases = 1 amino acid.
  • tRNA delivers amino acids; the ribosome links them into a protein.
  • The DNA base order sets the amino acid order, which determines the protein.

✏️ Example 1: Transcription, Base by Base

📝 Copy a DNA template into mRNA

Take the DNA template strand T A C. Transcription builds a complementary mRNA strand, using U instead of T:

  • DNA T → mRNA A
  • DNA A → mRNA U (not T — RNA uses U)
  • DNA C → mRNA G

So DNA TAC becomes mRNA AUG. (And AUG is the start codon!)

🔤 Example 2: Translation, Codon by Codon

📝 Read mRNA three bases at a time

Take the mRNA AUG GGA UUU. The ribosome reads it in codons (groups of three):

  • AUG → Methionine (Met) — the start
  • GGA → Glycine (Gly)
  • UUU → Phenylalanine (Phe)

tRNA brings each amino acid, and they link into the protein Met–Gly–Phe. The order of codons set the order of amino acids.

🩸 Example 3: Why One Base Matters (Sickle Cell)

📝 A tiny DNA change can change a protein

  • In sickle cell anemia, a single DNA base change in the gene for hemoglobin changes one codon.
  • That changes one amino acid in the hemoglobin protein.
  • The altered protein makes red blood cells bend into a sickle shape, which can block blood flow.

Big idea: because DNA → mRNA → protein, even a small change in the DNA can change the protein and affect health. This is why the base sequence matters so much.

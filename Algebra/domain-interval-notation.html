Algebra / Precalculus · Domains

Find what breaks the function; the rest is the domain.

A domain is every input the function can actually take. Only a few things break it: dividing by zero, or taking an even root of a negative. Find those critical x-values, mark them on a number line, and read off the allowed regions in interval notation. Polynomials break nowhere — they're all real numbers.

What restricts the domain

Rational — exclude where the denominator is 0.
Even radical — the radicand must be \(\ge 0\).
Sum of radicals — intersect each condition.
Radical in a denominator — radicand \(> 0\) (strict).

Worked examples

One reasoning pattern at a time.

Rational · factor the cubic denominator

\(f(x) = \dfrac{x+1}{x^3 - 7x + 6}\)
Factor the denominator
\(x^3-7x+6 = (x-1)(x-2)(x+3)\).
Exclude the zeros
The denominator is 0 at \(x=1,\,2,\,-3\). Those are the only restrictions.
Write the domain
\((-\infty,-3)\cup(-3,1)\cup(1,2)\cup(2,\infty)\). Every break is an open parenthesis.

Radical · quadratic radicand

\(f(x) = \sqrt{x^2 - x - 6}\)
Radicand ≥ 0
Need \(x^2-x-6 \ge 0\), i.e. \((x-3)(x+2)\ge 0\).
Sign analysis
The product is \(\ge 0\) outside the roots: \(x\le -2\) or \(x\ge 3\).
Write the domain
\((-\infty,-2]\cup[3,\infty)\). The boundaries are included — square brackets.

Sum of radicals · intersect

\(f(x) = \sqrt{x-2} + \sqrt{5-x}\)
Each radicand ≥ 0
\(x-2\ge 0 \Rightarrow x\ge 2\), and \(5-x\ge 0 \Rightarrow x\le 5\).
Intersect (both must hold)
Both conditions at once: \(2\le x\le 5\). It's an intersection, not a union.
Write the domain
\([2,5]\).

Rational + radical · combine ≥ and ≠

\(f(x) = \dfrac{\sqrt{x+1}}{x-3}\)
Radicand ≥ 0
\(x+1\ge 0 \Rightarrow x\ge -1\), so \([-1,\infty)\).
Denominator ≠ 0
\(x-3\ne 0 \Rightarrow x\ne 3\). Punch that point out.
Write the domain
\([-1,3)\cup(3,\infty)\). Closed at \(-1\) (allowed), open at \(3\) (excluded).
Cheat sheet

Critical points first, brackets second.

Polynomial

Nothing breaks it. Domain is (-∞, ∞) — all real numbers.

Rational

Factor the denominator, set each factor to 0, exclude those x with open parentheses.

Even radical

Set the radicand \(\ge 0\) and solve. Boundaries are included — square brackets.

Combinations

Two radicals → intersect. Radical over a polynomial → \(\ge 0\) and \(\ne 0\). Radical in the denominator → strict \(> 0\).

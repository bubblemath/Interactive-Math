An exponential or log graph is pinned down by three anchor points and one asymptote. Learn to take the parent's anchors, push each one through the transformation rule, and read off exactly where the new graph lives — using a table of values, not guesswork.

The parent & its skeleton

Three anchors and one asymptote

The exponential parent is $f(x)=b^{x}$ for a base $b>0,\ b\neq 1$. Whatever the base, the curve always threads the same three anchor points, and it never crosses one horizontal line — its asymptote.

PARENT · f(x) = b^x
name$x$$y=b^{x}$why it's an anchor
left anchor$-1$$\tfrac{1}{b}$$b^{-1}$ — one step below the pivot
pivot$0$$1$every base passes through $(0,1)$
right anchor$1$$b$$b^{1}$ — the base itself

Horizontal asymptote: $y=0$ (the curve flattens toward the $x$-axis on one side).

The logarithm parent is $f(x)=\log_b(x)$ — the inverse of $b^{x}$. Reflect the exponential across the line $y=x$ and you get the log. Its three anchors are the exponential's anchors with the coordinates swapped, and its asymptote turns from horizontal into vertical.

PARENT · f(x) = log_b(x)
name$x$$y=\log_b x$why it's an anchor
low anchor$\tfrac{1}{b}$$-1$$\log_b b^{-1}=-1$
root$1$$0$every base passes through $(1,0)$
high anchor$b$$1$$\log_b b=1$

Vertical asymptote: $x=0$ (the curve dives down the $y$-axis).

INVERSE BRIDGE

Because $b^{x}$ and $\log_b x$ are inverses, every fact transfers by swapping coordinates and swapping axes. The pivot $(0,1)$ becomes the root $(1,0)$; the right anchor $(1,b)$ becomes the high anchor $(b,1)$; the horizontal asymptote $y=0$ becomes the vertical asymptote $x=0$. Master one family and you've nearly mastered both.

Bases you'll meet here include small integers ($2,3,5$), the reciprocal base $\tfrac12$, the common base $10$, and the natural base $e\approx 2.718$. The anchors are defined the same way for all of them — only the numbers change.

One rule moves every anchor

From parent point to transformed point

Write the transformed exponential in full general form:

$f(x)=a\cdot b^{\,c\,(x-h)}+k$

Write the transformed logarithm in full general form:

$f(x)=a\cdot \log_b\!\big(c\,(x-h)\big)+k$

Every parameter does one job. Read them straight off the equation:

a — vertical

Multiplies the $y$-coordinate. $|a|>1$ stretches vertically, $0<|a|<1$ compresses, and $a<0$ reflects across the $x$-axis.

c — horizontal

Acts inside on the input. The $x$-coordinate gets divided by $c$. $|c|>1$ compresses horizontally, $0<|c|<1$ stretches, and $c<0$ reflects across the vertical.

h — shift

Slides the graph right by $h$. Note the sign trap: $x-3$ means $h=3$ (right), $x+3$ means $h=-3$ (left).

k — shift

Slides the graph up by $k$. This is the parameter that moves an exponential's asymptote.

Now the punchline. Take any parent anchor $(x_0,\,y_0)$ and the same rule sends it to:

$(x_0,\ y_0)\ \xrightarrow{\ \ }\ \left(\,h+\dfrac{x_0}{c}\,,\ \ a\,y_0+k\,\right)$

Inputs (the first coordinate) are governed by $h$ and $c$; outputs (the second coordinate) by $a$ and $k$. The horizontal factor lands in a denominator — divide by $c$, don't multiply.

ASYMPTOTE

The parent's horizontal asymptote $y=0$ only feels the vertical shift, so it moves to $y=k$. The range becomes $(k,\infty)$ when $a>0$ and $(-\infty,k)$ when $a<0$.

ASYMPTOTE

The parent's vertical asymptote $x=0$ sits where the inside hits zero: $c(x-h)=0$. That's $x=h$ regardless of $a$, $c$, or $k$ — so the asymptote moves to $x=h$. The domain is everything on the side of $x=h$ where $c(x-h)>0$.

⚠ Four ways this goes wrong

  • Sign of $h$. $x-4$ shifts right 4, not left. The shift is the opposite sign of what's written.
  • Multiplying by $c$ instead of dividing. The input coordinate becomes $h+\frac{x_0}{c}$. A horizontal compression ($|c|>1$) pulls anchors toward $x=h$.
  • Wrong axis for the asymptote. Exponentials keep a horizontal asymptote ($y=k$); logs keep a vertical one ($x=h$). Don't swap them.
  • Order of operations. Apply the inside ($c$, then $h$) to the input and the outside ($a$, then $k$) to the output. Mixing a vertical shift into the exponent is the classic slip.

Worked examples

Three anchors, one asymptote, every time
EXAMPLE 1 · EXPONENTIAL

$f(x)=-2\cdot 3^{\,2(x-1)}+4$

1
Read the parameters. Matching $a\,b^{c(x-h)}+k$: $\;a=-2,\ b=3,\ c=2,\ h=1,\ k=4$. Base $3$, so the parent $3^{x}$ has anchors $(-1,\tfrac13),(0,1),(1,3)$ and asymptote $y=0$.
2
Apply the rule $(x_0,y_0)\to\big(1+\tfrac{x_0}{2},\,-2y_0+4\big)$ to each anchor:
parent pointinput: $1+\tfrac{x_0}{2}$output: $-2y_0+4$new point
$(-1,\tfrac13)$$1-\tfrac12=\tfrac12$$-\tfrac23+4=\tfrac{10}{3}$$(\tfrac12,\tfrac{10}{3})$
$(0,1)$$1+0=1$$-2+4=2$$(1,2)$
$(1,3)$$1+\tfrac12=\tfrac32$$-6+4=-2$$(\tfrac32,-2)$
3
Move the asymptote. Horizontal stays horizontal: $y=0\to y=k=4$. Since $a=-2<0$ the graph is flipped and sits below the line, so the range is $(-\infty,4)$.
EXAMPLE 2 · LOGARITHM

$f(x)=2\,\log_2\!\big(3(x-4)\big)-1$

1
Read the parameters. Matching $a\,\log_b(c(x-h))+k$: $\;a=2,\ b=2,\ c=3,\ h=4,\ k=-1$. Base $2$, so the parent $\log_2 x$ has anchors $(\tfrac12,-1),(1,0),(2,1)$ and asymptote $x=0$.
2
Apply the rule $(x_0,y_0)\to\big(4+\tfrac{x_0}{3},\,2y_0-1\big)$:
parent pointinput: $4+\tfrac{x_0}{3}$output: $2y_0-1$new point
$(\tfrac12,-1)$$4+\tfrac{1}{6}=\tfrac{25}{6}$$-2-1=-3$$(\tfrac{25}{6},-3)$
$(1,0)$$4+\tfrac13=\tfrac{13}{3}$$0-1=-1$$(\tfrac{13}{3},-1)$
$(2,1)$$4+\tfrac23=\tfrac{14}{3}$$2-1=1$$(\tfrac{14}{3},1)$
3
Move the asymptote. Vertical stays vertical: it sits where the inside is zero, $3(x-4)=0\Rightarrow x=h=4$. Since $c=3>0$, the inside is positive for $x>4$, so the domain is $(4,\infty)$.
EXAMPLE 3 · NATURAL BASE e

$f(x)=e^{\,x+1}-2$

1
Read the parameters. Rewrite $e^{x+1}$ as $e^{\,1\cdot(x-(-1))}$: $\;a=1,\ b=e,\ c=1,\ h=-1,\ k=-2$. Parent $e^{x}$ anchors are $(-1,\tfrac1e),(0,1),(1,e)$ — keep $e$ symbolic, don't round yet.
2
Apply the rule $(x_0,y_0)\to\big(-1+x_0,\,y_0-2\big)$:

$(0,1)$$-1$$-1$$(-1,-1)$

Only round at the very end if you need to plot: $\tfrac1e-2\approx-1.63$ and $e-2\approx0.72$.

3
Move the asymptote. $y=0\to y=k=-2$; with $a=1>0$ the range is $(-2,\infty)$.

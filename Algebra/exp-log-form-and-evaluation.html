Logarithms · form & meaning

A logarithm is just an exponent in disguise.

Every log statement is an exponential statement wearing different clothes. Learn to read the same three numbers — base, exponent, result — in either form, flip between them at will, and read a log's value straight off the powers of its base.

b^{y}=x \quad\Longleftrightarrow\quad \log_b x = y

with $b>0$, $b\neq 1$, and $x>0$

The one equivalence

Same three numbers, two outfits

A logarithm answers a question about exponents: “what power turns the base into the result?” The base stays the base in both forms. The exponent in $b^{y}=x$ is exactly the value of the log. And the result $x$ becomes the thing you take the log of — the argument.

Converting between forms

Move the numbers, don't recompute them

Converting is pure bookkeeping — no arithmetic required. Find the three roles, then drop each into its slot in the other form. The base keeps its subscript spot; the exponent and the value trade places around the equals sign.

EXPONENTIAL → LOG

$b^{y}=x \;\Rightarrow\; \log_b x = y$

1
Label the roles in $4^{3/2}=8$: base 4, exponent 3/2, result 8.
2
Place them: base stays the subscript, the result becomes the argument, the exponent becomes the value → $\log_4 8 = \tfrac32$.
3
Negatives ride along. $2^{-3}=\tfrac18 \;\Rightarrow\; \log_2\!\left(\tfrac18\right) = -3$.
LOG → EXPONENTIAL

$\log_b x = y \;\Rightarrow\; b^{y}=x$

1
Label the roles in $\log_5 125 = 3$: base 5, value 3, argument 125.
2
Rebuild the power: base to the value equals the argument → $5^{3}=125$.
3
Fractional bases too. $\log_{1/2} 8 = -3 \;\Rightarrow\; \left(\tfrac12\right)^{-3}=8$.
NAMED LOGS

Two bases get special symbols, but they convert exactly the same way. Common log: $\log x$ means $\log_{10} x$, so $\log x = y \Leftrightarrow 10^{y}=x$. Natural log: $\ln x$ means $\log_{e} x$, so $\ln x = y \Leftrightarrow e^{y}=x$.

Evaluating simple logs

Ask one question every time

To evaluate $\log_b x$, ask: “$b$ to what power gives $x$?” That sentence turns every evaluation into a power you already know. Listing the powers of the base makes the answer jump out.

EXAMPLE · INTEGER POWER

$\log_2 16 = \;?$  →  “2 to what power is 16?”

$2^{1}$=2
$2^{2}$=4
$2^{3}$=8
$2^{4}$=16 ✓

The rung that lands on 16 is $2^{4}$, so $\log_2 16 = 4$.

ROOTS → FRACTIONS

A fractional answer means a root. $\log_{25} 5 = \tfrac12$ because $25^{1/2}=\sqrt{25}=5$. Likewise $\log_8 4 = \tfrac23$ since $8^{2/3}=(\sqrt[3]{8})^{2}=2^{2}=4$.

RECIPROCALS → NEGATIVES

A value below 1 means a negative exponent. $\log_2\!\left(\tfrac18\right) = -3$ because $2^{-3}=\tfrac{1}{2^{3}}=\tfrac18$. And $\log_{10} 0.01 = -2$ since $10^{-2}=0.01$.

Common pitfalls

Where points get lost

⚠ Four slips to avoid

  • Which number is the base? It's the small subscript on $\log_b$, and it matches the base of the power. In $\log_2 16$ the base is 2, not 16.
  • The answer is an exponent. A logarithm's output is the power, not the argument. $\log_2 16$ is 4, not 16.
  • You can't take a log of $0$ or a negative. Since $b^{y}$ is always positive, the argument $x$ must be $>0$.
  • The base can't be $1$ (or $\le 0$). $1^{y}=1$ for every $y$, so it could never single out a value.

Practice

Drill all four moves

Pick a mode. Fractions are fine (type 3/2 or -3); for base $e$ answers, type e or e^2.

Precalculus · Slope & equations of lines

Slope is just rise over run — until the run hits zero.

Every line's steepness is the change in \(y\) over the change in \(x\). Most of the time that's a tidy fraction. But when two points share a height the rise is zero (a flat line), and when they share an \(x\) the run is zero — and dividing by zero makes the slope undefined. Knowing which is which tells you whether the equation is \(y=\) or \(x=\).

The two special cases — remember HOY · VUX

H · O · Y
Horizontal → slope 0 → y = k
V · U · X
Vertical → Undefined slope → x = h

Skill 01 · Finding slope from two points

Drag the two points and watch the slope build itself.

Slope is \(m = \dfrac{\text{rise}}{\text{run}} = \dfrac{y_2 - y_1}{x_2 - x_1}\). Drag the points below: the colored staircase shows the run and the rise, and the panel keeps the slope reduced. Line the points up horizontally and the rise — and the slope — go to zero. Stack them vertically and the run goes to zero, making the slope undefined.

Drag either point. run (Δx)   rise (Δy)

Keep the order consistent: if \(y_2\) comes first on top, \(x_2\) comes first on the bottom. Subtracting in mismatched orders flips the sign.

Worked examples · all three kinds

Rational slope
Through \((-2,-1)\) and \((3,3)\): \[m=\frac{3-(-1)}{3-(-2)}=\frac{4}{5}\]
Zero slope (horizontal)
Through \((-3,2)\) and \((4,2)\): \[m=\frac{2-2}{4-(-3)}=\frac{0}{7}=0\] A flat line, \(y=2\).
Undefined slope (vertical)
Through \((5,-1)\) and \((5,4)\): \[m=\frac{4-(-1)}{5-5}=\frac{5}{0}\;\text{undefined}\] A vertical line, \(x=5\).

Skill 02 · Point-slope & slope-intercept

One slope and one point are all you need to write the line.

Drop the slope and a point straight into point-slope form, \(y - y_1 = m(x - x_1)\). Then, if you want slope-intercept form, just solve that for \(y\) to get \(y = mx + b\). Given two points instead? Find the slope first, then use either point — both lead to the same line.

The two oblique forms

\[ \text{point-slope: } y - y_1 = m(x - x_1) \qquad \text{slope-intercept: } y = mx + b \]

Watch the signs: a point like \((-1,\,-3)\) gives \(y-(-3)=m(x-(-1))\), i.e. \(y+3=m(x+1)\).

Worked example · from a slope and a point

Slope \(m=-2\) through \((1,-3)\)
Step 1 · Point-slope
\[y-(-3) = -2(x-1) \;\Rightarrow\; y+3 = -2(x-1)\]
Step 2 · Distribute
\[y+3 = -2x+2\]
Step 3 · Solve for y → slope-intercept
\[y = -2x - 1\]

Worked example · from two points

Through \((-1,5)\) and \((2,-1)\)
Step 1 · Slope first
\[m=\frac{-1-5}{2-(-1)}=\frac{-6}{3}=-2\]
Step 2 · Point-slope (using \((2,-1)\))
\[y-(-1) = -2(x-2) \;\Rightarrow\; y+1=-2(x-2)\]
Step 3 · Slope-intercept
\[y = -2x+4-1 \;\Rightarrow\; y=-2x+3\]

Skill 03 · Horizontal & vertical lines

The two lines that don't fit \(y = mx + b\) the usual way.

A horizontal line has slope \(0\): every point shares the same \(y\), so the equation is simply \(y = k\). A vertical line has undefined slope: every point shares the same \(x\), so it's \(x = h\) — and it has no slope-intercept form at all. Remember HOY (Horizontal, 0, Y=) and VUX (Vertical, Undefined, X=).

Read the points: equal \(y\)-coordinates → horizontal → \(y=\) that value. Equal \(x\)-coordinates → vertical → \(x=\) that value.

Worked example · horizontal

Cheat sheet

Slope first, then pick the form.

Slope

\(m=\dfrac{y_2-y_1}{x_2-x_1}\). Same order top and bottom. \(0\) rise → slope 0; \(0\) run → undefined.

Point-slope

\(y-y_1=m(x-x_1)\). Drop in a slope and any point on the line; mind double negatives.

Slope-intercept

\(y=mx+b\). Solve point-slope for \(y\), or find \(b\) by plugging a point into \(y=mx+b\).

HOY · VUX

Horizontal → slope 0 → \(y=k\). Vertical → undefined → \(x=h\) (no slope-intercept form).

Precalculus · Parallel & perpendicular lines

Same slope to run parallel; flip-and-negate to turn perpendicular.

Both tasks start by reading the given line's slope. To go parallel, keep that slope exactly. To go perpendicular, take its negative reciprocal — flip the fraction and change the sign — so the two slopes multiply to \(-1\). Then run the new line through the given point. Horizontal and vertical lines follow the same idea by orientation.

The two slope rules

Parallel ∥
same slope: \(m_{\parallel} = m\)
Lines never meet; identical steepness.
Perpendicular ⟂
negative reciprocal: \(m_{\perp} = -\dfrac{1}{m}\)
Slopes multiply to \(-1\); they meet at a right angle.

Skill 02 · Writing a parallel line

Keep the slope, then steer it through the given point.

Read the slope of the given line and use that exact slope. Drop it and the given point into point-slope form, then solve for \(y\). Parallel to a horizontal line is still horizontal (\(y=\) the point's \(y\)); parallel to a vertical line is still vertical (\(x=\) the point's \(x\)).

Parallel keeps orientation. Same slope for oblique lines; same horizontal/vertical type for special lines. The new constant comes from the given point.

Worked example · parallel to an oblique line

Parallel to \(y = 2x + 1\) through \((3, -2)\)
Step 1 · Same slope
The given slope is \(2\), so the parallel line also has slope \(2\).
Step 2 · Point-slope
\[y-(-2) = 2(x-3) \;\Rightarrow\; y+2 = 2(x-3)\]
Step 3 · Slope-intercept
\[y = 2x - 6 - 2 \;\Rightarrow\; y = 2x - 8\]

Worked example · parallel to horizontal / vertical

Parallel to \(y = 4\) through \((1,-3)\); and parallel to \(x = 5\) through \((2,7)\)
Parallel to a horizontal line
It stays horizontal; use the point's \(y\): \[y = -3\]
Parallel to a vertical line
It stays vertical; use the point's \(x\): \[x = 2\]

Skill 03 · Writing a perpendicular line

Negative reciprocal for the slope, then through the given point.

Take the given slope, flip it, and change the sign — that's the perpendicular slope. Use it with the given point in point-slope form and solve for \(y\). Perpendicular to a horizontal line is vertical (\(x=\) the point's \(x\)); perpendicular to a vertical line is horizontal (\(y=\) the point's \(y\)).

Perpendicular swaps orientation. Oblique → negative reciprocal slope. Horizontal ↔ vertical for the special lines. Never write \(-1/0\); reason by the right angle instead.

Worked example · perpendicular to an oblique line

Perpendicular to \(y = 2x + 1\) through \((4, 1)\)
Step 1 · Negative reciprocal
Flip \(2 = \tfrac{2}{1}\) to \(\tfrac{1}{2}\), change the sign: slope \(=-\tfrac{1}{2}\).
Step 2 · Point-slope
\[y-1 = -\tfrac{1}{2}(x-4)\]
Step 3 · Slope-intercept
\[y = -\tfrac{1}{2}x + 2 + 1 \;\Rightarrow\; y = -\tfrac{1}{2}x + 3\]

Worked example · perpendicular to horizontal / vertical

Perpendicular to \(y = 4\) through \((2,-1)\); and perpendicular to \(x = 5\) through \((3,6)\)
Perpendicular to a horizontal line → vertical
A right angle with a flat line is straight up; use the point's \(x\): \[x = 2\]
Perpendicular to a vertical line → horizontal
A right angle with an upright line is flat; use the point's \(y\): \[y = 6\]

Cheat sheet

Read the slope, set the new slope, run through the point.

Parallel

Same slope. \(m_{\parallel}=m\). Then point-slope through the given point and solve for \(y\).

Perpendicular

Negative reciprocal: flip the fraction and change the sign. \(m_{\perp}=-\tfrac{1}{m}\); slopes multiply to \(-1\).

Parallel to H / V

Keeps orientation. Parallel to \(y=k\) is \(y=\) (point's y); parallel to \(x=h\) is \(x=\) (point's x).

Perpendicular to H / V

Swaps orientation. ⟂ to \(y=k\) is \(x=\) (point's x); ⟂ to \(x=h\) is \(y=\) (point's y).

Precalculus · Graphing lines

A slope is a set of steps you walk on the grid.

To graph a line you only need a place to start and a direction to go. With two points, plot both and connect them. With a slope and a point, plot the point, then let the slope walk you to the next one: the top number is the rise (up or down), the bottom number is the run (right). Then draw the line through them and extend it.

Three kinds of slope, three kinds of step

Rational
\(m=\tfrac{2}{3}\): up 2, right 3
Zero
rise 0 → flat, \(y=\) const
Undefined
run 0 → straight up, \(x=\) const

Set a point and a slope, and watch the staircase build the line.

Drag the point, then choose a slope. The colored staircase shows the run (right) and the rise (up or down) that step you to the next lattice point — connect them and extend, and that's the line. Switch to slope 0 to see it flatten into a horizontal line, or undefined to see it stand straight up with no horizontal run at all.

Worked examples · slope & a point

Rational slope
Slope \(\tfrac{3}{4}\) through \((-2,-1)\): plot \((-2,-1)\), then go up 3 and right 4 to \((2,2)\). Connect and extend.
Negative slope
Slope \(-\tfrac{1}{2}\) through \((0,2)\): from \((0,2)\) go down 1 and right 2 to \((2,1)\). (Or up 1, left 2.)
Zero slope
Slope \(0\) through \((1,-3)\): rise is 0, so the line is flat — a horizontal line, \(y=-3\).
Undefined slope
Undefined slope through \((4,2)\): run is 0, so you can't step sideways — a vertical line, \(x=4\).
Skill 02 · Graphing from two points

Plot both points, then connect and extend.

With two points there's no stepping to do — plot each one, draw the straight line through them, and extend it past both with arrows. Click the grid below to plot the two given points; the line appears once both are placed correctly.

Skill 03 · Graphing from a slope and a point

The point is placed for you — step the slope to a second point.

The given point is already on the grid. Use the slope to walk to another point on the line and click it: rise then run for a rational slope, horizontally for slope 0, vertically for an undefined slope. Any correct second point works. The line draws once you land on one.

Cheat sheet

Start at a point; let the slope do the walking.

Two points

Plot both, connect with a straight line, and extend past both ends with arrows.

Rational slope

Top = rise (up if +, down if −). Bottom = run (right). An integer \(m\) is \(\tfrac{m}{1}\): run 1.

Zero slope

Rise 0 → horizontal line. Same \(y\) everywhere: \(y=\) the point's \(y\).

Undefined slope

Run 0 → vertical line. Same \(x\) everywhere: \(x=\) the point's \(x\). No sideways step.

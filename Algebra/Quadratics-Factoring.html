📚 The AC Method for Factoring Quadratics

Goal: Factor \(ax^2 + bx + c = 0\) and solve for \(x\) using the Zero Product Property.

Key Reminder: Always check for a Greatest Common Factor (GCF) first!

🔮 The Discriminant Connection

Before factoring, you can check if a quadratic IS factorable using the discriminant:

\(D = b^2 - 4ac\)

If \(D\) is a perfect square (0, 1, 4, 9, 16, 25, ...), the quadratic is factorable over the integers!

  • \(D = 25\) → \(\sqrt{25} = 5\) ✓ Factorable!
  • \(D = 17\) → \(\sqrt{17} \approx 4.12\) ✗ Not factorable (use Quadratic Formula)

🔄 The AC Method Steps

1

Write in standard form
\(ax^2 + bx + c = 0\)

2

Multiply
\(a \cdot c\)

3

Find two numbers:
Product = \(ac\), Sum = \(b\)

4

Split the middle term using those numbers

5

Factor by grouping

6

Set each factor = 0
and solve for \(x\)

📊 Visual Grouping Animation

Watch how \(2x^2 + 7x + 3 = 0\) factors step by step:

Step 1: \(a \cdot c = 2 \cdot 3 = 6\). Find two numbers: product = 6, sum = 7 → 1 and 6
Step 2: Split middle term: \(2x^2 + \color{blue}{1x} + \color{red}{6x} + 3 = 0\)
Step 3: Group: \((\color{blue}{2x^2 + 1x}) + (\color{red}{6x + 3}) = 0\)
Step 4: Factor each group: \(\color{blue}{x(2x + 1)} + \color{red}{3(2x + 1)} = 0\)
Step 5: Factor out common binomial: \(\color{green}{(2x + 1)}(x + 3) = 0\)
Step 6: Solve: \(2x + 1 = 0\) → \(x = -\frac{1}{2}\) OR \(x + 3 = 0\) → \(x = -3\)

💡 When to Use Factoring

  • The discriminant \(D = b^2 - 4ac\) is a perfect square
  • You can find integer factors that work
  • The equation is already in or easily converted to standard form

If factoring doesn't work: Use the Quadratic Formula instead!

📝 Worked Examples with Factor Charts

Example 1: When \(a = 1\)

Solve: \(x^2 + 7x + 12 = 0\)

Step 1: Identify \(a = 1\), \(b = 7\), \(c = 12\). Calculate \(a \cdot c = 1 \cdot 12 = 12\)
Step 2: Find factor pairs of 12:
FactorsSum
1 × 1213
2 × 68
3 × 47 ✓
Step 3: Split middle term: \(x^2 + 3x + 4x + 12 = 0\)
Step 4: Group: \((x^2 + 3x) + (4x + 12) = 0\)
Step 5: Factor: \(x(x + 3) + 4(x + 3) = 0\) → \((x + 3)(x + 4) = 0\)
Step 6: Solve: \(x + 3 = 0\) → \(x = -3\) OR \(x + 4 = 0\) → \(x = -4\)
Solutions: \(x = -3\) or \(x = -4\)

Example 2: When \(a \neq 1\)

Solve: \(2x^2 + 7x + 3 = 0\)

Step 1: Identify \(a = 2\), \(b = 7\), \(c = 3\). Calculate \(a \cdot c = 2 \cdot 3 = 6\)
Step 2: Find factor pairs of 6:
FactorsSum
1 × 67 ✓
2 × 35
Step 3: Split middle term: \(2x^2 + 1x + 6x + 3 = 0\)
Step 4: Group: \((2x^2 + x) + (6x + 3) = 0\)
Step 5: Factor: \(x(2x + 1) + 3(2x + 1) = 0\) → \((2x + 1)(x + 3) = 0\)
Step 6: Solve: \(2x + 1 = 0\) → \(x = -\frac{1}{2}\) OR \(x + 3 = 0\) → \(x = -3\)
Solutions: \(x = -\frac{1}{2}\) or \(x = -3\)

Example 3: Negative Coefficients

Solve: \(x^2 - x - 12 = 0\)

Step 1: Identify \(a = 1\), \(b = -1\), \(c = -12\). Calculate \(a \cdot c = 1 \cdot (-12) = -12\)
Step 2: Find factor pairs of -12 (one positive, one negative):
FactorsSum
1 × (-12)-11
(-1) × 1211
2 × (-6)-4
(-2) × 64
3 × (-4)-1 ✓
(-3) × 41
Step 3: Split middle term: \(x^2 + 3x - 4x - 12 = 0\)
Step 4: Group: \((x^2 + 3x) + (-4x - 12) = 0\)
Step 5: Factor: \(x(x + 3) - 4(x + 3) = 0\) → \((x + 3)(x - 4) = 0\)
Step 6: Solve: \(x + 3 = 0\) → \(x = -3\) OR \(x - 4 = 0\) → \(x = 4\)
Solutions: \(x = -3\) or \(x = 4\)

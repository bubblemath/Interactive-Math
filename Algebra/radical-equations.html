Precalculus · Solving radical equations

Square both sides — but the equation might lie to you.

Raising both sides to a power undoes a radical, but it isn't reversible: it can hand you solutions that don't actually work. That's why every radical equation ends with one non-negotiable step — substitute each answer back and confirm it. This page drills the solving and the checking together.

Why checking is mandatory

\(\sqrt{x+6} = x \;\Rightarrow\; x+6 = x^2 \;\Rightarrow\; x^2-x-6=0 \;\Rightarrow\; x = 3 \text{ or } x = -2\)
x = 3 → √9 = 3 ✓ keep  ·  x = −2 → √4 = 2 ≠ −2 ✗ extraneous, throw it out

Skill 01 · A single radical

Isolate the radical, raise both sides to its power, then check.

The whole method is four moves: get the radical alone on one side, raise both sides to the index (square for a square root), solve the resulting equation, and verify every answer in the original. Isolating first matters — squaring a sum like \((\sqrt{x}+3)^2\) creates a messy middle term, while squaring an isolated radical cleanly removes it.

The order that saves you: isolate → power → solve → verify. Never square before the radical is alone.

The pattern

\[ \sqrt{\,\text{stuff}\,} = E \;\xrightarrow{\;(\,\cdot\,)^2\;}\; \text{stuff} = E^{2} \]

Squaring is the step that can introduce extraneous solutions, so it obligates you to check. A square root is never negative, so if \(E\) is negative for a candidate, that candidate fails.

Worked example · with an extraneous solution

Solve \(\;\sqrt{x+6} = x\)
Step 1 · Radical already isolated
The square root stands alone on the left, so we can square directly.
Step 2 · Square both sides ⚠ may add extraneous roots
\[(\sqrt{x+6})^2 = x^2 \;\Rightarrow\; x+6 = x^2\]
Step 3 · Solve the quadratic
\[x^2 - x - 6 = 0 \Rightarrow (x-3)(x+2)=0 \Rightarrow x = 3,\; x = -2\]
Step 4 · Verify in the ORIGINAL
\(x=3:\ \sqrt{9}=3\) ✓    \(x=-2:\ \sqrt{4}=2 \ne -2\) ✗
Solution
\[x = 3 \quad(\,x=-2 \text{ is extraneous}\,)\]

Skill 02 · Two radicals

Isolate one radical, square, then repeat — and mind the middle term.

With two radicals you usually can't clear them in one shot. Isolate one radical, square both sides, and you'll often be left with a single remaining radical. Isolate that one and square again. The classic mistake is squaring a binomial like \((1+\sqrt{x})^2\) and forgetting the cross term — it is not just \(1 + x\).

Watch the cross term: \((a+\sqrt{u})^2 = a^2 + \color{#0E8C8B}{2a\sqrt{u}} + u\). That highlighted middle piece is where points are lost.

The rhythm

\[ \text{isolate} \to \text{square} \to \text{isolate again} \to \text{square again} \to \text{solve} \to \text{verify} \]

Each squaring can add an extraneous solution, so verification is doubly important here.

Worked example

Solve \(\;\sqrt{x+5} = 1 + \sqrt{x}\)
Step 1 · One radical already isolated
The left side is a single radical; square both sides.
Step 2 · Square — keep the cross term
\[x+5 = 1 + \color{#0E8C8B}{2\sqrt{x}} + x\]The highlighted middle term is what makes this still contain a radical.
Step 3 · Isolate the remaining radical
\[5 - 1 = 2\sqrt{x} \;\Rightarrow\; 4 = 2\sqrt{x} \;\Rightarrow\; \sqrt{x} = 2\]
Step 4 · Square again, then verify
\(x = 4\). Check: \(\sqrt{9} = 1 + \sqrt{4} \Rightarrow 3 = 3\) ✓
Solution
\[x = 4\]

Skill 03 · Verify & catch extraneous solutions

A candidate is only a solution if it survives the original equation.

Substitute each candidate back into the equation you started with — not a squared version — and compute both sides independently. If they match, keep it. If they don't, it's extraneous and gets discarded. Below, decide for yourself whether each candidate is valid, then reveal the substitution.

The graph · why extraneous solutions exist

Squaring secretly invites the negative branch to the party.

The true solutions of \(\sqrt{\text{stuff}} = \text{line}\) are where the radical curve meets the line. But squaring both sides also satisfies \(-\sqrt{\text{stuff}} = \text{line}\) — the reflected, negative branch the original never allowed. Cheat sheet

Keep this beside you while you practice.

One radical

Isolate the radical, raise both sides to the index, solve, then verify. Never square before isolating.

Two radicals

Isolate one, square (keep the \(2a\sqrt{u}\) cross term), isolate the leftover radical, square again, solve, verify.

Verify

Substitute into the original equation and compute each side separately. A square root is never negative.

Extraneous

A candidate that solves the squared equation but fails the original. Common when one side could be negative.

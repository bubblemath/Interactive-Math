Algebra / Precalculus · Composition

Feed \(x\) through \(g\), then through \(f\) — and check both gates.

A composition \((f\circ g)(x)=f(g(x))\) runs \(x\) through the inner function first, then the outer one. The domain has two conditions, not one: \(x\) must be allowed into \(g\), and the output \(g(x)\) must be a legal input to \(f\). Simplifying the final expression can hide that second gate, so always trace the domain back to the original functions.

Domain of a composition

Gate 1 — \(x\) must be in \(\text{dom}(g)\).
Gate 2 — \(g(x)\) must be in \(\text{dom}(f)\).
Order matters — \(f\circ g\) is usually not \(g\circ f\).
Don't trust the simplified form — \(x^2\circ\sqrt{x}=x\) but the domain is \([0,\infty)\).

Worked examples

Four compositions, four domain stories.

Radical outer · domain from gate 2

\((f\circ g)(x)\) for \(f(x)=\sqrt{x},\ g(x)=x-5\)
Compose
\(f(g(x))=\sqrt{x-5}\).
Both gates
Gate 1: \(\text{dom}(g)=(-\infty,\infty)\). Gate 2: need \(g(x)\ge 0\Rightarrow x-5\ge 0\Rightarrow x\ge 5\).
Answer
\(\sqrt{x-5}\), domain \([5,\infty)\).

The simplification trap · \(x^2\circ\sqrt{x}\)

\((f\circ g)(x)\) for \(f(x)=x^2,\ g(x)=\sqrt{x}\)
Compose & simplify
\(f(g(x))=(\sqrt{x})^2=x\).
Domain comes first
Gate 1: \(\text{dom}(g)=[0,\infty)\). The simplified \(x\) looks defined everywhere, but \(g\) was only defined for \(x\ge 0\).
Answer
\(x\), domain \([0,\infty)\) — not all reals.

Rational outer · gate 2 is \(g(x)\ne\) value

\((f\circ g)(x)\) for \(f(x)=\dfrac{1}{x},\ g(x)=x-2\)
Compose
\(f(g(x))=\dfrac{1}{x-2}\).
Both gates
Gate 1: \(\text{dom}(g)=(-\infty,\infty)\). Gate 2: \(f\) needs a nonzero input, so \(g(x)\ne 0\Rightarrow x\ne 2\).
Answer
\(\dfrac{1}{x-2}\), domain \((-\infty,2)\cup(2,\infty)\).

Order matters · \(f\circ g\) vs \(g\circ f\)

\(f(x)=x^2,\ g(x)=x+1\)
\((f\circ g)(x)\)
\(f(g(x))=(x+1)^2=x^2+2x+1\).
\((g\circ f)(x)\)
\(g(f(x))=x^2+1\).
Different functions
\(x^2+2x+1\ne x^2+1\). Composition is not commutative — the order of the gates changes the result.
Cheat sheet

Compose inside-out; gate the domain twice.

What composition means

\((f\circ g)(x)=f(g(x))\): inner first. Substitute \(g(x)\) wherever \(f\) has its input, then simplify.

Domain — two gates

\(x\in\text{dom}(g)\) and \(g(x)\in\text{dom}(f)\). Square-root outer ⇒ \(g(x)\ge 0\); reciprocal outer ⇒ \(g(x)\ne 0\).

Table or graph

Find \(g(a)\) first, then \(f\) of that value. If \(g(a)\) isn't a legal input to \(f\), it's undefined.

Don't trust the tidy form

Simplifying can erase the inner restriction. Read the domain off \(f\), \(g\), and the order — before you simplify.

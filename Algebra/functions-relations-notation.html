Algebra · Functions

A function gives each input exactly one output — then notation lets you feed it anything.

Two skills live here. First, reading a relation: pull the domain (the inputs) and range (the outputs) from a table or a set of points, and decide whether it's a function — which depends only on the inputs. Second, function notation: substitute a number, a variable, or a whole expression into every \(x\) and simplify.

Only inputs decide. A repeated output is fine — three inputs can all map to \(7\). What breaks a function is one input pointing to two different outputs.

Worked examples

Set of points
\(\{(1,2),(3,4),(1,5)\}\): the input \(1\) maps to both \(2\) and \(5\) → not a function. Domain \(\{1,3\}\), range \(\{2,4,5\}\).
A table
\(x: -2,0,1,3\) with \(y: 5,5,2,7\): every \(x\) is different → function. Domain \(\{-2,0,1,3\}\), range \(\{2,5,7\}\) (the repeated \(5\) is listed once).
Don't get fooled
The repeated \(5\) in the range does not break it — only a repeated input with a different output would.

Skill 02 · Evaluating function notation

Whatever sits in the parentheses goes into every \(x\).

Pick a function and an input — a number, a variable like \(a\) or \(-x\), or an expression like \(x+1\) — and watch it drop into each \(x\) and simplify. The quadratic is where care pays off: \(f(x+1)\) needs the whole binomial squared, and \(f(-x)\) turns on the fact that \((-x)^2 = x^2\).

Expand, don't drop. \(f(x+1)\) for \(f(x)=x^2\) is \((x+1)^2 = x^2+2x+1\), never \(x^2+1\). And substitute into every \(x\), not just one.

Worked examples

Linear · f(x) = 2x − 3
\(f(4)=2(4)-3=5\); \(f(a)=2a-3\); \(f(x+1)=2(x+1)-3=2x-1\).
Quadratic · f(x) = x² − x
\(f(-2)=4+2=6\); \(f(-x)=(-x)^2-(-x)=x^2+x\); \(f(x+1)=(x+1)^2-(x+1)=x^2+x\).
Rational · f(x) = 1/(x − 2)
\(f(5)=\tfrac{1}{3}\); \(f(a)=\tfrac{1}{a-2}\); \(f(x+1)=\tfrac{1}{(x+1)-2}=\tfrac{1}{x-1}\). Note \(x\neq2\).

Type ^ for powers (x^2), and / for fractions. Equivalent forms are accepted.

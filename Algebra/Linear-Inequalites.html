📍 Understanding Number Line Graphs

\(x \geq 2\) (Closed circle, shade right)

\(x < -1\) (Open circle, shade left)

○ Open Circle

Used for \(<\) and \(>\)

The endpoint is NOT included

Uses parentheses ( ) in interval notation

● Closed Circle

Used for \(\leq\) and \(\geq\)

The endpoint IS included

Uses brackets [ ] in interval notation

📐 Three-Part (Compound) Inequalities

A three-part inequality has a variable expression between two bounds:

\(-2 \leq x < 5\)

Algebraic

\(-2 \leq x < 5\)

Interval Notation

\([-2, 5)\)

Meaning

x is between −2 and 5

Note: The left bracket [ means −2 is included (≤), while the right parenthesis ) means 5 is NOT included (<).

🚨 Critical Rule: Flipping the Inequality Sign

When you multiply or divide both sides by a NEGATIVE number, you must REVERSE the inequality symbol!

Example: Solve \(-2x > 6\)

Divide both sides by −2:

\(x < -3\) ← Sign flipped from > to <

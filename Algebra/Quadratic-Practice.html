📚 Solving Quadratic Equations: Method Guide

Every quadratic equation can be written as: \(ax^2 + bx + c = 0\)

There are four main methods to solve quadratics. The key is choosing the most efficient method for each problem!

🧩 Factoring

  • Write as product of binomials
  • Use Zero Product Property
  • Set each factor = 0
Best when: Discriminant is a perfect square (factorable over integers)

√ Square Root Method

  • Isolate the squared term
  • Take ± square root
  • Solve for x
Best when: No middle term (b = 0) or already in \((x + h)^2 = k\) form

⬜ Completing the Square

  • Move constant, divide by a
  • Add \((b/2)^2\) to both sides
  • Factor perfect square, solve
Best when: Need vertex form, or when other methods are messy

🧪 Quadratic Formula

  • Identify a, b, c
  • Substitute into formula
  • Simplify carefully
Best when: Doesn't factor nicely, or you want a guaranteed method

🔍 The Discriminant: Your Decision Helper

\(D = b^2 - 4ac\)

+
D > 0: Two distinct real solutions
If D is a perfect square → Factorable!
0
D = 0: One repeated real solution
Perfect square trinomial → Factorable!
−
D < 0: Two complex solutions
No real x-intercepts → Use Quadratic Formula

Algebra · Piecewise functions

Make the table, then let it become the graph.

A clean piecewise graph comes straight out of a table. For each piece you compute a couple of points — including the transition value, where you decide if it's included (closed dot) or excluded (open dot). Then you plot those points and connect each set: a segment when the piece is bounded, a ray with an arrow when it runs on forever.

The whole pipeline

Fill the table — each point lands on the graph — then connect.

Compute each \(y\). At a transition, choose closed if the inequality includes it (\(\le, \ge\)) or open if it doesn't (\(<, >\)). Check the table and the points appear. Then tell each piece whether it's a segment or a ray, and draw it.

closed = included open = excluded → ray runs to infinity
Worked example

Ray, segment, ray — all three at once.

A three-piece function gives one of everything: two rays running outward and a segment in the middle. Watch the transition dots and the segment-vs-ray calls.

Solve the transition values
At each boundary, plug \(x\) into the piece on each side. The values differ — a jump — so each boundary has two points at the same \(x\).
Open or closed
The piece whose inequality includes the boundary gets the closed dot; the other gets the open dot.
Connect
The middle piece is bounded → a segment. The two outer pieces run forever → rays, each with an arrow on the infinite side.
Practice 1 · The table

Solve every value; mark every transition.

Fill each \(y\), and at the transitions choose open or closed from the inequality. This is the half where mistakes start, so it's worth fast reps.

Practice 2 · Plot & connect

Plot the table, then join the right points.

The table is done for you. Click the grid to drop each point (it snaps to the lattice). When every point is down, click two dots from the same piece to connect them — bounded pieces become segments, unbounded pieces become rays with arrows.

Cheat sheet

Table first; the graph follows.

Transition value

Plug the boundary x into the piece on each side. They usually differ — that's the jump, two points at one x.

Open vs closed

≤ or ≥ includes the boundary → closed dot. < or > excludes it → open dot.

Segment vs ray

Bounded on both ends → a segment (dot on each end). Runs to ±∞ → a ray with an arrow.

Don't connect a jump

Only join points from the same piece. Across a jump, lift your pencil — leave the gap.

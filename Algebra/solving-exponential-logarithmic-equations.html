Logarithms · putting it all together

Four equations, four strategies — pick the right one first.

Exponential and log equations aren't hard once you name the situation. The whole game is reading the equation, choosing one of four moves, solving the simple algebra that's left — and, for log equations, checking that your answers don't break a domain. This page ties together form conversion, the log properties, and change of base.

Diagnose first

Which of the four situations is this?

Before touching algebra, classify the equation. Each situation has exactly one natural first move:

Exponential · common base
$b^{\,f(x)}=b^{\,g(x)}$ — or rewrite both sides to a shared base.
Do: set the exponents equal, $f(x)=g(x)$.
Exponential · uncommon base
$b^{\,f(x)}=c$ where $c$ isn't a power of $b$.
Do: take $\ln$ of both sides → change of base.
Logarithmic · log = log
A single log on each side: $\log_b M=\log_b N$.
Do: one-to-one gives $M=N$, then check the domain.
Logarithmic · log = number
One log equal to a constant: $\log_b M=k$.
Do: rewrite as $M=b^{k}$, solve, then check the domain.

Try it. Read each equation and pick its method — you don't have to solve it.

Exponential equations

Common base: make the bases match, then equate exponents

If both sides can be written as powers of the same base, the exponents must be equal. Sometimes they already match; more often you rewrite one or both sides first.

SAME BASE · $2^{x+1}=32$
1
Match the base. $32=2^{5}$, so $2^{x+1}=2^{5}$.
2
Equate exponents. $x+1=5$.
3
Solve. $x=4$.
REWRITE TO A COMMON BASE · $8^{x}=16$
1
Common base 2. $8=2^{3}$ and $16=2^{4}$, so $2^{3x}=2^{4}$.
2
Equate exponents. $3x=4$.
3
Solve. $x=\tfrac{4}{3}$.
Uncommon base: take a log of both sides

When the two sides can't share a base — like $2^{x}=7$, since 7 is not a power of 2 — take $\ln$ (or $\log$) of both sides. The power rule drops the exponent down, and change of base finishes it.

UNCOMMON BASE · $2^{x}=7$
1
Take $\ln$ of both sides. $\ln\!\left(2^{x}\right)=\ln 7$.
2
Power rule. The exponent comes down: $x\ln 2=\ln 7$.
3
Solve, exactly then approximately. $x=\dfrac{\ln 7}{\ln 2}=\log_2 7\approx 2.807$.

Graphically, $x\approx 2.807$ is exactly where $y=2^{x}$ crosses $y=7$ — try it on the calculator in the practice section.

Logarithmic equations

log = log: the one-to-one property

If you can get a single logarithm (same base) on each side, then the insides must be equal: $\log_b M=\log_b N \Rightarrow M=N$. Condense first if needed — then always check the domain, since logs only accept positive arguments.

LOG = LOG · $\log_2 x+\log_2(x-2)=\log_2 15$
1
Condense the left. Product rule: $\log_2\!\big(x(x-2)\big)=\log_2 15$.
2
One-to-one. $x(x-2)=15 \Rightarrow x^{2}-2x-15=0$.
3
Solve. $(x-5)(x+3)=0 \Rightarrow x=5$ or $x=-3$.
4
Check the domain. $x=5$: arguments $5$ and $3$ are positive ✓. $x=-3$: argument $x-2=-5<0$ ✗ — reject. Solution: $x=5$.
log = number: rewrite in exponential form

A single log equal to a constant converts straight back to an exponential statement: $\log_b M=k \Rightarrow M=b^{k}$. Solve the result, then check the domain again.

LOG = NUMBER · $\log_3 x+\log_3(x+6)=3$
1
Condense. $\log_3\!\big(x(x+6)\big)=3$.
2
Exponential form. $x(x+6)=3^{3}=27$.
3
Solve. $x^{2}+6x-27=0 \Rightarrow (x+9)(x-3)=0 \Rightarrow x=3$ or $x=-9$.
4
Check the domain. $x=3$ ✓ (arguments $3$, $9$). $x=-9$ ✗ (negative). Solution: $x=3$.

Extraneous solutions

The algebra can lie — the domain can't

A logarithm only accepts a positive argument. When you condense logs or rewrite to a polynomial, the new equation can have roots that the original logs would reject. Those are extraneous: real roots of the algebra, but illegal for the logs. Every candidate must be plugged back into each log's argument.

Common pitfalls

Where solutions go wrong

⚠ Five traps

  • Skipping the domain check. For any log equation, an unchecked answer is an unfinished answer. Negative or zero arguments mean rejection.
  • Equating exponents across different bases. $2^{x}=3^{x}$ is not $x=x$. Equate exponents only after the bases truly match.
  • Taking a log when you could match bases. $4^{x}=8$ is cleaner as $2^{2x}=2^{3}$ than by logarithms — look for a common base first.
  • One-to-one with leftover terms. $\log_b M=\log_b N \Rightarrow M=N$ only when each side is a single log. Condense first.
  • Dropping the exact form. $\dfrac{\ln 7}{\ln 2}$ is the exact answer; $2.807$ is a rounded stand-in. Keep both when asked.

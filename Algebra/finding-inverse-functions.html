Find the function that undoes yours

One method handles all four: write y = f(x), swap x and y, then solve for y. The algebra changes with the function — a cube root here, a domain restriction there, a rational staying rational — but the move never does. And every inverse is just your graph reflected across the line y = x.

They swap: D(f) = R(f−1).

Two cautions ride along. A quadratic isn’t one-to-one, so it only has an inverse after you restrict its domain (say x ≥ h) — the restriction decides whether the inverse takes the + or − square-root branch. And swapping x and y is exactly reflecting the graph over y = x, which is why the domain and range trade places.

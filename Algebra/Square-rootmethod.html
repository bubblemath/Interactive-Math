📐 Solving Quadratics: Square Root Method

Learn to solve equations by isolating the squared term and taking square roots

🎯 When to Use the Square Root Method

The square root method works best when your quadratic equation:

  • Has no linear term (no "bx" term)
  • Can be written in the form x² = k or (x + a)² = k
If x² = k, then x = ±√k
⚠️ Don't forget the ±! When you take the square root of both sides, you must include both the positive AND negative solutions.

💡 Strategy Tip: To solve using the square root method: (1) Isolate x² on one side, (2) Take the square root of both sides, (3) Remember ± for both solutions!

📋 The Four Steps

  1. Isolate the squared term on one side
  2. Take the square root of both sides
  3. Remember ± for both solutions
  4. Simplify if possible

Choose an Example:

Solve: x² - 16 = 0
1 Isolate the squared term
🤔 Think: What operation will get x² alone on one side?

Add 16 to both sides to get x² alone:

x² - 16 + 16 = 0 + 16
x² = 16
2 Take the square root of both sides
🤔 Think: How do you "undo" a square?

Apply the square root to both sides:

√(x²) = √16
3 Remember ± for both solutions
🤔 Think: Why are there TWO solutions?

Don't forget both positive and negative roots:

x = ±√16

Because both 4² = 16 and (-4)² = 16!

4 Simplify
🤔 Think: What is √16?

Calculate the final answer:

x = ±4

Solutions: x = 4 or x = -4

Solve: 3x² - 27 = 0
1 Isolate the squared term
🤔 Think: There's a coefficient! What TWO operations do you need?

First, add 27 to both sides:

3x² - 27 + 27 = 0 + 27
3x² = 27

Then divide both sides by 3:

3x² ÷ 3 = 27 ÷ 3
x² = 9
2 Take the square root of both sides
🤔 Think: Now that x² is isolated, what's next?

Apply the square root to both sides:

√(x²) = √9
3 Remember ± for both solutions
🤔 Think: What numbers squared give you 9?

Include both positive and negative roots:

x = ±√9
4 Simplify
🤔 Think: Simplify √9

Calculate the final answer:

x = ±3

Solutions: x = 3 or x = -3

Solve: (x + 3)² = 25
1 Check if squared term is isolated
🤔 Think: Is the squared expression already alone?

The squared term (x + 3)² is already isolated on the left side!

(x + 3)² = 25 ✓
2 Take the square root of both sides
🤔 Think: What happens when you take √ of (x+3)²?

Apply the square root to both sides:

√((x + 3)²) = √25
3 Remember ± for both solutions
🤔 Think: Write out both equations you need to solve

Include both positive and negative roots:

x + 3 = ±5

This gives us TWO equations to solve!

4 Solve for x
🤔 Think: Solve x + 3 = 5 AND x + 3 = -5

Subtract 3 from both equations:

x + 3 = 5 → x = 2
x + 3 = -5 → x = -8

Solutions: x = 2 or x = -8

Solve: 2x² - 50 = 0
1 Isolate the squared term
🤔 Think: What operations will isolate x²?

First, add 50 to both sides:

2x² - 50 + 50 = 0 + 50
2x² = 50

Then divide both sides by 2:

2x² ÷ 2 = 50 ÷ 2
x² = 25
2 Take the square root of both sides
🤔 Think: Apply the inverse operation of squaring

Apply the square root to both sides:

√(x²) = √25
3 Remember ± for both solutions
🤔 Think: Both 5² and (-5)² equal what?

Include both positive and negative roots:

x = ±√25
4 Simplify
🤔 Think: What is √25?

Calculate the final answer:

x = ±5

Solutions: x = 5 or x = -5

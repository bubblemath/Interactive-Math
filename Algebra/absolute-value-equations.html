Precalculus · Solving absolute value equations

One symbol, three possible endings — the right side decides.

An absolute value is a distance, so it's never negative. That single fact tells you, before you solve a thing, how many answers to expect: isolate the absolute value, then read what it equals. This page teaches the method and the prediction together.

Skill 01 · One side, two solutions

The main event: \(|X| = a\) with \(a > 0\) splits into two cases.

This is the workhorse. Once the absolute value is isolated and equals a positive number, the inside can be that number or its negative — two equations, two solutions. Distance reading: \(|x-c| = d\) means "\(x\) is a distance \(d\) from \(c\)," so the answers are \(c \pm d\), always symmetric about \(c\).

Isolate first, always. Strip away any coefficient or added constant before you split. Splitting too early is the number-one mistake.

The split

\[ |X| = a \;(a>0) \;\Longrightarrow\; X = a \quad\text{or}\quad X = -a \]

Two cases, because a number and its opposite share the same distance from zero.

Worked example

Solve \(\;3|x+2| - 4 = 11\)
Step 1 · Isolate the absolute value
\[3|x+2| = 15 \;\Rightarrow\; |x+2| = 5\]
Step 2 · Right side is positive → two cases
Case 1\(x+2 = 5\)
\(x = 3\)
or
Case 2\(x+2 = -5\)
\(x = -7\)
Solutions
\[x = 3 \quad\text{and}\quad x = -7\] Symmetric about the vertex \(x=-2\), each a distance 5 away.

Skill 02 · One side, one solution

The special case: \(|X| = 0\) gives exactly one answer.

When the isolated absolute value equals zero, the two cases collapse into one — because \(+0\) and \(-0\) are the same number. So \(|X| = 0\) means simply \(X = 0\), a single solution. Spotting this saves you from writing a phantom "second" answer.

Spot it when: after isolating, the absolute value equals \(0\). One case, one solution.

The collapse

\[ |X| = 0 \;\Longrightarrow\; X = 0 \quad(\text{the two cases coincide}) \]

Worked example

Solve \(\;2|x-1| - 3 = -3\)
Step 1 · Isolate
\[2|x-1| = 0 \;\Rightarrow\; |x-1| = 0\]
Step 2 · Zero → a single case
\[x - 1 = 0 \;\Rightarrow\; x = 1\]
Solution
\[x = 1 \quad(\text{just one})\]

Skill 03 · One side, no solution

The dead end: \(|X| = a\) with \(a < 0\) has no solution.

A distance can't be negative, so no value of \(x\) can make \(|X|\) equal a negative number. The instant your isolated absolute value equals something below zero, you're done — there is no solution. Don't split into cases; there's nothing to solve.

Stop when: after isolating, the absolute value equals a negative number. Answer: no solution.

The impossibility

\[ |X| = a \;(a<0) \;\Longrightarrow\; \text{no solution} \]

Be careful: you must isolate first. \(|x|-9=-2\) is not hopeless — it isolates to \(|x|=7\). Only the isolated right side counts.

Worked example

Solve \(\;|2x-1| + 6 = 2\)
Step 1 · Isolate
\[|2x-1| = 2 - 6 = -4\]
Step 2 · Right side is negative → stop
An absolute value is never negative, so nothing works.
Conclusion
No solution.

Skill 04 · Absolute values on both sides

\(|X| = |Y|\): set the insides equal, or equal to the opposite.

When both sides are absolute values, two expressions with the same size are either equal or opposite. So \(|X| = |Y|\) splits into \(X = Y\) or \(X = -Y\) — note the negative wraps the entire second side. Sometimes one branch collapses into a contradiction, leaving a single solution.

The twist: in the second case the minus sign applies to all of \(Y\): \(X = -(Y)\). Distribute it carefully.

The split

\[ |X| = |Y| \;\Longrightarrow\; X = Y \quad\text{or}\quad X = -Y \]

Worked example

Solve \(\;|2x-1| = |x+4|\)
Step 1 · Two cases
Case 1 · equal\(2x-1 = x+4\)
\(x = 5\)
or
Case 2 · opposite\(2x-1 = -(x+4)\)
\(3x = -3 \Rightarrow x = -1\)
Step 2 · Verify both
\(x=5:\ |9|=|9|\) ✓    \(x=-1:\ |{-3}|=|3|\) ✓
Solutions
\[x = 5 \quad\text{and}\quad x = -1\]

Cheat sheet

Isolate, read the right side, then split.

\(|X| = a,\ a>0\)

Two solutions. \(X = a\) or \(X = -a\). Answers are symmetric about the vertex.

\(|X| = 0\)

One solution. The cases coincide, so just solve \(X = 0\).

\(|X| = a,\ a<0\)

No solution. A distance is never negative. Isolate first before judging.

\(|X| = |Y|\)

\(X = Y\) or \(X = -Y\). The minus wraps all of \(Y\). A branch may be a contradiction.
